వికీ సింపర్ రిఫరెన్స్

వికీ సింటాక్స్ అనేది ప్రాథమిక అవసరాలను (శీర్షికలు, పేరాలు, కోట్‌లు, జాబితాలు...) కవర్ చేయడానికి అధ్యయనం చేయబడిన కనీస ట్యాగ్‌ల సెట్‌తో మీ వచనాన్ని మెరుగుపరచడానికి ఒక మార్గం.

బ్లాక్ అంశాలు
  • రెండు విధమైన బ్లాక్ల మధ్య ఖాళీ రేఖ వదిలించుము.
  • పేరా: ఉచిత వచనం, మరొక పేరా అనుసరించాలంటే ఖాళీ లైన్‌తో ముగుస్తుంది.
  • శీర్షిక: !!! శీర్షిక, !! శీర్షిక లేదా ! శీర్షిక, మీరు మూడు వేర్వేరు స్థాయిల శీర్షికలను ఉపయోగించడానికి అనుమతిస్తుంది.
  • Horizontal line: ----
  • జాబితాలు: ప్రతి పంక్తిని * లేదా # తో వరుసగా నంబర్ లేని లేదా సంఖ్యా జాబితాల కోసం ప్రారంభించండి. జాబితా గుర్తులను ఈ విధంగా కలపడం ద్వారా జాబితా ఇంబ్రికేషన్ చేయబడుతుంది: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • నిర్వచన జాబితాలు: ప్రారంభమైన ప్రతి వరుసను = తో నిర్వచించటానికి మరియు : వరకు నిర్వచించబడే నిర్వచనం.

HTML Code
మీరు వికీ సింటాక్స్‌ని ఎంచుకున్నప్పటికీ, మీకు కొన్నిసార్లు మరింత శక్తివంతమైన ఫార్మాటింగ్ అవసరం కావచ్చు, అంటే HTML సింటాక్స్. ఈ క్రింది విధంగా చేయండి: 
/// html
<p style="color:red">నా పాఠం ఎరుపు</p>
 // /
