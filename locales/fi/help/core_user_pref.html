Profiilini

Sukunimi, Etunimi
If the display name is empty, the author's name will be displayed as his first name and last name as set here.
Näytä nimi
Here you can choose a nickname that will be displayed in your posts instead of your last name and first name.
Sähköposti
E-mail address used when you request a new password.
New comment notifications will be sent to this address as well.(If you update this field, you must enter your current password in the appropriate field at the bottom of the form)
URL:
The user's website. If set, the author name will be displayed as a link to the said website.
Käyttöliittymäni kieli
The language used when publishing entries. If the translation is available, the interface will also be displayed in that language.
Aikavyöhykkeeni
Tämä valinta määrittää julkaisun jälkeen näytettävän ajan.
Vaihda salasanasi
To change your password, write down twice your password in the fields New password and Confirm password. The minimal password length is 6 characters. If no new password is provided, the current one is kept. (If you update this field, you must enter your current password in the appropriate field at the bottom of the form)

Vaihtoehtoni

Liitäntä
Aktivoi laajennettu latausohjelma medianhallinnassa
Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Poista käytöstä Javascript-käyttöinen vedä ja pudota kohteiden tilaamisessa
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
Mediamanagerissa sivulla näkyvien elementtien määrä
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Älä käytä tavallista faviconia
Mahdollistaa oletussuosikkikuvakkeen poistamisen käytöstä verkkoselaimen osoitepalkissa.
Painos
Suositeltava muoto
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Oletusarvoisen merkinnän tila
Uusien merkintöjen oletustilaksi voidaan asettaa:
  • Vireillä: Julkaisun tilaa ei ole vielä päätetty.
  • Aikataulutettu: Published on -kentässä annettuun aikaan ja päivämäärään mennessä.
  • Julkaisematon: Offline-merkintä.
  • Julkaistu: Julkaistu: Online entry.
Merkinnän muokkauskentän korkeus
Asettaa merkinnän painoskentän korkeuden. Oletusarvoisesti tämä parametri on 24.
Ota WYSIWYG-tila käyttöön
Mahdollistaa visuaalisen editorin (WYSIWYG: What You See Is What You Get) käytön merkintöjen, sivujen ja kategorioiden kuvauskentissä.
Muut vaihtoehdot
Tunnisteiden luettelomuoto
Voit valita, haluatko nähdä merkintää muokatessasi kaikki käytetyt tunnisteet (Laajennettu) vai vain eniten käytetyt (Lyhyt).
Syntaksin korostus teemaeditorissa
Ottaa käyttöön syntaksin korostuksen teemaeditoria käytettäessä (HTML- ja CSS-tiedostot).

Kojelautani

Suosikkini
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Määritä oletussuosikeiksi (super administrator)
Ota käyttöön tällä hetkellä määritetyn suosikkiluettelon asettaminen kaikkien käyttäjien oletussuosikkiluetteloksi. Tätä listaa ehdotetaan käyttäjille, jos he eivät ole vielä määritelleet omia suosikkejaan.
Muut saatavilla olevat suosikit
Kaikki liitännäiset, jotka voidaan laittaa suosikeiksi, näkyvät tässä. Tarkista vastaavat laatikot ja lisää ne kojelautaan painamalla Add tp my favorites.
Näytä suosikit valikon yläosassa
Jos tämä on käytössä, näyttää suosikit valikon yläosassa.
Ohjaustaulun moduulit
Tällä alueella tehdyt valinnat näkyvät blogisi kojelaudalla.
Näytä dokumentaatiolinkit
Ota käyttöön dokumentaatiolinkkien näyttäminen kojelaudan aloitussivulla.
Näytä Dotclearin uutiset
Ota käyttöön Dotclearin uutisten näyttäminen kojelaudan aloitussivulla.
Näytä pikamerkintälomake
Ota käyttöön "quick entry" -lomakkeen näyttäminen kojelaudan aloitussivulla.
Näyttää myöhässä olevien tehtävien laskurin huoltotyön kojelaudan kuvakkeessa.
Mahdollistaa myöhässä olevien (eli käsiteltävien) tehtävien määrän näyttämisen Maintenance -kuvakkeen alla. Huolto -kuvakkeen on oltava asetettu suosikkeihisi, jotta voit käyttää tätä ominaisuutta.
Näytä erääntyneiden tehtävien luettelo kojelaudan kohteissa
Voit näyttää luettelon erääntyneistä tehtävistä (eli tehtävistä, jotka pitäisi käsitellä) omassa osiossaan Dashboard-laskeutumissivulla.
