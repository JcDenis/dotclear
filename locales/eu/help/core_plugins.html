Instalatutako pluginak

Lerro bakoitza menuan (pantailaren ezkerraldean) aurkituko duzun pluginaren logotipoarekin hasten da.

Ekintza-botoien ondoan dagoen Dotclear logo batek adierazten du plugina zuzenean Dotclear-ekin datorrela eta bere osagaietako bat dela.

Aktibatutako pluginak

Zerrenda honek aktibatutako pluginei buruzko informazioa erakusten du: izena, bertsioa, deskribapen laburra. Bi ekintza posible dira: Desaktibatu (aldi baterako eragiketa) edo Ezabatu (eragiketa iraunkorra).

Desaktibatuta dauden pluginak

Zerrenda honek erabiltzaileak desaktibatu dituen pluginak erakusten ditu. Bakoitzerako, bi ekintza posible dira: Aktibatu edo Ezabatu (eragiketa iraunkorra).

Gehitu pluginak

Orrialde honek DotAddict biltegia arakatzeko eta hortik zuzenean plugin bat instalatzeko aukera ematen du.

Bi bilaketa metodo daude eskuragarri:

  • Pluginaren izena Bilatu biltegian eremuan idatzita,
  • Aurkibidea arakatuz

Aurkibidean, plugin bat bere izenaren ondoko gezian klik eginez gero, bere DotAddict orrialdera sartzeko edo laguntza-webgune batera joateko aukera emango dizu pluginaren egileak aukeratu badu.

Instalatu edo berritu eskuz

DotAddict-en eskuragarri ez dauden zure pluginak edo pluginak instala ditzakezu orri hau erabiliz.

Horiek instalatzeko bi modu daude:

  • Kargatu zip fitxategi bat zure ordenagailutik. Egin klik Arakatu fitxategia hautatzeko, sartu pasahitza eta sakatu Kargatu.
  • Deskargatu zip fitxategi bat zuzenean Webetik bere URLa sartuta, ondoren sartu zure pasahitza eta sakatu Deskargatu.
