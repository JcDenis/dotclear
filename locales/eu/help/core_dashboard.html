Goiburua

Instantzia honetan atzi ditzakezun hainbat blog badaude, goitibeherako zerrenda batek landu nahi duzuna hautatzeko aukera ematen du. Blog bakarra badago, bere izena bistaratzen du.

Hurrengo estekak uneko blogera joateko aukera ematen du (fitxa berri batean irekitzen da).

Beste estekek panel honetara itzultzeko, zure hobespenetara sartzeko eta saioa amaitzeko aukera ematen dute.

Aginte-panelaren pertsonalizazioa

Gogokoak
Orrialdearen erdian dauden ikono handiek zure administratzaile gogokoen orrietara azkar sartzeko aukera ematen dute.
Arbel moduluak
Gogokoen azpian bistaratzen diren moduluak lehenespenez instalatzen dira edo pluginek gehitzen dituzte.

Gogokoen eta moduluen bistaratzea pertsonaliza dezakezu Nire panela fitxan Nire hobespenak orrialdean.

Menu nagusia

Menu hau edonoiz bistaratu edo ezkutatu dezakezu bere eskuineko aldean klik eginda (edo Ezkutatu menu nagusia esteka).

Menuaren lehen zatian erabiltzailearen orrialde gogokoenak zerrendatzen dira. Zerrenda hau bistaratzea ala ez hauta dezakezu Nire panela fitxan Nire hobespenak orrialdean.

Gainerako menuan administrazio-orriak ageri dira. Hiru ataletan banatuta dago:

  • Bloga uneko blogaren edukiari eta parametroei buruzko guztiarentzat.
  • Sistemaren ezarpenak instalazio globala kudeatzeko.
  • Pluginak sistema osoari eragiten dioten luzapenetarako.
