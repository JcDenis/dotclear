Bi faktoreko autentifikazioa
Two factors authentication uses One Time Password authentication and is used only in addition to standard login+password authentication. To activate this feature, scan QR code with an authentication application or use a browser extension, then enter the verification code. On login, when prompted, enter code given by the application.
Authentication keys
Authentication key is used instead of standard login+password authentication, it can be physical key or browser extension or password manager. Javascript and navigator credential API must be enabled to use this feature. Multiple keys can be registered.
Autentifikazio aplikazioak
If configured, various third party applications are available. It is used instead of standard login+password authentication. You can grant or revoke access to these applications if you have accounts on their plateforms, then you can login in using these account.

Nire aukerak

Interfazea
Aktibatu kargatzaile hobetua multimedia-kudeatzailean
Multimedia-kudeatzailean interfaze aurreratua aktibatzeko aukera ematen du, hainbat fitxategi aldi berean kargatzeko aukera ematen duena (Oharra: zure web-arakatzaileak Javascript-a onartu behar du funtzio hau erabili ahal izateko).
Desgaitu Javascript bidezko arrastatu eta jaregin elementuak eskatzeko
Elementuak ordenatzeko "arrastatu eta jaregin" desgaitzeko aukera ematen du. Arrastatu eta jaregin zenbakizko eremuekin ordezkatuko da, eremuen posizioa sartzeko aukera emango dizutenak.
Multimedia-kudeatzailean orrialde bakoitzeko bistaratzen diren elementu kopurua
Multimedia-kudeatzailean bistaratuko den multimedia kopurua zehazten du (karpetak ez dira kontuan hartzen elementuak zenbatzeko orduan).
Ez erabili favicon estandarra
Web arakatzailearen helbide-barran favicon lehenetsia desgaitzeko aukera ematen du.
Edizioa
Formatu hobetsia
Aukeratu sarreraren sintaxia. Wiki sintaxi sinplifikatua da eta baliozko HTML bihurtuko da. HTMLa ezin hobeto ulertzen ez baduzu behintzat, Wiki sintaxia aukeratzea gomendatzen dizugu. Ikus Wiki sintaxiaren erreferentzia informazio gehiago lortzeko.
Sarrerako egoera lehenetsia
Sarrera berrien egoera lehenetsia honela ezar daiteke:
  • Zain: Argitalpenaren egoera ez da oraindik erabaki.
  • Programatua: Sarrera sarean ezarriko da eremuan argitaratutako ordu eta datan.
  • Argitaratu gabea: Lineaz kanpoko sarrera.
  • Argitaratu: Sareko sarrera.
Sarrera editatzeko eremuaren altuera
Sarrera baten edizio-eremuaren altuera ezartzen du. Lehenespenez, parametro hau 24an ezartzen da.
Gaitu WYSIWYG modua
Ikusizko editorea (WYSIWYG: What You See Is What You Get) erabiltzeko aukera ematen du sarreretarako, orrietarako eta kategorien deskribapen-eremuetarako.

Nire gogokoenak
Zona honek aukeratutako gogokoenak bistaratzen ditu blogaren aginte-panelean. Zure arakatzailean Javascript aktibatuta badago saguarekin berrantola daitezke. Bestela, eremu batek ordena-zenbaki bat emateko aukera ematen du. Amaitutakoan, ez ahaztu aldaketak gordetzea sakatuz Gorde ordena.
Gogokoak ere ezaba ditzakezu hautatuta eta Ezabatu hautatutako gogokoaksakatuz.
Definitu gogoko lehenetsi gisa (super administratzailea)
Gaitu unean definitutako gogokoen zerrenda erabiltzaile guztientzat gogokoen zerrenda lehenetsi gisa ezartzeko. Zerrenda hau erabiltzaileei proposatuko zaie oraindik gogokoenak zehaztu ez badituzte.
Eskuragarri dauden beste gogokoenak
Hemen gogoko gisa jartzea ahalbidetzen duten plugin guztiak bistaratzen dira. Markatu dagozkion laukiak eta gehitu panelean Gehitu nire gogokoenaksakatuz.
Erakutsi gogokoak menuaren goialdean
Gaituta badago, gogokoak bistaratzen ditu menuaren goialdean.
Arbel moduluak

Erakutsi sarrera azkarreko formularioa

Erakutsi atzeratutako zereginen kontagailua mantentze-panelaren ikonoan
Atzeratutako zereginen kopurua (hau da, prozesatu beharrekoak) bistaratzeko gaitzen du Mantentze-lanak ikonoaren azpian. Mantentzea ikonoa gogokoetan ezarri behar da funtzio hau erabiltzeko.
Bistaratu atzeratutako zereginen zerrenda paneleko elementuetan
Gaitu atzeratutako zereginen zerrenda (hau da, prozesatu beharko liratekeenak) Arbelaren helmuga-orriko atal dedikatu batean bistaratzeko.
