Referéncia de sintaxi Wiki

La sintaxi Wiki es un biais de melhorar vòstre tèxte amb un ensemble minimal d'etiquetas estudiadas per cobrir los besonhs de basa (títols, paragrafs, citacions, listas...).

Elements de blòt
  • Daissatz una linha voida entre dos blòts similars.
  • Paragraf: Tèxte liure, acabat per una linha voida se un autre paragraf deu seguir.
  • Títol: !!! títol, !! title o ! títol, vos permetent d'utilizar tres nivèls diferents d'entèsta.
  • Horizontal line: ----
  • Listas: Començatz cada linha amb * o # per las listas non numerotadas o numerotadas respectivament. L'imbricacion de lista se fa en mesclant de marcadors de lista d'aquesta manièra: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Listas de definicions: Començatz cada linha per = pel tèrme a definir e amb : per la definicion del tèrme. 
      = tèrme
 : descripcion del tèrme
  • Block quote: each line must start with a semicolon (>).
  • Block quote: Cada linha deu començar per un >.
  • Blòt lateral: Cada linha deu començar per un ).
Formatatge de las balisas
  • Emphasis: two quotes ''text''
  • Strong emphasis: two underscore __text__
  • New line:
  • Insertion: two plusses ++text++ %%%
  • Deletion: two minuses --text--
  • Link: [url], [name|url], [name|url|language] or [name|url|languagee|title]
  • Ligam: [url], [name|url], [name|url|lenga] o [nom|url|lenga|títol]
  • Imatge:((url|tèxte alternatiu)), ((url|tèxte alternatiu|posicion)) o ((url| tèxte alternatiu|posicion|descripcion longa)).
    La posicion pòt èsser siá L (esquèrra), R (drecha) siá C (centrada).
  • Acronym: ??acronym|title??
  • Inline HTML: two backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} or {{quote|language|url}}
  • Cita en linha: {{quote}}, {{quote|language}} o {{quote|language|url}}
  • Footnotes: $$footnote$$
  • Notas: $$footnote$$
  • Superscript : ^superscript^
  • Subscript : ,,subscript,,
Tèxte non formatat
Se volètz pas qu'un caractèr de formatatge siá interpretat coma tal, apondètz un \ just abans. Per exemple: \[Tèxte entre parentèlas sens èsser un ligam\]
HTML Code
Quitament se avètz causit la sintaxi Wiki, podètz de còps aver besonh d'un formatatge mai poderós, es a dire la sintaxi HTML. Fasètz-lo de la manièra seguenta: 
///html
<p style="color:red">mon tèxte en roge</p>
///
