Editing an entry

エントリのタイトル
エントリのタイトルを入力します。 このフィールドは必須です。
抜粋
このフィールドの内容は、ホームページやカテゴリ ページなど、エントリをリストとして表示するページに表示され、その後に「続きを読む」リンクが表示されます。また、投稿全体を表示するページの先頭にも表示されます。空白のままにすると、 コンテンツ フィールドは、ホームページやその他のエントリ リスト ページに完全に表示されます。
内容
エントリの内容を記入します。 このフィールドは必須です。
注記
このテキストエリアは、個人使用、メモ用です。ここに入力した内容はブログには表示されません。
カテゴリ
エントリのステータスを選択できます：
  • 保留中: 出版状況はまだ決定されていない。
  • スケジュール日: 投稿は、 公開日時 フィールドに設定されたオンラインで設定されます。
  • 未発表: 投稿はオフラインです。
  • 公開日: 投稿はオンラインです。
エントリの状態
ここで、エントリの公開日時を変更できます。エントリのステータスが スケジュール済みの場合、指定された日時にオンラインになります。
掲載の日時
エントリの言語コードを選択します。デフォルトでは (ユーザーとしての) 言語に設定されていますが、任意の言語コード (例: 「en」または「es-ar」) に変更できます。このコードはエントリの表示に使用されます。
テキストの書式
エントリの構文を選択します。Wiki は簡略化された構文であり、有効な HTML に変換されます。HTML を完全に理解していない場合は、Wiki 構文を選択することをお勧めします。詳細については、Wiki 構文リファレンスを参照してください。
コメントを許可
このボックスにチェックを入れると、訪問者がエントリにコメントを付けられるようになります。 これに対応する全域的な設定は「ブログの設定」にあります。
トラックバックを許可
エントリーのカテゴリ。新しいカテゴリを作成するには、 カテゴリ セクションを参照してください。 (なし) 項目を選択すると、エントリーにカテゴリを設定せずに残すことができます。
イチ押しのエントリ
「イチ押しのエントリ」は、ブログメニューの「私のイチ押し」の下で一覧表示されます。
エントリのパスワード
エントリ用のパスワードを入力することができます。 パスワード保護されたエントリはブログには表示されず、エントリのURL（「エントリの表示」リンクを参照）とパスワードを与えられた人のみがアクセスできるようになります。
ベース名
添付ファイルは、エントリに追加されたメディア ファイルです。エントリに追加するメディア ファイルを選択するには、[ このエントリにファイルを追加 ] ボタンを使用します。
このオプションは、エントリを保存した後に表示されます。
詳細については、メディア マネージャーのヘルプを確認してください。
エントリの言語
エントリの2文字の言語コードです。 デフォルトではあなたの言語（ユーザとして）になりますが、"en" や "fr-qc" のような好みの言語コードに変更することもできます。 このコードはエントリの表示に使用されます。
添付ファイル
トラックバックは、あなたのエントリーの一部を別のブログにコメントとして投稿する方法です。他の人があなたのエントリーをトラックバックできるようにするには、このボックスをチェックしてください。対応するグローバル設定は、ブログ設定にあります。
パスワード
エントリーにパスワードを入力できます。パスワードで保護されたエントリーはブログに表示されず、エントリーの URL ( エントリーを表示 リンクを参照) とパスワードを伝えたユーザーのみがアクセスできるようになります。
Basename
右側の小さなロックをクリックして、あなたのエントリの別のURLを選択することで、このフィールドの保護を解除することができます。 使用しようとしているURLがすでに別のエントリで使用されている場合は、そのエントリに番号が追加されます。

コメント

コメント一覧
コメント タブでは、 エントリのコメント (およびトラックバック) のステータスを読んだり変更したりできます。ブログ管理者から許可されている場合は、コメント (およびトラックバック) をオンラインまたはオフラインに設定したり、削除したり、ジャンクとしてマークしたりできます。
コメントの追加
「コメントを追加」タブで、バックエンドから出ずにエントリのコメントの1つに返信することができます。 ここで使用される構文は、プレーンかつ無制限の HTML です。 コメントを編集するように、フィールドを使用してください。

コメントを追加

Pingを送信するURL
Pingを送信したい投稿で見つけたトラックバックURLを貼り付けます。
送信する抜粋
このフィールドはエントリの最初の数行がデフォルトとなります。 これは投稿へのリンクと共に、Ping送信するブログへ送信されます。 このフィールドを編集することで、抜粋を修正することができます。
Ping URLを自動検出
エントリが特定の投稿にリンクし、それらの投稿を掲載しているブログが許可している場合、このツールはPing送信するURLを見つけます。
コメント
コメントの内容。このフィールドは HTML 構文を使用します。
