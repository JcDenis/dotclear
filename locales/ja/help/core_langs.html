インストールされている言語

管理プロファイルとブログで利用可能なインストール済み言語の一覧がここに表示されます。これらの言語はそれぞれ削除できます。

言語が存在しない場合は、管理部分のほか、ブログのデフォルトで英語が使用されます。

言語のインストールまたはアップグレード

利用可能な言語

Dotclear のウェブサイトで利用可能な言語のリストがここにあります。 これらの言語のいずれかをインストールするには、ドロップダウンリストで選択し、パスワードを入力し、 Install language を押します。

選択した言語がダウンロードされ、システムにインストールされます。

ZIP ファイルをアップロード

圧縮されたアーカイブ (zip 形式) を使用して言語をインストールすることもできます。 コンピュータ上の対応するファイルを選択し、パスワードを入力し、 言語をアップロードを押します。
