このページには、システム上のすべてのユーザーが表示されます。

ユーザーの権限を追加(または変更)するには、 パスワード フィールドにパスワードが必要になります。

ユーザー一覧フィルタ

検索
検索はユーザ ID に基づいています。 * ワイルドカードはユーザ ID の一部のみを検索するために使用できます。
並び順
ユーザーリストは、 ID、 Last Name、 First Name、 Display Name （ニックネーム）、または Number of Entries （作成したエントリー数）によって並べ替えることができます。
並べ替え
結果リストに適用されるソート方法を示します (昇順 または 降順 ) 。
ユーザ/ページ
結果リストの各ページに表示されるユーザ数を設定。

ユーザー一括操作

複数のユーザーに対して一括操作を行うことが可能です。

  • 権限の設定: 1つ以上のブログの権限を設定します。
  • 削除: 選択したユーザをシステムから削除します。エントリを作成したユーザは削除できません。

ユーザーの権限を設定する場合、システムで利用可能なブログを一覧表示する新しいページにリダイレクトされます。 権限を設定したいブログを確認し、 権限の設定 を押します。

ユーザーの権限

各ブログに次の権限を設定できます：

  • Administrator: ブログを完全に管理できます。
  • 独自のエントリとコメントを管理する: ユーザーは、エントリを作成し、作成したエントリのコメントを管理することができます。
  • エントリとコメントの公開: ユーザーは自身のエントリとコメントの公開状況を変更できます。
  • エントリとコメントの削除: ユーザーは自分のエントリとそのエントリのコメントを削除することができます。
  • すべてのエントリとコメントを管理する: このブログのすべてのエントリは、彼が作者かどうかに関係なく、ユーザーが管理することができます。
  • カテゴリ管理: カテゴリの作成、変更、削除ができます。
  • 独自のメディア項目を管理する: ユーザーはメディア項目を追加したり、変更したり、自分のメディアを削除することができます。
  • すべてのメディア項目を管理: ユーザーは、任意のユーザーからメディア項目を管理できます。
  • ページの管理: ユーザーはページを追加および変更できます。
  • ブログロールの管理: ユーザーは現在のブログロールを管理できます。

Super administrator のステータスは、ユーザープロファイルで設定されます。
