트랙백

트랙백을 만들려면 핑 블로그 링크를 클릭합니다. 이 링크가 표시되지 않으면 항목 상태가 게시됨으로 설정되어 있는지 확인하세요.

핑할 URL
핑하려는 글에서 찾은 트랙백 URL을 여기에 붙여넣습니다.
발췌하여 보내기
기본적으로 이 필드에는 글의 시작 부분이 포함됩니다. 이 내용이 글 링크와 함께 핑하는 블로그에 전송됩니다. 이 필드를 수정하여 발췌문을 수정할 수 있습니다.
핑 URL 자동 검색
항목이 특정 게시물에 연결되어 있고 해당 게시물을 호스팅하는 블로그가 이를 허용하도록 설정된 경우 이 도구는 핑할 URL을 찾아줍니다.
