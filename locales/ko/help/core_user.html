사용자 프로필

사용자 ID
문자, 숫자 또는 기호를 사용하여 2자 이상 입력합니다(공백 없음). 이 필드는 필수 입력 사항입니다.
새 비밀번호
비밀번호를 변경하려면 새 비밀번호 및 비밀번호 확인 필드에 새 비밀번호를 두 번 입력합니다. 비밀번호는 6자 이상이어야 합니다. 두 필드 모두 필수 입력 사항입니다.
연결하려면 비밀번호 변경 필요
이 확인란을 선택하면 새 사용자가 연결할 때 자신의 비밀번호를 직접 선택해야 합니다.
슈퍼 관리자
이렇게 하면 현재 사용자에게 시스템에서 호스팅되는 모든 블로그에 대한 모든 권한이 부여됩니다.
성, 이름
표시 이름이 제공되지 않으면 표시되는 작성자 이름에는 이 두 필드에 입력한 성(姓)과 이름(名)이 포함됩니다.
표시 이름
글에 서명할 때 사용할 닉네임을 선택할 수 있습니다. 비어 있지 않으면 성 및 이름 필드 대신 이 필드가 사용됩니다.
이메일
이 이메일 주소는 닷클리어의 인증 화면에서 비밀번호를 복구해야 하는 경우에 사용됩니다.
URL
사용자의 웹사이트를 나타냅니다. 이 필드가 비어 있지 않으면 작성자의 이름이 이 웹사이트로 연결되는 클릭 가능한 링크가 됩니다.

옵션

인터페이스
언어
새 항목의 기본 언어를 설정합니다. 이 언어에 대한 번역이 있는 경우 블로그 인터페이스도 이 언어로 번역됩니다.
표준 시간대
이 옵션은 항목의 게시 시간이 표시되는 방식에 영향을 줍니다.
에디션
기본 형식
항목의 구문을 선택합니다. Wiki는 단순화된 구문이며 유효한 HTML로 변환됩니다. HTML을 완벽하게 이해하고 있지 않다면 Wiki 구문을 선택하는 것이 좋습니다. 자세한 내용은 Wiki 구문 참조를 참조하세요.
기본 항목 상태
새 항목의 기본 상태는 다음과 같이 설정할 수 있습니다:
  • 보류 중입니다: 아직 게시 상태가 결정되지 않았습니다.
  • 예약됨: 출품작은 에 게시됨 필드에 제공된 시간과 날짜에 온라인으로 설정됩니다.
  • 게시되지 않음: 오프라인 항목.
  • 게시됨: 게시됨: 온라인 응모.
항목 편집 필드 높이
항목의 에디션 필드 높이를 설정합니다. 기본적으로 이 매개 변수는 24로 설정되어 있습니다.
태그 목록 형식
항목을 편집하는 동안 사용된 모든 태그(확장)를 볼지, 가장 많이 사용된 태그(단축)만 볼지 선택할 수 있습니다.
