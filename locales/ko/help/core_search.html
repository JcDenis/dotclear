검색 옵션

텍스트 필드에 찾고자 하는 단어나 용어를 지정하고 검색 범위를 선택합니다:

항목에서 검색
검색은 기사 제목, 요약 및 내용 항목의 필드에서만 수행됩니다.
댓글로 검색
댓글에 대해서만 검색이 이루어집니다.

검색 결과

결과

검색 결과에서 문서의 링크를 클릭하여 문서를 편집할 수 있습니다.

배치 작업

한 번에 여러 문서에 대해 일괄 작업을 수행할 수 있습니다. 가능한 작업은 사용자의 권한에 따라 다릅니다:

  • 게시합니다: 게시: 글을 게시합니다.
  • 게시 취소: 글을 오프라인 상태로 전환합니다.
  • 스케줄: 게시 날짜에 출품작이 온라인에 게시되도록 예약합니다.
  • 보류 중으로 표시합니다: 게시 대기 중입니다.
  • 선택한 것으로 표시합니다: 선택한 항목을 강조 표시합니다.
  • 선택되지 않은 것으로 표시합니다: 선택한 항목의 하이라이트(있는 경우)를 제거합니다.
  • 카테고리 변경하기: 카테고리 목록에서 리디렉션하여 선택한 글의 카테고리를 변경합니다.
  • 언어를 변경합니다: 언어 목록에서 리디렉션하여 항목 언어를 수정합니다.
  • 작성자 변경: 새 작성자의 사용자 ID를 지정하여 항목 작성자를 수정할 수 있습니다.
  • 삭제: 항목을 삭제합니다. 선택을 확인하는 팝업 창이 표시됩니다.
  • 태그를 추가합니다: 선택한 항목에 태그를 추가합니다.
  • 태그를 제거합니다: 선택한 항목에서 하나 이상의 태그를 제거합니다.
