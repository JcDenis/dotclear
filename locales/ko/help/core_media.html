미디어 목록

미디어 관리화면의 기본화면에서는, 현재 선택된 블로그에 사용가능한 미디어파일 목록을 볼 수 있습니다.

폴더나 하위폴더에 있는 미디어파일을 보시려면 그 파일의 아이콘이나 이름을 클릭하시면 됩니다.

MP3 음악 파일을 클릭하시면, 곧바로 플레이어가 나타나고 음악이 연주됩니다. 이 플레이어가 정상 작동하려면, 브라우저에 플래시 플러그인이 설치되어 있어야 합니다.

Selecting a MP3 sound file displays a player to immediately play the file. The Flash plugin must be installed on your browser for this player to operate.

파일 추가

파일 선택
여기에서 로컬 파일 중 하나를 선택하여 블로그로 보낼 수 있습니다. 파일이 최대 크기보다 클 경우 전송되지 않습니다.
제목
보내고자하는 파일에 붙일 제목으로 선택사항입니다.
미디어 관리화면에서 감춤
파일을 비공개로 설정하려면(즉, 소유자만 액세스할 수 있고 게시될 때까지 블로그에서 액세스할 수 없도록 하려면 이 옵션을 선택하십시오.)
파일 이름
로컬 파일 중 하나 또는 여러 개를 선택하여 블로그에 보낼 수 있습니다. 파일이 최대 크기보다 클 경우 전송되지 않습니다.
여러 로컬 파일을 선택하면 대기열에 추가됩니다. 파일 선택을 다시 클릭하면 더 많은 파일을 추가할 수 있습니다. 해당 빨간색 십자 모양 아이콘을 클릭하여 대기열에서 파일을 제거할 수도 있습니다.
준비가 되면 업로드를 클릭하세요.

파일 삭제

꼬맹이사진(Thumbnail)이 준비되어 있을 경우, 이미지들은 그것을 이용해 화면에 보입니다.

새 폴더

이 양식을 사용하면 현재 표시된 폴더에 새 하위 폴더를 만들 수 있습니다. 자신의 컴퓨터에서 하듯이 이름을 선택하기만 하면 됩니다.

미디어 세부 정보

이 페이지는 현재 선택한 파일의 전체 정보를 보여주고, 그 파일에 대한 몇 가지 작업을 하실 수 있게 해줍니다.

파일 이름
현재 파일의 파일 이름을 변경합니다.
파일 제목
파일에 제목을 지정하거나 현재 제목을 변경합니다.
파일 날짜
파일 날짜를 변경합니다.
비공개
미디어 관리자에서 소유자 외의 다른 사람에게 파일을 숨기도록 지정합니다.
새 디렉토리
파일을 다른 폴더로 옮길 수 있습니다.
파일 변경
이 양식을 사용하면 모든 속성(이름, 제목...)을 유지하면서 파일을 바꿀 수 있습니다. 언제나 그렇듯이 최대 파일 크기를 준수해야 합니다.

특정 사례: 이미지

이미지 파일을 추가하실 때, 파일 크기에 따라 아래와 같이 최대 네 종류의 버전을 더 만들게 됩니다.

  • 아이콘: 48x48픽셀 크기의 아이콘
  • 꼬맹이사진: 세로나 가로 중, 큰 쪽이 100 픽셀인 축소이미지 (Thumbnail)
  • 작은 크기: 큰 쪽이 240 픽셀인 축소이미지.
  • 중간 크기: 큰 쪽이 500 픽셀인 축소이미지.
  • 원래 크기: 원본, 변경 안된 이미지.

당신의 글꼭지에 쉽게 파일을 첨부하실 수 있습니다. 먼저, 글꼭지를 생성하고 저장하세요. 그런 다음, "이 글꼭지에 파일 추가" 링크를 눌러 미디어 관리화면으로 갑니다.

항목에 파일 첨부하기

거기서 [+] 아이콘을 클릭하세요 (이 파일을 글꼭지에 첨부). 그러면 파일이 글꼭지에 첨부됩니다.

[+] 아이콘을 클릭하기만 하면 됩니다(이 파일을 항목에 첨부). 그러면 파일이 게시물에 첨부됩니다.

글꼭지에 첨부된 모든 파일들은 당신의 타래소식(RSS피드)에도 실리게 되므로, 어떤 형식의 파일도 쉽게 포드캐스팅할 수 있게 됩니다.

팟캐스트 및 음악 파일 방송

첨부된 모든 파일은 블로그 피드에 포함되므로 모든 유형의 파일을 쉽게 팟캐스트할 수 있습니다.
