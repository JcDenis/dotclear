내 프로필

성과 이름
필명이 비어있으면, 여기서 지정한 이름과 성이 글쓴이 이름으로 표시됩니다.
필명
여기서 당신이 쓴, 글꼭지나 댓글 등, 포스트에 쓰일 글쓴이 이름을 지정해줍니다.
이메일
새 댓글이 달렸음을 알리는 쪽지를 보낼 이메일주소.
URL
사용자의 웹사이트. 이 값이 지정되어있으면, 글쓴이 이름이 이 웹사이트를 지시하는 링크로 표시됩니다.
선호하는 편집방식
당신이 선호하는 구문규칙을 지정해줍니다. 위키는 단순화된 구문규칙으로서, 올바른 xhtml로 변환됩니다. HTML에 대해 완벽히 알고 있지 않는 한, 위키 구문규칙을 사용하실 것을 권합니다.
글꼭지 공개상태 기본값
이 선택에 따라 게시글에 표시되는 시간이 정의됩니다.
글꼭지 편집칸 높이
글꼭지 편집칸의 높이를 지정합니다. 기본값은 24 입니다.

내 옵션

인터페이스
미디어 관리자에서 향상된 업로더 활성화
미디어 관리자에서 고급 인터페이스를 활성화하여 한 번에 여러 파일을 업로드할 수 있습니다(참고: 이 기능을 사용하려면 웹 브라우저에서 자바스크립트 를 지원해야 합니다).
품목 주문 시 자바스크립트 기반 드래그 앤 드롭 사용 안 함
항목 주문 시 '드래그 앤 드롭'을 비활성화할 수 있습니다. 드래그 앤 드롭은 대신 필드 위치를 입력할 수 있는 숫자 필드로 대체됩니다.
미디어 관리자에서 페이지당 표시되는 요소 수
미디어 관리자에 표시할 미디어의 수를 정의합니다( 폴더는 항목 수 계산 시 고려되지 않음).
표준 파비콘을 사용하지 마세요.
웹 브라우저 주소 표시줄에서 기본 파비콘을 비활성화할 수 있습니다.
에디션
기본 형식
항목의 구문을 선택합니다. Wiki는 단순화된 구문 이며 유효한 HTML로 변환됩니다. HTML에 대해 완벽하게 이해하지 않는 한 위키 구문을 선택하는 것이 좋습니다. 자세한 내용은 Wiki 구문 참조를 참조하세요.
기본 항목 상태
새 항목의 기본 상태는 다음과 같이 설정할 수 있습니다:
  • 보류 중입니다: 아직 게시 상태가 결정되지 않았습니다.
  • 예약됨: 출품작은 에 게시됨 필드에 제공된 시간과 날짜에 온라인으로 설정됩니다.
  • 게시되지 않음: 오프라인 항목.
  • 게시됨: 게시됨: 온라인 응모.
항목 편집 필드 높이
항목의 에디션 필드 높이를 설정합니다. 기본적으로 이 매개 변수는 24로 설정되어 있습니다.
위지와이그 모드 활성화
항목, 페이지 및 카테고리 설명 필드에 시각적 편집기(위지위그: 보이는 그대로 표시)를 사용할 수 있습니다.
기타 옵션
태그 목록 형식
항목을 편집하는 동안 사용된 모든 태그(확장)를 볼지, 가장 많이 사용된 태그(단축)만 볼지 선택할 수 있습니다.
테마 편집기에서 구문 강조 표시
테마 편집기 사용 시 구문 강조 표시를 활성화합니다(HTML 및 CSS 파일).

내 대시보드

내 즐겨 찾기
이 영역에는 블로그의 대시보드에 선택한 즐겨찾기가 표시됩니다. 브라우저에서 자바스크립트가 활성화된 경우 마우스로 즐겨찾기를 재주문할 수 있습니다. 그렇지 않으면 필드를 통해 주문 번호를 입력할 수 있습니다. 완료되면 주문 저장을 눌러 변경 사항을 저장하는 것을 잊지 마세요.
즐겨찾기를 선택하고 선택한 즐겨찾기 삭제를 눌러 삭제할 수도 있습니다.
기본 즐겨찾기로 정의(슈퍼 관리자)
현재 정의된 즐겨찾기 목록을 모든 사용자의 기본 즐겨찾기 목록으로 설정할 수 있습니다. 사용자가 아직 즐겨찾기를 정의하지 않은 경우 이 목록이 사용자에게 제안됩니다.
기타 사용 가능한 즐겨찾기
즐겨찾기로 추가할 수 있는 모든 플러그인이 여기에 표시됩니다. 내 즐겨찾기에 추가를 눌러 해당 상자를 체크하고 대시보드에 추가합니다.
메뉴 상단에 즐겨찾기를 표시합니다.
활성화하면 메뉴 상단에 즐겨찾기를 표시합니다.
대시보드 모듈
이 영역에서 선택한 내용은 블로그 대시보드에 반영됩니다.
문서 링크 표시
대시보드 랜딩 페이지에 문서 링크를 표시하도록 설정합니다.
닷클리어 뉴스 표시
대시보드 랜딩 페이지에 닷클리어 뉴스를 표시하도록 설정합니다.
빠른 입력 양식 표시
대시보드 랜딩 페이지에 '빠른 입력' 양식을 표시하도록 설정합니다.
유지 관리 대시보드 아이콘에 기한이 지난 작업 카운터 표시
유지 관리 아이콘 아래에 기한이 지난 작업(즉, 처리해야 하는 작업)의 수를 표시할 수 있습니다. 이 기능을 사용하려면 즐겨찾기에 유지 관리 아이콘을 설정해야 합니다.
대시보드 항목에 기한이 지난 작업 목록 표시
대시보드 랜딩 페이지의 전용 섹션에 기한이 지난 작업(즉, 처리해야 하는 작업) 목록을 표시하도록 설정합니다.
