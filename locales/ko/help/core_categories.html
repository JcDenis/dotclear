카테고리 목록

새 분류 만들기
카테고리 삭제를 클릭하여 삭제하려는 카테고리를 선택하세요. 이 작업을 확인하라는 팝업이 표시됩니다.
카테고리에 기사가 포함되어 있지 않은 경우에만 카테고리를 삭제할 수 있습니다.
카테고리를 비우려면 항목을 이동할 새 카테고리를 드롭다운 목록에서 선택하여 항목을 이동할 수 있습니다. 이 선택을 확인하려면 OK 를 클릭하세요.
분류 순서 바꾸기
드래그 앤 드롭을 사용하여 카테고리 순서를 수정할 수 있습니다. 카테고리를 상위 카테고리로 이동할 수도 있습니다.
주문이 만족스러우면 카테고리 저장 순서를 클릭하세요.
최상위 레벨의 모든 카테고리 재정렬
모든 범주를 최상위 수준으로 다시 이동하려면 이 옵션을 사용합니다.
카테고리 순서를 더 정확하게 수정하려면 다음을 수행하세요.
  • 드래그 앤 드롭을 사용하여 이동합니다,
  • 각 카테고리 이름을 클릭하고 상위 카테고리 또는 형제 카테고리에 대한 위치를 선택합니다.
