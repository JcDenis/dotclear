Actualizaciones

Esta página le permite actualizar Dotclear automáticamente. Si hay una nueva versión disponible, se le preguntará si desea actualizarla. Antes de hacerlo, puede comprobar qué cambios aporta esta actualización haciendo clic en Información sobre esta versión.

Puede forzar la comprobación de la nueva versión haciendo clic en Forzar la comprobación de actualizaciones de Dotclear.

Si todo es correcto, es decir, si no ha modificado manualmente ningún archivo de instalación, se iniciará el procedimiento de actualización:

  • Cree una copia de seguridad de su instalación actual en el directorio de trabajo actual con el nombre backup-x.y.z.zip (donde x.y.z son los números de versión, por ejemplo 2.5.3).
  • Descarga la nueva versión.

Una vez hecho esto, aparecerá un mensaje como el siguiente: "Enhorabuena, estás a un clic de finalizar la actualización".
Haga clic en Finalizar la actualización. Se cerrará la sesión. La base de datos se actualizará, si es necesario, cuando vuelva a conectarse.

Si ha modificado manualmente algunos archivos, aparecerán en la lista y se le recomendará que los actualice manualmente.

Gestionar archivos de copia de seguridad

Desde esta pestaña, puede eliminar los archivos de copia de seguridad antiguos.

También puede volver a una versión anterior. Considera la posibilidad de revertir a una versión anterior sólo si tienes problemas graves con la nueva versión.
