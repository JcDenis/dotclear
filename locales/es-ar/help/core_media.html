Lista de medios de comunicación

The media manager's main page displays a list of available files (medias) for the currently selected blog.

You can click the icons or names of the medias in the folder or any subfolder to display them.

Selecting a MP3 sound file displays a player to immediately play the file. The Flash plugin must be installed on your browser for this player to operate.

Las imágenes se muestran con una miniatura siempre que es posible.

Añadir archivos

Elegir archivo
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Título
Título opcional del fichero que desea enviar.
Privado
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
File name
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

Eliminar archivos

This form allows you to create an new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Nueva carpeta

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Detalles del medio

When you add an image, depending of its size, up to four versions will be made available:

Nombre del archivo
Cambia el nombre del archivo actual.
Título del archivo
Dar un título al fichero o cambiar el actual.
Fecha del archivo
Cambia la fecha del archivo.
Privado
Especifica que el archivo estará oculto para cualquiera excepto su propietario en el gestor de medios.
Nuevo directorio
Permite mover el archivo a otra carpeta.
Cambiar el archivo
With this form, you can replace the file while keeping all the attributes (name, title...). As always, you have to respect the maximum file size.

Caso concreto: imágenes

You can easily attach a file to one of your posts. First, create a new post and save it. Then click on "Add files to this entry" to open the media manager.

  • square: a 48x48 pixels square thumbnail,
  • thumbnail: the image reduced to 100 pixels on its longest side.
  • small: the longuest side is reduced to 240 pixels.
  • medium: the longuest side is reduced to 500 pixels.
  • original: the original, unchanged image.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Adjuntar un archivo a una entrada

The attached files will be shown on the post page, as a list beside the post content. The MP3 files will be made available through the Flash player, so that they can easily by read by visitors.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Los archivos adjuntos se mostrarán en la página de la entrada, como una lista después del contenido de la entrada. Los archivos MP3 estarán disponibles a través de un reproductor Flash, para que puedan ser reproducidos fácilmente por sus visitantes.

Podcast y difusión de archivos musicales

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
