Perfil del usuario

ID de usuario
Al menos 2 caracteres utilizando letras, números o símbolos (sin espacios). Este campo es obligatorio.
Nueva contraseña
Si desea cambiar su contraseña, introduzca la nueva contraseña dos veces en los campos Nueva contraseña y Confirmar contraseña. Las contraseñas deben contener al menos 6 caracteres. Ambos campos son obligatorios.
Se necesita un cambio de contraseña para conectar
Marque esta casilla para que un nuevo usuario tenga que elegir su propia contraseña cuando se conecte.
Super administrador
Esto dará al usuario actual todos los permisos sobre todos los blogs alojados en el sistema.
Apellido, Nombre
Si no se proporciona ningún nombre para mostrar, el nombre del autor mostrado contendrá el apellido y el nombre introducidos en estos dos campos.
Nombre mostrado
Puede seleccionar el apodo que desea utilizar para firmar sus artículos. Si no está vacío, este campo se utilizará en lugar de los campos de apellido y nombre.
Correo electrónico
Esta dirección de correo electrónico se utilizará si necesita recuperar su contraseña desde la pantalla de autenticación de Dotclear.
URL
Indica el sitio web del usuario. Si este campo no está vacío, el nombre del autor se convertirá en un enlace clicable a este sitio web.

Opciones

Interfaz
Idioma
Establece el idioma por defecto para las nuevas entradas. Si hay una traducción disponible para este idioma, la interfaz del blog se traducirá también a este idioma.
Zona horaria
Esta opción influye en la forma en que se muestra la hora de publicación de las entradas.
Edición
Formato predilecto
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Estado predeterminado para las entradas
El estado por defecto para las nuevas entradas se puede establecer en:
  • Pendiente: El estado de publicación aún no se ha decidido.
  • Programada: La entrada se pondrá en línea a la hora y fecha indicadas en el campo Publicado en.
  • No publicado: Entrada sin conexión.
  • Publicado: Entrada en línea.
Alto del campo de edición de entradas
Establece la altura del campo de edición de una entrada. Por defecto, este parámetro se establece en 24.
Formato de la lista de etiquetas
Puede seleccionar si desea ver, mientras edita una entrada, todas las etiquetas utilizadas (Extendido) o sólo las más utilizadas (Corto).
