Referință de sintaxă Wiki

Sintaxa Wiki este o modalitate de a vă îmbunătăți textul cu un set minim de etichete studiate pentru a acoperi nevoile de bază (titluri, paragrafe, citate, liste...).

Blocare elemente
  • Lăsați o linie goală între două blocuri similare.
  • Paragraf: Text liber, încheiat cu o linie goală dacă urmează un alt paragraf.
  • Titlu: !!! titlu, !! titlu sau ! titlu, permițându-vă să utilizați trei niveluri diferite de titlu.
  • Horizontal line: ----
  • Liste: Începeți fiecare rând cu * sau # pentru listele nenumerotate sau, respectiv, numerotate. Imbricarea listei se face prin amestecarea markerilor listei astfel: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Lista de definiții: Începe fiecare linie cu = pentru ca termenul să definească și cu : pentru definiția termenului. 
      = termenul
  : descrierea termenului
  • Block quote: each line must start with a semicolon (>).
  • Ofertă bloc: Fiecare linie trebuie să înceapă cu >.
  • Bloc lateral: Fiecare linie trebuie să înceapă cu ).
Formatare tag-uri
  • Emphasis: two quotes ''text''
  • Strong emphasis: two underscore __text__
  • New line:
  • Insertion: two plusses ++text++ %%%
  • Deletion: two minuses --text--
  • Link: [url], [name|url], [name|url|language] or [name|url|languagee|title]
  • Link: [url], [nume|url], [nume| url|limba] sau [nume|url|limba|titlu]
  • Imaginea:(url alternative text)), (url alternative text position)) sau (url alternative text position long description)).
    Poziţia poate fi fie L (stânga), R (dreapta) sau C (centrată).
  • Acronym: ??acronym|title??
  • Inline HTML: two backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} or {{quote|language|url}}
  • Citate în linie: {{quote}}, {{quote|language}} sau {{quote|language|url}}
  • Footnotes: $$footnote$$
  • Note de subsol: $$footnote$$
  • Superscript : ^superscript^
  • Abonament : ,,abonament,,
Text neformatat
Dacă nu doriți ca un caracter de formatare să fie interpretat ca atare, adăugați un \ chiar înaintea acestuia. De exemplu: \[Text între paranteze fără a fi un link\]
HTML Code
Chiar dacă ați ales sintaxa Wiki, uneori este posibil să aveți nevoie de o formatare mai puternică, adică sintaxa HTML. Faceți-o în felul următor: 
//html
<p style="color:red">textul meu cu roșu</p>
////
