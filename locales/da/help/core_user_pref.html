Min profil

Name, First name
If the display name is empty, the author's name will be displayed as his first name and last name as set here.
Vis navn
Here you can choose a nickname that will be displayed in your posts instead of your last name and first name.
E-mail
E-mail address used when you request a new password.
New comment notifications will be sent to this address as well.(If you update this field, you must enter your current password in the appropriate field at the bottom of the form)
URL
The user's website. If set, the author name will be displayed as a link to the said website.
Preferred format
The language used when publishing entries. If the translation is available, the interface will also be displayed in that language.
Default entry status
Dette valg vil definere den tid, der vises på post offentliggørelse.
Entry edit field height
To change your password, write down twice your password in the fields New password and Confirm password. The minimal password length is 6 characters. If no new password is provided, the current one is kept. (If you update this field, you must enter your current password in the appropriate field at the bottom of the form)

Mine muligheder

Grænseflade
Aktiver forbedret uploader i mediehåndtering
Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Deaktivér træk og slip drevet af Javascript til bestilling af varer
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
Antal elementer, der vises pr. side i mediehåndtering
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Brug ikke standard favicon
Tillader at deaktivere standard favicon i webbrowserens adresselinje.
Udgave
Foretrukket format
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Standard indtastningsstatus
Standardstatus for nye poster kan indstilles til:
  • Afventer: Offentliggørelsesstatus er endnu ikke afgjort.
  • Planlægges: Indgangen vil blive sat online på det tidspunkt og den dato der er angivet i Publiceret på feltet.
  • Ikke offentliggjort: Offline post.
  • Publiceret: Online post.
Indtastnings redigeringsfeltets højde
Sætter højden af redigeringsfeltet for en indgang. Denne parameter er som standard sat til 24.
Aktiver WYSIWYG tilstand
Tillader brug af den visuelle editor (WYSIWYG: Hvad du ser er hvad du får) for indgange, sider og kategorier beskrivelsesfelter.
Andre muligheder
Tags liste-format
Du kan vælge, om du vil se, mens du redigerer en post, alle de anvendte tags (Udvidet) eller kun de mest anvendte (Kort).
Syntaksfremhævning i temaeditoren
Aktiverer syntaksfremhævning når temaeditoren bruges (HTML- og CSS-filer)

Mit betjeningspanel

Mine favoritter
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Definér som standard favoritter (super administrator)
Aktiver for at indstille favoritter listen som standard favoritter liste for alle brugere. Denne liste vil blive foreslået til brugere, hvis de ikke har defineret deres egne favoritter endnu.
Andre tilgængelige favoritter
Alle plugins, der giver mulighed for at blive sat som favoritter, bliver spillet her. Tjek de tilsvarende felter og tilføj dem til instrumentbrættet ved at trykke på Tilføj tp mine favoritter.
Vis favoritter øverst i menuen
Hvis aktiveret, vises favoritter øverst i menuen.
Dashboard moduler
Valg lavet i denne zone vil blive reflekteret på din blogs dashboard.
Vis dokumentationslinks
Aktiver for at vise dokumentationslinks på dashboard destinationsside.
Vis Dotclear nyheder
Aktiver for at vise Dotclear nyheder på dashboard destinationsside.
Vis hurtig-indtastningsformular
Aktiver for at vise "Hurtig indtastning" formular på dashboard destinationsside.
Vis forsinkede opgaver tæller på vedligeholdelse dashboard ikon
Aktiverer visning af antallet af forfaldne opgaver (dvs. det skal behandles) under Vedligeholdelse ikonet. Vedligeholdelse -ikonet skal indstilles i dine favoritter for at bruge denne funktion.
Vis liste over forfaldne opgaver på dashboard elementer
Aktiver for at vise listen over forfaldne opgaver (dvs. som skal behandles) i en dedikeret sektion på destinationssiden Dashboard.
