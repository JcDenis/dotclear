Мой профиль

Фамилия, Имя
Если отображаемое имя пустое, имя автора будет отображаться как его имя и фамилия, как указано здесь.
Отображать имя
Здесь вы можете выбрать псевдоним, который будет отображаться в ваших сообщениях вместо вашей фамилии и имя.
Почта
Адрес электронной почты, используемый при запросе нового пароля.
На этот адрес также будут отправляться уведомления о новых комментариях.(Если вы обновляете это поле, вы должны ввести свой текущий пароль в соответствующее поле в нижней части формы)
URL
Веб-сайт пользователя. Если задано, имя автора будет отображаться как ссылка на указанный сайт.
Язык интерфейса
Язык, используемый при публикации записей. Если перевод доступен, интерфейс также будет отображаться на этом языке.
Мой часовой пояс
Этот выбор определит время, которое будет отображаться в публикации.
Изменить пароль
Чтобы изменить пароль, впишите пароль дважды в поля Новый пароль и Подтвердите пароль. Длина минимального пароля составляет 6 символов. Если новый пароль не введен, то текущий пароль сохраняется. (Если вы обновите это поле, вы должны ввести текущий пароль в соответствующее поле внизу формы)

Мои настройки

Интерфейс
Активировать расширенный загрузчик в медиа-менеджере
Позволяет активировать расширенный интерфейс в медиа-менеджере, который позволяет загружать несколько файлов одновременно (Примечание: ваш веб-браузер должен поддерживать Javascript , чтобы использовать эту функцию).
Отключить перетаскивание на Javascript для заказа элементов
Позволяет отключить "перетаскивание" для сортировки элементов. Перетаскивание будет заменено числовыми полями, которые позволят вводить поля.
Количество отображаемых элементов на странице в медиа-менеджере
Определяет количество медиафайлов, которые будут отображаться в менеджере мультимедиа (папки не учитываются при подсчёте элементов).
Не использовать стандартный фавикон
Позволяет отключить стандартный значок в адресной строке браузера
Редактирование
Предпочитаемый формат
Выберите синтаксис записи. Вики является упрощённым синтаксисом и будут преобразованы в корректный HTML. Если у вас нет идеального понимания HTML, мы рекомендуем вам выбрать Wiki синтаксис. Подробнее см. в вики-справочнике по синтаксису.
Статус записи по умолчанию
Статус по умолчанию для новых записей может быть установлен в:
  • В ожидании: Статус публикации еще не был принят.
  • Запланировано: Запись будет установлена в режиме онлайн в то время и дату, указанную в поле Опубликовано на.
  • Не опубликовано: Запись оффлайн.
  • Опубликовано: Онлайн-запись.
Редактирование поля ввода высота
Устанавливает высоту поля редактирования записи. По умолчанию значение параметра равно 24.
Включить режим WYSIWYG
Позволяет использовать визуальный редактор (WYSIWYG: что вы видите для записей, страниц и категорий поля.
Другие опции
Формат списка тегов
Вы можете выбрать, если хотите увидеть, при редактировании записи, все используемые теги (Расширенный) или только самые используемые (Краткий).
Подсветка синтаксиса в редакторе тем
Включает подсветку синтаксиса при использовании редактора темы (HTML и CSS файлов)

Моя панель управления

Избранное
Эта зона отображает избранные статьи на панели блога. Их можно упорядочить с помощью мыши, если в браузере активирован Javascript. В противном случае в поле можно указать номер заказа. Когда вы закончите, не забудьте сохранить изменения, нажав Сохранить заказ.
Вы также можете удалить избранное, выделив его и нажав Удалить избранное.
Определить как избранное по умолчанию (супер администратор)
Включите, чтобы установить список избранного по умолчанию для всех пользователей. Этот список будет предложен пользователям, если они еще не определили избранное.
Другие доступные избранные
Все плагины позволяют поместить в избранное размещены здесь. Проверьте соответствующие поля и добавьте их на приборную панель, нажав Добавить в избранное.
Отображать избранное в верхней части меню
Если включено, отображает избранное в верхней части меню.
Модули панели инструментов
Выбор сделанный в этой зоне будет отражен на панели управления вашего блога.
Показать ссылки на документацию
Включите, чтобы отображать ссылки на документацию на главной странице.
Отображать новости точечной очистки
Включите, чтобы отображать новости Dotclear на главной странице.
Отображать форму быстрой записи
Включите, чтобы отобразить форму «быстрый ввод» на главной странице.
Отображать счетчик просроченных задач на иконке панели обслуживания
Позволяет отображать количество просроченных задач (т.е. должно быть обработано) под значком Техническое обслуживание . Значок Техническое обслуживание должен быть установлен в избранном, чтобы использовать эту функцию.
Отображать список просроченных задач на панели инструментов
Включите, чтобы отобразить список просроченных задач (т.е. должен быть обработан) в выделенном разделе на главной странице панели.
