Фильтровать список блогов

Искать
Поиск основан на идентификаторе блога. Вы можете использовать * шаблон, чтобы указать только часть названия.
Сортировать по
Последнее обновление блога, имя блога, ID блога.
Сортировка
Укажите сортировку, которую вы хотите использовать для фильтрации результатов (по убыванию или по возрастанию).
Блоги на страницу
Выбирает количество блогов, отображаемых на странице в этом списке.
