Список файлов

На главной странице менеджера медиа отображен список доступных файлов (медиа) для текущего блога.

Вы можете нажать на иконки или названия медиафайлов в папке или любой подпапке для их отображения.

Выбор звукового файла MP3 отображает проигрыватель для немедленного воспроизведения файла. Плагин Flash должен быть установлен в Вашем браузере, чтобы этот плеер работал.

Изображения отображаются с эскизами, когда это возможно.

Добавить файлы

Выберите файл
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Заголовок
Необязательный заголовок для файла, который вы хотите отправить.
Приватный
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
Расширенный загрузчик
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

Удалить файлы

Чтобы удалить файл, установите флажок на кнопку Удалить выбранный файл. Вы можете удалить несколько файлов сразу.

Новая папка

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Конкретный случай: изображения

При добавлении изображения, в зависимости от его размера, будут доступны до четырех версий (в зависимости от размера исходного изображения):

  • Квадрат: Квадрат квадрата размером 48x48 пикселей.
  • Миниатюра: Изображение уменьшено до 100 пикселей на его самой длинной стороне.
  • Маленько: Самая длинная сторона уменьшена до 240 пикселей.
  • Средний: Самая длинная сторона снижена до 500 пикселей.
  • Оригинал: Исходное изображение, без изменений.

Размер эскизов по умолчанию, малые и средние версии могут быть изменены в настройках вашего блога.

Прикрепить файл к записи

Вы можете легко прикрепить файл к вашему сообщению. Сначала создайте новое сообщение и сохраните его. Затем нажмите на кнопку "Добавить файлы к этой записи", чтобы открыть медиа-менеджер.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Прикрепленные файлы будут показаны на странице записи, как список после содержания сообщения. MP3 файлы будут доступны через Flash Player, так что они могут быть легко сыграны вашими посетителями.

Подкаст и трансляция музыкальных файлов

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
