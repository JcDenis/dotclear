Синтаксическая ссылка на вики

Вики-синтаксис - это способ улучшить текст с минимальным набором тегов для удовлетворения основных потребностей (заголовков, Пункты, кавычки, списки. .).

Блокировать элементы
  • Оставьте пустую строку между двумя похожими блоками.
  • Абзац: Свободный текст, заканчивающийся пустой строкой, если другой абзац следует за ним.
  • Название: !!! название, !! название или ! название, позволяющее использовать три различных уровня заголовка .
  • Горизонтальная линия: ----
  • Списки: Начать каждую строку с * или # для ненумерованных или нумерованных списков соответственно. Имбрикация списка выполняется методом смешивания маркеров списка: 
        * Пункт 1
    ** пункт 1.1
    * пункт 2
    *# пункт 2.1
...
  • Списки определений: Начать каждую строку с = для определения термина и с : для определения терма. 
      = термин
  : описание термина
  • Предформатированный текст: Каждая строка должна начинаться с пробела.
  • Блок кавычки: Каждая строка должна начинаться с >.
  • Боковой блок: Каждая строка должна начинаться с ).
Форматирование тегов
  • Внимание: Два цитаты ''текст''
  • Сильное подчеркивание: Два подчеркивания __text__
  • Выделиться: Два двойных кавычки ""text""
  • Новая строка: %%%
  • Вставка: Два плюса ++text++
  • Удаление: Два минуты --text--
  • Ссылка: [url], [name|url], [name|url|language] или [name|url|language|title]
  • Image: ((url|alternative text)), ((url|alternative text|position)) or ((url|alternative text|position|long description)).
    The position can be either L (left), R (right) or C (centered).
  • Якорь: ~anchor~
  • Acronym: ??acronym|title??
  • Встроенный HTML: Два backquotes ``HTML code``
  • Цитата: {{quote}}, {{quote|language}} или {{quote|language|url}}
  • Код: @@code@@
  • Примечания: $$footnote$$
  • Superscript : ^superscript^
  • Подскрипт : ,подписок,
Неформатированный текст
Если вы не хотите, чтобы один символ форматирования интерпретировался как таковый, добавьте \ непосредственно перед этим. Например: \[Текст в скобках без ссылки\]
HTML-код
Даже если вы выбрали Wiki синтаксис, вам иногда потребуется более мощное форматирование, то есть HTML-синтаксис. Выполните следующие действия: 
///html
<p style="color:red">мой текст красным</p>
///
