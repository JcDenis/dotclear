Синтаксическая ссылка на вики

Синтаксис Wiki — это способ улучшить ваш текст с помощью минимального набора тегов, изученных для удовлетворения основных потребностей (заголовки, абзацы, цитаты, списки...).

Блокировать элементы
  • Оставьте пустую строку между двумя похожими блоками.
  • Абзац: Свободный текст, заканчивающийся пустой строкой, если за ним следует другой абзац.
  • Название: !!! заголовок, !! заголовок или ! title, позволяющий использовать три разных уровня заголовка.
  • Горизонтальная линия: ----
  • Списки: Начинайте каждую строку с * или # для ненумерованных или нумерованных списков соответственно. Укладка списка выполняется путем смешивания маркеров списка следующим образом: 
        * Пункт 1
    ** пункт 1.1
    * пункт 2
    *# пункт 2.1
...
  • Списки определений: Начать каждую строку с = для определения термина и с : для определения терма. 
      = термин
  : описание термина
  • Предформатированный текст: Каждая строка должна начинаться с пробела.
  • Блок кавычки: Каждая строка должна начинаться с >.
  • Боковой блок: Каждая строка должна начинаться с ).
Форматирование тегов
  • Внимание: Два цитаты ''текст''
  • Сильное подчеркивание: Два подчеркивания __text__
  • Выделиться: Два двойных кавычки ""text""
  • Новая строка: %%%
  • Вставка: Два плюса ++text++
  • Удаление: Два минуты --text--
  • Ссылка: [url], [имя|url], [имя| url|язык] или [имя|url|язык|название]
  • Image:((url|alternative text)), ((url|alternative text|position)) или (url|alternative text|position|long description)).
    Позиция может быть либо L (слева), R (справа) или C (по центру).
  • Якорь: ~anchor~
  • Acronym: ??acronym|title??
  • Встроенный HTML: Два backquotes ``HTML code``
  • Встроенная цитата: {{quote}}, {{quote|language}} или {{quote|language|url}}
  • Код: @@code@@
  • Примечания: $$footnote$$
  • Superscript : ^superscript^
  • Подскрипт : ,подписок,
Неформатированный текст
Если вы не хотите, чтобы один символ форматирования интерпретировался как таковой, добавьте \ непосредственно перед ним. Например: \[Текст в скобках, не являющийся ссылкой\]
HTML-код
Даже если вы выбрали синтаксис Wiki, иногда вам может понадобиться более мощное форматирование, например синтаксис HTML. Сделайте это следующим образом: 
///html
<p style="color:red">мой текст красным</p>
///
