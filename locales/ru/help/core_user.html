Профиль пользователя

ID пользователя
Не менее 2 символов с использованием букв, цифр или символов (без пробела). Это поле является обязательным.
Новый пароль
Если вы хотите изменить свой пароль, введите новый пароль дважды в поле Новый пароль и Подтвердите пароль . Пароль должен содержать не менее 6 символов. Оба поля обязательны для заполнения.
Для подключения требуется смена пароля
Установите этот флажок, чтобы новый пользователь мог выбрать свой собственный пароль, когда он подключится.
Супер администратор
Это даст текущему пользователю доступ ко всем блогам, размещенным в системе.
Фамилия, Имя
Если отображаемое имя не указано, отображаемое имя автора будет содержать фамилию и имя, введенное в этих двух полях.
Отображать имя
Вы можете выбрать псевдоним, который хотите подписать ваши статьи. Если не пусто, это поле будет использоваться вместо поля фамилия и имя.
Почта
Этот адрес электронной почты будет использован, если вам необходимо восстановить пароль с экрана аутентификации Dotcleare.
URL
Указывает сайт пользователя. Если это поле не пустое, имя автора станет кликабельной ссылкой на этот сайт.

Варианты

Интерфейс
Язык
Установить язык по умолчанию для новых записей. Если перевод доступен для этого языка, интерфейс блога также будет переведен на этот язык.
Timezone
Этот параметр влияет на то, как отображается время публикации записей.
Редактирование
Предпочитаемый формат
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Статус записи по умолчанию
Статус по умолчанию для новых записей может быть установлен в:
  • В ожидании: Статус публикации еще не был принят.
  • Запланировано: Запись будет установлена в режиме онлайн в то время и дату, указанную в поле Опубликовано на.
  • Не опубликовано: Запись оффлайн.
  • Опубликовано: Онлайн-запись.
Редактирование поля ввода высота
Устанавливает высоту поля редактирования записи. По умолчанию значение параметра равно 24.
Формат списка тегов
Вы можете выбрать, если хотите увидеть, при редактировании записи, все используемые теги (Расширенный) или только самые используемые (Краткий).
