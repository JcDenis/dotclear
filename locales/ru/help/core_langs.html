Установленные языки

Здесь вы найдете список языков, доступных для вашего профиля администрирования и для вашего блога. Каждый из этих языков может быть удален.

В том случае, если ни один язык не присутствует, английский будет использоваться по умолчанию как для административной части, так и для блога.

Установить или обновить языки

Доступные языки

Вы найдете здесь список языков, доступных на сайте Dotcle. Чтобы установить один из этих языков, выберите его в раскрывающемся списке, введите пароль и нажмите Установить язык.

Выбранный язык будет загружен и установлен в системе.

Загрузить zip-файл

Вы также можете использовать сжатый архив (zip-формат) для установки языка. Выберите соответствующий файл на вашем компьютере, введите свой пароль и нажмите Загрузить язык.
