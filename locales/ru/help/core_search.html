Параметры поиска

Укажите слова или термины, которые вы ищете, в текстовом поле и выберите область поиска:

Поиск в записях
Поиск будет производиться только по заголовку статьи, Резюме и Контент поля записи.
Искать в комментариях
Поиск будет производиться только по комментариям.

Результаты поиска

Результат

Результат поиска позволяет щелкнуть по ссылке одной из статей для её редактирования.

Пакетные действия

Можно совершать несколько действий на несколько статей в один раз. Возможные действия зависят от прав пользователя:

  • Опубликовать: Опубликовать статью.
  • Снять с публикации: Поставить статью оффлайн.
  • График: Запланировать записи для размещения в сети на дату публикации.
  • Отметить в ожидании: В ожидании публикации.
  • Отметить как выбранный: Выделить выбранные записи.
  • Отметить как не выбранный: Удалить подсветку (если есть) для выбранных записей.
  • Изменить категорию: Перенаправить в списке категорий, чтобы изменить категорию выбранных материалов.
  • Изменить язык: Перенаправить в списке языков для изменения языков.
  • Автор изменения: Позволяет изменить автора записей, указав ID пользователя нового автора.
  • Удалить: Удалите записи. Для подтверждения будет показано всплывающее окно.
  • Добавить теги: Добавить теги к выбранным записям.
  • Удалить теги: Удалить один или несколько тегов из выбранных записей.
