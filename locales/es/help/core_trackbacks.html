Retroenlaces

Para hacer un trackback, haz clic en el enlace Ping blogs . Si no ves este enlace, asegúrate de que el estado de tu entrada es publicada.

URL a las que enviar un ping
Pega aquí las URL de los trackbacks que has encontrado en las entradas a las que quieres hacer ping.
Extracto para enviar
Por defecto, este campo contiene el comienzo de tu entrada. Esto es lo que se enviará al blog al que está haciendo ping, junto con un enlace a su entrada. Puedes modificar el extracto editando este campo.
Descubrimiento automático de las URL ping
Si su entrada enlaza con publicaciones específicas y si el blog que aloja esas publicaciones está configurado para permitirlo, esta herramienta encontrará las URL para hacer ping por usted.
