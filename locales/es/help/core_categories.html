Lista de categorías

Create a new category
Elija la categoría que desea eliminar haciendo clic en Eliminar la categoría. Una ventana emergente le pedirá que confirme esta acción.
Una categoría sólo puede suprimirse si no contiene ningún artículo.
Para vaciar una categoría, puede mover las entradas seleccionando (en la lista desplegable) la nueva categoría a la que se moverán. Haga clic en OK para confirmar esta elección.
Change category order
Puede modificar el orden de las categorías mediante arrastrar y soltar. También puede mover una categoría a una categoría superior.
Cuando esté satisfecho con el orden, haga clic en Guardar el orden de las categorías.
Reordenar todas las categorías en el nivel superior
Utilice esta opción para volver a colocar todas las categorías en el nivel superior.
Si desea modificar el orden de las categorías con mayor precisión, puede :
  • o bien moverlos mediante arrastrar y soltar,
  • Haga clic en el nombre de cada categoría y seleccione su categoría padre o su posición respecto a sus hermanas.
