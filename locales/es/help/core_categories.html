Lista de categorías

Create a new category
Elija la categoría que desea eliminar haciendo clic en Eliminar la categoría. Una ventana emergente le pedirá que confirme esta acción.
Una categoría solo se puede eliminar si no contiene ningún artículo.
Para vaciar una categoría, puede mover las entradas seleccionando (en la lista desplegable) la nueva categoría a la que se moverán. Haga clic en Aceptar para confirmar esta elección.
Change category order
Puede modificar el orden de las categorías arrastrando y soltando. También puede mover una categoría a una categoría principal.
Cuando esté satisfecho con el pedido, haga clic en Guardar orden de categorías.
Reordenar todas las categorías en el nivel superior
Utilice esta opción para retroceder todas las categorías al nivel superior.
Si desea modificar el orden de las categorías con mayor precisión, puede:
  • o bien moverlos mediante arrastrar y soltar,
  • Haga clic en el nombre de cada categoría y seleccione su categoría padre o su posición respecto a sus hermanas.
