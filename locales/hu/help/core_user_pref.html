Profilom

Name, First name
If the display name is empty, the author's name will be displayed as his first name and last name as set here.
Név megjelenítése
Here you can choose a nickname that will be displayed in your posts instead of your last name and first name.
E-mail
E-mail address used when you request a new password.
New comment notifications will be sent to this address as well.(If you update this field, you must enter your current password in the appropriate field at the bottom of the form)
URL
The user's website. If set, the author name will be displayed as a link to the said website.
Preferred format
The language used when publishing entries. If the translation is available, the interface will also be displayed in that language.
Default entry status
Ez a választás határozza meg a közzététel után megjelenő időpontot.
Entry edit field height
To change your password, write down twice your password in the fields New password and Confirm password. The minimal password length is 6 characters. If no new password is provided, the current one is kept. (If you update this field, you must enter your current password in the appropriate field at the bottom of the form)

A lehetőségeim

Interfész
Fokozott feltöltő aktiválása a médiakezelőben
Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Javascript-alapú drag and drop letiltása a tételek rendeléséhez
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
A médiakezelőben oldalanként megjelenített elemek száma
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Ne használjon szabványos favicont
Lehetővé teszi az alapértelmezett favicon letiltását a webböngésző címsorában.
Kiadás
Előnyös formátum
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Alapértelmezett belépési státusz
Az új bejegyzések alapértelmezett állapota a következőre állítható:
  • Függőben: A közzététel státuszáról még nem született döntés.
  • Tervezett: A bejegyzés a Published on mezőben megadott időpontban és dátummal kerül online beállításra.
  • Kiadatlan: Offline bejegyzés.
  • Megjelent: Online bejegyzés.
Bejegyzés szerkesztési mező magassága
Beállítja egy bejegyzés kiadási mezőjének magasságát. Alapértelmezés szerint ez a paraméter 24-re van beállítva.
WYSIWYG mód engedélyezése
Lehetővé teszi a vizuális szerkesztő (WYSIWYG: What You See Is What You Get) használatát a bejegyzések, oldalak és kategóriák leíró mezőihez.

Kiválaszthatja, hogy egy bejegyzés szerkesztése közben az összes használt címkét (Extended) vagy csak a leggyakrabban használtakat (Short) szeretné-e látni.
Szintaxis kiemelés a témaszerkesztőben
Szintaxis kiemelés engedélyezése a témaszerkesztő használatakor (HTML és CSS fájlok)

Az én műszerfalam

A kedvenceim
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Alapértelmezett kedvencekként definiálni (szuper adminisztrátor)
Engedélyezi, hogy az aktuálisan meghatározott kedvencek listáját az összes felhasználó számára alapértelmezett kedvencek listájává tegye. Ez a lista lesz javasolt a felhasználóknak, ha még nem határozták meg saját kedvenceiket.
Egyéb elérhető kedvencek
Itt jelenik meg minden olyan bővítmény, amely lehetővé teszi a kedvencek közé helyezést. Jelölje be a megfelelő négyzeteket, és adja hozzá őket a műszerfalhoz a Add tp my favoritesgomb megnyomásával.
Kedvencek megjelenítése a menü tetején
Ha engedélyezve van, a kedvencek a menü tetején jelennek meg.
Műszerfal modulok
Az ebben a zónában tett döntések megjelennek a blog műszerfalán.
Dokumentációs hivatkozások megjelenítése
Engedélyezze a dokumentációs hivatkozások megjelenítését a műszerfal céloldalán.
Dotclear hírek megjelenítése
Engedélyezze a Dotclear hírek megjelenítését a műszerfal céloldalán.
Gyorsbeviteli űrlap megjelenítése
Engedélyezze a "gyorsbejegyzés" űrlap megjelenítését a műszerfal céloldalán.
Késedelmes feladatok számlálójának megjelenítése a karbantartási műszerfal ikonján
Lehetővé teszi a késedelmes (azaz feldolgozandó) feladatok számának megjelenítését a Karbantartás ikon alatt. A funkció használatához a Karbantartás ikont be kell állítani a kedvencek közé.
Késedelmes feladatok listájának megjelenítése a műszerfal elemein
Engedélyezze a késedelmes (azaz feldolgozandó) feladatok listájának megjelenítését a Dashboard céloldalon egy külön erre a célra szolgáló szakaszban.
