Medialijst

De hoofdpagina van de mediabeheerder toont een lijst met beschikbare bestanden (media) voor het huidige geselecteerde blog.

Je kunt op de pictogrammen of namen van de media in de map of een submap klikken om ze weer te geven.

Als je een MP3-geluidsbestand selecteert, wordt er een speler weergegeven die het bestand onmiddellijk afspeelt. De Flash-plugin moet geïnstalleerd zijn op je browser om deze speler te laten werken.

De afbeeldingen worden indien mogelijk weergegeven met een miniatuur.

Bestanden toevoegen

Kies een bestand
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Titel
Optionele titel voor het bestand dat je wilt verzenden.
Privé
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
Verbeterde uploader
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

Bestanden verwijderen

Om een bestand te verwijderen, schakel je het selectievakje in en klik je op Verwijder geselecteerde media. Je kunt meerdere bestanden tegelijk verwijderen.

Nieuwe map

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Media details

Deze pagina geeft een reeks informatie weer over het momenteel geselecteerde bestand en stelt je in staat om er verschillende bewerkingen op uit te voeren.

Bestandsnaam
De bestandsnaam van het huidige bestand wijzigen.
Titel bestand
Geef een titel aan het bestand of wijzig de huidige titel.
Datum bestand
Wijzig de datum van het bestand.
Privé
Geef aan dat het bestand verborgen moet worden voor iedereen behalve de eigenaar in de mediabeheerder.
Nieuwe map
Hiermee kunt u het bestand verplaatsen naar een andere map.
Bestand wijzigen
With this form, you can replace the file while keeping all the attributes (name, title...). As always, you have to respect the maximum file size.

Specifiek geval: afbeeldingen

Wanneer je een afbeelding toevoegt, zijn er afhankelijk van de grootte tot vier versies beschikbaar (afhankelijk van de originele afbeeldingsgrootte):

  • Vierkant: Een vierkante miniatuur van 48 x 48 pixels.
  • Miniatuurafbeelding: De afbeelding verkleind tot 100 pixels aan de langste zijde.
  • Klein: De langste zijde is verkleind tot 240 pixels.
  • Medium: De langste zijde is verkleind tot 500 pixels.
  • Origineel: De originele, ongewijzigde afbeelding.

Standaardformaten voor de thumbnail, kleine en middelgrote versies kunnen worden aangepast in je bloginstellingen.

Een bestand aan een invoer toevoegen

Je kunt eenvoudig een bestand toevoegen aan een van je berichten. Maak eerst een nieuw bericht aan en sla het op. Klik vervolgens op "Bestanden aan dit bericht toevoegen" om de mediabeheerder te openen.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

De bijgevoegde bestanden worden getoond op de postpagina, als een lijst na de inhoud van de post. De MP3-bestanden worden beschikbaar gemaakt via een Flash-speler, zodat je bezoekers ze gemakkelijk kunnen afspelen.

Podcast en uitzending van muziekbestanden

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
