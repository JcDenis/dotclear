Mijn profiel

Achternaam, Voornaam
If the display name is empty, the author's name will be displayed as his first name and last name as set here.
Naam weergeven
Here you can choose a nickname that will be displayed in your posts instead of your last name and first name.
E-mail
E-mail address used when you request a new password.
New comment notifications will be sent to this address as well.(If you update this field, you must enter your current password in the appropriate field at the bottom of the form)
URL
The user's website. If set, the author name will be displayed as a link to the said website.
Taal voor mijn interface
The language used when publishing entries. If the translation is available, the interface will also be displayed in that language.
Mijn tijdzone
Deze keuze bepaalt de tijd die wordt weergegeven bij de publicatie.
Wijzig je wachtwoord
To change your password, write down twice your password in the fields New password and Confirm password. The minimal password length is 6 characters. If no new password is provided, the current one is kept. (If you update this field, you must enter your current password in the appropriate field at the bottom of the form)

Mijn opties

Interface
Verbeterde uploader activeren in mediabeheerder
Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Slepen en neerzetten met Javascript uitschakelen voor het bestellen van items
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
Aantal weergegeven elementen per pagina in mediabeheerder
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Gebruik geen standaard favicon
Hiermee kan de standaard favicon in de adresbalk van de webbrowser worden uitgeschakeld.
Editie
Voorkeur formaat
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Standaard ingangsstatus
De standaardstatus voor nieuwe items kan worden ingesteld op:
  • In behandeling: De publicatiestatus is nog niet bepaald.
  • Gepland: De invoer wordt online gezet op de tijd en datum die zijn opgegeven in het veld Gepubliceerd op.
  • Ongepubliceerd: Offline invoer.
  • Gepubliceerd: Online invoer.
Hoogte invoerveld bewerken
Stelt de hoogte van het editieveld van een invoer in. Standaard is deze parameter ingesteld op 24.
WYSIWYG-modus inschakelen
Maakt het gebruik van de visuele editor (WYSIWYG: What You See Is What You Get) mogelijk voor de beschrijvingsvelden van vermeldingen, pagina's en categorieÃ«n.

Mijn dashboard

Mijn favorieten
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Definieer als standaard favorieten (superbeheerder)
Inschakelen om de huidige favorietenlijst in te stellen als standaard favorietenlijst voor alle gebruikers. Deze lijst zal aan gebruikers worden voorgesteld als ze nog geen eigen favorieten hebben gedefinieerd.
Andere beschikbare favorieten
Alle plugins die je als favoriet kunt instellen worden hier weergegeven. Vink de bijbehorende vakjes aan en voeg ze toe aan het dashboard door op Add tp my favorites te drukken.
Favorieten bovenaan het menu weergeven
Indien ingeschakeld, worden favorieten bovenaan het menu weergegeven.
Dashboardmodules
Keuzes die je in deze zone maakt, worden weergegeven op het dashboard van je blog.
Documentatielinks weergeven
Inschakelen om documentatielinks weer te geven op de dashboard landingspagina.
Dotclear nieuws weergeven
Laat Dotclear nieuws zien op de dashboard landingspagina.
Snel invoerformulier weergeven
Inschakelen om een formulier voor snelle invoer weer te geven op de dashboard landingspagina.
Teller achterstallige taken weergeven op pictogram onderhoudsdashboard
Maakt het mogelijk om het aantal achterstallige taken (die verwerkt moeten worden) weer te geven onder het Onderhoud-pictogram. Het onderhoudspictogram moet zijn ingesteld in je favorieten om deze functie te kunnen gebruiken.
Lijst met achterstallige taken weergeven op dashboard-items
Inschakelen om de lijst met achterstallige taken (d.w.z. die moeten worden verwerkt) weer te geven in een speciale sectie op de Dashboard landingspagina.
