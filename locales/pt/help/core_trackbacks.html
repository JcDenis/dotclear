Trackbacks

Para fazer um rastreador, clique no link Ping blogs . Se você não vê esse link, certifique-se de que o status da sua postagem está definido como publicado.

URLs a pingar
Cole aqui as URLs de rastreamento que você encontrou nos posts que você deseja.
Resumo para enviar
Por padrão, este campo contém o início de sua entrada. Isto é o que será enviado para o blog que você está pingando, junto com um link para o seu post. Você pode modificar o trecho editando este campo.
Procura automático das URLs para ping
Se a sua entrada estiver vinculada a postagens específicas e se o blog que hospeda essas postagens estiver configurado para permitir isso, esta ferramenta encontrará os URLs para fazer ping para você.
