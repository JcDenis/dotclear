Muuda oma salasõna
Parooli muutmiseks kirjutage oma parool kaks korda üles väljadele Uus parool ja Kinnita parool. Parooli minimaalne pikkus on 6 tähemärki. Kui uut parooli ei sisestata, säilitatakse praegune. (Kui värskendate seda välja, peate sisestama oma praeguse parooli vastavale väljale vormi allosas)
Kaks tegurit autentimine
Two factors authentication uses One Time Password authentication and is used only in addition to standard login+password authentication. To activate this feature, scan QR code with an authentication application or use a browser extension, then enter the verification code. On login, when prompt, enter code given by the application.
Autentimisvõtmed
Autentimisvõtit kasutatakse tavalise sisselogimise+sõna autentimise asemel, see võib olla füüsiline võti või brauseripikendus või paroolihaldur. Selle funktsiooni kasutamiseks peab olema lubatud Javascript ja navigeerija credential API. Registreerida saab mitu võtit.
Autentimisrakendused
If configured, various third party applications are availabled. It is used instead of standard login+password authentication. You can grant or revoke access to these applications if you have accounts on their plateforms, then you can login in using these account.

Minu võimalused

Kasutajaliides
Aktiveerige täiustatud üleslaadimisfunktsioon meediahalduris
Võimaldab aktiveerida meediahalduris täiustatud liidese, mis võimaldab korraga üles laadida mitu faili (Märkus. Selle funktsiooni kasutamiseks peab teie veebibrauser toetama Javascripti).
Lülita Javascriptiga toimuva lohistamise ja loovutamise võimalus esemete tellimiseks välja
Võimaldab keelata üksuste tellimise "pukseerimise". Pukseerimine asendatakse numbriväljadega, mis võimaldavad teil selle asemel sisestada väljade asukohta.
Meediahalduris lehe kohta kuvatavate elementide arv
Määrab meediumihalduris kuvatavate meediumide arvu (katalooge ei võeta üksuste loendamisel arvesse).
Ärge kasutage standardset favicon'i
Võimaldab keelata veebibrauseri aadressiriba vaikimisi favikooni.
Väljaanne
Eelistatud vorming
Valige kirje süntaks. Wiki on lihtsustatud süntaks ja see teisendatakse kehtivaks HTML-iks. Kui te ei mõista HTML-i täiuslikult, soovitame teil valida Wiki süntaksi. Lisateabe saamiseks vaadake Wiki süntaksi viidet.
Vaikimisi sissekande staatus
Uute kirjete vaikimisi olekuks saab määrata:
  • Ootel: Avaldamise staatus ei ole veel otsustatud.
  • Planeeritud: Väljal Avaldatud on esitatud kuupäeval ja kellaajal.
  • Avaldamata: Offline kanne.
  • Avaldatud: Online kanne.
Sissekande redigeerimisvälja kõrgus
Määrab kirjete välja kõrgust. Vaikimisi on selle parameetri väärtuseks 24.
WYSIWYG-režiimi sisselülitamine
Võimaldab visuaalse redaktori (WYSIWYG: What You See Is What You Get) kasutamist kirjete, lehekülgede ja kategooriate kirjeldusväljade jaoks.

Minu armatuurlaud

Minu lemmikud
See tsoon kuvab valitud lemmikud ajaveebi armatuurlaual. Kui Javascript on teie brauseris aktiveeritud, saab neid hiirega ümber järjestada. Vastasel juhul võimaldab väli anda neile tellimisnumbri. Kui olete lõpetanud, ärge unustage muudatusi salvestada, vajutades Salvesta tellimus.
Saate ka lemmikuid kustutada, valides need ja vajutades Kustutage valitud lemmikud.
Määrata vaikimisi lemmikuteks (superadministraator)
Võimaldab määrata hetkel määratud lemmikute nimekirja vaikimisi lemmikute nimekirjaks kõigile kasutajatele. Seda nimekirja pakutakse kasutajatele, kui nad ei ole veel oma lemmikuid määratlenud.
Muud saadaval olevad lemmikud
Siin kuvatakse kõik pluginad, mida saab panna lemmikuteks. Märkige vastavad kastid ja lisage need armatuurlauale, vajutades Add tp my favorites.
Näita lemmikuid menüü ülaosas
Kui see on lubatud, kuvab lemmikud menüü ülaosas.
