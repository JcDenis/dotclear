Linguaggi installati

Troverai qui l'elenco delle lingue installate disponibili per il tuo profilo di amministrazione e per il tuo blog. Ciascuna di queste lingue può essere rimossa.

Nel caso in cui non fosse presente alcuna lingua, l'inglese sarebbe utilizzato per impostazione predefinita per la parte amministrativa e per il blog.

Installa o aggiorna le lingue

Lingue disponibili

Troverete qui un elenco di lingue disponibili sul sito Dotcleare. Per installare una di queste lingue, selezionala nell'elenco a discesa, inserisci la tua password e premi Installa lingua.

La lingua selezionata verrà quindi scaricata e installata sul tuo sistema.

Carica un file zip

È inoltre possibile utilizzare un archivio compresso (formato zip) per installare una lingua. Seleziona il file corrispondente sul tuo computer, inserisci la password e premi Lingua di caricamento.
