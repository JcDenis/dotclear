Riferimento sintassi Wiki

La sintassi Wiki è un modo per arricchire il tuo testo con un set minimo di tag studiato per coprire le esigenze di base (titoli, paragrafi, virgolette, elenchi...).

Blocca elementi
  • Lascia una linea vuota tra due blocchi simili.
  • Paragrafo: Testo libero, terminato con una riga vuota se deve seguire un altro paragrafo.
  • Titolo: !!! titolo, !! titolo o ! titolo, che ti consente di utilizzare tre diversi livelli di intestazione.
  • Horizontal line: ----
  • Elenchi: Inizia ogni riga con * o # rispettivamente per gli elenchi non numerati o numerati. L'embricazione delle liste viene eseguita mescolando i marcatori delle liste in questo modo: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Elenco delle definizioni: Inizia ogni riga con = per il termine da definire e con : per la definizione del termine. 
      = term
  : description of the term
  • Block quote: each line must start with a semicolon (>).
  • Preventivo blocco: Ogni riga deve iniziare con un >.
  • Blocco laterale: Ogni riga deve iniziare con un ).
Formattazione tag
  • Emphasis: two quotes ''text''
  • Strong emphasis: two underscore __text__
  • New line:
  • Insertion: two plusses ++text++ %%%
  • Deletion: two minuses --text--
  • Link: [url], [name|url], [name|url|language] or [name|url|languagee|title]
  • Link: [url], [nome|url], [nome| url|lingua] o [nome|url|lingua|titolo]
  • Immagine:((url testo alternativo)), ((url posizione alternativa text)) o (((url alternativa text position long description)).
    La posizione può essere L (sinistra), R (destra) o C (centrata).
  • Acronym: ??acronym|title??
  • Inline HTML: two backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} or {{quote|language|url}}
  • Citazione in linea: {{quote}}, {{quote|language}} o {{quote|language|url}}
  • Footnotes: $$footnote$$
  • Note a piè di pagina: $$footnote$$
  • Apice : ^superscript^
  • Subscript : ,,subscript,,
Testo non formattato
Se non vuoi che un carattere di formattazione venga interpretato come tale, aggiungi \ subito prima. Ad esempio: \[Testo tra parentesi senza essere un collegamento\]
HTML Code
Anche se scegli la sintassi Wiki, a volte potresti aver bisogno di una formattazione più potente, ad esempio la sintassi HTML. Fallo nel modo seguente: 
///html
<p style="color:red">il mio testo in rosso</p>
///
