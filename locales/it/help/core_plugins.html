Aggiungi plugin

Questa pagina consente di esplorare il repository DotAddict e di installare un plugin direttamente da esso.

Sono disponibili due metodi di ricerca:

  • Digitando il nome del plugin nel campo Cerca nel repository
  • Navigando nell'indice

Nell'indice, dispiegare un plugin facendo clic sulla freccia accanto al suo nome ti permetterà di accedere alla sua pagina DotAddict o di andare su un sito web di supporto se l'autore del plugin ha scelto uno.

Installa o aggiorna manualmente

È possibile installare i propri plugin o plugin che non sono disponibili su DotAddict utilizzando questa pagina.

Ci sono due modi per installarli:

  • Carica un file zip dal tuo computer. Fare clic su Sfoglia per selezionare il file, inserire la password e premere Upload.
  • Scarica un file zip direttamente dal Web inserendo il suo URL, quindi inserisci la password e premi Scarica.
