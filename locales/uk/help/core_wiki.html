Вікі-довідник з синтаксису

Синтаксис Wiki – це спосіб покращити ваш текст за допомогою мінімального набору тегів, вивчених для задоволення основних потреб (заголовки, абзаци, цитати, списки...).

Елементи блоку
  • Між двома однаковими блоками залишайте порожній рядок.
  • Абзац: Довільний текст, закінчений порожнім рядком, якщо слідує інший абзац.
  • Назва: !!! назва, !! назва або ! title, що дозволяє використовувати три різні рівні заголовка.
  • Горизонтальна лінія: ----
  • Списки: Кожен рядок починайте з * або # для ненумерованих або пронумерованих списків відповідно. Змішування списків здійснюється шляхом змішування маркерів списків таким чином: 
        * пункт 1
    ** пункт 1.1
    * пункт 2
    *# пункт 2.1
  ...
  • Списки визначень: Починайте кожен рядок з = для терміну, який потрібно визначити, і з : для визначення терміну. 
      = термін
  : опис терміну
  • Попередньо відформатований текст: Кожен рядок повинен починатися з пробілу.
  • Блоковані цитати: Кожен рядок повинен починатися з >.
  • Бічний блок: Кожен рядок повинен починатися з ).
Форматування тегів
  • Наголос: Дві лапки ''текст''
  • Сильний наголос: Два підкреслення __text__
  • Виділено: Дві подвійні лапки ""текст""
  • Нова лінія: %%%
  • Вставка: Два плюси ++текст++
  • Видалення: Два мінуси --text--
  • Посилання: [url], [ім’я|url], [ім’я| url|мова] або [назва|url|мова|назва]
  • Зображення:((url|alternative text)), (url|альтернативний текст|позиція)) або (url|alternative text|position|long description)).
    Положення може бути або L (зліва), R (праворуч) або C (по центру).
  • Якір: ~anchor~
  • Абревіатура: ?? абревіатура|назва??
  • Вбудований HTML: Дві зворотні лапки ``HTML-код``
  • Вбудована цитата: {{quote}}, {{quote|language}} або {{quote|language|url}}
  • Код: @@code@@
  • Сноски: $$footnote$$
  • Надрядковий знак : ^superscript^
  • Підписка : ,,підписка,,
Неформатований текст
Якщо ви не хочете, щоб один символ форматування інтерпретувався як такий, додайте \ безпосередньо перед ним. Наприклад: \[Текст у дужках без посилання\]
HTML-код
Навіть якщо ви вибрали синтаксис Wiki, іноді вам може знадобитися більш потужне форматування, тобто синтаксис HTML. Зробіть це так: 
///html
<p style="color:red">мій текст червоним</p>
///
