Вікі-довідник з синтаксису

Вікі-синтаксис - це спосіб покращити ваш текст за допомогою мінімального набору тегів, вивчених для задоволення основних потреб (заголовки, абзаци, цитати, списки...).

Елементи блоку
  • Між двома однаковими блоками залишайте порожній рядок.
  • Абзац: Вільний текст, що закінчується порожнім рядком, якщо наступний абзац має бути наступним.
  • Заголовок: !!! заголовок, !!! заголовок або !!! заголовок, що дозволяє використовувати три різні рівні заголовка .
  • Горизонтальна лінія: ----
  • Списки: Починайте кожен рядок з * або # для ненумерованих або нумерованих списків відповідно. Вбудовування списків відбувається шляхом змішування маркерів списків у такий спосіб: 
        * пункт 1
    ** пункт 1.1
    * пункт 2
    *# пункт 2.1
  ...
  • Списки визначень: Починайте кожен рядок з = для терміну, який потрібно визначити, і з : для визначення терміну. 
      = термін
  : опис терміну
  • Попередньо відформатований текст: Кожен рядок повинен починатися з пробілу.
  • Блоковані цитати: Кожен рядок повинен починатися з >.
  • Бічний блок: Кожен рядок повинен починатися з ).
Форматування тегів
  • Наголос: Дві лапки ''текст''
  • Сильний наголос: Два підкреслення __text__
  • Виділено: Дві подвійні лапки ""текст""
  • Нова лінія: %%%
  • Вставка: Два плюси ++текст++
  • Видалення: Два мінуси --text--
  • Посилання: [url], [назва|URL], [назва|URL|мова] або [назва|URL|мова|назва]
  • Зображення: ((url|альтернативний текст)), ((url|альтернативний текст|позиція)) або ((url|альтернативний текст|позиція|довгий опис)).
    Позиція може бути L (ліворуч), R (праворуч) або C (по центру).
  • Якір: ~anchor~
  • Абревіатура: ?? абревіатура|назва??
  • Вбудований HTML: Дві зворотні лапки ``HTML-код``
  • Вбудована цитата: {{quote}}, {{quote|language}} або {{quote|language|url}}
  • Код: @@code@@
  • Сноски: $$footnote$$
  • Надрядковий знак : ^superscript^
  • Підписка : ,,підписка,,
Неформатований текст
Якщо ви не хочете, щоб один символ форматування інтерпретувався як такий, додайте \ безпосередньо перед ним. Наприклад: \[Текст у дужках, який не є посиланням\]
HTML-код
Навіть якщо ви вибрали синтаксис Вікі, іноді вам може знадобитися більш потужне форматування, тобто синтаксис HTML. Зробіть це наступним чином: 
///html
<p style="color:red">мій текст червоним</p>
///
