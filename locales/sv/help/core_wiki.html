Wiki-syntaxreferens

Wiki-syntaxen är ett sätt att förbättra din text med en minimal uppsättning taggar som studeras för att täcka de grundläggande behoven (titlar, stycken, citat, listor...).

Blockera element
  • Lämna en tom linje mellan två liknande block.
  • Stycke: Fritext, avslutas med en tom rad om ytterligare ett stycke ska följa.
  • Titel: !!! titel, !! titel eller ! title, så att du kan använda tre olika rubriknivåer.
  • Horizontal line: ----
  • Listor: Börja varje rad med * eller # för onumrerade respektive numrerade listor. Listimbricering görs genom att blanda listmarkörer på detta sätt: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Definitionslistor: Starta varje rad med = för termen att definiera och med : för definitionen av termen. 
      = term
  : beskrivning av termen
  • Block quote: each line must start with a semicolon (>).
  • Block quote: Varje rad måste börja med >.
  • Sidoblock: Varje rad måste börja med en ).
Formatera taggar
  • Emphasis: two quotes ''text''
  • Strong emphasis: two underscore __text__
  • New line:
  • Insertion: two plusses ++text++ %%%
  • Deletion: two minuses --text--
  • Link: [url], [name|url], [name|url|language] or [name|url|languagee|title]
  • Länk: [url], [namn|url], [namn| url|språk] eller [namn|url|språk|titel]
  • Bild:((url|alternativ text)), ((url|alternativ text|position)) eller ((url| alternativ text|position|lång beskrivning)).
    Positionen kan vara antingen L (vänster), R (höger) eller C (centrerad).
  • Acronym: ??acronym|title??
  • Inline quote: {{quote}}, {{quote|language}} or {{quote|language|url}}
  • Code: @@code@@
  • Inline-citat: {{quote}}, {{quote|language}} eller {{quote|language|url}}
  • Kod: @@code@@
  • Fotnoter: $$footnote$$
  • Superscript : ^superscript ^
  • Subscript : ,,subscript,,
Oformaterad text
Om du inte vill att ett formateringstecken ska tolkas som sådant, lägg till en \ strax före det. Till exempel: \[Text inom parentes utan att vara en länk\]
HTML Code
Även om du valde Wiki-syntaxen kan du ibland behöva en kraftfullare formatering, dvs HTML-syntax. Gör det på följande sätt: 
///html
<p style="color:red">min text i rött</p>
//
