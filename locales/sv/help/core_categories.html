Lista över kategorier

Create a new category
Välj den kategori du vill ta bort genom att klicka på Ta bort kategorin. Ett popup-fönster kommer att be dig bekräfta denna åtgärd.
En kategori kan bara tas bort om den inte innehåller någon artikel i den.
För att tömma en kategori kan du flytta posterna genom att välja (i rullgardinsmenyn) den nya kategorin de ska flyttas till. Klicka på OK för att bekräfta detta val.
Change category order
Du kan ändra kategoriernas ordning genom att dra och släppa. Du kan också flytta en kategori till en överordnad kategori.
När du är nöjd med beställningen klickar du på Spara kategoribeställning.
Ordna om alla kategorier på toppnivå
Använd det här alternativet för att flytta tillbaka alla kategorier till översta nivån.
Om du vill ändra kategoriernas ordning mer exakt kan du:
  • antingen flytta dem med dra och släpp,
  • antingen klicka på varje kategorinamn och välj dess överordnade kategori eller dess position till dess syskon.
