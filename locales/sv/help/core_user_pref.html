Min profil

Name, First name
If the display name is empty, the author's name will be displayed as his first name and last name as set here.
Visa namn
Here you can choose a nickname that will be displayed in your posts instead of your last name and first name.
E-post
E-mail address used when you request a new password.
New comment notifications will be sent to this address as well.(If you update this field, you must enter your current password in the appropriate field at the bottom of the form)
URL
The user's website. If set, the author name will be displayed as a link to the said website.
Preferred format
The language used when publishing entries. If the translation is available, the interface will also be displayed in that language.
Default entry status
Det valet kommer att definiera den tid som visas på post publicering.
Entry edit field height
To change your password, write down twice your password in the fields New password and Confirm password. The minimal password length is 6 characters. If no new password is provided, the current one is kept. (If you update this field, you must enter your current password in the appropriate field at the bottom of the form)

Mina alternativ

Gränsgränssnitt
Aktivera förbättrad uppladdning i mediehanteraren
Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Inaktivera Javascript powered drag and drop för att beställa objekt
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
Antal element som visas per sida i mediehanteraren
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Använd inte standard favicon
Tillåter att inaktivera standard favicon i webbläsarens adressfält.
Upplaga
Föredraget format
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Standard poststatus
Standardstatus för nya poster kan ställas in till:
  • Väntande: Publiceringsstatus har ännu inte beslutats.
  • Schemalagd: Posten kommer att ställas in online vid den tidpunkt och datum som anges i Publicerad på fältet.
  • Avpublicerat: Offline post.
  • Publicerad: Online-post.
Ändra fältets höjd
Sätter höjden på versionsfältet för en post. Som standard är denna parameter satt till 24.
Aktivera WYSIWYG-läge
Tillåter användning av den visuella redigeraren (WYSIWYG: Vad du ser är vad du får) för poster, sidor och kategorier beskrivning fält.
Andra alternativ
Etiketter listformat
Du kan välja om du vill se, medan du redigerar en post, alla taggar som används (Utökad) eller bara de mest använda (Kort).
Syntaxmarkering i tema-editorn
Aktiverar syntaxmarkering vid användning av tema-redigeraren (HTML och CSS-filer)

Min instrumentpanel

Mina favoriter
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Definiera som standardfavoriter (superadministratör)
Aktivera för att ställa in just nu definierade favoriter som standardlista för alla användare. Denna lista kommer att föreslås till användare om de inte har definierat sina egna favoriter ännu.
Andra tillgängliga favoriter
Alla plugins som gör det möjligt att sätta som favoriter delas här. Markera motsvarande rutor och lägg till dem i instrumentpanelen genom att trycka på Lägg till mina favoriter.
Visa favoriter högst upp på menyn
Om aktiverad, visar favoriter högst upp på menyn.
Dashboard moduler
Val som görs i denna zon kommer att återspeglas på din bloggs instrumentpanel.
Visa dokumentationslänkar
Aktivera för att visa dokumentationslänkar på instrumentpanelens landningssida.
Visa Dotclear nyheter
Aktivera för att visa Dotclear nyheter på instrumentpanelens landningssida.
Visa snabbformulär
Aktivera för att visa formuläret "quick entry" på instrumentbrädans landningssida.
Visa förfallna uppgifter räknare på underhåll instrumentpanelen ikonen
Möjliggör för att visa antalet försenade uppgifter (dvs att de ska behandlas) under Underhåll -ikonen. Underhållsikonen måste ställas in i dina favoriter för att använda denna funktion.
Visa listan över förfallna uppgifter på instrumentpanelens objekt
Aktivera för att visa listan över förfallna uppgifter (dvs som ska behandlas) i en dedikerad avdelning på Dashboard landningssidan.
