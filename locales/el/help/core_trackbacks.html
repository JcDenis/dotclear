Κομμάτια

Για να κάνετε μια πίστα, κάντε κλικ στο Ping blogs σύνδεσμο. Αν δεν βλέπετε αυτόν τον σύνδεσμο, βεβαιωθείτε ότι η κατάσταση εισόδου σας έχει οριστεί σε δημοσιευμένη.

URLs για ping
Επικολλήστε εδώ τα URL trackback που βρήκατε στις δημοσιεύσεις που θέλετε να πληκτρολογήσετε.
Απόσπασμα για αποστολή
Από προεπιλογή, αυτό το πεδίο περιέχει την αρχή της καταχώρησής σας. Αυτό είναι που θα αποσταλεί στο blog που πίνετε, μαζί με ένα σύνδεσμο για την ανάρτησή σας. Μπορείτε να τροποποιήσετε το απόσπασμα με την επεξεργασία αυτού του πεδίου.
Αυτόματος εντοπισμός διευθύνσεων URL
Εάν η καταχώρισή σας συνδέεται με συγκεκριμένες αναρτήσεις και εάν το ιστολόγιο που φιλοξενεί αυτές τις αναρτήσεις έχει ρυθμιστεί να το επιτρέπει, αυτό το εργαλείο θα βρει τις διευθύνσεις URL για ping για εσάς.
