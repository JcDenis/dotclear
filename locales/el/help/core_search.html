Επιλογές αναζήτησης

Καθορίστε τις λέξεις ή τους όρους που ψάχνετε στο πεδίο κειμένου και επιλέξτε το πεδίο αναζήτησης:

Αναζήτηση στις καταχωρήσεις
Η αναζήτηση θα γίνει μόνο στα πεδία Τίτλος άρθρου, Περίληψη και Περιεχομένου μιας καταχώρησης.
Αναζήτηση σε σχόλια
Η αναζήτηση θα γίνει μόνο σε σχόλια.

Αποτελέσματα αναζήτησης

Αποτέλεσμα

Ένα αποτέλεσμα αναζήτησης σας επιτρέπει να κάνετε κλικ σε έναν από τους συνδέσμους του άρθρου για να τον επεξεργαστείτε.

Μαζικές ενέργειες

Είναι δυνατό να κάνετε μια παρτίδα ενεργειών σε διάφορα άρθρα σε ένα χρόνο. Οι πιθανές ενέργειες εξαρτώνται από τις άδειες του χρήστη:

  • Δημοσίευση: Δημοσίευση του άρθρου.
  • Αδημοσίευση: Βάλτε το άρθρο εκτός σύνδεσης.
  • Χρονοδιάγραμμα: Προγραμματισμός των καταχωρήσεων που θα τεθούν σε απευθείας σύνδεση κατά την ημερομηνία δημοσίευσης.
  • Σήμανση ως εκκρεμού: Εκκρεμεί για δημοσίευση.
  • Σήμανση ως επιλεγμένα: Επισήμανση των επιλεγμένων καταχωρήσεων.
  • Σημειώστε ως μη επιλεγμένο: Αφαιρέστε την επισήμανση (αν υπάρχει) στις επιλεγμένες καταχωρήσεις.
  • Αλλαγή κατηγορίας: Ανακατεύθυνση στη λίστα κατηγοριών για να αλλάξετε την κατηγορία των επιλεγμένων άρθρων.
  • Αλλαγή γλώσσας: Ανακατεύθυνση στη λίστα γλωσσών για να τροποποιήσετε τις γλώσσες καταχωρήσεων.
  • Αλλαγή συγγραφέα: Επιτρέπει την τροποποίηση του συγγραφέα καταχωρήσεων καθορίζοντας το αναγνωριστικό χρήστη του νέου συγγραφέα.
  • Διαγραφή: Διαγράψτε τις καταχωρήσεις. Θα εμφανιστεί ένα αναδυόμενο παράθυρο για να επιβεβαιώσετε την επιλογή σας.
  • Προσθήκη ετικετών: Προσθήκη ετικετών στις επιλεγμένες καταχωρήσεις.
  • Αφαίρεση ετικετών: Αφαίρεση μιας ή περισσότερων ετικετών από τις επιλεγμένες καταχωρήσεις.
