Προφίλ χρήστη

Id Χρήστη
Τουλάχιστον 2 χαρακτήρες χρησιμοποιώντας γράμματα, αριθμούς ή σύμβολα (χωρίς κενό). Αυτό το πεδίο είναι υποχρεωτικό.
Νέος κωδικός πρόσβασης
Αν θέλετε να αλλάξετε τον κωδικό πρόσβασής σας, εισάγετε τον νέο κωδικό πρόσβασης δύο φορές στα πεδία Νέος κωδικός πρόσβασης και Επιβεβαίωση κωδικού πρόσβασης . Οι κωδικοί πρόσβασης πρέπει να περιέχουν τουλάχιστον 6 χαρακτήρες. Και τα δύο πεδία είναι υποχρεωτικά.
Απαιτείται αλλαγή κωδικού πρόσβασης για σύνδεση
Ενεργοποιήστε αυτό το πλαίσιο έτσι ώστε ένας νέος χρήστης να πρέπει να επιλέξει το δικό του κωδικό πρόσβασης όταν θα συνδεθεί.
Υπερδιαχειριστής
Αυτό θα δώσει στον τρέχοντα χρήστη όλα τα δικαιώματα σε όλα τα blogs που φιλοξενούνται στο σύστημα.
Επώνυμο, Όνομα
Εάν δεν παρέχεται κανένα εμφανιζόμενο όνομα, το εμφανιζόμενο όνομα του συγγραφέα θα περιέχει το επώνυμο και το όνομα που αναγράφεται σε αυτά τα δύο πεδία.
Εμφανιζόμενο όνομα
Μπορείτε να επιλέξετε το ψευδώνυμο που θέλετε να χρησιμοποιήσετε για να υπογράψετε τα άρθρα σας. Αν δεν είναι κενό, αυτό το πεδίο θα χρησιμοποιηθεί αντί για το επώνυμο και τα πεδία ονομάτων.
E-mail
Αυτή η διεύθυνση ηλεκτρονικού ταχυδρομείου θα χρησιμοποιηθεί αν χρειαστεί να ανακτήσετε τον κωδικό πρόσβασής σας από την οθόνη ελέγχου ταυτότητας του Dotclear.
URL
Δηλώνει την ιστοσελίδα του χρήστη. Αν αυτό το πεδίο δεν είναι κενό, το όνομα του συγγραφέα θα γίνει ένας σύνδεσμος με δυνατότητα κλικ σε αυτήν την ιστοσελίδα.

Επιλογές

Διεπαφή
Γλώσσα
Ορίστε την προεπιλεγμένη γλώσσα για νέες καταχωρήσεις. (Automatic Translation) Εάν μια μετάφραση είναι διαθέσιμη για αυτή τη γλώσσα, το blog interface θα μεταφραστεί και σε αυτή τη γλώσσα.
Timezone
Αυτή η επιλογή επηρεάζει τον τρόπο εμφάνισης του χρόνου δημοσίευσης των καταχωρήσεων.
Έκδοση
Προτιμώμενη μορφή
Επιλέξτε τη σύνταξη της καταχώρισης. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρο HTML. Αν δεν έχετε τέλεια κατανόηση του HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη του Wiki. Δείτε την αναφορά σύνταξης Wiki για περισσότερες πληροφορίες.
Προεπιλεγμένη κατάσταση καταχώρησης
Η προεπιλεγμένη κατάσταση για τις νέες καταχωρήσεις μπορεί να οριστεί σε:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί.
  • Προγραμματισμός: Η καταχώρηση θα οριστεί online την ώρα και την ημερομηνία που παρέχεται στο πεδίο Δημοσιεύθηκε στο.
  • Αδημοσίευτο: Καταχώρηση χωρίς σύνδεση.
  • Δημοσιεύθηκε: Ηλεκτρονική καταχώρηση.
Ύψος πεδίου επεξεργασίας εγγραφής
Ορίζει το ύψος του πεδίου έκδοσης μιας καταχώρησης. Από προεπιλογή, αυτή η παράμετρος έχει οριστεί σε 24.
Μορφή λίστας ετικετών
Μπορείτε να επιλέξετε αν θέλετε να δείτε, κατά την επεξεργασία μιας καταχώρησης, all the tags used (Extended) or only the most used ones (Short).
