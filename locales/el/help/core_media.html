Λίστα πολυμέσων

Η κύρια σελίδα του διαχειριστή πολυμέσων εμφανίζει μια λίστα με τα διαθέσιμα αρχεία (πολυμέσα) για το τρέχον επιλεγμένο blog.

Μπορείτε να κάνετε κλικ στα εικονίδια ή τα ονόματα των μέσων στο φάκελο ή σε οποιοδήποτε υποφάκελο για να τα εμφανίσετε.

Επιλέγοντας ένα αρχείο ήχου MP3 εμφανίζει έναν παίκτη για να παίξει αμέσως το αρχείο. Το πρόσθετο Flash πρέπει να εγκατασταθεί στο πρόγραμμα περιήγησής σας για να λειτουργήσει αυτός ο παίκτης.

Οι εικόνες εμφανίζονται με μια μικρογραφία όταν είναι δυνατόν.

Προσθήκη αρχείων

Επιλέξτε ένα αρχείο
Εδώ μπορείτε να επιλέξετε ένα από τα τοπικά σας αρχεία για να το στείλετε στο blog σας. Αν το αρχείο είναι μεγαλύτερο από το μέγιστο μέγεθος, δεν θα σταλεί.
Τίτλος
Προαιρετικός τίτλος για το αρχείο που θέλετε να στείλετε.
Ιδιωτικό
Ενεργοποιήστε αυτή την επιλογή αν θέλετε το αρχείο να είναι ιδιωτικό, δηλαδή προσβάσιμο μόνο από τον ιδιοκτήτη του και δεν είναι προσβάσιμο από το blog (μέχρι να δημοσιευθεί).
Ενισχυμένος αποστολέας
Επιτρέπει την επιλογή ενός ή περισσότερων από τα τοπικά σας αρχεία για να τα στείλετε στο ιστολόγιό σας. Αν οποιοδήποτε αρχείο είναι μεγαλύτερο από το μέγιστο μέγεθος, δεν θα σταλεί.
Όταν επιλέγετε διάφορα τοπικά αρχεία, προστίθενται στην ουρά. Μπορείτε να προσθέσετε περισσότερα αρχεία κάνοντας κλικ ξανά στο Επιλέξτε αρχεία. Μπορείτε επίσης να αφαιρέσετε αρχεία από την ουρά κάνοντας κλικ στο αντίστοιχο κόκκινο εικονίδιο σταυροειδούς σχήματος.
Όταν είστε έτοιμοι, κάντε κλικ στο Μεταφόρτωση.

Διαγραφή αρχείων

Για να διαγράψετε ένα αρχείο, ελέγξτε το πλαίσιο ελέγχου του και κάντε κλικ στο Αφαίρεση επιλεγμένων μέσων. Μπορείτε να διαγράψετε πολλά αρχεία ταυτόχρονα.

Νέος φάκελος

Αυτή η φόρμα σας επιτρέπει να δημιουργήσετε έναν νέο υποφάκελο στον τρέχοντα φάκελο που εμφανίζεται. Απλά επιλέξτε ένα όνομα για αυτό όπως θα κάνατε στον υπολογιστή σας.

Λεπτομέρειες πολυμέσων

Αυτή η σελίδα εμφανίζει ένα σύνολο πληροφοριών σχετικά με το επιλεγμένο αρχείο και σας επιτρέπει να εκτελέσετε διάφορες λειτουργίες σε αυτό.

Όνομα αρχείου
Αλλάξτε το όνομα του αρχείου του τρέχοντος αρχείου.
Τίτλος αρχείου
Δώστε έναν τίτλο στο αρχείο ή αλλάξτε τον τρέχοντα.
Ημερομηνία αρχείου
Αλλαγή της ημερομηνίας του αρχείου.
Ιδιωτικό
Καθορίστε ότι το αρχείο θα είναι κρυμμένο σε οποιονδήποτε εκτός από τον ιδιοκτήτη του στο διαχειριστή πολυμέσων.
Νέος κατάλογος
Σας επιτρέπει να μετακινήσετε το αρχείο σε άλλο φάκελο.
Αλλαγή αρχείου
Με αυτή τη φόρμα, μπορείτε να αντικαταστήσετε το αρχείο διατηρώντας όλα τα χαρακτηριστικά (όνομα, τίτλος. ). Όπως πάντα, πρέπει να σεβαστείτε το μέγιστο μέγεθος αρχείου.

Συγκεκριμένη περίπτωση: εικόνες

Όταν προσθέτετε μια εικόνα, ανάλογα με το μέγεθός της, θα διατίθενται έως και τέσσερις εκδόσεις (ανάλογα με το αρχικό μέγεθος της εικόνας):

  • Τετράγωνο: Μια τετράγωνη μικρογραφία 48x48 εικονοστοιχείων.
  • Μικρογραφία: Η εικόνα μειώθηκε σε 100 εικονοστοιχεία στην μακρύτερη πλευρά της.
  • Μικρό: Η μεγαλύτερη πλευρά μειώνεται σε 240 εικονοστοιχεία.
  • Μεσαίο: Η μεγαλύτερη πλευρά μειώνεται σε 500 pixels.
  • Πρωτότυπο: Η πρωτότυπη, αμετάβλητη εικόνα.

Τα προεπιλεγμένα μεγέθη για τις μικρογραφίες, μικρές και μεσαίες εκδόσεις μπορούν να τροποποιηθούν στις ρυθμίσεις του blog σας.

Επισύναψη αρχείου σε μια καταχώρηση

Μπορείτε εύκολα να επισυνάψετε ένα αρχείο σε μία από τις δημοσιεύσεις σας. Αρχικά, δημιουργήστε μια νέα δημοσίευση και αποθηκεύστε το. Στη συνέχεια, κάντε κλικ στο "Προσθήκη αρχείων σε αυτήν την καταχώρηση" για να ανοίξετε το διαχειριστή πολυμέσων.

Απλά κάντε κλικ στο εικονίδιο [+] (Βοηθήστε αυτό το αρχείο για καταχώρηση). Το αρχείο στη συνέχεια θα προσαρτηθεί στην ανάρτηση.

Τα συνημμένα αρχεία θα εμφανίζονται στη σελίδα του άρθρου, ως μια λίστα μετά το περιεχόμενο του ταχυδρομείου. Τα αρχεία MP3 θα διατεθούν μέσω ενός Flash player, έτσι ώστε να μπορούν εύκολα να παιχτούν από τους επισκέπτες σας.

Podcast και μετάδοση αρχείων μουσικής

Όλα τα συνημμένα αρχεία θα πρέπει να περιλαμβάνονται στις ροές blog σας, επιτρέποντας μια εύκολη podcasting οποιουδήποτε τύπου αρχείου.
