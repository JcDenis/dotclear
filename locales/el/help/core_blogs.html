Φιλτράρισμα της λίστας blogs

Αναζήτηση
Η αναζήτηση βασίζεται στο αναγνωριστικό ιστολογίου. Μπορείτε να χρησιμοποιήσετε τον μπαλαντέρ * για να καθορίσετε μόνο ένα μέρος του ονόματος.
Ταξινόμηση κατά
Τελευταία ενημέρωση του blog, blog όνομα, blog ID.
Ταξινόμηση
Υποδείξτε τη διαλογή που θέλετε να χρησιμοποιήσετε για να φιλτράρετε τα αποτελέσματα (φθίνουσα ή αύξουσα).
Ιστολόγια ανά σελίδα
Επιλογή του αριθμού των blogs που εμφανίζονται ανά σελίδα σε αυτή τη λίστα.
