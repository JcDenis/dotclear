Ενημέρωση

Αυτή η σελίδα σας επιτρέπει να ενημερώσετε το Dotclear αυτόματα. Αν είναι διαθέσιμη μια νέα έκδοση, θα ερωτηθείτε αν θέλετε να ενημερώσετε. Πριν το κάνετε αυτό, μπορείτε να ελέγξετε τι αλλαγές κάνουν αυτή η ενημέρωση φέρει, κάνοντας κλικ στο Πληροφορίες σχετικά με αυτή την έκδοση.

Μπορείτε να επιβάλετε τον έλεγχο της νέας έκδοσης κάνοντας κλικ στο Επιβολή Dotclear έλεγχο ενημερώσεων.

Αν όλα είναι εντάξει, δηλαδή αν δεν έχετε τροποποιήσει χειροκίνητα οποιαδήποτε αρχεία εγκατάστασης, η διαδικασία ενημέρωσης θα είναι:

  • Δημιουργήστε ένα αντίγραφο ασφαλείας της τρέχουσας εγκατάστασής σας στον τρέχοντα κατάλογο εργασίας με όνομα backup-x.y.z.zip (όπου x.y.z είναι οι αριθμοί έκδοσης, π.χ. 2.5.3).
  • Κατεβάστε τη νέα έκδοση.

Μόλις γίνει αυτό, θα εμφανιστεί ένα μήνυμα όπως το ακόλουθο: "Συγχαρητήρια, απέχετε ένα κλικ από το τέλος της ενημέρωσης".
Κάντε κλικ στο Ολοκλήρωση της ενημέρωσης. Θα αποσυνδεθείτε. Η βάση δεδομένων θα ενημερωθεί, εάν χρειάζεται, όταν θα συνδεθείτε ξανά.

Αν τροποποιήσετε χειροκίνητα κάποια αρχεία, θα εμφανίζονται και θα σας συμβουλεύουμε να τα ενημερώσετε χειροκίνητα
