Δημιουργία νέας κατηγορίας

Όνομα
Επιλέξτε το όνομα που θέλετε να δώσετε στη νέα κατηγορία σας. Αυτό το πεδίο είναι υποχρεωτικό.
Γονικός
Είναι δυνατή η δημιουργία υποκατηγοριών. Σε αυτή την περίπτωση, πρέπει να επιλέξετε τη γονική κατηγορία στο πεδίο Γονική.
URL
Το Dotclear δημιουργεί ένα προεπιλεγμένο URL για την κατηγορία, καθιστώντας το προσβάσιμο με μια διαδρομή όπως http://myblog. ld/category/My-category. Κάνοντας κλικ στο μικρό κλείδωμα στα δεξιά, μπορείτε να το τροποποιήσετε όπως θέλετε.
Περιγραφή
Αυτό το περιεχόμενο πεδίου περιγραφής θα εμφανίζεται όταν εισάγετε μια κατηγορία στο blog. Μπορείτε να το συμπληρώσετε με οποιοδήποτε είδος περιεχομένου (παραγράφους, λίστες, κλπ.). Η μορφή που χρησιμοποιείται για αυτό το πεδίο είναι HTML.

Μετακίνηση αυτής της κατηγορίας

Γονική κατηγορία
Είναι δυνατή η δημιουργία υποκατηγοριών. Σε αυτή την περίπτωση, πρέπει να επιλέξετε τη γονική κατηγορία στο πεδίο Γονική.
Κατηγορία εξευγενισμού
Μπορείτε να μετακινήσετε την κατηγορία σε οποιαδήποτε άλλη, η οποία θα γίνει μητρική της. Μπορείτε επίσης να καθορίσετε τη θέση του στα άλλα αδέλφια μετακινώντας την πριν ή μετά από ένα από αυτά.
