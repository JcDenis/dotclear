Αυτό γίνεται για να παρακάμψει τα μηνύματα σφάλματος που εμφανίζονται όταν προσπαθείτε να αναβαθμίσετε το dotclear, ενώ ορισμένα βασικά αρχεία έχουν τροποποιηθεί.

Πριν προχωρήσετε, πρέπει να λάβετε υπόψη ότι οποιοδήποτε βασικό αρχείο που ενδέχεται να έχετε τροποποιήσει θα αντικατασταθεί από τα νέα κατά την εκτέλεση μιας αναβάθμισης. Οποιαδήποτε προσαρμοσμένη αλλαγή στη συνέχεια θα χαθεί!
