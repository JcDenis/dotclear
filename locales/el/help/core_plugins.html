Εγκατεστημένες πρόσθετες λειτουργίες

Κάθε γραμμή ξεκινά με το λογότυπο του plugin θα βρείτε στο μενού (στα αριστερά της οθόνης σας).

Ένα λογότυπο Dotclear δίπλα στα κουμπιά δράσης δείχνει ότι το plugin έρχεται απευθείας με Dotclear και είναι ένα από τα συστατικά του.

Ενεργοποιημένες πρόσθετες λειτουργίες

Αυτή η λίστα εμφανίζει πληροφορίες σχετικά με ενεργοποιημένα πρόσθετα: Όνομα, έκδοση, σύντομη περιγραφή. Είναι δυνατές δύο ενέργειες: Απενεργοποιήστε (προσωρινή λειτουργία) ή Διαγραφή (μόνιμη λειτουργία).

Απενεργοποιημένα πρόσθετα

Αυτή η λίστα εμφανίζει τα πρόσθετα που έχουν απενεργοποιηθεί από το χρήστη. Για κάθε μία, είναι δυνατές δύο ενέργειες: Ενεργοποίηση ή Διαγραφή (μόνιμη λειτουργία).

Προσθήκη προσθέτων

Αυτή η σελίδα σας επιτρέπει να περιηγηθείτε στο αποθετήριο DotAddict και να εγκαταστήσετε ένα πρόσθετο απευθείας από αυτό.

Δύο μέθοδοι αναζήτησης είναι διαθέσιμες:

  • Πληκτρολογώντας το όνομα του πρόσθετου στο πεδίο Αναζήτηση στο χώρο αποθήκευσης,
  • Με την περιήγηση στο ευρετήριο

Στο δείκτη, ξεδιπλώνοντας ένα plugin κάνοντας κλικ στο βέλος δίπλα στο όνομά του, θα σας επιτρέψει να έχετε πρόσβαση στη σελίδα DotAddict ή να μεταβείτε σε μια ιστοσελίδα υποστήριξης αν ο συγγραφέας plugin επιλέχθηκε.

Χειροκίνητη εγκατάσταση ή αναβάθμιση

Μπορείτε να εγκαταστήσετε τα δικά σας πρόσθετα ή πρόσθετα που δεν είναι διαθέσιμα στο DotAddict χρησιμοποιώντας αυτή τη σελίδα.

Υπάρχουν δύο τρόποι για να τους εγκαταστήσετε:

  • Ανεβάστε ένα αρχείο zip από τον υπολογιστή σας. Κάντε κλικ στο Περιήγηση για να επιλέξετε το αρχείο, εισάγετε τον κωδικό πρόσβασής σας και πατήστε Αποστολή.
  • Κατεβάστε ένα αρχείο zip απευθείας από τον Ιστό εισάγοντας τη διεύθυνση URL, στη συνέχεια εισάγετε τον κωδικό πρόσβασής σας και πατήστε Λήψη.
