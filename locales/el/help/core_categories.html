Λίστα κατηγοριών

Διαγραφή κατηγορίας
Επιλέξτε την κατηγορία που θέλετε να διαγράψετε κάνοντας κλικ στο Διαγραφή της κατηγορίας. Ένα αναδυόμενο παράθυρο θα σας ζητήσει να επιβεβαιώσετε αυτήν την ενέργεια.
Μια κατηγορία μπορεί να διαγραφεί μόνο εάν δεν περιέχει κανένα άρθρο σε αυτήν.
Για να αδειάσετε μια κατηγορία, μπορείτε να μετακινήσετε τις εγγραφές επιλέγοντας (στην αναπτυσσόμενη λίστα) τη νέα κατηγορία στην οποία θα μετακινηθούν. Κάντε κλικ στο OK για να επιβεβαιώσετε αυτήν την επιλογή.
Αναδιάταξη κατηγοριών
Μπορείτε να τροποποιήσετε τη σειρά των κατηγοριών χρησιμοποιώντας μεταφορά και απόθεση. Μπορείτε επίσης να μετακινήσετε μια κατηγορία σε μια γονική κατηγορία.
Όταν είστε ικανοποιημένοι με την παραγγελία, κάντε κλικ στο Αποθήκευση παραγγελίας κατηγοριών.
Αναδιάταξη όλων των κατηγοριών στο ανώτερο επίπεδο
Χρησιμοποιήστε αυτήν την επιλογή για να επαναφέρετε όλες τις κατηγορίες στο ανώτερο επίπεδο.
Εάν θέλετε να τροποποιήσετε τη σειρά των κατηγοριών με μεγαλύτερη ακρίβεια, μπορείτε:
  • είτε μετακινήστε τους χρησιμοποιώντας drag and drop,
  • είτε κάντε κλικ σε κάθε όνομα κατηγορίας και επιλέξτε τη μητρική κατηγορία ή τη θέση της στα αδέλφια της.
