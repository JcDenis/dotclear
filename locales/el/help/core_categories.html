Λίστα κατηγοριών

Διαγραφή κατηγορίας
Επιλέξτε την κατηγορία που θέλετε να διαγράψετε κάνοντας κλικ στο Διαγραφή της κατηγορίας. Ένα αναδυόμενο παράθυρο θα σας ζητήσει να επιβεβαιώσετε αυτή την ενέργεια.
Μια κατηγορία μπορεί να διαγραφεί μόνο εάν δεν περιέχει κανένα άρθρο.
Για να αδειάσετε μια κατηγορία, μπορείτε να μετακινήσετε τα λήμματα επιλέγοντας (στην αναπτυσσόμενη λίστα) τη νέα κατηγορία στην οποία θα μετακινηθούν. Κάντε κλικ στο OK για να επιβεβαιώσετε αυτή την επιλογή.
Αναδιάταξη κατηγοριών
Μπορείτε να τροποποιήσετε τη σειρά των κατηγοριών με drag and drop. Μπορείτε επίσης να μετακινήσετε μια κατηγορία σε μια γονική κατηγορία.
Όταν είστε ικανοποιημένοι με τη σειρά, κάντε κλικ στο Αποθήκευση σειράς κατηγοριών.
Αναδιάταξη όλων των κατηγοριών στο ανώτερο επίπεδο
Χρησιμοποιήστε αυτή την επιλογή για να μετακινήσετε όλες τις κατηγορίες πίσω στο κορυφαίο επίπεδο.
Αν θέλετε να τροποποιήσετε τη σειρά των κατηγοριών με μεγαλύτερη ακρίβεια, μπορείτε να :
  • είτε μετακινήστε τους χρησιμοποιώντας drag and drop,
  • είτε κάντε κλικ σε κάθε όνομα κατηγορίας και επιλέξτε τη μητρική κατηγορία ή τη θέση της στα αδέλφια της.
