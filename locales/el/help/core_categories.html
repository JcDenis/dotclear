Λίστα κατηγοριών

Διαγραφή κατηγορίας
Επιλέξτε την κατηγορία που θέλετε να διαγράψετε κάνοντας κλικ στο Διαγράψτε την κατηγορία. Ένα αναδυόμενο παράθυρο θα σας ζητήσει να επιβεβαιώσετε αυτήν την ενέργεια.
Η κατηγορία A μπορεί να διαγραφεί μόνο αν δεν περιέχει κανένα άρθρο.
Για να αδειάσετε μια κατηγορία, μπορείτε να μετακινήσετε τις καταχωρήσεις επιλέγοντας (στην αναπτυσσόμενη λίστα) τη νέα κατηγορία στην οποία θα μετακινηθούν. Πατήστε OK για να επιβεβαιώσετε αυτή την επιλογή.
Αναδιάταξη κατηγοριών
Μπορείτε να τροποποιήσετε τη σειρά κατηγοριών χρησιμοποιώντας drag and drop. Μπορείτε επίσης να μετακινήσετε μια κατηγορία σε μια γονική κατηγορία.
Όταν είστε ικανοποιημένοι με την παραγγελία, κάντε κλικ στο κουμπί Αποθήκευση παραγγελίας κατηγοριών.
Αναδιάταξη όλων των κατηγοριών στο ανώτερο επίπεδο
Χρησιμοποιήστε αυτή την επιλογή για να μετακινήσετε πίσω όλες τις κατηγορίες στο ανώτερο επίπεδο.
Εάν θέλετε να τροποποιήσετε τη σειρά κατηγοριών με μεγαλύτερη ακρίβεια, μπορείτε :
  • είτε μετακινήστε τους χρησιμοποιώντας drag and drop,
  • είτε κάντε κλικ σε κάθε όνομα κατηγορίας και επιλέξτε τη μητρική κατηγορία ή τη θέση της στα αδέλφια της.
