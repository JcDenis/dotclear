Επεξεργασία μιας καταχώρησης

Τίτλος καταχώρισης
Πληκτρολογήστε τον τίτλο της καταχώρησης. Αυτό το πεδίο είναι υποχρεωτικό.
Απόσπασμα
Το περιεχόμενο αυτού του πεδίου θα εμφανίζεται σε σελίδες που εμφανίζουν καταχωρήσεις ως μια λίστα, όπως η αρχική σελίδα ή οι σελίδες κατηγοριών, ακολουθούμενη από έναν σύνδεσμο "συνεχίστε την ανάγνωση" . Θα είναι επίσης ορατό στην αρχή της σελίδας που εμφανίζει ολόκληρη την ανάρτηση . Αν το αφήσετε κενό, το πεδίο περιεχόμενο θα εμφανιστεί εξ ολοκλήρου στην αρχική σελίδα και σε άλλες σελίδες λίστας εισαγωγών.
Περιεχόμενο
Το περιεχόμενο της καταχώρησής σας. Αυτό το πεδίο είναι υποχρεωτικό.
Προσωπικές σημειώσεις
Αυτή η περιοχή κειμένου είναι για προσωπική χρήση, για σημειώσεις ή σημειώσεις. Αυτό που εισάγετε εδώ δεν θα εμφανιστεί ποτέ στο blog.
Κατάσταση εισόδου
Σας επιτρέπει να επιλέξετε την κατάσταση της καταχώρησης:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί.
  • Προγραμματισμός: Η ανάρτηση θα οριστεί online την ώρα και την ημερομηνία που ορίστηκε στο πεδίο Published στο.
  • Αδημοσίευτο: Η δημοσίευση είναι εκτός σύνδεσης.
  • Δημοσιεύθηκε: Η δημοσίευση είναι online.
Ημερομηνία και ώρα δημοσίευσης
Εδώ μπορείτε να αλλάξετε την ημερομηνία και την ώρα δημοσίευσης της καταχώρισης. Εάν η κατάσταση της καταχώρησης είναι Προγραμματισμένη, θα είναι online στη δεδομένη ημερομηνία και ώρα.
Γλώσσα εισόδου
Επιλέξτε τον κωδικό γλώσσας της καταχώρησής σας. Είναι προεπιλεγμένο στη γλώσσα σας (ως χρήστη) αλλά μπορείτε να την αλλάξετε σε όποιον κωδικό θέλετε, π. χ. . "en" ή "es-ar". Αυτός ο κώδικας χρησιμοποιείται για την προβολή της καταχώρησης.
Μορφοποίηση κειμένου
Επιλέξτε τη σύνταξη της καταχώρησης. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρη HTML. Αν δεν έχετε μια τέλεια κατανόηση της HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη Wiki. Δείτε την αναφορά σύνταξης του Wiki για περισσότερες πληροφορίες.
Επιλεγμένη καταχώρηση
Οι επιλεγμένες καταχωρήσεις θα εμφανίζονται στο μενού του blog σας, με τον τίτλο "Best of me".
Κατηγορία
Η κατηγορία σας. Για να δημιουργήσετε μια νέα κατηγορία, ανατρέξτε στην ενότητα Κατηγορίες . Μπορείτε να επιλέξετε να αφήσετε την εγγραφή σας χωρίς μια κατηγορία επιλέγοντας το στοιχείο (κανένα).
Προσθέστε μια νέα κατηγορία
Μπορείτε να δημιουργήσετε μια νέα κατηγορία εισάγοντας τον τίτλο της και επιλέγοντας τη γονική της κατηγορία (ή επιλέξτε (κανένα) για να δημιουργήσετε μια κατηγορία ανωτάτου επιπέδου).
Ετικέτες
Προσθέστε ετικέτες διαχωρισμένες με κόμματα, και πατήστε εντάξει όταν τελειώσετε. Μπορείτε επίσης να επιλέξετε υπάρχουσες ετικέτες κάνοντας κλικ στο Επιλέξτε από τη λίστα.
Συνημμένα
Τα συνημμένα είναι αρχεία πολυμέσων που προστίθενται στην καταχώρηση. Μπορείτε να χρησιμοποιήσετε το κουμπί Προσθήκη αρχείων σε αυτήν την καταχώρηση για να επιλέξετε αρχεία πολυμέσων για να προσθέσετε στην καταχώρηση.
Αυτή η επιλογή θα εμφανιστεί μετά την αποθήκευση της καταχώρησής σας.
Παρακαλούμε ελέγξτε τη βοήθεια του διαχειριστή πολυμέσων για περισσότερες πληροφορίες.
Αποδοχή σχολίων
Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε στους επισκέπτες σας να σχολιάσουν αυτήν την καταχώρηση. Η αντίστοιχη καθολική ρύθμιση μπορεί να βρεθεί στις ρυθμίσεις ιστολογίου.
Αποδοχή trackbacks
Ένα trackback είναι ένας τρόπος για να αφήσετε ένα τμήμα της καταχώρησης σας ως σχόλιο για άλλο blog. Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε σε άλλους να παρακολουθούν την καταχώρησή σας. Η αντίστοιχη καθολική ρύθμιση θα βρεθεί στις ρυθμίσεις ιστολογίου.
Κωδικός
Μπορείτε να εισάγετε έναν κωδικό πρόσβασης για την καταχώρησή σας. Μια καταχώρηση με προστασία κωδικού πρόσβασης δεν θα εμφανίζεται στο ιστολόγιό σας, θα είναι προσβάσιμο μόνο σε εκείνους που θα δώσει το URL της καταχώρησης για να (δείτε το σύνδεσμο προβολή σύνδεση) και τον κωδικό πρόσβασης.
Βασικό
Μπορείτε να καταργήσετε την προστασία αυτού του πεδίου, κάνοντας κλικ στο μικρό κλείδωμα στα δεξιά του και επιλέγοντας μια άλλη διεύθυνση URL για την καταχώρησή σας. Αν το URL που προσπαθείτε να χρησιμοποιήσετε χρησιμοποιείται ήδη από άλλη καταχώρηση, ένας αριθμός θα προστεθεί σε αυτό.

Λίστα σχολίων
Στην καρτέλα Σχόλια , μπορείτε να διαβάσετε και να αλλάξετε την κατάσταση των σχολίων της καταχώρησής σας (και trackback). Αν σας επιτρέπεται από τον διαχειριστή του ιστολογίου, θα είστε σε θέση να ορίσετε τα σχόλιά σας (και trackbacks) σε απευθείας σύνδεση ή εκτός σύνδεσης, να τα διαγράψετε ή να τα επισημάνετε ως σκουπίδια.
Προσθέστε ένα σχόλιο
Στην καρτέλα Προσθέστε ένα σχόλιο , μπορείτε να απαντήσετε σε ένα από τα σχόλια της καταχώρησής σας χωρίς να αφήσετε το backend του ιστολογίου σας. Η σύνταξη που θα χρησιμοποιηθεί εδώ είναι απλή, απεριόριστη HTML. Χρησιμοποιήστε τα πεδία σαν να τροποποιήσατε ένα σχόλιο.
