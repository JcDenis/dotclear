Επεξεργασία μιας καταχώρησης

Τίτλος καταχώρισης
Πληκτρολογήστε τον τίτλο της καταχώρησης. Αυτό το πεδίο είναι υποχρεωτικό.
Απόσπασμα
Το περιεχόμενο αυτού του πεδίου θα εμφανίζεται σε σελίδες που εμφανίζουν καταχωρήσεις ως λίστα, όπως η αρχική σελίδα ή οι σελίδες κατηγοριών, ακολουθούμενες από έναν σύνδεσμο "συνέχεια ανάγνωσης". Θα είναι επίσης ορατό στην αρχή της σελίδας που δείχνει ολόκληρη την ανάρτηση. Εάν το αφήσετε κενό, το πεδίο περιεχόμενο θα εμφανίζεται εξ ολοκλήρου στην αρχική σελίδα και σε άλλες σελίδες της λίστας καταχωρήσεων.
Περιεχόμενο
Το περιεχόμενο της καταχώρησής σας. Αυτό το πεδίο είναι υποχρεωτικό.
Προσωπικές σημειώσεις
Αυτή η περιοχή κειμένου είναι για προσωπική χρήση, για σημειώσεις ή σημειώσεις. Ό,τι εισάγετε εδώ δεν θα εμφανίζεται ποτέ στο ιστολόγιο.
Κατάσταση εισόδου
Σας επιτρέπει να επιλέξετε την κατάσταση της καταχώρησης:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί.
  • Προγραμματισμός: Η ανάρτηση θα οριστεί online την ώρα και την ημερομηνία που ορίστηκε στο πεδίο Published στο.
  • Αδημοσίευτο: Η δημοσίευση είναι εκτός σύνδεσης.
  • Δημοσιεύθηκε: Η δημοσίευση είναι online.
Ημερομηνία και ώρα δημοσίευσης
Εδώ μπορείτε να αλλάξετε την ημερομηνία και την ώρα δημοσίευσης της καταχώρισης. Εάν η κατάσταση της καταχώρισης είναι Προγραμματισμένη, θα είναι συνδεδεμένη τη δεδομένη ημερομηνία και ώρα.
Γλώσσα εισόδου
Επιλέξτε τον κωδικό γλώσσας της συμμετοχής σας. Είναι προεπιλεγμένη στη γλώσσα σας (ως χρήστης), αλλά μπορείτε να την αλλάξετε σε όποιον κωδικό γλώσσας θέλετε, π.χ. "en" ή "es-ar". Αυτός ο κωδικός χρησιμοποιείται για την εμφάνιση της καταχώρισης.
Μορφοποίηση κειμένου
Επιλέξτε τη σύνταξη της καταχώρισης. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρο HTML. Αν δεν έχετε τέλεια κατανόηση του HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη του Wiki. Δείτε την αναφορά σύνταξης Wiki για περισσότερες πληροφορίες.
Επιλεγμένη καταχώρηση
Οι επιλεγμένες καταχωρήσεις θα εμφανίζονται στο μενού του blog σας, με τον τίτλο "Best of me".
Κατηγορία
Η κατηγορία της καταχώρισής σας. Για να δημιουργήσετε μια νέα κατηγορία, ανατρέξτε στην ενότητα Κατηγορίες . Μπορείτε να επιλέξετε να αφήσετε την καταχώρισή σας χωρίς κατηγορία επιλέγοντας το στοιχείο (καμία) .
Προσθέστε μια νέα κατηγορία
Μπορείτε να δημιουργήσετε μια νέα κατηγορία εισάγοντας τον τίτλο της και επιλέγοντας τη γονική της κατηγορία (ή επιλέξτε (κανένα) για να δημιουργήσετε μια κατηγορία ανωτάτου επιπέδου).
Ετικέτες
Προσθέστε ετικέτες διαχωρισμένες με κόμματα, και πατήστε εντάξει όταν τελειώσετε. Μπορείτε επίσης να επιλέξετε υπάρχουσες ετικέτες κάνοντας κλικ στο Επιλέξτε από τη λίστα.
Συνημμένα
Τα συνημμένα είναι αρχεία πολυμέσων που προστίθενται στην καταχώρηση. Μπορείτε να χρησιμοποιήσετε το κουμπί Προσθήκη αρχείων σε αυτήν την καταχώρηση για να επιλέξετε αρχεία πολυμέσων για προσθήκη στην καταχώρηση.
Αυτή η επιλογή θα εμφανιστεί αφού αποθηκεύσετε την καταχώρισή σας.
Ελέγξτε τη βοήθεια του Media Manager για περισσότερες πληροφορίες.
Αποδοχή σχολίων
Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε στους επισκέπτες σας να σχολιάσουν αυτήν την καταχώρηση. Η αντίστοιχη καθολική ρύθμιση μπορεί να βρεθεί στις ρυθμίσεις ιστολογίου.
Αποδοχή trackbacks
Το trackback είναι ένας τρόπος για να αφήσετε ένα μέρος της καταχώρισής σας ως σχόλιο σε άλλο ιστολόγιο. Επιλέξτε αυτό το πλαίσιο για να επιτρέψετε σε άλλους να παρακολουθήσουν την καταχώρισή σας. Η αντίστοιχη καθολική ρύθμιση βρίσκεται στις ρυθμίσεις του ιστολογίου.
Κωδικός
Μπορείτε να εισαγάγετε έναν κωδικό πρόσβασης για την είσοδό σας. Μια καταχώριση που προστατεύεται με κωδικό πρόσβασης δεν θα εμφανίζεται στο ιστολόγιό σας, θα είναι προσβάσιμη μόνο σε αυτούς στους οποίους θα δώσετε τη διεύθυνση URL εισόδου (δείτε τον σύνδεσμο προβολής ) και τον κωδικό πρόσβασης.
Βασικό
Μπορείτε να καταργήσετε την προστασία αυτού του πεδίου, κάνοντας κλικ στο μικρό κλείδωμα στα δεξιά του και επιλέγοντας μια άλλη διεύθυνση URL για την καταχώρησή σας. Αν το URL που προσπαθείτε να χρησιμοποιήσετε χρησιμοποιείται ήδη από άλλη καταχώρηση, ένας αριθμός θα προστεθεί σε αυτό.

Σχόλια

Λίστα σχολίων
Στην καρτέλα Σχόλια , μπορείτε να διαβάσετε και να αλλάξετε την κατάσταση των των σχολίων (και των trackbacks) της καταχώρισής σας. Εάν σας το επιτρέψει ο διαχειριστής του ιστολογίου, θα μπορείτε να ορίσετε τα σχόλιά σας (και τα trackbacks) στο διαδίκτυο ή εκτός σύνδεσης, να τα διαγράψετε ή να τα επισημάνετε ως ανεπιθύμητα.
Προσθέστε ένα σχόλιο
Στην καρτέλα Προσθέστε ένα σχόλιο , μπορείτε να απαντήσετε σε ένα από τα σχόλια της καταχώρησής σας χωρίς να αφήσετε το backend του ιστολογίου σας. Η σύνταξη που θα χρησιμοποιηθεί εδώ είναι απλή, απεριόριστη HTML. Χρησιμοποιήστε τα πεδία σαν να τροποποιήσατε ένα σχόλιο.
