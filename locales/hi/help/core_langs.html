संस्थापित भाषाएँ

आप यहाँ आपके प्रशासन प्रोफ़ाइल के लिए उपलब्ध स्थापित भाषाओं की सूची और अपने ब्लॉग के लिए उपलब्ध होगा. इन भाषाओं में से प्रत्येक को हटा दिया जा सकता है।

ऐसे मामले में जहां कोई भाषा मौजूद नहीं होगी, अंग्रेजी का उपयोग प्रशासन के भाग के साथ-साथ ब्लॉग के लिए भी डिफ़ॉल्ट रूप से किया जाएगा।

भाषाओं को संस्थापित या अद्यतन करें

उपलब्ध भाषाएँ

You will find here a list of languages available on Dotclear website. To install one of these languages, select it in the dropdown list, enter your password and press Install language.

तब चयनित भाषा को डाउनलोड करके आपके सिस्टम पर स्थापित किया जाएगा.

एक ज़िप फ़ाइल अपलोड करें

You can also use a compressed archive (zip format) to install a language. Select the corresponding file on your computer, enter your password and press Upload language.
