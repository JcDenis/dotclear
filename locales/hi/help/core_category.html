नया वर्ग बनाएँ

नाम
नाम चुनें जिसे आप अपनी नई श्रेणी में देना चाहते हैं. यह क्षेत्र अनिवार्य है.
जनक
It is possible to create sub-categories. In that case, you have to select its parent category in the Parent field.
यूआरएल
Dotclear creates a default URL for the category, making it reachable with a path such as http://myblog.tld/category/My-category. By clicking the little lock on the right, you can modify it the way you like.
वर्णन
इस विवरण क्षेत्र सामग्री को ब्लॉग में एक श्रेणी में प्रवेश करते समय प्रदर्शित किया जाएगा. आप इसे किसी भी प्रकार की सामग्री के साथ भर कर सकते हैं (पैराग्राफ, सूची, आदि). इस क्षेत्र के लिए प्रयुक्त प्रारूप HTML है.

इस श्रेणी को खिसकाएँ

मातृ श्रेणी
It is possible to create sub-categories. In that case, you have to select its parent category in the Parent field.
सीलिंग श्रेणी
आप इस श्रेणी को किसी भी अन्य व्यक्ति में ले जा सकते हैं, जो उसका जनक बन जाएगा. आप इसे अपने दूसरे भाई-बहनों में से किसी एक के बाद या उसके बाद भी निर्दिष्ट कर सकते हैं.
