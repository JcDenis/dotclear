Référence de la syntaxe wiki

La syntaxe Wiki est un moyen d'enrichir votre texte avec un ensemble minimal de balises étudiées pour couvrir les besoins de base (titres, paragraphes, citations, listes...).

Éléments de bloc
  • Laissez une ligne vide entre chaque bloc de même nature.
  • Paragraphe : Texte libre, terminé par une ligne vide si un autre paragraphe doit suivre.
  • Titre : ! !! titre, ! ! titre ou ! titre, ce qui vous permet d'utiliser trois niveaux de titre différents.
  • Trait horizontal : ----
  • Listes : Commencez chaque ligne par * ou # pour les listes non numérotées ou numérotées respectivement. L'imbrication des listes se fait en mélangeant les marqueurs de liste de cette manière : 
        * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Listes de définition : ligne débutant par = pour le terme à définir et par : pour la définition du terme. 
      = terme
  : définition du terme
  • Texte préformaté : espace avant chaque ligne de texte.
  • Bloc de citation : > devant chaque ligne de texte.
  • Bloc de côté : ) devant chaque ligne de texte
Éléments de formatage
  • Emphase : deux apostrophes ''texte''
  • Forte emphase : deux soulignés __texte__
  • Surligné : Deux guillemets ""texte""
  • Nouvelle ligne : %%%
  • Insertion : Deux plus ++texte++
  • Suppression : Deux moins --text--
  • Lien : [url], [nom|url], [nom|url|langue] ou [nom|url|langue|titre]
  • Image :((url|texte alternatif)), ((url|texte alternatif|position)) ou ((url|texte alternatif|position|longue description)).
    La position peut être L (gauche), R (droite) ou C (centré).
  • Ancrage : ~anchor~
  • Acronyme : ??acronyme|titre ??
  • HTML en ligne : Deux guillemets ``Code HTML``
  • Citation en ligne: {{quote}}, {{quote|language}} ou {{quote|language|url}}
  • Code : @@code@@
  • Notes de bas de page : $$footnote$$
  • Exposant : ^ exposant ^
  • Indice : ,,indice,,
Empêcher le formatage du texte
Si vous ne voulez pas qu'un caractère de formatage soit interprété comme tel, ajoutez un \ juste avant. Par exemple : \[Texte entre parenthèses sans lien]
Insérer du code HTML
Même si vous avez choisi la syntaxe Wiki, vous pouvez parfois avoir besoin d'un formatage plus puissant, c'est-à-dire la syntaxe HTML. Procédez de la manière suivante : 
///html
<p style="color:red">mon texte en rouge</p>
///
