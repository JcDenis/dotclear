ইনস্টল করা ভাষা

আপনি এখানে আপনার প্রশাসন প্রোফাইলের জন্য এবং আপনার ব্লগের জন্য ইনস্টল করা ভাষার তালিকা খুঁজে পাবেন. এই ভাষাগুলোর প্রত্যেকটি অপসারণ করা যায়.

যেখানে কোন ভাষা উপস্থিত থাকত না, সেক্ষেত্রে ইংরেজরা প্রশাসন অংশের জন্য ডিফল্টরূপে ব্যবহার করা হত এবং ব্লগের জন্য.

You will find here a list of languages available on Dotclear website. To install one of these languages, select it in the dropdown list, enter your password and press Install language.
