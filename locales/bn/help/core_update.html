আপডেট

This page allows you to update Dotclear automatically. If a new version is available, you will be asked if you want to update. Before doing so, you can check what changes do this update bring by clicking on Information about this version.

You can force the new version check by clicking on Force Dotclear update checking.

যদি সবকিছু ঠিক হয়ে থাকে, অর্থাৎ যদি আপনি কোন ইনস্টলেশন ফাইল পরিবর্তন না করেন, তাহলে আপডেট প্রক্রিয়া হবে:

  • Backup-x.y.z.zip নামের বর্তমান কাজের ডিরেক্টরিতে আপনার বর্তমান ইনস্টলেশনের একটি ব্যাকআপ তৈরি করুন (যেখানে x.y.z সংস্করণ সংখ্যা, যেমন ২.৫.৩).
  • নতুন সংস্করণ ডাউনলোড করুন.

Once this is done, a message such as the following will be displayed: "Congratulations, you're one click away from the end of the update."
Click on Finish the update. You will be logged out. The database will be updated, if needed, when you will log back in.

আপনি যদি কিছু ফাইল পরিবর্তন করেন, তাহলে তালিকাভুক্ত করা হবে এবং তাদেরকে ম্যানুয়ালে আপডেট করার পরামর্শ দেয়া হবে.

ব্যাকআপ ফাইল পরিচালনা করুন

এই ট্যাব থেকে আপনি পুরোনো ব্যাকআপ ফাইল অপসারণ করতে পারেন.

আপনি একটি পূর্ববর্তী সংস্করণেও রোল করতে পারেন. নতুন সংস্করণের সাথে মারাত্মক ঝামেলা থাকলে কেবল রোলিং করার কথা বিবেচনা করা হবে.
