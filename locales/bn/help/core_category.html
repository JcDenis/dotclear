নাম:
আপনার নতুন শ্রেণীতে আপনি যে নাম দিতে চান তা বেছে নিন. এই মাঠ বাধ্যতামূলক.
প্যারেন্ট
It is possible to create sub-categories. In that case, you have to select its parent category in the Parent field.
CITY NAME (OPTIONAL,
Dotclear creates a default URL for the category, making it reachable with a path such as http://myblog.tld/category/My-category. By clicking the little lock on the right, you can modify it the way you like.
বর্ণনা
এই বিবরণ ফিল্ড কন্টেন্ট ব্লগে একটি শ্রেণীতে প্রবেশ করলে প্রদর্শিত হবে. আপনি যে কোন ধরনের বিষয়বস্তু (অনুচ্ছেদ, তালিকা ইত্যাদি) দিয়ে তা পূরণ করতে পারেন. এই ক্ষেত্রের জন্য ব্যবহৃত ফরম্যাট হল এইচটিএমএল.

এই শ্রেণীবিভাগ স্থানান্তর করো

প্যারেন্ট শ্রেণীবিভাগ
It is possible to create sub-categories. In that case, you have to select its parent category in the Parent field.
সিলিং ক্যাটাগরী
আপনি এই শ্রেণীটিকে অন্য যেকোন একটিতে স্থানান্তর করতে পারেন, যা তার পিতামাতা হয়ে উঠবে. আপনি তার অন্য ভাইবোনদের কাছে এর অবস্থান উল্লেখ করতে পারেন এর আগে বা তার পরে.
