Medienliste

Der Mediamanager zeigt eine Übersicht aller Medien, die Du in Deinem Blog, in dem Du gerade arbeitest, verwendest oder hochgeladen hast.

Du klickst auf den Namen oder das Icon oder den Unterordner, um diese ansehen zu können.

Wenn Du ein MP3 anklickst, zeigt sich ein kleiner Player und Du kannst die Datei abspielen. Allerdings muss das Flash plugin in Deinem Browser installiert sein, um MP3 abspielen zu koennen.

Bilder werden sofern möglich mit einem Thumbnail dargestellt.

Neuer Ordner

Details zu den Medien

Wenn Du ein Bild hinzufügst, 4 verschiedene Grössen werden möglich sein:

Name der Datei
Ändern Sie den Dateinamen der aktuellen Datei.
Titel der Datei
Geben Sie der Datei einen Titel oder ändern Sie den aktuellen Titel.
Datum der Datei
Ändern Sie das Dateidatum.
Privat
Legen Sie fest, dass die Datei im Medienmanager für alle außer dem Eigentümer verborgen wird.
Neues Verzeichnis
Ermöglicht es Ihnen, die Datei in einen anderen Ordner zu verschieben.
Datei ändern
Mit diesem Formular können Sie die Datei ersetzen und dabei alle Attribute (Name, Titel...) beibehalten. Wie immer müssen Sie die maximale Dateigröße beachten.

Sonderfall: Bilder

Um eine Datei einem Beitrag zuzufügen, musst Du zuerst einen Blogbeitrag erstellen. Klicke auf "Datei zu diesem Beitrag hinzufügen" und es öffnet sich der Medienmanager.

  • Quadrat: ein 48x48 Pixel Vorschaubild
  • Vorschaubild: das Bild wird zu einer Breite von 100 Pixel in der Breite reduziert.
  • Klein: Die Breite ist 240 Pixel
  • Mittel: Die Breite beträgt 500 Pixel.
  • Original: die Originalgrösse

Klicke auf das [+] Icon (Datei zu diesem Beitrag hinzufügen). Die Datei wird dem Beitrag hinzugefügt.

Eine Datei an einen Eintrag anhängen

Die angefügten Dateien werden rechts neben dem Beitrag angezeigt. MP3 Dateien werden durch den mitgelieferten Flash Player für Besucher abspielbar sein.

Alle Dateien sind in Deinem RSS Feed eingefügt und erlaubt jeden Dateityp anzubieten.

Die angehängten Dateien werden auf der Beitragsseite in einer Liste nach dem Inhalt des Beitrags angezeigt. Die MP3-Dateien werden über einen Flash-Player zur Verfügung gestellt, so dass sie von Ihren Besuchern einfach abgespielt werden können.

Podcast und Ausstrahlung von Musikdateien

Alle angehängten Dateien werden in Ihre Blog-Feeds aufgenommen, was ein einfaches podcasting jeder Art von Datei ermöglicht.
