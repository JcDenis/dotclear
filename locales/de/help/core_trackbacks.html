Trackbacks

Um einen Trackback zu erstellen, klicken Sie auf den Link Ping blogs . Wenn Sie diesen Link nicht sehen, vergewissern Sie sich, dass der Status Ihres Eintrags auf veröffentlichteingestellt ist.

URLs zum Anpingen
Fügen Sie hier die Trackback-URLs ein, die Sie in den Beiträgen gefunden haben, die Sie anpingen möchten.
Auszug zu senden
Dieses Feld enthält standardmäßig den Anfang Ihres Eintrags. Dieser wird zusammen mit einem Link zu Ihrem Beitrag an den Blog, den Sie anpingen, gesendet. Sie können den Auszug ändern, indem Sie dieses Feld bearbeiten.
Automatische Erkennung von Ping-URLs
Wenn Ihr Eintrag auf bestimmte Beiträge verlinkt und der Blog, der diese Beiträge hostet, so eingestellt ist, dass er dies zulässt, findet dieses Tool die URLs, die Sie anpingen können.
