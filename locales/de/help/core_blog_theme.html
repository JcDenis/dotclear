Aktualisierung der Themen

Wenn neue Versionen der in Ihrem Blog installierten Themen in das Dotaddict-Ressourcenzentrum hochgeladen wurden, gibt es auf der Seite eine zusätzliche Registerkarte mit dem Namen Theme Updates, auf der die Themen mit der installierten Versionsnummer und der Versionsnummer des Updates aufgeführt sind.

Verwenden Sie dann einfach die Aktionstaste Update.

Wenn mehrere Aktualisierungen verfügbar sind, ermöglicht eine Aktionsschaltfläche am unteren Rand der Seite die gleichzeitige Aktualisierung aller Themen .

Bitte beachten Sie: Alle Änderungen, die Sie an den Themendateien vorgenommen haben, gehen bei der Aktualisierung des Themas verloren.

Installierte Themen

Auf dieser Registerkarte wird die Liste der installierten Themen angezeigt. Sie können ein Thema auswählen, indem Sie es anklicken und auf Dieses Thema verwendendrücken.

Das aktuell ausgewählte Thema kann über die Schaltfläche Themen-Dateien bearbeiten geändert werden. Wenn das ausgewählte Thema einen Themen-Editor enthält, kann es über die Schaltfläche Thema konfigurieren leicht angepasst werden.

Sie können ein Thema auch deaktivieren (z. B. wenn Sie Kompatibilitätsprobleme damit haben). Es erscheint eine Registerkarte Deaktivierte Themen mit einer Liste dieser Themen.

Themen hinzufügen

Auf dieser Registerkarte können Sie das DotAddict-Theme-Repository entweder in alphabetischer Reihenfolge oder mit Hilfe des Suchfeldes durchsuchen. Die bereits installierten Themen erscheinen nicht in der Liste.

Manuelle Installation oder Aktualisierung

Wenn sich das Thema, das Sie installieren (oder aktualisieren) möchten, nicht im DotAddict-Repository befindet, können Sie es entweder von Ihrer Festplatte hochladen oder seine Installations-URL eingeben. Ein Theme ist immer als Zip-Datei verpackt.
