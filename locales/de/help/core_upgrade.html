Update

Diese Seite ermöglicht es Ihnen, Dotclear automatisch zu aktualisieren. Wenn eine neue Version verfügbar ist, werden Sie gefragt, ob Sie aktualisieren möchten. Bevor Sie dies tun, können Sie überprüfen, welche Änderungen dieses Update mit sich bringt, indem Sie auf Informationen über diese Versionklicken.

Sie können die Prüfung der neuen Version erzwingen, indem Sie auf Force Dotclear update checkingklicken.

Wenn alles in Ordnung ist, d. h. wenn Sie keine Installationsdateien manuell geändert haben, wird der Aktualisierungsvorgang durchgeführt:

  • Erstellen Sie eine Sicherungskopie Ihrer aktuellen Installation im aktuellen Arbeitsverzeichnis unter dem Namen backup-x.y.z.zip (wobei x.y.z die Versionsnummern sind, z. B. 2.5.3).
  • Laden Sie die neue Version herunter.

Once this is done, a message such as the following will be displayed: "Congratulations, you're one click away from the end of the update."
Click on Finish the update. You will be logged out. The database will be updated, if needed, when you will log back in.

Wenn Sie einige Dateien manuell geändert haben, werden diese aufgelistet und Sie werden aufgefordert, sie manuell zu aktualisieren.
