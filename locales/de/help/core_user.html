Benutzerprofil

Benutzer-ID
Mindestens 2 Zeichen mit Buchstaben, Zahlen oder Symbolen (kein Leerzeichen). Dieses Feld ist obligatorisch.
Neues Passwort
Wenn Sie Ihr Passwort ändern möchten, geben Sie das neue Passwort zweimal in die Felder Neues Passwort und Passwort bestätigen ein. Passwörter müssen mindestens 6 Zeichen enthalten. Beide Felder sind Pflichtfelder.
Passwortänderung für Verbindung erforderlich
Markieren Sie dieses Kästchen, damit ein neuer Benutzer sein eigenes Kennwort wählen muss, wenn er eine Verbindung herstellt.
Super-Verwalter
Dadurch erhält der aktuelle Benutzer alle Berechtigungen für alle Blogs, die auf dem System gehostet werden.
Nachname, Vorname
Wird kein Anzeigename angegeben, enthält der angezeigte Autorenname den Nachnamen und den Vornamen, die in diese beiden Felder eingegeben werden.
Name anzeigen
Sie können den Spitznamen auswählen, mit dem Sie Ihre Artikel unterschreiben möchten. Wenn dieses Feld nicht leer ist, wird es anstelle der Felder für den Nachnamen und den Vornamen verwendet.
E-Mail
Diese E-Mail-Adresse wird verwendet, wenn Sie Ihr Kennwort über den Authentifizierungsbildschirm von Dotclear wiederherstellen müssen.
URL
Gibt die Website des Benutzers an. Wenn dieses Feld nicht leer ist, wird der Name des Autors zu einem anklickbaren Link zu dieser Website.

Optionen

Schnittstelle
Sprache
Legen Sie die Standardsprache für neue Einträge fest. Wenn eine Übersetzung für diese Sprache verfügbar ist, wird auch die Blog-Oberfläche in diese Sprache übersetzt.
Zeitzone
Diese Option wirkt sich auf die Art und Weise aus, wie die Veröffentlichungszeit der Einträge angezeigt wird.
Ausgabe
Bevorzugtes Format
Wählen Sie die Syntax des Eintrags. Wiki ist eine vereinfachte Syntax und wird in gültiges HTML umgewandelt. Wenn Sie nicht über perfekte HTML-Kenntnisse verfügen, raten wir Ihnen, die Wiki-Syntax zu wählen. Weitere Informationen finden Sie in der Wiki-Syntax-Referenz.
Standard-Eintragsstatus
Der Standardstatus für neue Einträge kann auf gesetzt werden:
  • Anhängig: Über den Veröffentlichungsstatus wurde noch nicht entschieden.
  • Geplant: Der Eintrag wird zu dem Datum und der Uhrzeit online gestellt, die im Feld angegeben sind. Veröffentlicht auf .
  • Unveröffentlicht: Offline-Eintrag.
  • Veröffentlicht: Online-Eintrag.
Höhe des Eingabefeldes
Legt die Höhe des Ausgabefelds eines Eintrags fest. Standardmäßig ist dieser Parameter auf 24 eingestellt.
Tags Listenformat
Sie können auswählen, ob Sie bei der Bearbeitung eines Eintrags alle verwendeten Tags (Extended) oder nur die am häufigsten verwendeten Tags (Short) sehen möchten.
