Wiki-Syntax-Referenz

Die Wiki-Syntax ist eine Möglichkeit, Ihren Text mit einem minimalen Satz von Tags zu verbessern, die die grundlegenden Bedürfnisse abdecken (Überschriften, Absätze, Zitate, Listen...).

Blockelemente
  • Lass eine leere Zeile zwischen zwei gleichen Blocks
  • Absatz: Freier Text, der mit einer Leerzeile endet, wenn ein weiterer Absatz folgen soll.
  • Titel: !!! Titel, !! Titel oder ! Titel, so dass Sie drei verschiedene Ebenen von Überschriften verwenden können.
  • Horizontale Linie: ----
  • Listen: Beginnen Sie jede Zeile mit * oder # für unnummerierte bzw. nummerierte Listen. Die Listeneinbettung erfolgt durch Mischen von Listenmarkierungen auf diese Weise: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Definitionslisten: Beginnen Sie jede Zeile mit = für den zu definierenden Begriff und mit : für die Definition des Begriffs. 
      = term
  : Beschreibung des Begriffs
  • Zitat: Jede Zeile muss mit einem Semikolon beginnen (>).
  • Blocksatz: Jede Zeile muss mit einem >beginnen.
  • Seitenblock: Jede Zeile muss mit einem )beginnen.
Tagformate
  • Betonung: zwei Anführungszeichen ''text''
  • Starke Betonung: zwei Unterstriche __text__
  • Neue Zeile:
  • Einschub: zwei Pluszeichen ++text++ %%%
  • Löschung: zwei Minuszeichen --text--
  • Link: [url], [name|url], [name|url|language] oder [name|url|language|title]
  • Link: [url], [name|url], [name|url|Sprache] oder [name|url|Sprache|Titel]
  • Bild:((url|alternativer Text)), ((url|alternativer Text|Position)) oder ((url|alternativer Text|Position|Lange Beschreibung)).
    Die Position kann entweder L (links), R (rechts) oder C (zentriert) sein.
  • Acronym: ??acronym|title??
  • Inline HTML: zwei backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} oder {{quote|language|url}}
  • Inline-Zitat: {{quote}}, {{quote|language}} oder {{quote|language|url}}
  • Fussnote: $$footnote$$
  • Fußnoten: $$footnote$$
  • Hochgestellt : ^hochgestellt^
  • Tiefgestellt : ,,tiefgestellt,,
Unformatierter Text
Wenn Sie nicht wollen, dass ein Formatierungszeichen als solches interpretiert wird, fügen Sie ein \ direkt davor ein. Zum Beispiel: \[Text in Klammern, ohne ein Link zu sein\]
HTML Code
Auch wenn Sie sich für die Wiki-Syntax entschieden haben, benötigen Sie manchmal eine leistungsfähigere Formatierung, z. B. die HTML-Syntax. Gehen Sie dabei wie folgt vor: 
///html
<p style="color:red">mein text in rot</p>
///
