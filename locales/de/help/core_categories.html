Liste der Kategorien

Neue Kategorie erstellen
Wählen Sie die Kategorie, die Sie löschen möchten, indem Sie auf Kategorie löschenklicken. In einem Pop-up-Fenster werden Sie aufgefordert, diesen Vorgang zu bestätigen.
Eine Kategorie kann nur gelöscht werden, wenn sie keine Artikel enthält.
Um eine Kategorie zu leeren, können Sie die Einträge verschieben, indem Sie (in der Dropdown-Liste) die neue Kategorie auswählen, in die sie verschoben werden sollen. Klicken Sie auf OK, um diese Auswahl zu bestätigen.
Wechseln der Kategoriezuordnung
Sie können die Reihenfolge der Kategorien durch Ziehen und Ablegen ändern. Sie können auch eine Kategorie in eine übergeordnete Kategorie verschieben.
Wenn Sie mit der Reihenfolge zufrieden sind, klicken Sie auf Kategorienreihenfolge speichern.
Neuordnung aller Kategorien auf der obersten Ebene
Verwenden Sie diese Option, um alle Kategorien wieder auf die oberste Ebene zu verschieben.
Wenn Sie die Reihenfolge der Kategorien genauer ändern möchten, können Sie :
  • entweder durch Ziehen und Ablegen verschieben,
  • Klicken Sie entweder auf den Namen jeder Kategorie und wählen Sie die übergeordnete Kategorie oder die Position zu ihren Geschwistern.
