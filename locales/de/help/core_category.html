Eine neue Kategorie erstellen

Name
Wählen Sie den Namen, den Sie Ihrer neuen Kategorie geben möchten. Dieses Feld ist obligatorisch.
Elternteil
Es ist möglich, Unterkategorien zu erstellen. In diesem Fall müssen Sie die übergeordnete Kategorie in dem Feld Parent auswählen.
URL
Dotclear erstellt eine Standard-URL für die Kategorie, so dass sie über einen Pfad wie http://myblog.tld/category/My-categoryerreichbar ist. Durch Anklicken des kleinen Schlosses auf der rechten Seite können Sie die URL nach Ihren Wünschen ändern.
Beschreibung
Der Inhalt dieses Beschreibungsfeldes wird angezeigt, wenn eine Kategorie im Blog eingegeben wird. Sie können es mit jeder Art von Inhalt füllen (Absätze, Listen, usw.). Das für dieses Feld verwendete Format ist HTML.

Diese Kategorie verschieben

Übergeordnete Kategorie
Es ist möglich, Unterkategorien zu erstellen. In diesem Fall müssen Sie die übergeordnete Kategorie in dem Feld Parent auswählen.
Kategorie Geschwister
Sie können die Kategorie in eine beliebige andere Kategorie verschieben, die dann zu ihrer übergeordneten Kategorie wird. Sie können auch die Position der Kategorie zu ihren anderen Geschwistern festlegen, indem Sie sie vor oder nach einem von ihnen verschieben.
