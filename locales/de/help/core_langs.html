Installierte Sprachen

Hier finden Sie die Liste der installierten Sprachen, die für Ihr Administrationsprofil und für Ihren Blog verfügbar sind. Jede dieser Sprachen kann entfernt werden.

Wenn keine Sprache vorhanden ist, wird standardmäßig Englisch für den Verwaltungsteil und für den Blog verwendet.

Sprachen installieren oder aktualisieren

Verfügbare Sprachen

Hier finden Sie eine Liste der auf der Dotclear-Website verfügbaren Sprachen. Um eine dieser Sprachen zu installieren, wählen Sie sie in der Dropdown-Liste aus, geben Sie Ihr Passwort ein und drücken Sie Sprache installieren.

Die ausgewählte Sprache wird dann heruntergeladen und auf Ihrem System installiert.

Hochladen einer Zip-Datei

Sie können auch ein komprimiertes Archiv (Zip-Format) verwenden, um eine Sprache zu installieren. Wählen Sie die entsprechende Datei auf Ihrem Computer aus, geben Sie Ihr Passwort ein und drücken Sie Sprache hochladen.
