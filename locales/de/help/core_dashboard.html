Kopfzeile

Wenn es mehrere Blogs gibt, auf die Sie auf dieser Instanz zugreifen können, können Sie über eine Dropdown-Liste den Blog auswählen, den Sie bearbeiten möchten. Wenn es nur ein Blog gibt, wird nur dessen Name angezeigt.

Über den folgenden Link können Sie den aktuellen Blog aufrufen (öffnet in einem neuen Tab).

Über die anderen Links können Sie zu diesem Dashboard zurückkehren, auf Ihre Einstellungen zugreifen und sich abmelden.

Personalisierung des Dashboards

Favoriten
Mit den großen Symbolen in der Mitte der Seite können Sie schnell auf Ihre bevorzugten Verwaltungsseiten zugreifen.
Dashboard-Module
Die unter den Favoriten angezeigten Module werden standardmäßig installiert oder durch Plugins hinzugefügt.

Sie können die Anzeige der Favoriten und Module personalisieren, indem Sie die Registerkarte Mein Dashboard auf der Seite Meine Einstellungen aufrufen.

Das Hauptmenü

Sie können dieses Menü jederzeit ein- oder ausblenden, indem Sie auf die rechte Seite klicken (oder indem Sie den Link Hauptmenü ausblenden aktivieren).

Im ersten Teil des Menüs werden alle Lieblingsseiten des Benutzers aufgelistet. Sie können auswählen, ob diese Liste auf der Registerkarte Mein Dashboard auf der Seite Meine Einstellungen angezeigt werden soll oder nicht.

Der Rest des Menüs listet die Verwaltungsseiten auf. Es ist in drei Abschnitte unterteilt:

  • Blog für alles, was sich auf den aktuellen Bloginhalt und die Parameter bezieht.
  • Systemeinstellungen zur Verwaltung der globalen Installation.
  • Plugins für Erweiterungen, die das gesamte System betreffen.
