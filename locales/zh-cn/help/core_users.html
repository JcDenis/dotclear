这个页面列出了系统中的所有用户。

要添加（或修改）用户的权限，在 你的密码 ，你的密码将被要求。

用户列表过滤器

搜索
搜索是基于用户ID的。 * 通配符可以用来只查找用户ID的一部分。
订购方式
用户列表可以按照他们的 ID、他们的 Last Name、他们的 First Name、他们的 Display Name （昵称）或他们创建的 Number of Entries 来排序。
分类
表示将应用于结果列表的排序方法（Ascending 或 Descending）。
每页用户数
设置结果列表中每页显示的用户数。

用户批量行动

可以在同一时间对几个用户进行批量操作。

  • 设置权限： 为一个或多个博客设置权限。
  • 删除： 从系统中删除所选用户。创建条目的用户不能被删除。

如果你决定设置用户的权限，你会被转到一个新的页面，列出你系统中的可用博客。勾选你想设置权限的博客，然后按 设置权限。

用户权限

对于每个博客，你可以设置以下权限：

  • 管理员： 可以完全管理该博客。
  • 管理他们自己的条目和评论： 用户可以创建条目并管理他所创建的条目的评论。
  • 发布条目和评论： 用户可以修改他自己的条目和他的条目评论的发布状态。
  • 删除条目和评论： 用户可以删除自己的条目以及他的条目评论。
  • 管理所有条目和评论： 用户可以管理这个博客的所有条目，无论他是否是作者。
  • 管理类别： 用户可以创建、修改和删除类别。
  • 管理他们自己的媒体项目： 用户可以添加媒体项目并修改或删除自己的媒体。
  • 管理所有媒体项目： 用户可以管理来自任何用户的媒体项目。
  • 管理页面： 用户可以添加和修改页面。
  • 管理博客列表： 用户可以管理当前博客的博客列表。

超级管理员 状态是在用户配置文件中设置的。
