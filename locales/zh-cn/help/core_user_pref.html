我的简介

名字，姓氏
如果显示名称为空，则作者的姓名将显示为此处设置的名字和姓氏。
显示的名字
您可以在此处选择一个昵称，该昵称将显示在您的帖子中，而不是您的姓氏和名字。
邮箱
申请新密码时使用的电子邮件地址。
新评论通知也将发送到此地址。（如果您更新此字段，则必须在表单底部的相应字段中输入当前密码）
URL
用户的网站。如果设置，作者姓名将显示为该网站的链接。
喜好的格式
发布条目时使用的语言。如果有翻译，界面也会以该语言显示。
默认的文章状态
这一选择将定义在帖子发表时显示的时间。
文章编辑区域高度
要更改密码，请在 新密码 和 确认密码字段中写下两次密码。密码最短长度为 6 个字符。如果没有提供新密码，则保留当前密码。 （如果您更新此字段，则必须在表单底部的相应字段中输入当前密码）
双因素认证
Two factors authentication uses One Time Password authentication and is used only in addition to standard login+password authentication. To activate this feature, scan QR code with an authentication application or use a browser extension, then enter the verification code. On login, when prompt, enter code given by the application.
验证密钥
使用验证密钥代替标准的登录+密码验证，可以是物理密钥或浏览器扩展或密码管理器。要使用此功能，必须启用 Javascript 和导航凭据 API。可注册多个密钥。
认证应用
If configured, various third party applications are availabled. It is used instead of standard login+password authentication. You can grant or revoke access to these applications if you have accounts on their plateforms, then you can login in using these account.

我的选择

介面
在媒体管理器中激活增强型上传器
允许激活媒体管理器中的高级界面，以便一次上传多个文件（注意：您的 Web 浏览器必须支持 Javascript 才能使用此功能）。
禁用由Javascript驱动的拖放订购项目的功能
允许禁用“拖放”来排序项目。拖放将被数字字段取代，让您输入字段位置。
媒体管理器中每页显示的元素数量
定义媒体管理器中显示的媒体数量（计算项目时不考虑文件夹）。
不要使用标准的favicon
允许禁用网络浏览器地址栏中的默认favicon。
版本
首选的格式
选择条目的语法。Wiki 是一种简化的语法，将转换为有效的 HTML。除非您对 HTML 有完美的理解，否则我们建议您选择 Wiki 语法。有关更多信息，请参阅 Wiki 语法参考。
默认进入状态
新条目的默认状态可以设置为：
  • 待定： 出版状态尚未决定。
  • 预定的： 该条目将在 Published on 字段中提供的时间和日期在线设置。
  • 未发表： 脱机条目。
  • 已公布： 在线输入。
条目编辑字段高度
设置一个条目的编辑字段的高度。默认情况下，这个参数被设置为24。
激活所见即所得模式
允许使用可视化编辑器（WYSIWYG: What You See Is What You Get）来编辑条目、页面和类别描述栏。
其他选择
标签列表格式
你可以选择在编辑一个条目时，是要看所有使用过的标签(Extended) 还是只看最常用的标签(Short)。
主题编辑器中的语法高亮
在使用主题编辑器（HTML和CSS文件）时，启用语法高亮。

我的仪表板

我的最爱
此区域在博客的仪表板上显示所选的收藏夹。如果您的浏览器中启用了 Javascript，则可以使用鼠标重新排序它们。否则，字段允许为它们提供排序号。完成后，请不要忘记按 保存更改 保存顺序。
您也可以通过选择收藏夹并按 删除所选收藏夹来删除它们。
定义为默认收藏夹（超级管理员）
启用设置当前定义的收藏夹列表作为所有用户的默认收藏夹列表。如果用户还没有定义自己的收藏夹，这个列表将被推荐给用户。
其他可用的最爱
所有允许作为收藏夹的插件都显示在这里。勾选相应的方框，并按 添加到我的最爱，将它们添加到仪表板。
在菜单的顶部显示收藏夹
如果启用，在菜单的顶部显示收藏夹。
仪表板模块
在这个区域做出的选择将反映在你的博客仪表板上。
显示文件链接
启用在仪表板登陆页上显示文档链接。
显示Dotclear的新闻
启用显示Dotclear新闻的仪表板登陆页面。
显示快速录入表
启用在仪表板登陆页上显示 "快速输入 "表格。
在维护仪表板图标上显示过期任务计数器
能够在 维护 图标下显示过期的任务（即应该处理的任务）数量。 维护 图标必须设置在你的收藏夹中才能使用这个功能。
在仪表板项目上显示过期任务列表
启用在仪表板登陆页面的专门部分显示逾期任务（即应该处理的任务）的列表。
