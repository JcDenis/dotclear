רשימת מדיה

הדף הראשי של מנהל המדיה מציג רשימה של קבצים זמינים (מדיה) עבור הבלוג הנבחר הנוכחי.

תוכלו ללחוץ על האיקונים או שמות המדיה בתיקיה או בכל תת-תיקיה כדי להציג אותם.

בחירת קובץ קול MP3 מציגה שחקן שינגן מיד את הקובץ. תוסף Flash חייב להיות מותקן בדפדפן כדי ששחקן זה יפעל.

התמונות מוצגות עם תמונה ממוזערת במידת האפשר.

הוספת קבצים

בחירת קובץ
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
כותרת
כותרת אופציונלית עבור הקובץ שברצונכם לשלוח.
פרטי
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
מעב משופר
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

מחיקת קבצים

כדי למחוק קובץ, סמנו את תיבת הסימון שלו ולחצו על סילוק המדיה שנבחרה. תוכלו למחוק כמה קבצים בבת אחת.

תיקיה חדשה

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

פרטי מדיה

דף זה מציג קבוצה של מידע על הקובץ הנבחר הנוכחי ומאפשר לכם לבצע מספר פעולות עליו.

שם קובץ
שנו את שם הקובץ של הקובץ הנוכחי.
כותרת קובץ
העניקו כותרת לקובץ או שנו את הכותרת הנוכחית.
תאריך קובץ
שנו את תאריך הקובץ.
פרטי
ציינו שהקובץ יוסתר לכל אחד מלבד הבעלים שלו במנהל המדיה.
מדריך חדש
מאפשר להעביר את הקובץ בתיקיה אחרת.
שינוי קובץ
With this form, you can replace the file while keeping all the attributes (name, title...). As always, you have to respect the maximum file size.

מקרה ספציפי: תמונות

כשתוסיפו תמונה, בהתאם לגודל שלה, יוצגו עד ארבע גרסאות (בהתאם לגודל התמונה המקורית):

  • ריבוע: פיקסלים מרובעים של 48x48 פיקסלים.
  • אצבעונית: התמונה הצטמצמה ל-100 פיקסלים בצד הארוך ביותר שלו.
  • קטנים: הצד הארוך ביותר מופחת ל-240 פיקסלים.
  • בינוני: הצד הארוך ביותר מופחת ל-500 פיקסלים.
  • מקורי: התמונה המקורית, ללא שינוי.

ניתן לשנות את גדלי ברירת המחדל עבור תמונות ממוזערות, קטנות ובינוניות בהגדרות הבלוג.

צירוף קובץ לרשומה

תוכלו לחבר בקלות קובץ לאחת מהפרסומים שלכם. ראשית, צרו הודעה חדשה ושמרו אותה. לאחר מכן לחצו על "הוספת קבצים לרשומה זו" כדי לפתוח את מנהל המדיה.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

הקבצים המצורפים יוצגו בדף 'פרסום', בתור רשימה לאחר פרסום התוכן. קבצי MP3 יהיו זמינים באמצעות נגן Flash, כך שהם יכולים להיות בקלות שיחק על ידי המבקרים שלך.

פודקאסט ושידורי קבצי מוזיקה

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
