Antarmuka

Allows to activate the advanced interface in media manager that enables to upload multiple files at a time (Note: Your Web browser must support Javascript to be able to use this feature).
Nonaktifkan seret dan lepas yang didukung Javascript untuk memesan item
Allows to disable "drag and drop" for item ordering. Drag and drop will be replaced by numeric fields that will let you enter fields position instead.
Jumlah elemen yang ditampilkan per halaman di pengelola media
Defines the number of media that will be displayed in media manager (folders are not taken into account when counting items).
Jangan gunakan favicon standar
Memungkinkan untuk menonaktifkan favicon default di bilah alamat browser Web.
Edisi
Format yang disukai
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Status entri default
Status default untuk entri baru dapat diatur ke:
  • Tertunda: Status publikasi belum diputuskan.
  • Terjadwal: Entri akan ditetapkan secara online pada waktu dan tanggal yang disediakan di Dipublikasikan di bidang .
  • Tidak dipublikasikan: Entri offline.
  • Diterbitkan: Entri online.
Tinggi bidang edit entri
Mengatur tinggi bidang edisi entri. Secara default, parameter ini diatur ke 24.
Mengaktifkan mode WYSIWYG
Memungkinkan penggunaan editor visual (WYSIWYG: Apa yang Anda Lihat Adalah Apa yang Anda Dapatkan) untuk entri, halaman, dan bidang deskripsi kategori.

Format daftar tag

Penyorotan sintaksis di editor tema
Mengaktifkan penyorotan sintaksis saat menggunakan editor tema (file HTML dan CSS)

Dasbor saya

Favorit saya
This zone displays the chosen favorites on the blog's dashboard. They can be reordered with your mouse if Javascript is activated in your browser. Otherwise, a field allows to give them an ordering number. When you are done, do not forget to save your changes by pressing Save order.
You may also delete favorites by selecting them and pressing Delete selected favorites.
Tetapkan sebagai favorit default (administrator super)
Memungkinkan untuk menetapkan daftar favorit yang saat ini ditentukan sebagai daftar favorit default untuk semua pengguna. Daftar ini akan diusulkan kepada pengguna jika mereka belum menentukan favorit mereka sendiri.
Favorit lain yang tersedia
Semua plugin yang memungkinkan untuk dijadikan favorit ditampilkan di sini. Centang kotak yang sesuai dan tambahkan ke dasbor dengan menekan Tambahkan ke favorit saya.
Menampilkan favorit di bagian atas menu
Jika diaktifkan, menampilkan favorit di bagian atas menu.
