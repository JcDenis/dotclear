Daftar media

Halaman utama pengelola media menampilkan daftar file (media) yang tersedia untuk blog yang sedang dipilih.

Anda dapat mengeklik ikon atau nama media dalam folder atau subfolder mana pun untuk menampilkannya.

Memilih file suara MP3 akan menampilkan pemutar untuk segera memutar file tersebut. Plugin Flash harus diinstal pada browser Anda agar pemutar ini dapat beroperasi.

Gambar ditampilkan dengan gambar mini apabila memungkinkan.

Menambahkan file

Pilih file
Di sini Anda dapat memilih salah satu file lokal Anda untuk dikirimkan ke blog . Jika ukuran file lebih besar dari ukuran maksimum, maka file tidak akan dikirim.
Judul
Judul opsional untuk file yang ingin Anda kirim.
Pribadi
Centang ini jika Anda ingin file tersebut bersifat pribadi, yaitu hanya dapat diakses oleh pemiliknya dan tidak dapat diakses dari blog (hingga dipublikasikan).
Pengunggah yang disempurnakan
Memungkinkan untuk memilih satu atau beberapa file lokal Anda untuk dikirim ke blog Anda. Jika ada file yang lebih besar dari ukuran maksimum, file tersebut tidak akan dikirim.
Ketika Anda memilih beberapa file lokal, file-file tersebut akan ditambahkan ke dalam antrean. Anda dapat menambahkan lebih banyak file dengan mengklik lagi Pilih file. Anda juga dapat menghapus file dari antrean dengan mengklik ikon berbentuk palang merah yang sesuai.
Jika sudah siap, klik Unggah.

Menghapus file

Untuk menghapus file, centang kotak centang dan klik Hapus media yang dipilih. Anda dapat menghapus beberapa file sekaligus.

Folder baru

Formulir ini memungkinkan Anda untuk membuat subfolder baru di folder yang sedang ditampilkan. Cukup pilih nama untuk folder tersebut seperti yang Anda lakukan pada komputer Anda sendiri.

Detail media

Halaman ini menampilkan serangkaian informasi tentang file yang sedang dipilih dan memungkinkan Anda melakukan beberapa operasi pada file tersebut.

Nama file
Mengubah nama file dari file yang sedang dibuka.
Judul file
Berikan judul pada file atau ubah judul yang ada.
Tanggal pengajuan
Mengubah tanggal file.
Pribadi
Tentukan bahwa file akan disembunyikan bagi siapa pun kecuali pemiliknya di pengelola media.
Direktori baru
Memungkinkan Anda memindahkan file ke folder lain.
Ubah file
Dengan formulir ini, Anda dapat mengganti file dengan tetap mempertahankan semua atribut (nama, judul...). Seperti biasa, Anda harus menghormati ukuran file maksimum.

Kasus khusus: gambar

Apabila Anda menambahkan gambar, tergantung dari ukurannya, hingga empat versi akan tersedia (tergantung ukuran gambar aslinya):

  • Persegi: Gambar mini persegi 48x48 piksel.
  • Gambar kecil: Gambar diperkecil hingga 100 piksel pada sisi terpanjangnya.
  • Kecil: Sisi terpanjang diperkecil hingga 240 piksel.
  • Sedang: Sisi terpanjang dikurangi hingga 500 piksel.
  • Asli: Gambar asli yang tidak diubah.

Ukuran default untuk gambar mini, versi kecil dan menengah dapat dimodifikasi dalam pengaturan blog Anda.

Melampirkan file ke entri

Anda dapat dengan mudah melampirkan file ke salah satu postingan Anda. Pertama, buat postingan baru dan simpan. Kemudian klik "Tambahkan file ke entri ini" untuk membuka pengelola media.

Cukup klik ikon [+] (Lampirkan file ini ke entri). File tersebut akan dilampirkan ke postingan.

File yang dilampirkan akan ditampilkan di halaman posting, sebagai daftar setelah konten posting. File MP3 akan tersedia melalui pemutar Flash, sehingga dapat dengan mudah diputar oleh pengunjung Anda.

Podcast dan penyiaran file musik

Semua file yang dilampirkan akan disertakan dalam feed blog Anda, sehingga memungkinkan podcasting semua jenis file dengan mudah.
