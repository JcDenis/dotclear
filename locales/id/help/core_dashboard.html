Header

Jika ada beberapa blog yang dapat Anda akses pada contoh ini, daftar tarik-turun memungkinkan Anda untuk memilih blog yang ingin Anda kerjakan. Jika hanya ada satu blog, maka hanya akan menampilkan namanya saja.

Tautan berikut ini memungkinkan Anda untuk membuka blog saat ini (terbuka di tab baru).

Tautan lainnya memungkinkan Anda untuk kembali ke dasbor ini, untuk mengakses preferensi Anda dan keluar.

Personalisasi dasbor

Favorit
Ikon besar di tengah halaman memungkinkan Anda mengakses halaman admin favorit Anda dengan cepat.
Modul dasbor
Modul yang ditampilkan di bawah favorit diinstal secara default atau ditambahkan oleh plugin.

Anda dapat mempersonalisasi tampilan favorit dan modul dengan masuk ke tab Dasbor saya di halaman Preferensi saya .

Menu utama

Anda dapat menampilkan atau menyembunyikan menu ini kapan saja dengan mengeklik sisi kanannya (atau dengan mengaktifkan tautan Sembunyikan menu utama ).

Bagian pertama dari menu ini mencantumkan semua halaman favorit pengguna. Anda dapat memilih untuk menampilkan daftar ini atau tidak di tab Dasbor saya di halaman Preferensi saya .

Sisa menu lainnya berisi daftar halaman administrasi. Halaman ini dibagi dalam tiga bagian:

  • Blog untuk segala sesuatu yang berhubungan dengan konten dan parameter blog saat ini.
  • Pengaturan sistem untuk mengelola penginstalan global.
  • Plugin untuk ekstensi yang memengaruhi seluruh sistem.
