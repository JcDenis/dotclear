Memperbarui

Halaman ini memungkinkan Anda untuk memperbarui Dotclear secara otomatis. Jika versi baru tersedia, Anda akan ditanya apakah Anda ingin memperbarui. Sebelum melakukannya, Anda dapat memeriksa perubahan apa saja yang dibawa oleh pembaruan ini dengan mengklik Informasi tentang versi ini.

Anda dapat memaksa pemeriksaan versi baru dengan mengeklik Memaksa pemeriksaan pembaruan Dotclear.

Jika semuanya baik-baik saja, yaitu jika Anda tidak mengubah file instalasi secara manual, prosedur pembaruan akan melakukannya:

  • Buatlah cadangan dari instalasi Anda saat ini pada direktori kerja yang bernama backup-x.y.z.zip (di mana x.y.z adalah nomor versinya, misalnya 2.5.3).
  • Unduh versi baru.

Once this is done, a message such as the following will be displayed: "Congratulations, you're one click away from the end of the update."
Click on Finish the update. You will be logged out. The database will be updated, if needed, when you will log back in.

Jika Anda memodifikasi beberapa file secara manual, file-file tersebut akan terdaftar dan Anda akan disarankan untuk memperbaruinya secara manual.

Mengelola file cadangan

Dari tab ini, Anda dapat menghapus file cadangan lama.

Anda juga dapat kembali ke versi sebelumnya. Hanya pertimbangkan untuk kembali ke versi sebelumnya jika Anda mengalami masalah serius dengan versi baru.
