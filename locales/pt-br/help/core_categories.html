Lista de categorias

Create a new category
Escolha a categoria que deseja excluir clicando em Excluir a categoria. Uma janela pop-up solicitará que você confirme essa ação.
Uma categoria só pode ser excluída se não contiver nenhum artigo.
Para esvaziar uma categoria, você pode mover as entradas selecionando (na lista suspensa) a nova categoria para a qual elas serão movidas. Clique em OK para confirmar essa escolha.
Change category order
Você pode modificar a ordem das categorias usando o recurso de arrastar e soltar. Você também pode mover uma categoria para uma categoria principal.
Quando estiver satisfeito com a ordem, clique em Save categories order.
Reordenar todas as categorias no nível superior
Use essa opção para mover todas as categorias de volta para o nível superior.
Se quiser modificar a ordem das categorias com mais precisão, você pode :
  • ou mova-os usando arrastar e soltar,
  • ou clique em cada nome de categoria e selecione sua categoria principal ou sua posição em seus irmãos.
