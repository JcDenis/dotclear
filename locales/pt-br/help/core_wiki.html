Referência de sintaxe Wiki

A sintaxe Wiki é uma forma de aprimorar seu texto com um conjunto mínimo de tags estudadas para cobrir as necessidades básicas (títulos, parágrafos, citações, listas...).

Elementos de bloco
  • Deixe uma linha vazia entre dois blocos similares.
  • Parágrafo: Texto livre, finalizado por uma linha vazia se outro parágrafo for seguido.
  • Título: !!! título, !! título ou ! title, permitindo que você use três níveis diferentes de título.
  • Horizontal line: ----
  • Listas: Comece cada linha com * ou # para listas não numeradas ou numeradas, respectivamente. A imbricação de lista é feita misturando marcadores de lista desta forma: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Listas de definição: Comece cada linha com = para o termo a definir e com : para a definição do termo. 
      = termo
  : descrição do termo
  • Block quote: each line must start with a semicolon (>).
  • Taxa de bloco: Cada linha deve começar com um >.
  • Bloco lateral: Cada linha deve começar com um ).
Tags de formatação
  • Emphasis: two quotes ''text''
  • Strong emphasis: two underscore __text__
  • New line:
  • Insertion: two plusses ++text++ %%%
  • Deletion: two minuses --text--
  • Link: [url], [name|url], [name|url|language] or [name|url|languagee|title]
  • Link: [url], [nome|url], [nome| url|idioma] ou [nome|url|idioma|título]
  • Imagem:((url alternativa)), ((url├alternativo position)) ou ((url├alternativa de texto (posicionamento).
    A posição pode ser L (esquerda), R (direita) ou C (centralizada).
  • Acronym: ??acronym|title??
  • Inline HTML: two backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} or {{quote|language|url}}
  • Citação embutida: {{quote}}, {{quote|language}} ou {{quote|language|url}}
  • Footnotes: $$footnote$$
  • Notas de rodapés: $$footnote$$
  • Sobrescrito : ^superscript^
  • Subscript : ,,,subscript,,
Texto não formatado
Se você não deseja que um caractere de formatação seja interpretado como tal, adicione um \ logo antes dele. Por exemplo: \[Texto entre colchetes sem ser um link\]
HTML Code
Mesmo se você escolher a sintaxe Wiki, às vezes você pode precisar de uma formatação mais poderosa, ou seja, sintaxe HTML. Faça da seguinte maneira: 
///html
<p style="color:red">meu texto em vermelho</p>
///
