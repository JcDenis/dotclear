Perfil do usuário

ID de usuário
Pelo menos 2 caracteres usando letras, números ou símbolos (sem espaço). Este campo é obrigatório.
Nova senha
Se quiser alterar a sua senha, digite a nova senha duas vezes nos campos Nova Senha e Confirmar Senha . As senhas devem conter pelo menos 6 caracteres. Ambos os campos são obrigatórios.
Alteração de senha necessária para conectar
Marque esta caixa para que um novo usuário tenha que escolher a sua própria senha quando se conectar.
Super administrador
Isto dará ao usuário atual todas as permissões em todos os blogs hospedados no sistema.
Último nome, Primeiro Nome
Se nenhum nome de exibição for fornecido, o nome do autor será exibido e o primeiro nome inserido nestes dois campos.
Apelido
Você pode selecionar o apelido que você deseja usar para assinar seus artigos. Se não estiver vazio, este campo será usado em vez do sobrenome e primeiro nome.
e-mail
Este endereço de e-mail será usado se você precisar recuperar sua senha da tela de autenticação de Dotclear.
URL:
Indica o site do usuário. Se este campo não estiver vazio, o nome do autor se tornará um link clicável para este site.

Opções

Interface
IDIOMA
Defina o idioma padrão para novas entradas. Se uma tradução estiver disponível para este idioma, a interface de blog será traduzida para este idioma também.
Timezone
Esta opção afeta a forma como o tempo de publicação das postagens é exibido.
Edição
Formato de edição preferido
Escolha a sintaxe da entrada. Wiki é uma sintaxe simplificada e será convertida em HTML válido. A menos que você tenha um conhecimento perfeito de HTML, aconselhamos que escolha a sintaxe Wiki. Consulte a referência de sintaxe do Wiki para obter mais informações.
Status padrão dos posts
O status padrão para novas entradas pode ser definido para:
  • Pendente: O estado da publicação ainda não foi decidido.
  • Programado: A entrada será definida online no momento e data fornecida no campo publicado no campo.
  • Não publicado: Entrada off-line.
  • Publicado: Entrada online.
Tamanho da zona de edição
Define a altura do campo de edição de uma entrada. Por padrão, esse parâmetro é definido como 24.
Formato da lista de tags
Você pode selecionar se você quer ver, ao editar uma entrada, todas as tags usadas (Estendidas) ou apenas as mais usadas (Curtas).
