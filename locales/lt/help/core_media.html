Žiniasklaidos sąrašas

Der Mediamanager zeigt eine Übersicht aller Medien, die Du in Deinem Blog, in dem Du gerade arbeitest, verwendest oder hochgeladen hast.

Du klickst auf den Namen oder das Icon oder den Unterordner, um diese ansehen zu können.

Wenn Du ein MP3 anklickst, zeigt sich ein kleiner Player und Du kannst die Datei abspielen. Allerdings muss das Flash plugin in Deinem Browser installiert sein, um MP3 abspielen zu koennen.

Bilder werden sofern möglich mit einem Thumbnail dargestellt.

Pridėti failus

Auswählen der Datei
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Titel
Opptional, ob Du deiner Datei einen Titel geben möchtest.
Privat
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
Dateiname
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

Ištrinti failus

Du kannst Unterordner in Deinem Mediamanger erstellen. Setze einen Namen ein und speichere es ab.

Naujas aplankas

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Žiniasklaidos informacija

Wenn Du ein Bild hinzufügst, 4 verschiedene Grössen werden möglich sein:

Failo pavadinimas
Pakeiskite dabartinio failo pavadinimą.
Failo pavadinimas
Suteikite failui pavadinimą arba pakeiskite esamą pavadinimą.
Bylos data
Pakeiskite failo datą.
Privatus
Nurodykite, kad failas bus paslėptas visiems, išskyrus jo savininką, medijos tvarkytuve.
Naujas katalogas
Leidžia perkelti failą į kitą aplanką.
Pakeisti failą
With this form, you can replace the file while keeping all the attributes (name, title...). As always, you have to respect the maximum file size.

Konkretus atvejis: vaizdai

Um eine Datei einem Beitrag zuzufügen, musst Du zuerst einen Blogbeitrag erstellen. Klicke auf "Datei zu diesem Beitrag hinzufügen" und es öffnet sich der Medienmanager.

  • Quadrat: ein 48x48 Pixel Vorschaubild
  • Vorschaubild: das Bild wird zu einer Breite von 100 Pixel in der Breite reduziert.
  • Klein: Die Breite ist 240 Pixel
  • Mittel: Die Breite beträgt 500 Pixel.
  • Original: die Originalgrösse

Klicke auf das [+] Icon (Datei zu diesem Beitrag hinzufügen). Die Datei wird dem Beitrag hinzugefügt.

Pridėti failą prie įrašo

Die angefügten Dateien werden rechts neben dem Beitrag angezeigt. MP3 Dateien werden durch den mitgelieferten Flash Player für Besucher abspielbar sein.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Pridėti failai bus rodomi pranešimo puslapyje, kaip sąrašas po pranešimo turinio. MP3 failai bus prieinami per "Flash" grotuvą, kad lankytojai galėtų juos lengvai paleisti.

Podcast ir muzikos failų transliavimas

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
