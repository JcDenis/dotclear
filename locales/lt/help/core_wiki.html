"Wiki" sintaksės nuoroda

„Wiki“ sintaksė – tai būdas patobulinti tekstą naudojant minimalų ištirtų žymų rinkinį, kad būtų patenkinti pagrindiniai poreikiai (pavadinimai, pastraipos, citatos, sąrašai...).

Blockelemente
  • Lass eine leere Zeile zwischen zwei gleichen Blocks
  • Pastraipa: Laisvas tekstas, baigiamas tuščia eilute, jei reikia sekti kitą pastraipą.
  • Pavadinimas: !!! pavadinimas, !! pavadinimas arba ! pavadinimas, leidžiantis naudoti tris skirtingus antraštės lygius.
  • Horizontale Linie: ----
  • Sąrašai: Pradėkite kiekvieną eilutę su * arba # atitinkamai nenumeruotiems arba sunumeruotiems sąrašams. Sąrašo žymėjimas atliekamas maišant sąrašo žymeklius tokiu būdu: 
        * * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Apibrėžčių sąrašai: Kiekvieną eilutę pradėkite nuo = , jei norite apibrėžti terminą, ir nuo : , jei norite apibrėžti terminą. 
      = terminas
  : termino aprašymas
  • Zitat: Jede Zeile muss mit einem Semikolon beginnen (>).
  • Bloko citata: Kiekviena eilutė turi prasidėti >.
  • Šoninis blokas: Kiekviena eilutė turi prasidėti ).
Tagformate
  • Betonung: zwei Anführungszeichen ''text''
  • Starke Betonung: zwei Unterstriche __text__
  • Neue Zeile:
  • Einschub: zwei Pluszeichen ++text++ %%%
  • Löschung: zwei Minuszeichen --text--
  • Link: [url], [name|url], [name|url|language] oder [name|url|language|title]
  • Nuoroda: [url], [vardas|url], [vardas| url|kalba] arba [vardas|url|kalba|pavadinimas]
  • Vaizdas:((url|alternatyvus tekstas)), ((url|alternatyvus tekstas|pozicija)) arba ((url| alternatyvus tekstas|padėtis|ilgas aprašymas)).
    Padėtis gali būti L (kairėje), R (dešinėje) arba C (centre).
  • Acronym: ??acronym|title??
  • Inline HTML: zwei backquotes ``HTML code``
  • Inline quote: {{quote}}, {{quote|language}} oder {{quote|language|url}}
  • Eilutinė citata: {{quote}}, {{quote|language}} arba {{quote|language|url}}
  • Fussnote: $$footnote$$
  • Pastabos: $$footnote$$
  • Viršutinis indeksas : ^viršutinis indeksas^
  • Indeksas : ,,indeksas,,,
Unformatierter Text
Jei nenorite, kad vienas formatavimo simbolis būtų interpretuojamas kaip toks, prieš pat jį pridėkite \ . Pavyzdžiui: \[Tekstas skliausteliuose, bet ne nuoroda\]
HTML Code
Net jei pasirinkote Wiki sintaksę, kartais gali prireikti galingesnio formatavimo, ty HTML sintaksės. Atlikite tai tokiu būdu: 
///html
<p style="color:red">mein text in rot</p>
///
