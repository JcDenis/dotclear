Medya listesi

The media manager's main page displays a list of available files (medias) for the currently selected blog.

You can click the icons or names of the medias in the folder or any subfolder to display them.

Selecting a MP3 sound file displays a player to immediatly play the file. The Flash plugin must be installed on your browser for this player to operate.

Görüntüler mümkün olduğunda bir küçük resimle birlikte görüntülenir.

Dosya ekleme

Bir dosya seçin
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Başlık
Göndermek istediğiniz dosya için isteğe bağlı başlık.
Özel
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog.
File name
Yerel dosyalarınızdan birini veya birkaçını seçerek blogunuza göndermenizi sağlar. Herhangi bir dosya maksimum boyuttan büyükse gönderilmeyecektir.
Birkaç yerel dosya seçtiğinizde, bunlar kuyruğa eklenir. Dosya seçüzerine tekrar tıklayarak daha fazla dosya ekleyebilirsiniz. Ayrıca ilgili kırmızı çarpı şeklindeki simgeye tıklayarak dosyaları kuyruktan kaldırabilirsiniz.
Hazır olduğunuzda, Yükleüzerine tıklayın.

Dosyaları sil

This form allows you to create an new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Yeni klasör

This page displays a set of informations about the currently selected file and allows you to perform a few operations.

Medya detayları

When you add an image, depending of its size, up to four versions will be made available:

Dosya adı
Geçerli dosyanın dosya adını değiştirin.
Dosya başlığı
Dosyaya bir başlık verin veya mevcut başlığı değiştirin.
Dosya tarihi
Dosya tarihini değiştirin.
Özel
Dosyanın medya yöneticisinde sahibi dışında herkese gizleneceğini belirtin.
Yeni dizin
Dosyayı başka bir klasöre taşımanızı sağlar.
Dosyayı değiştir
Bu form ile, (isim, başlık...) tüm nitelikleri koruyarak dosyayı değiştirebilirsiniz. Her zaman olduğu gibi, maksimum dosya boyutuna uymanız gerekir.

Özel durum: görüntüler

You can easily attach a file to one of your posts. First, create a new post and save it. Then click on "Add files to this entry" to open the media manager.

  • square: a 48x48 pixels square thumbnail,
  • thumbnail: the image reduced to 100 pixels on its longest side.
  • small: the longuest side is reduced to 240 pixels.
  • medium: the longuest side is reduced to 500 pixels.
  • original: the original, unchanged image.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Bir girişe dosya ekleme

The attached files will be shown on the post page, as a list beside the post content. The MP3 files will be made available through the Flash player, so that they can easily by read by visitors.

All attached files will be included in your RSS feeds, allowing for easy podcasting of any type of file.

Ekli dosyalar yazı sayfasında, yazı içeriğinden sonra bir liste olarak gösterilecektir. MP3 dosyaları, ziyaretçileriniz tarafından kolayca oynatılabilmeleri için bir Flash oynatıcı aracılığıyla sunulacaktır.

Podcast ve müzik dosyalarının yayınlanması

Ekli tüm dosyalar blog beslemelerinize dahil edilecek ve her tür dosyanın adresinden kolayca podcast'lenmesine olanak tanıyacaktır.
