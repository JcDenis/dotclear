Uwierzytelnianie dwuskładnikowe
Two factors authentication uses One Time Password authentication and is used only in addition to standard login+password authentication. To activate this feature, scan QR code with an authentication application or use a browser extension, then enter the verification code. On login, when prompted, enter code given by the application.
Klucze uwierzytelniające
Authentication key is used instead of standard login+password authentication, it can be physical key or browser extension or password manager. Javascript and navigator credential API must be enabled to use this feature. Multiple keys can be registered.
Aplikacje uwierzytelniające
If configured, various third party applications are available. It is used instead of standard login+password authentication. You can grant or revoke access to these applications if you have accounts on their plateforms, then you can login in using these account.

Moje opcje

Interfejs
Aktywuj ulepszony uploader w menedżerze mediów
Umożliwia aktywację zaawansowanego interfejsu w menedżerze multimediów, który umożliwia przesyłanie wielu plików jednocześnie (Uwaga: Twoja przeglądarka internetowa musi obsługiwać JavaScript, aby móc korzystać z tej funkcji).
Wyłącz przeciąganie i upuszczanie zasilane Javascript dla zamawiania przedmiotów
Pozwala wyłączyć opcję „przeciągnij i upuść” przy zamawianiu przedmiotów. Przeciąganie i upuszczanie zostanie zastąpione polami numerycznymi, które pozwolą zamiast tego wprowadzić położenie pól.
Liczba elementów wyświetlanych na stronie w menedżerze mediów
Określa liczbę multimediów, które będą wyświetlane w menadżerze multimediów (foldery nie są brane pod uwagę przy zliczaniu sztuk).
Nie używaj standardowej favicon
Pozwala na wyłączenie domyślnego favicona w pasku adresu przeglądarki internetowej.
Edycja
Preferowany format
Wybierz składnię wpisu. Wiki ma uproszczoną składnię i zostanie przekonwertowana na prawidłowy kod HTML. Jeśli nie znasz doskonale HTML-a, radzimy wybrać składnię Wiki. Więcej informacji znajdziesz w dokumentacji składni Wiki.
Domyślny status wpisu
Domyślny status dla nowych wpisów można ustawić na:
  • W toku: Status publikacji nie został jeszcze ustalony.
  • Zaplanowane: Wpis zostanie ustawiony online w czasie i dacie w polu opublikowanym na.
  • Nieopublikowano: Wpis offline.
  • Opublikowano: Wpis online.
Wysokość pola edycji
Ustawia wysokość pola edycji wpisu. Domyślnie ten parametr jest ustawiony na 24.
Włącz tryb WYSIWYG
Pozwala na używanie edytora wizualnego (WYSIWYG: Co widzisz) dla pól opisowych wpisów, stron i kategorii.
Inne opcje
Format listy tagów
Możesz wybrać jeśli chcesz zobaczyć, podczas edycji wpisu, wszystkie użyte tagi (Extended) lub tylko te najczęściej używane (Short).
Podświetlanie składni w edytorze motywów
Włącza podświetlanie składni podczas używania edytora motywów (pliki HTML i CSS)

Mój panel

Moje ulubione
W tej strefie wybrane ulubione znajdują się na dashboardzie bloga. Można je zmienić za pomocą myszy, jeśli w przeglądarce jest włączony JavaScript. W przeciwnym razie pole pozwala na nadanie im numeru porządkowego. Kiedy skończysz, nie zapomnij zapisać zmian, naciskając Zapisz zamówienie.
Możesz także usunąć ulubione, zaznaczając je i naciskając Usuń wybrane ulubione.
Zdefiniuj jako domyślne ulubione (super administrator)
Włącz aby ustawić aktualnie zdefiniowaną listę ulubionych jako domyślną listę ulubionych dla wszystkich użytkowników. Ta lista zostanie zaproponowana użytkownikom, jeśli nie zdefiniowali jeszcze swoich ulubionych.
Inne dostępne ulubione
Wszystkie wtyczki pozwalające na umieszczenie jako ulubione są tutaj odtwarzane. Zaznacz odpowiednie pola i dodaj je do kokpitu menedżerskiego, naciskając Dodaj moje ulubione.
Wyświetl ulubione w górnej części menu
Jeśli włączone, wyświetla ulubione w górze menu.
Moduły panelu ustawień
Wybór dokonany w tej strefie będzie odzwierciedlony na pulpicie twojego bloga.
Wyświetl linki do dokumentacji
Włącz wyświetlanie linków dokumentacji na stronie lądowania pulpitu.
Wyświetl aktualności Dotclear
Włącz, aby wyświetlić wiadomości Dotclear na stronie startowej pulpitu.
Wyświetl formularz szybkiego wpisu
Włącz, aby wyświetlić formularz "szybki wpis" na stronie startowej pulpitu.
Wyświetlaj zaległy licznik zadań na ikonie panelu obsługi technicznej
Umożliwia wyświetlanie liczby zaległych zadań (tj. które powinny być przetworzone) pod ikoną Obsługa . Konserwacja musi być ustawiona w ulubionych aby korzystać z tej funkcji.
Wyświetl listę zaległych zadań na pulpicie
Włącz, aby wyświetlić listę zaległych zadań (tj. które powinny być przetworzone) w dedykowanej sekcji na stronie lądowania pulpitu.
