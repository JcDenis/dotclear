Η τροποποίηση των ρυθμίσεων του θέματος Blowup σας επιτρέπει να προσαρμόσετε το θέμα σας πολύ εύκολα. Για να το κάνετε αυτό, απλά συμπληρώστε τα πεδία διαμόρφωσης ή επιλέξτε ένα προκαθορισμένο στυλ.

Προκαθορισμένα στυλ

You can choose a predefined style in the Predefined styles drop-down list. Once you have chosen the style, you have to submit the form to apply the modifications.

Στη συνέχεια, μπορείτε να τροποποιήσετε το προκαθορισμένο στυλ σύμφωνα με το γούστο και τις ανάγκες σας.

Χρώματα

Όταν πρέπει να εισαγάγετε μια τιμή χρώματος, θα πρέπει να χρησιμοποιείτε τη δεκαεξαδική μορφή. Για παράδειγμα: #FF0000 θα σας δώσει κόκκινο χρώμα. Μπορείτε επίσης να χρησιμοποιήσετε τον επιλογέα χρωμάτων που βρίσκεται δίπλα σε κάθε πεδίο χρώματος.

Αν σας λείπει η έμπνευση, επισκεφθείτε τις παρακάτω σελίδες:

  • Κατάλογος κωδικών χρωμάτων (Wikipedia)
  • Adobe Kuler
  • ColorBlender
  • ColorCombos

Μονάδες μέτρησης

Όταν πρέπει να αναφέρετε ένα μέγεθος, πρέπει να ακολουθείται από μια μονάδα μέτρησης. Για παράδειγμα: 1em. Εάν δεν συμπεριλάβετε τη μονάδα, τα pixels θα χρησιμοποιηθούν ως προεπιλεγμένη μονάδα μέτρησης.

Είναι διαθέσιμες οι ακόλουθες μονάδες μέτρησης:

  • px
  • em
  • ex
  • pt
  • %

Εικόνες κεφαλίδας

Μπορείτε να επιλέξετε μια εικόνα κεφαλίδας από τη λίστα για να αντικαταστήσετε την προεπιλεγμένη.

Επιλέγοντας Custom... στη λίστα εικόνων, θα μπορείτε να ανεβάσετε τη δική σας εικόνα. Το αρχείο εικόνας πρέπει να είναι σε μορφή JPG ή PNG και να έχει πλάτος ακριβώς 800px.

Αν ανεβάσετε μια εικόνα JPG, θα προστεθεί ένα περίγραμμα στην εικόνα, κάτι που δεν συμβαίνει με τις εικόνες PNG (η διαφάνεια των οποίων επίσης θα διατηρηθεί).

Εισαγωγή και εξαγωγή ρυθμίσεων

Στο κάτω μέρος της σελίδας, μπορείτε να εμφανίσετε μια περιοχή που ονομάζεται Εισαγωγή/εξαγωγή παραμέτρων. Αυτή η περιοχή κειμένου περιέχει τη διαμόρφωση που χρησιμοποιείτε αυτή τη στιγμή. Μπορείτε να την αντιγράψετε για να την μοιραστείτε με άλλους.

Για να εφαρμόσετε (να εισαγάγετε) μια διαμόρφωση, πρέπει απλώς να αντικαταστήσετε το περιεχόμενο της περιοχής κειμένου με τη διαμόρφωση που θέλετε να χρησιμοποιήσετε. Μην ξεχάσετε να κάνετε κλικ στο Εφαρμογή κωδικού.
