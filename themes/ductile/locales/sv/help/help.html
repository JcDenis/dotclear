Personalisering av innehållet

På den här fliken kan du anpassa innehållet i din blogg.

Rubrik
Markera rutan "Dölj bloggens beskrivning" för att inte visa bloggens beskrivning som vanligtvis visas under titeln.

Om insticksprogrammet "Enkel meny" är installerat och aktivt kan du klicka på den angivna länken för att konfigurera menyn i sidhuvudet. (huvudmeny?)

Ange en URL för logotypen som ska användas istället för standardlogotypen. Om URL:en inte börjar med / kommer den att vara en relativ URL från mappen img i temat (att lägga logo.png, till exempel, är liknande som att använda standardlogotypen). Rensa detta fält för att återgå till standardkonfigurationen.

Klistermärken
Klistermärkena kommer att visas högst upp på skärmen som små bilder på stora skärmar och längst ner på sidan på mycket små skärmar. Alla bilder vars filnamn motsvarar sticker-*.png-schemat och placerade i temats undermapp img listas. Du kan ange för varje etikett och motsvarande url. (Lämna ett av båda fälten tomt för att inte visa det.)
Du kan också ändra visningsordningen med ett dra och släpp om javascript är aktiverat och om du inte har inaktiverat det i dina inställningar. Om javascript är avaktiverat låter ett fält dig ange visningsordningen (1 till n).
Typer och begränsningar för listor över inlägg
Några av visningskontexterna för din blogg presenteras i form av en lista med inlägg. Det är till exempel fallet för hemsidan eller för månadsarkivet. För var och en av de sammanhang som listas i tabellen kan du ange om innehållet i varje visat inlägg endast ska vara som en titel, ett kort formulär (innehållet i extraktfältet med de första 300 tecknen) eller deras fullständiga form. OBS! På startsidan visas alltid det första inlägget i sin helhet.
Dessutom, för vissa av dessa sammanhang, kan du också ange antalet inlägg som visas per sida. Om du lämnar det här fältet tomt kommer värdet som anges i de allmänna inställningarna för bloggen som kommer att användas.

Personlig anpassning av presentationen

På den här fliken kan du anpassa vissa presentationselement för din blogg.

Allmänna parametrar

Dessa parametrar kommer att användas på alla bloggens visningsbredder.

Typsnitt
Här kan du välja den huvudsakliga typsnittsfamiljen som ska användas för de flesta texterna. Du kan också ange en andra typsnittsfamilj (sekundär) som kommer att användas för vissa titlar på den andra nivån och kommentarer. Alternativet "standard" gör det möjligt att behålla det ursprungliga valet.
Titel på blogg
Här kan du ange om bloggtiteln måste visas i fet stil, dess storlek, (standard är em, erkända enheter är %, pt, px, em och ex) samt färgen, med formatet #rrggbb med rr, gg och bb vald inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och gå tillbaka till standardinställningen.
Titlar på inlägg
Du kan här ange om inläggstitlarna ska visas i fetstil, deras storlek (standard är em, erkända enheter är %, pt, px, em och ex) samt deras färg med hjälp av färgväljaren som är tillgänglig genom att klicka på väljarikonen eller genom att ange färgen i formatet #rrggbb med rr, gg och bb valda inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och gå tillbaka till det ursprungliga.
Titlar utan länk
Du kan här ange färgen på titlar utan länk (i huvudsak: titeln på ett inlägg när det visas av sig självt, underrubriker i inlägg, det aktiva objektet i den horisontella menyn) genom att använda färgväljaren som är tillgänglig genom att klicka på väljarikon eller genom att ange färgen i formatet #rrggbb med rr, gg och bb valda inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och gå tillbaka till standardinställningen.
Länkar i inlägg
Du kan ange här om länkarna i dina inlägg ska visas i fetstil eller inte och definiera deras färg i ett normalt tillstånd såväl som ett "besökt" tillstånd, deras färg när de är aktiva eller när de hålls på eller tar fokus , genom att använda färgväljaren som är tillgänglig genom att klicka på väljarikonen eller genom att ange färgen i formatet #rrggbb med rr, gg och bb valda inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och gå tillbaka till standardinställningen.

Tillgänglighet och kontrast mellan färger

När en färg har valts kommer kontrastförhållandet att visas nära fältet och specificeras, när så är fallet, att denna kontrast är i överensstämmelse med rekommendationerna i WCAG 2.0. För att vara giltig bör kontrasten mellan bakgrundsfärgen och teckensnittsfärgen ha följande värden:

  • Minst 4,5:1 för text av normal storlek för nivå AA ;
  • Minst 7:1 för texter av normal storlek för nivå AAA ;
  • Minst 3:1 för texter med förstorad text (150 % eller 120 % fet stil) för nivå AA ;
  • Minst 4,5:1 för förstorad text (150 % eller 120 % fet stil) för nivå AAA.
