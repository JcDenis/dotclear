Personalisering av innehållet

På den här fliken kan du anpassa innehållet i din blogg.

Rubrik
Markera rutan "Dölj bloggens beskrivning" för att inte visa bloggens beskrivning som vanligtvis visas under titeln.

Om insticksprogrammet "Enkel meny" är installerat och aktivt kan du klicka på den angivna länken för att konfigurera menyn i sidhuvudet. (huvudmeny?)

Ange en URL för logotypen som ska användas istället för standardlogotypen. Om URL:en inte börjar med / kommer den att vara en relativ URL från mappen img i temat (att lägga logo.png, till exempel, är liknande som att använda standardlogotypen). Rensa detta fält för att återgå till standardkonfigurationen.

Klistermärken
Klistermärkena visas överst på skärmen som små bilder på stora skärmar och längst ner på sidan på mycket små skärmar. Alla bilder vars filnamn motsvarar schemat sticker-*.png och som placerats i temats undermapp img listas. Du kan ange en etikett och motsvarande webbadress för varje bild. (Lämna ett av de båda fälten tomt för att inte visa den.)
Du kan också ändra visningsordningen med dra och släpp om javascript är aktiverat och om du inte har inaktiverat det i dina inställningar. Om javascript är avaktiverat kan du i ett fält ange visningsordningen (1 till n).
Typer och begränsningar för listor över inlägg
Vissa av visningssammanhangen på din blogg presenteras i form av en lista med inlägg. Det är till exempel fallet för startsidan eller för månadsarkiven. För varje kontext som listas i tabellen kan du ange om innehållet i varje inlägg som visas endast ska vara en titel, en kort form (innehållet i extraktfältet med de första 300 tecknen) eller deras fullständiga form. OBS. På startsidan visas alltid det första inlägget i sin helhet.
För vissa av dessa sammanhang kan du dessutom ange hur många inlägg som ska visas per sida. Om du lämnar detta fält tomt används det värde som anges i de allmänna inställningarna för den blogg som kommer att användas.

Personlig anpassning av presentationen

På den här fliken kan du anpassa vissa presentationselement för din blogg.

Allmänna parametrar

Dessa parametrar kommer att användas på alla bloggens visningsbredder.

Typsnitt
Här kan du välja den huvudsakliga typsnittsfamiljen som ska användas för de flesta texterna. Du kan också ange en andra typsnittsfamilj (sekundär) som kommer att användas för vissa titlar på den andra nivån och kommentarer. Alternativet "standard" gör det möjligt att behålla det ursprungliga valet.
Titel på blogg
Här kan du ange om bloggtiteln ska visas med fet stil, dess storlek (standard är em, erkända enheter är %, pt, px, em och ex) samt färg, med formatet #rrggbb där rr, gg och bb väljs inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och återgå till standardinställningen.
Titlar på inlägg
Här kan du ange om inläggstitlarna ska visas med fetstil, deras storlek (standard är em, erkända enheter är %, pt, px, em och ex) samt deras färg med hjälp av färgväljaren som finns genom att klicka på väljarsymbolen eller genom att ange färgen i formatet #rrggbb med rr, gg och bb valda inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och gå tillbaka till ditt ursprungliga val.
Titlar utan länk
Du kan här ange färgen på titlar utan länk (i huvudsak: titeln på ett inlägg när det visas ensamt, undertitlar i inlägg, det aktiva objektet i den horisontella menyn) genom att använda färgväljaren som finns genom att klicka på väljarsymbolen eller genom att ange färgen i formatet #rrggbb med rr, gg och bb valda inom 0 och ff.
Töm ett eller alla fält om du vill avbryta ditt val och återgå till standardinställningen.
Länkar i inlägg
Här kan du ange om länkarna i dina inlägg ska visas med fetstil eller inte och definiera deras färg i normalt tillstånd och i "besökt"-tillstånd, deras färg när de är aktiva, när de hovras över eller får fokus, genom att använda färgväljaren som finns genom att klicka på ikonen eller genom att ange färgen i formatet #rrggbb där rr, gg och bb väljs inom 0 och ff.
Töm ett eller alla fält för att avbryta ditt val och återgå till standardinställningen.

Tillgänglighet och kontrast mellan färger

När en färg har valts kommer kontrastförhållandet att visas nära fältet och specificeras, när så är fallet, att denna kontrast är i överensstämmelse med rekommendationerna i WCAG 2.0. För att vara giltig bör kontrasten mellan bakgrundsfärgen och teckensnittsfärgen ha följande värden:

  • Minst 4,5:1 för text av normal storlek för nivå AA ;
  • Minst 7:1 för texter av normal storlek för nivå AAA ;
  • Minst 3:1 för texter med förstorad text (150 % eller 120 % fet stil) för nivå AA ;
  • Minst 4,5:1 för förstorad text (150 % eller 120 % fet stil) för nivå AAA.
