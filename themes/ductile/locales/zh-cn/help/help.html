通过修改 Ductile 主题的配置，您可以非常轻松地个性化您的主题

内容个性化

在此选项卡上，您可以个性化设置博客内容。

页眉
选中 "隐藏博客描述 "复选框，以便不显示通常显示在标题下的博客描述。

如果 "简单菜单 "插件已安装并激活，您可以点击指定链接来配置页眉的菜单。(主菜单？）

指定徽标的 URL，以取代默认徽标。如果 URL 不是以 / 开头，那么它将是主题 img 文件夹中的相对 URL（例如，将 logo.png与使用默认徽标类似）。清除此字段即可返回默认配置。

贴纸
在大屏幕上，贴纸将以小图片的形式显示在屏幕顶部；在极小屏幕上，贴纸将以小图片的形式显示在页面底部。所有文件名与贴纸-*.png 方案一致，并放置在主题的 img 子文件夹中的图片都会被列出。您可以为每张图片指定一个标签和相应的网址。(如果两个字段都为空，则不显示。）
如果激活了 javascript，且没有在偏好设置中禁用，还可以通过拖放修改显示顺序。如果停用了 javascript，则可以在一个字段中指定显示顺序（1 到 n）。
职位列表的类型和限制
博客的某些显示内容是以帖子列表的形式呈现的。例如首页或每月存档。对于表中列出的每种上下文，您都可以指定显示的每篇文章的内容应是标题、简短形式（提取字段前 300 个字符的内容）还是完整形式。注意。在主页上，第一篇文章总是显示全文。
此外，在某些情况下，您还可以指定每页显示的文章数量。如果该字段为空，则将使用博客常规设置中指定的值。

个性化演示

在此选项卡上，您可以个性化设置博客的一些展示元素。

一般参数

这些参数将用于博客的所有显示宽度。

字体
您可以在此选择主要字体系列，大部分文本都将使用该系列。您还可以指定第二种字体（次要字体），用于第二层的某些标题和注释。默认 "选项允许保留原来的选择。
博客标题
您可以在此指定博客标题是否必须以粗体显示、大小（默认为 em，可识别的单位有 %、pt、ptx、em 和 ex）以及颜色，格式为 #rrggbb，rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段可取消选择并返回默认值。
职位名称
您可以在此说明是否要以粗体显示文章标题、标题大小（默认为 em，可识别的单位有 %、pt、px、em 和 ex）以及标题的颜色，可通过点击拾色器图标使用颜色拾取器，或以 #rrggbb 格式指定颜色，在 0 和 ff 之间选择 rr、gg 和 bb。
清空一个或所有字段可取消选择并返回到原来的字段。
无链接的标题
您可以通过点击拾色器图标，使用拾色器或以 #rrggbb 格式指定颜色（rr、gg 和 bb 在 0 和 ff 之间选择），来指定无链接标题（主要是：文章单独显示时的标题、文章中的副标题、水平菜单中的活动项目）的颜色。
清空一个或所有字段可取消选择并返回默认值。
帖子内的链接
您可以通过点击拾色器图标，使用拾色器或以 #rrggbb 格式指定颜色（rr、gg 和 bb 在 0 和 ff 之间选择），在此说明帖子中包含的链接是否应以粗体显示，并定义其在正常状态和 "访问 "状态下的颜色、激活时的颜色、悬停或聚焦时的颜色。
清空一个或所有字段可取消选择并返回默认值。

移动电话的具体参数

这些参数仅在显示宽度小于 480 像素时使用（例如在手机上）。如果指定了这些参数，它们将取代上述一般参数中指定的相应参数。

博客标题
您可以在此指定博客标题是否必须以粗体显示、大小（默认为 em，可识别的单位有 %、pt、ptx、em 和 ex）以及颜色，格式为 #rrggbb，rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段可取消选择并返回默认值。
职位名称
您可以在此说明是否要以粗体显示文章标题、标题大小（默认为 em，可识别的单位有 %、pt、px、em 和 ex）以及标题的颜色，可通过单击拾色器图标使用颜色拾取器，或以 #rrggbb 格式指定颜色，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段可取消选择并返回默认值。

无障碍环境和色彩对比

选定颜色后，对比度将显示在字段附近，并说明该对比度符合 WCAG 2.0 的建议。背景颜色和字体颜色的对比度应符合以下要求：

  • AA 级正常大小的文字至少为 4.5:1；
  • AAA 级正常大小的文本至少为 7:1；
  • AA 级放大（150 % 或 120 % 粗体）文本的比例至少为 3:1；
  • 对于 AAA 级的放大文本（粗体为 150 % 或 120 %），比例至少为 4.5:1。
