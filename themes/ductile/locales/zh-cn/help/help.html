通过修改 Ductile 主题的配置，您可以非常轻松地个性化您的主题

内容个性化

在此选项卡上，您可以个性化设置博客内容。

页眉
选中 "隐藏博客描述 "复选框，以便不显示通常显示在标题下的博客描述。

如果 "简单菜单 "插件已安装并激活，您可以点击指定链接来配置页眉的菜单。(主菜单？）

指定徽标的 URL，以取代默认徽标。如果 URL 不是以 / 开头，那么它将是主题 img 文件夹中的相对 URL（例如，将 logo.png与使用默认徽标类似）。清除此字段即可返回默认配置。

贴纸
在大屏幕上，贴纸将以小图像的形式显示在屏幕顶部，而在非常小的屏幕上，贴纸将显示在页面底部。所有文件名与 sticker-*.png 方案相对应并放置在主题的子文件夹 img 中的图像都会列出。您可以为每个图像指定一个标签和相应的网址。（将两个字段之一留空以不显示它。）
如果启用了 javascript 并且您没有在首选项中禁用它，您还可以通过拖放来修改显示顺序。如果停用了 javascript，则字段允许您指示显示顺序（1 到 n）。
职位列表的类型和限制
您的博客的某些显示上下文以帖子列表的形式呈现。例如，主页或每月存档就是这种情况。对于表中列出的每个上下文，您可以指定每个显示帖子的内容是否应仅为标题、简短形式（前 300 个字符的摘录字段的内容）或完整形式。注意：在主页上，第一篇帖子始终完整显示。
此外，对于其中一些上下文，您还可以指定每页显示的帖子数。如果将此字段留空，则将使用博客常规设置中指定的值。

个性化演示

在此选项卡上，您可以个性化设置博客的一些展示元素。

一般参数

这些参数将用于博客的所有显示宽度。

字体
您可以在此选择主要字体系列，大部分文本都将使用该系列。您还可以指定第二种字体（次要字体），用于第二层的某些标题和注释。默认 "选项允许保留原来的选择。
博客标题
您可以在这里指定博客标题是否必须以粗体显示、其大小（默认为 em，可识别的单位为 %、pt、px、em 和 ex）以及颜色，使用格式 #rrggbb，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回默认设置。
职位名称
您可以在这里指示帖子标题是否应以粗体显示、其大小（默认为 em，可识别的单位为 %、pt、px、em 和 ex）以及它们的颜色，方法是使用颜色选择器，单击选择器图标或以 #rrggbb 格式指定颜色，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回到原始字段。
无链接的标题
您可以在这里指定没有链接的标题的颜色（本质上：帖子单独显示时的标题、帖子内的副标题、水平菜单中的活动项目），方法是单击选择器图标，使用可用的颜色选择器，或者以 #rrggbb 格式指定颜色，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回默认设置。
帖子内的链接
您可以在这里指示您的帖子中包含的链接是否应以粗体显示，并定义它们在正常状态和“访问”状态下的颜色，它们在活动状态或悬停或获得焦点时的颜色，方法是使用可用的颜色选择器，单击选择器图标或以 #rrggbb 格式指定颜色，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回默认设置。

移动电话的具体参数

这些参数仅在显示宽度小于 480 像素时使用（例如在手机上）。如果指定了这些参数，它们将取代上述一般参数中指定的相应参数。

博客标题
您可以在这里指定博客标题是否必须以粗体显示、其大小（默认为 em，可识别的单位为 %、pt、px、em 和 ex）以及颜色，使用格式 #rrggbb，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回默认设置。
职位名称
您可以在这里指示帖子标题是否应以粗体显示、其大小（默认为 em，可识别的单位为 %、pt、px、em 和 ex）以及它们的颜色，方法是使用颜色选择器，单击选择器图标或以 #rrggbb 格式指定颜色，其中 rr、gg 和 bb 在 0 和 ff 之间选择。
清空一个或所有字段以取消您的选择并返回默认设置。

无障碍环境和色彩对比

选定颜色后，对比度将显示在字段附近，并说明该对比度符合 WCAG 2.0 的建议。背景颜色和字体颜色的对比度应符合以下要求：

  • AA 级正常大小的文字至少为 4.5:1；
  • AAA 级正常大小的文本至少为 7:1；
  • AA 级放大（150 % 或 120 % 粗体）文本的比例至少为 3:1；
  • 对于 AAA 级的放大文本（粗体为 150 % 或 120 %），比例至少为 4.5:1。
