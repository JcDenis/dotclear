콘텐츠 개인화

이 탭에서 블로그 콘텐츠를 맞춤 설정할 수 있습니다.

헤더
일반적으로 제목 아래에 표시되는 블로그의 설명을 표시하지 않으려면 '블로그 설명 숨기기' 확인란을 선택합니다.

플러그인 "심플 메뉴"가 설치되어 활성화된 경우 지정된 링크를 클릭하여 헤더의 메뉴를 구성할 수 있습니다. (메인 메뉴?)

기본 로고 대신 사용할 로고의 URL을 지정합니다. URL이 / 로 시작하지 않으면 테마의 img 폴더의 상대 URL이 됩니다(예를 들어 logo.png를 입력하는 것은 기본 로고를 사용하는 것과 유사합니다). 기본 구성으로 돌아가려면 이 필드의 선택을 취소합니다.

스티커
스티커는 대형 화면에서는 화면 상단에 작은 이미지로 표시되고, 매우 작은 화면에서는 페이지 하단에 표시됩니다. 파일 이름이 스티커-*.png 구성표에 해당하고 테마의 하위 폴더 img에 있는 모든 이미지가 나열됩니다. 각각의 라벨과 해당 URL을 표시할 수 있습니다. (표시하지 않으려면 두 필드를 모두 비워 두세요.)
자바스크립트가 활성화되어 있고 기본 설정에서 비활성화하지 않은 경우 드래그 앤 드롭으로 표시 순서를 수정할 수도 있습니다. 자바스크립트가 비활성화된 경우 필드를 통해 표시 순서(1~n)를 지정할 수 있습니다.
글 목록의 유형 및 제한
블로그의 일부 표시 컨텍스트는 게시물 목록 형식으로 표시됩니다. 예를 들어 홈 페이지나 월간 아카이브의 경우입니다. 표에 나열된 각 컨텍스트에 대해 표시되는 각 게시물의 콘텐츠가 제목으로만 표시되어야 하는지, 짧은 형식(처음 300자의 추출 필드 내용) 또는 전체 형식으로 표시되어야 하는지 지정할 수 있습니다. 주의. 홈 페이지에서는 첫 번째 게시물이 항상 전체 내용으로 표시됩니다.
또한 일부 상황에서는 페이지당 표시되는 게시물 수를 지정할 수도 있습니다. 이 필드를 비워두면 블로그의 일반 설정에 지정된 값이 사용됩니다.

접근성 및 색상 간 대비

색상을 선택하면 필드 근처에 대비 비율이 표시되며, 이 경우 이 대비가 WCAG 2.0의 권장 사항을 준수하도록 지정됩니다. 배경색과 글꼴 색의 대비가 유효하려면 다음 값을 가져야 합니다:

  • 레벨 AA의 일반 크기 텍스트의 경우 최소 4,5:1입니다;
  • 레벨 AAA의 일반 크기 텍스트의 경우 최소 7:1입니다;
  • 레벨 AA의 경우 확대된 크기(150% 또는 120% 굵게)의 텍스트의 경우 최소 3:1입니다;
  • 레벨 AAA의 경우 확대된 크기(150% 또는 120% 굵게)의 텍스트의 경우 최소 4,5:1입니다.
