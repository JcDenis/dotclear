Τροποποιώντας τις ρυθμίσεις του θέματος Ductile, μπορείτε να προσαρμόσετε το θέμα σας πολύ εύκολα.

Εξατομίκευση του περιεχομένου

Σε αυτή την καρτέλα μπορείτε να εξατομικεύσετε το περιεχόμενο του ιστολογίου σας.

Κεφαλίδα
Επιλέξτε το πλαίσιο "Απόκρυψη της περιγραφής του ιστολογίου" για να μην εμφανίζεται η περιγραφή του ιστολογίου που συνήθως εμφανίζεται κάτω από τον τίτλο.

Εάν το πρόσθετο "Απλό μενού" είναι εγκατεστημένο και ενεργό, μπορείτε να κάνετε κλικ στον καθορισμένο σύνδεσμο για να ρυθμίσετε το μενού της κεφαλίδας. (κύριο μενού;)

Καθορίστε μια διεύθυνση URL του λογότυπου που θα χρησιμοποιηθεί αντί του προεπιλεγμένου. Εάν η διεύθυνση URL δεν ξεκινά με ένα / τότε θα είναι μια σχετική διεύθυνση URL από το φάκελο img του θέματος (η τοποθέτηση logo.png, για παράδειγμα, είναι παρόμοια με τη χρήση του προεπιλεγμένου λογότυπου). Καθαρίστε αυτό το πεδίο για να επιστρέψετε στην προεπιλεγμένη διαμόρφωση.

Αυτοκόλλητα
Τα αυτοκόλλητα θα εμφανίζονται στο επάνω μέρος της οθόνης ως μικρές εικόνες σε μεγάλες οθόνες και στο κάτω μέρος της σελίδας σε πολύ μικρές οθόνες. Όλες οι εικόνες των οποίων το όνομα αρχείου αντιστοιχούν στο σχήμα sticker-*.png και τοποθετούνται στον υποφάκελο img του θέματος παρατίθενται. Μπορείτε να υποδείξετε για κάθε μια ετικέτα και το αντίστοιχο url. (Αφήστε το κενό και από τα δύο πεδία για να μην εμφανιστεί.)
Μπορείτε επίσης να τροποποιήσετε τη σειρά εμφάνισης με μεταφορά και απόθεση εάν η Javascript είναι ενεργοποιημένη και εάν δεν την έχετε απενεργοποιήσει στις προτιμήσεις σας. Εάν η javascript είναι απενεργοποιημένη, ένα πεδίο σάς επιτρέπει να υποδείξετε τη σειρά εμφάνισης (1 έως n).
Τύποι και όρια για λίστες θέσεων
Ορισμένα από τα περιβάλλοντα εμφάνισης του ιστολογίου σας παρουσιάζονται με τη μορφή λίστας αναρτήσεων. Είναι η περίπτωση για παράδειγμα για την αρχική σελίδα ή για τα μηνιαία αρχεία. Για καθένα από τα συμφραζόμενα που αναφέρονται στον πίνακα, μπορείτε να καθορίσετε εάν το περιεχόμενο κάθε εμφανιζόμενης ανάρτησης θα πρέπει να είναι μόνο ως τίτλος, μια σύντομη φόρμα (περιεχόμενο του πεδίου εξαγωγής των πρώτων 300 χαρακτήρων) ή η πλήρης μορφή τους. ΣΗΜ. Στην αρχική σελίδα, η πρώτη ανάρτηση εμφανίζεται πάντα πλήρης.
Επιπλέον, για ορισμένα από αυτά τα περιβάλλοντα, μπορείτε επίσης να καθορίσετε τον αριθμό των αναρτήσεων που εμφανίζονται ανά σελίδα. Εάν αφήσετε αυτό το πεδίο κενό, η τιμή που καθορίζεται στις γενικές ρυθμίσεις του ιστολογίου που θα χρησιμοποιηθεί.

Εξατομίκευση της παρουσίασης

Σε αυτή την καρτέλα μπορείτε να προσαρμόσετε ορισμένα στοιχεία παρουσίασης του ιστολογίου σας.

Γενικές παράμετροι

Αυτές οι παράμετροι θα χρησιμοποιηθούν σε όλα τα πλάτη εμφάνισης του ιστολογίου.

Γραμματοσειρές
Εδώ μπορείτε να επιλέξετε την κύρια οικογένεια γραμματοσειρών που θα χρησιμοποιηθεί για τα περισσότερα κείμενα. Μπορείτε επίσης να καθορίσετε μια δεύτερη οικογένεια γραμματοσειράς (δευτερεύουσα), η οποία θα χρησιμοποιείται για ορισμένους τίτλους του δεύτερου επιπέδου και για τα σχόλια. Η επιλογή "προεπιλογή" επιτρέπει να παραμείνει η αρχική επιλογή.
Τίτλος ιστολογίου
Μπορείτε να καθορίσετε εδώ εάν ο τίτλος του ιστολογίου πρέπει να εμφανίζεται με έντονη γραφή, το μέγεθός του (η προεπιλογή είναι em, οι αναγνωρισμένες μονάδες είναι %, pt, px, em και ex) καθώς και το χρώμα, χρησιμοποιώντας τη μορφή #rrggbb με rr, gg και ββ επιλεγμένα εντός 0 και ff.
Αδειάστε ένα ή όλα τα πεδία για να ακυρώσετε την επιλογή σας και να επιστρέψετε στην προεπιλογή.
Τίτλοι θέσεων
Μπορείτε να υποδείξετε εδώ εάν οι τίτλοι αναρτήσεων πρέπει να εμφανίζονται με έντονους χαρακτήρες, το μέγεθός τους (η προεπιλογή είναι em, οι αναγνωρισμένες μονάδες είναι %, pt, px, em και ex) καθώς και το χρώμα τους χρησιμοποιώντας το Color Picker που είναι διαθέσιμο κάνοντας κλικ στο εικονίδιο επιλογής ή καθορίζοντας το χρώμα στη μορφή #rrggbb με rr, gg και bb επιλεγμένα εντός 0 και ff.
Αδειάστε ένα ή όλα τα πεδία για να ακυρώσετε την επιλογή σας και να επιστρέψετε στο αρχικό σας.
Τίτλοι χωρίς σύνδεσμο
Μπορείτε να καθορίσετε εδώ το χρώμα των τίτλων χωρίς σύνδεσμο (ουσιαστικά : ο τίτλος μιας ανάρτησης όταν εμφανίζεται μόνη της, οι υπότιτλοι στις αναρτήσεις, το ενεργό στοιχείο στο οριζόντιο μενού) χρησιμοποιώντας το Color Picker που είναι διαθέσιμο κάνοντας κλικ στο εικονίδιο επιλογής ή καθορίζοντας το χρώμα στη μορφή #rrggbb με rr, gg και bb επιλεγμένα εντός 0 και ff.
Αδειάστε ένα ή όλα τα πεδία για να ακυρώσετε την επιλογή σας και να επιστρέψετε στην προεπιλογή.

Προσβασιμότητα και αντίθεση μεταξύ των χρωμάτων

Αφού επιλεγεί ένα χρώμα, ο λόγος αντίθεσης θα εμφανίζεται κοντά στο πεδίο και θα διευκρινίζεται, όταν αυτό συμβαίνει, ότι η αντίθεση αυτή είναι σύμφωνη με τις συστάσεις του WCAG 2.0. Για να είναι έγκυρη, η αντίθεση μεταξύ του χρώματος φόντου και του χρώματος της γραμματοσειράς πρέπει να έχει τις ακόλουθες τιμές:

  • Τουλάχιστον 4,5:1 για κείμενο κανονικού μεγέθους για το επίπεδο ΑΑ ,
  • Τουλάχιστον 7:1 για κείμενα κανονικού μεγέθους για το επίπεδο ΑΑΑ ,
  • Τουλάχιστον 3:1 για κείμενα μεγεθυμένου μεγέθους (150 % ή 120 % έντονη γραφή) για το επίπεδο ΑΑ ,
  • Τουλάχιστον 4,5:1 για κείμενα μεγεθυμένου μεγέθους (150 % ή 120 % έντονη γραφή) για το επίπεδο ΑΑΑ.
