থিম সম্পাদক

থিম সম্পাদক আপনাকে টেমপ্লেট ফাইল, stylesheets এবং আপনি বর্তমানে ব্যবহার করা থিমের জাভাস্ক্রিপ্ট ফাইল পরিবর্তন করতে পারবেন.

ডানদিকে ফাইল তালিকা চার ভাগে বিভক্ত:

  • টেমপ্লেট ফাইল
  • CSS ফাইল (stylesheets)
  • জাভাস্ক্রিপ্ট ফাইল
  • লোকেশন ফাইল (অনুবাদের জন্য ব্যবহৃত ফাইল)

ফাইলের প্রতিটি অংশে তিনটি উপধারা থাকতে পারে:

  • বর্তমান থিমটির অন্তর্গত ফাইল
  • প্যারেন্ট থিম এর ফাইলগুলি
  • টেম্পলেট সেটের ফাইলগুলো

প্রতিটি ফাইলের সামনে বুলেট তিনটি ভিন্ন রঙের হতে পারে:

  • Yellow means it belongs to the current theme.
  • Red means it belongs to a parent theme.
  • Black means it is in the template set.

বর্তমান থিমটির অন্তর্গত ফাইলগুলোও সাহসী.

ফাইল mofidication

To modify a file, click on its name. Its content will be displayed in the editor. If the file can be overwritten you will be able to save it by pressing the Save (s) button.

সতর্ক থাকুন: সংরক্ষণ করার পরপরই পরিবর্তন আবেদন করা হয়.

যদি আপনি একটি ফাইল পরিবর্তন করেন যা টেম্পেট সেট অথবা প্যারেন্ট থিম থেকে আসে, তবে ফাইলটি আপনার বর্তমান থিম-এ কপি করা হবে.
