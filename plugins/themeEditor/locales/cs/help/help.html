Editor motivů

Editor motivu umožňuje upravovat soubory šablony, soubory stylů, soubory Javascriptu, locales a některé soubory PHP aktuálně používaného motivu.

Seznam souborů vpravo je rozdělen do pěti oddílů:

  • Soubory šablon
  • Soubory CSS (soubory stylů)
  • Soubory Javascript
  • Soubory Locales (soubory používané pro překlad)
  • Soubory PHP (upravujte opatrně)

Každý oddíl souborů může mít čtyři pododdíly :

  • Přizpůsobené soubory
  • Soubory patřící k aktuálnímu tématu
  • Soubory patřící k nadřazenému tématu
  • Soubory patřící do sady šablon

Úprava souborů

Chcete-li soubor upravit, klikněte na jeho název. Jeho obsah se zobrazí v editoru. Pokud lze soubor přepsat, budete jej moci uložit stisknutím tlačítka Save (s).

Buďte opatrní: Úpravy se aplikují ihned po uložení.

Chcete-li obnovit úpravy souboru, klikněte na tlačítko Resetovat. Upravený soubor bude poté odstraněn.

Zvýraznění syntaxe

Zvýrazňování syntaxe můžete aktivovat v předvolbách (na kartě Moje možnosti).

Další čtení

Chcete-li upravit soubory šablon, podívejte se do seznamu značek šablon.
