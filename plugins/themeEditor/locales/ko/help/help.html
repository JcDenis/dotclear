테마 편집기

테마 편집기에서는 당신이 현재 사용 중인 테마의 템플릿파일들, 스타일시트 그리고 자바스크립트 파일 등을 편집/수정할 수 있습니다.

파일 목록은 다음 세 부분으로 나뉘어져 있습니다 :

  • 템플릿 파일들
  • CSS 파일들 : 스타일시트들
  • 자바스크립트 파일들
  • 로캘 파일(번역에 사용되는 파일)

파일 이름 앞의 노란 점은 파일이 현재 테마의 일부임을 표시합니다. 검은 점은 파일이 "기본" 테마의 일부임을 표시합니다.

  • 현재 테마에 속한 파일
  • 상위 테마에 속한 파일
  • 템플릿 세트에 속한 파일

"기본" 테마에 속하는 파일을 편집하면, 현재 사용중인 테마로 파일이 자동으로 복사됩니다.

  • 노란색 은 현재 테마에 속함을 의미합니다.
  • 빨간색 은 상위 테마에 속함을 의미합니다.
  • 검은색 은 템플릿 세트에 있음을 의미합니다.

파일을 편집하려면, 파일이름을 클릭하세요. 그러면 파일 내용이 편집칸에 보입니다. 파일에 쓰기허가가 되어있다면, "저장" 단추를 눌러 파일을 저장할 수 있습니다.

파일 모피디케이션

편집된 내용을 저장하면 블로그에 곧바로 적용되니, 편집과 저장에 주의하시기 바랍니다.

템플릿파일 편집에 도움이 필요하다면, 템플릿 태그 목록을 참조하십시요.

템플릿 세트 또는 상위 테마에서 가져온 파일을 수정하면 파일이 현재 테마에 복사됩니다.

구문 강조 표시

구문 강조 표시를 활성화하려면 내 기본 설정 ( 내 옵션 탭에서)에서 활성화할 수 있습니다.

추가 읽기

템플릿 파일을 편집하려면 템플릿 태그 목록을 확인하세요.
