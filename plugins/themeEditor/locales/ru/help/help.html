Редактор тем

Редактор тем позволяет изменять файлы шаблонов, таблицы стилей и файлы Javascript темы используемой в настоящее время.

Список файлов справа разделен на четыре раздела:

  • Файлы шаблонов
  • CSS-файлы (стили)
  • Файлы JavaScript
  • Файлы локалей (файлы, используемые для перевода)
  • PHP files (to be modified with caution)

Каждая секция файлов может иметь три подраздела :

  • Файлы, относящиеся к текущей теме
  • Файлы, относящиеся к родительской теме
  • Файлы, относящиеся к набору шаблонов

Мофификация файла

Чтобы изменить файл, нажмите на его имя. Его содержимое будет отображаться в редакторе. Если файл может быть перезаписан, вы сможете сохранить его, нажав кнопку Сохранить (Сохранить).

Будьте осторожны: изменения применяются сразу же после сохранения.

Если вы измените файл, который исходит из набора шаблонов или родительской темы, файл будет скопирован в вашу текущую тему.

Подсветка синтаксиса

You can activate the syntax highlighting in your preferences (in My options tab).
