Editor Tema

Editor Tema memungkinkan Anda untuk memodifikasi file template, stylesheet, dan file Javascript dari tema yang sedang Anda gunakan.

Daftar file di sebelah kanan dibagi dalam empat bagian:

  • File templat
  • File CSS (lembar gaya)
  • File Javascript
  • File lokal (file yang digunakan untuk terjemahan)

Tiap bagian file mungkin memiliki tiga sub-bagian:

  • File yang termasuk dalam tema saat ini
  • File milik tema induk
  • File yang termasuk dalam kumpulan templat

Peluru di depan tiap file bisa memiliki tiga warna berbeda:

  • Warna kuning berarti termasuk dalam tema saat ini.
  • Warna merah berarti tema ini termasuk dalam tema induk.
  • Hitam berarti ada di dalam set template.

File yang termasuk dalam tema saat ini, juga dicetak tebal.

Mofidikasi file

Untuk memodifikasi file, klik namanya. Isinya akan ditampilkan di editor. Jika file dapat ditimpa, Anda dapat menyimpannya dengan menekan tombol Simpan .

Hati-hati: Modifikasi diterapkan segera setelah menyimpan.

Jika Anda memodifikasi file yang berasal dari kumpulan templat atau tema induk, file tersebut akan disalin ke dalam tema Anda saat ini.

Penyorotan sintaksis

Anda dapat mengaktifkan penyorotan sintaksis di preferensi Anda (di tab Pilihan saya).

Bacaan lebih lanjut

Untuk mengedit file template Anda, silakan periksa daftar tag template.
