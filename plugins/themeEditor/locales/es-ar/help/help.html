Editor de temas

El Editor de temas le permite modificar los archivos de plantilla, las hojas de estilo, los archivos Javascript, las configuraciones regionales y algunos archivos PHP del tema que está utilizando actualmente.

La lista de archivos de la derecha está dividida en cinco secciones:

  • Ficheros template: las plantillas
  • Ficheros CSS: las hojas de estilo
  • Ficheros JavaScript
  • Archivos Locales (archivos utilizados para la traducción)
  • Ficheros PHP (a modificar con precaución)

Cada sección de archivos puede tener cuatro subsecciones :

  • Archivos personalizados
  • Archivos pertenecientes al tema actual
  • Archivos pertenecientes al tema principal
  • Archivos pertenecientes al conjunto de plantillas

Modificación de archivos

Las modificaciones se aplican inmediatamente tras guardar, preste atención.

Para modificar sus plantillas consulte la lista de etiquetas de plantilla.

Para restablecer las modificaciones de un fichero, haga clic en el botón restablecer. A continuación, se eliminará el archivo personalizado.

Resaltado sintáctico

Puede activar el resaltado de sintaxis en sus preferencias (en la pestaña Mis opciones).
