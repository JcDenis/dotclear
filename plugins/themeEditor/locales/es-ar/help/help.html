Editor de temas

El editor de temas le permite modificar los ficheros plantilla, las hojas de estilo y los ficheros Javascript de su tema actual.

La lista de ficheros se divide en tres partes:

  • Ficheros template: las plantillas
  • Ficheros CSS: las hojas de estilo
  • Ficheros JavaScript
  • Archivos Locales (archivos utilizados para la traducción)
  • PHP files (to be modified with caution)

Un topo amarillo antes del nombre de un fichero indica que éste forma parte del tema. Un topo rojo indica que se encuentra en el tema padre. Un topo negro indica que se encuentra en el tema «default»:

  • Archivos pertenecientes al tema actual
  • Archivos pertenecientes al tema principal
  • Archivos pertenecientes al conjunto de plantillas

Mofidicación de archivos

Las modificaciones se aplican inmediatamente tras guardar, preste atención.

Para modificar sus plantillas consulte la lista de etiquetas de plantilla.

Si modifica un archivo que proviene del conjunto de plantillas o del tema padre, el archivo se copiará en su tema actual.

Resaltado sintáctico

You can activate the syntax highlighting in your preferences (in My options tab).
