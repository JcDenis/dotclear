主题编辑器

主题编辑器允许您修改当前使用的主题的模板文件、样式表、Javascript 文件、本地化文件和一些 PHP 文件。

右侧的文件列表分为五个部分：

  • 模板文件：所有模板
  • CSS 文件：所有样式表
  • JavaScript 文件
  • 地域文件（用于翻译的文件）。
  • PHP 文件（需谨慎修改）

文件的每个部分可以有四个小节：

  • 定制文件
  • 属于当前主题的文件
  • 属于父主题的文件
  • 属于模板集的文件

文件修改

修改会在保存后立刻应用，所以要小心。

为了帮助修改您的模板，您可以参考模板标签列表。

要重置对文件的修改，请单击重置按钮。随后，自定义文件将被删除。

语法高亮

您可以在首选项（"我的选项"选项卡）中激活语法高亮。

进一步阅读

要编辑你的模板文件，请查看 ，模板标签列表。
