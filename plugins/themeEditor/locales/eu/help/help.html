Gaien editorea

Gaien editoreak erabiltzen ari zaren gaiaren txantiloi fitxategiak, estilo-orriak, Javascript fitxategiak, lokaleak eta PHP fitxategi batzuk aldatzeko aukera ematen dizu.

Eskuineko fitxategien zerrenda bost ataletan banatuta dago:

  • Txantiloi fitxategiak
  • CSS fitxategiak (estilo-orriak)
  • Javascript fitxategiak
  • Tokiko fitxategiak (itzultzeko erabiltzen diren fitxategiak)
  • PHP fitxategiak (kontuz aldatu beharrekoak)

Fitxategien atal bakoitzak lau azpiatal izan ditzake:

  • Pertsonalizatutako fitxategiak
  • Uneko gaiari dagozkion fitxategiak
  • Guraso gaiari dagozkion fitxategiak
  • Txantiloi multzoari dagozkion fitxategiak

Fitxategiaren aldaketa

Fitxategi bat aldatzeko, egin klik haren izenan. Bere edukia editorean bistaratuko da. Fitxategia gainidazten bada, gorde ahal izango duzu Gorde(k) botoia sakatuz.

Kontuz: aldaketak gorde eta berehala aplikatzen dira.

Fitxategi baten aldaketak berrezartzeko, egin klik berrezartzeko botoian. Pertsonalizatutako fitxategia kenduko da orduan.

Sintaxia nabarmentzea

Sintaxiaren nabarmentzea aktibatu dezakezu zure hobespenetan (Nire aukerak fitxan).

Irakurketa gehiago

Txantiloi-fitxategiak editatzeko, begiratu txantiloi-etiketen zerrenda.
