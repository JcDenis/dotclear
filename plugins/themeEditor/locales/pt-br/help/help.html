Editor de temas

O Theme Editor permite modificar os arquivos de modelo, as folhas de estilo, os arquivos Javascript, as localidades e alguns arquivos PHP do tema que você está usando no momento.

A lista de arquivos à direita está dividida em cinco seções:

  • Ficheiros de modelos
  • Ficheiros CSS (folhas de estilo)
  • Ficheiros Javascript
  • Ficheiros de localidades (ficheiros utilizados para tradução)
  • Arquivos PHP (a serem modificados com cautela)

Cada seção de arquivos pode ter quatro subseções:

  • Arquivos personalizados
  • Ficheiros pertencentes ao tema atual
  • Ficheiros pertencentes ao tema principal
  • Ficheiros pertencentes ao conjunto de modelos

Modificação de arquivos

Para modificar um ficheiro, clique no seu nome. O seu conteúdo será apresentado no editor. Se o ficheiro puder ser substituído, poderá guardá-lo premindo o botão Guardar (s) .

Cuidado: As modificações são aplicadas imediatamente após a gravação.

Para redefinir as modificações de um arquivo, clique no botão de redefinição. O arquivo personalizado será então removido.

Realce de sintaxe

Você pode ativar o realce de sintaxe em suas preferências (na guia Minhas opções).
