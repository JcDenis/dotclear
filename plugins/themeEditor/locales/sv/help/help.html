Temaredigerare

Med temaredigeraren kan du ändra mallfilerna, stilmallarna, Javascript-filerna, de lokala filerna och vissa PHP-filer i det tema som du använder för närvarande.

Fillistan till höger är indelad i fem avsnitt:

  • Mallfiler
  • CSS-filer (formatmallar)
  • Javascript-filer
  • Locales-filer (filer som används för översättning)
  • PHP-filer (ska ändras med försiktighet)

Varje sektion av filer kan ha fyra undersektioner :

  • Anpassade filer
  • Filer som hör till det aktuella temat
  • Filer som tillhör det överordnade temat
  • Filer som tillhör malluppsättningen

Filändring

För att ändra en fil, klicka på dess namn. Filens innehåll kommer att visas i editorn. Om filen kan skrivas över kan du spara den genom att trycka på knappen Save (s) .

Var försiktig: Ändringar tillämpas omedelbart efter att de sparats.

För att återställa ändringar av en fil, klicka på återställningsknappen. Den anpassade filen kommer då att tas bort.

Syntax markering

Du kan aktivera syntaxmarkeringen i dina inställningar (på fliken Mina alternativ).
