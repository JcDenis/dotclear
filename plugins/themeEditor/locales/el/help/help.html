Επεξεργαστής θέματος

Ο επεξεργαστής θέματος σας επιτρέπει να τροποποιήσετε τα αρχεία προτύπων, τα φύλλα στυλ και τα αρχεία Javascript του θέματος που χρησιμοποιείτε.

Η λίστα αρχείων στα δεξιά χωρίζεται σε τέσσερις ενότητες:

  • Αρχεία προτύπων
  • Αρχεία CSS (φύλλα στυλ)
  • Αρχεία Javascript
  • Αρχεία τοπικής γλώσσας (αρχεία που χρησιμοποιούνται για τη μετάφραση)

Κάθε ενότητα αρχείων μπορεί να έχει τρεις υποενότητες :

  • Αρχεία που ανήκουν στο τρέχον θέμα
  • Αρχεία που ανήκουν στο γονικό θέμα
  • Αρχεία που ανήκουν στο σύνολο προτύπων

Η σφαίρα μπροστά από κάθε αρχείο μπορεί να έχει τρία διαφορετικά χρώματα:

  • Το κίτρινο σημαίνει ότι ανήκει στο τρέχον θέμα.
  • Το κόκκινο σημαίνει ότι ανήκει σε ένα γονικό θέμα.
  • Μαύρο σημαίνει ότι υπάρχει στο σετ προτύπων.

Τα αρχεία που ανήκουν στο τρέχον θέμα είναι επίσης με έντονη γραφή.

Αρχείο mofidication

Για να τροποποιήσετε ένα αρχείο, κάντε κλικ στο όνομά του. Το περιεχόμενό του θα εμφανιστεί στον επεξεργαστή. Εάν το αρχείο μπορεί να αντικατασταθεί, θα μπορείτε να το αποθηκεύσετε πατώντας το κουμπί Save (s) .

Να είστε προσεκτικοί: Οι τροποποιήσεις εφαρμόζονται αμέσως μετά την αποθήκευση.

Αν τροποποιήσετε ένα αρχείο που προέρχεται από το σύνολο προτύπων ή το γονικό θέμα, το αρχείο θα αντιγραφεί στο τρέχον θέμα σας.

Υπογράμμιση σύνταξης

Μπορείτε να ενεργοποιήσετε την υπογράμμιση σύνταξης στο Οι προτιμήσεις σας (στην καρτέλα Οι επιλογές μου).

Περαιτέρω ανάγνωση

Για να επεξεργαστείτε τα αρχεία προτύπων σας, ελέγξτε τη λίστα με τις ετικέτες προτύπων.
