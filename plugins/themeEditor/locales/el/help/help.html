Επεξεργαστής θέματος

Ο Επεξεργαστής θέματος σας επιτρέπει να τροποποιήσετε τα αρχεία προτύπων, τα φύλλα στυλ, τα αρχεία Javascript, τις τοπικές ρυθμίσεις και ορισμένα αρχεία PHP του θέματος που χρησιμοποιείτε αυτή τη στιγμή.

Η λίστα αρχείων στα δεξιά χωρίζεται σε πέντε ενότητες:

  • Αρχεία προτύπων
  • Αρχεία CSS (φύλλα στυλ)
  • Αρχεία Javascript
  • Αρχεία τοπικής γλώσσας (αρχεία που χρησιμοποιούνται για τη μετάφραση)
  • Αρχεία PHP (πρέπει να τροποποιηθούν με προσοχή)

Κάθε ενότητα αρχείων μπορεί να έχει τέσσερις υποενότητες :

  • Προσαρμοσμένα αρχεία
  • Αρχεία που ανήκουν στο τρέχον θέμα
  • Αρχεία που ανήκουν στο γονικό θέμα
  • Αρχεία που ανήκουν στο σύνολο προτύπων

Τροποποίηση αρχείου

Για να τροποποιήσετε ένα αρχείο, κάντε κλικ στο όνομά του. Το περιεχόμενό του θα εμφανιστεί στον επεξεργαστή. Εάν το αρχείο μπορεί να αντικατασταθεί, θα μπορείτε να το αποθηκεύσετε πατώντας το κουμπί Save (s) .

Να είστε προσεκτικοί: Οι τροποποιήσεις εφαρμόζονται αμέσως μετά την αποθήκευση.

Για να επαναφέρετε τις τροποποιήσεις ενός αρχείου, κάντε κλικ στο κουμπί επαναφοράς. Το προσαρμοσμένο αρχείο θα αφαιρεθεί στη συνέχεια.

Υπογράμμιση σύνταξης

Μπορείτε να ενεργοποιήσετε την επισήμανση σύνταξης στις προτιμήσεις σας (στην καρτέλα Οι επιλογές μου).

Περαιτέρω ανάγνωση

Για να επεξεργαστείτε τα αρχεία προτύπων σας, ελέγξτε τη λίστα με τις ετικέτες προτύπων.
