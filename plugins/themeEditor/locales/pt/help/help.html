Editor do tema

O Editor de temas permite-lhe modificar os ficheiros de modelo, as folhas de estilo, os ficheiros Javascript, os locais e alguns ficheiros PHP do tema que está a utilizar atualmente.

A lista de ficheiros à direita está dividida em cinco secções:

  • Ficheiros template: o template mesmo
  • Ficheiros CSS: as folhas de estilo
  • Ficheiros JavaScript
  • Arquivos de localidade (arquivos usados para tradução)
  • Ficheiros PHP (a modificar com cuidado)

Cada secção de ficheiros pode ter quatro subsecções :

  • Ficheiros personalizados
  • Arquivos pertencentes ao tema atual
  • Arquivos pertencentes ao tema pai
  • Arquivos pertencentes ao conjunto de modelos

Modificação de ficheiros

As alterações são aplicadas imediatamente quando salvar, seja vigilantes!

Para ajuda na edição dos template, não hesite em consultar a lista dos tags de template.

Para anular as modificações de um ficheiro, clique no botão anular. O ficheiro personalizado será então removido.

Destaque de sintaxe

Pode ativar o realce de sintaxe nas suas preferências (no separador As minhas opções).

Leitura adicional

Para editar os seus ficheiros de modelos, consulte a lista de etiquetas de modelos.
