Editor do tema

O editor de tema permite modificar os ficheiros template, as folhas de estilo e os ficheiros JavaScript no seu tema activo.

A lista de ficheiros é dividido em três partes:

  • Ficheiros template: o template mesmo
  • Ficheiros CSS: as folhas de estilo
  • Ficheiros JavaScript
  • Arquivos de localidade (arquivos usados para tradução)

Um ícone amarelo contra o nome do ficheiro indica que é parte do tema.
Um ícone preto indica que é parte do tema principal (default theme).

  • Arquivos pertencentes ao tema atual
  • Arquivos pertencentes ao tema pai
  • Arquivos pertencentes ao conjunto de modelos

Ao editar um ficheiro que pertencia o tema principal este será copiado no seu tema activo.

  • Amarelo significa que ele pertence ao tema atual.
  • Vermelho significa que ele pertence a um tema pai.
  • Preto significa que está no conjunto de templates.

Para editar um ficheiro, clique em seu nome, o conteúdo será exibido numa área de edição. Se o ficheiro pode ser escrito, pode salvar-lo clicando em "gravar".

Filtragem de arquivo

As alterações são aplicadas imediatamente quando salvar, seja vigilantes!

Para ajuda na edição dos template, não hesite em consultar a lista dos tags de template.

Se você modificar um arquivo que vem do conjunto de modelos ou do tema principal, o arquivo será copiado para o tema atual.

Destaque de sintaxe

Você pode ativar o destaque de sintaxe em suas preferências (em Minhas opções guia).

Leitura adicional

Para editar os arquivos do template, por favor, verifique a lista de tags de template.
