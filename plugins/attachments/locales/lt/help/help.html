Priedai

Spustelėkite Add files to this entry , kad prie įrašo pridėtumėte failą iš medijos tvarkyklės. Pateksite į medijos tvarkyklę, kurioje galėsite peržiūrėti turimus failus. Radę failą, kurį norite pridėti, spustelėkite + (Attach this file to entry link) ženklą.

Norėdami pašalinti priedą, spustelėkite šiukšliadėžės piktogramą (Pašalinti priedo nuorodą).

Tinklaraštyje priedai bus rodomi po įrašo turinio.
