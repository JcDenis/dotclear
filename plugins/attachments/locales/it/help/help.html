Allegati

Fare clic su Aggiungi file a questa voce per allegare un file dal media manager alla tua voce. Sarai portato al media manager dove puoi navigare i file disponibili. Una volta trovato il file che si desidera allegare, clicca sul segno + (Allega questo file al link di inserimento).

Per rimuovere un allegato, fare clic sull'icona del cestino (Rimuovi link allegato).

Sul tuo blog, gli allegati appariranno dopo il contenuto dell'elemento.
