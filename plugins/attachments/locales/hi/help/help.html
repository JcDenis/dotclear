Click Add files to this entry to attach a file from the media manager to your entry. You will be taken to the media manager where you can browse the available files. Once you found the file you want to attach, click on the + (Attach this file to entry link) sign.

To remove an attachment, click on the trashcan icon (Remove attachment link).
