Συνημμένα

Κάντε κλικ στο Προσθήκη αρχείων σε αυτήν την καταχώρηση για να επισυνάψετε ένα αρχείο από το διαχειριστή πολυμέσων στην καταχώρησή σας. Θα μεταφερθείτε στο διαχειριστή πολυμέσων, όπου μπορείτε να περιηγηθείτε στα διαθέσιμα αρχεία. Μόλις βρείτε το αρχείο που θέλετε να επισυνάψετε, κάντε κλικ στο + (Παραγγείλετε αυτό το αρχείο για να καταχωρηθεί σύνδεσμος) σημάδι.

Για να καταργήσετε ένα συνημμένο, κάντε κλικ στο εικονίδιο απορριμάτων (Κατάργηση σύνδεσης συνημμένου).

Στο blog, τα συνημμένα θα εμφανίζονται μετά το περιεχόμενο της καταχώρησης.
