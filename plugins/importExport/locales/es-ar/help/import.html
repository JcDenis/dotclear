Importar archivo plano

Importa un blog o una instalación completa de Dotclear desde un archivo plano.

Blog individual

Selecciona el archivo de copia de seguridad (exportado desde otra instalación de Dotclear) o selecciona uno del directorio raíz de tu gestor multimedia y haz clic en el botón Importar para iniciar el proceso de importación.

Nota: el contenido importado durante este proceso se añadirá al contenido existente del blog actual, no se perderá nada.

Múltiples blogs

Selecciona el archivo de copia de seguridad completa de tu ordenador o selecciona uno de la carpeta raíz de tu gestor multimedia, introduce tu contraseña y haz clic en el botón Importar para iniciar el proceso de importación completa.

Advertencia: todo el contenido del blog actual será eliminado y sustituido por el contenido del archivo de copia de seguridad. Sólo se conservarán los usuarios actuales. Los usuarios adicionales del archivo de copia de seguridad se añadirán a la lista de usuarios actuales.

Importación de fuentes RSS o Atom

Añadir un contenido de alimentación al blog actual.

Introduzca la URL del feed RSS o Atom que desea importar y haga clic en Importar. El contenido encontrado en el feed se añadirá al contenido actual de tu blog.

Importación de Dotclear 1.2

Esta función le permite importar una instalación de Dotclear 1.2 a su blog actual. Este proceso reemplazará las categorías, blogrolls, entradas y comentarios del blog actual.

Introduzca la URL del servidor donde se encuentra la base de datos de Dotclear 1.2 que desea importar y, a continuación, introduzca el nombre de la base de datos, el nombre de usuario y la contraseña para conectarse a la base de datos y el prefijo de las tablas de la base de datos utilizadas en la instalación de Dotclear 1.2. Puede encontrar esta información en el archivo conf/config.php de su instalación de Dotclear 1.2.

Haga clic en el botón Importar mi blog ahora para iniciar el proceso de importación.

Puede disminuir o aumentar la cantidad de entradas importadas a la vez utilizando el campo Número de entradas a importar a la vez, dependiendo de la velocidad de acceso de su instalación de Dotclear 1.2.

Importación de Wordpress

Esta función le permite importar una instalación de Wordpress a su blog actual. Este proceso reemplazará las categorías, blogrolls, entradas y comentarios del blog actual.

Introduzca la URL del servidor donde se encuentra la base de datos de Wordpress que desea importar y, a continuación, introduzca el nombre de la base de datos, el nombre de usuario y la contraseña para conectarse a la base de datos y el prefijo de las tablas de la base de datos utilizadas en la instalación de Wordpress.

Puede configurar algunas opciones de importación relativas a la importación de categorías y al número de entradas a importar de una vez. También puedes definir la sintaxis utilizada para las entradas y el contenido de los comentarios (HTML, Wiki u otras dependiendo de las sintaxis actualmente instaladas y activadas).

Haga clic en Importar mi blog ahora para iniciar el proceso de importación.

Nota: las funciones de exportación se encuentran ahora en la páginaMantenimiento.
