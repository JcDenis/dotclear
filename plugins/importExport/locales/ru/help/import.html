Импортировать Flat файл

Импортирует блог или полную установку Dotclear из плоского файла.

Один блог

Выберите файл резервной копии (экспортированный из другой системы установки Dotcle) или выберите его из корневого каталога вашего менеджера медиа файлов и нажмите кнопку Импортировать , чтобы начать процесс импорта.

Примечание: содержимое, импортированное в ходе этого процесса, будет добавлено к существующему содержимому текущего блога, ничего не будет потеряно.

Несколько блогов

Выберите полный резервный файл с вашего компьютера или выберите его из корневой папки вашего медиа-менеджера, введите свой пароль и нажмите на кнопку Импорт для начала полного процесса импорта.

Предупреждение: всё содержимое текущего блога будет удалено и заменено содержимым файла резервной копии. Сохраняются только текущие пользователи. Дополнительные пользователи из файла резервной копии будут добавлены в текущий список пользователей.

Импорт RSS или Atom feed

Добавить содержимое ленты в текущий блог.

Введите URL RSS или Atom канала, который вы хотите импортировать и нажмите Импорт. Содержимое, найденное в ленте будет добавлено к вашему текущему блогу.

Импорт из Dotclear 1.2

Эта функция позволяет импортировать Dotclear 1.2 установки в ваш текущий блог. Этот процесс заменит категории, блоги, записи и комментарии текущего блога.

Введите URL сервера, где находится точка1 точки-1. база данных для импорта локальна, затем введите название базы данных имя пользователя и пароль для подключения к базе данных и префикса таблиц базы данных, используемого в Dotclear 1. инсталляция. Вы можете найти эту информацию в файле conf/config.php из вашей установки Dotclear 1.2.

Нажмите на кнопку Импорт моего блога сейчас для начала процесса импорта.

Вы можете уменьшить или увеличить количество импортированных записей за один раз, используя поле Количество импортируемых записей одновременно, в зависимости от скорости доступа вашей точки 1. установка.

Импорт Wordpress

Эта функция позволяет импортировать установку Wordpress в ваш текущий блог. Этот процесс заменит категории, блоги, записи и комментарии текущего блога.

Введите URL сервера, где находится Wordpress база данных для импорта, а затем введите имя базы данных имя пользователя и пароль для подключения к базе данных и префикса таблиц базы данных, используемых в установке Wordpress.

Вы можете настроить несколько вариантов импорта, касающихся импорта категорий и количества записей для импорта. Вы также можете определить синтаксис, используемый для записей и комментариев (HTML, Вики или другие в зависимости от установленных и активированных синтаксисов).

Нажмите Импортировать мой блог для начала процесса импорта.

Примечание: функции экспорта теперь расположены на страницеТехническое обслуживание.
