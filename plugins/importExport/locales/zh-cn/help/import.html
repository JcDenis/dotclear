纯文本导入

从一个平面文件导入一个博客或一个完整的Dotclear安装。

单一博客

选择备份文件（从另一个Dotclear安装中导出）或从你的媒体管理器的根目录中选择一个，然后点击 Import 按钮，开始导入过程。

注意：在这个过程中导入的内容将被添加到当前博客的现有内容中，不会有任何损失。

多个博客

从你的电脑中选择完整的备份文件，或从你的媒体管理器的根文件夹中选择一个，输入你的密码，然后点击 导入 按钮，开始完整的导入过程。

警告： ，当前博客的全部内容将被删除，并由备份文件的内容所取代。只有当前的用户会被保留。备份文件中的其他用户将被添加到当前用户列表中。

RSS或Atom feed导入

在当前的博客上添加一个feed内容。

输入你要导入的RSS或Atom feed的URL，然后点击 Import。在feed中找到的内容将被添加到你当前的博客内容中。

Dotclear 1.2 导入

这个功能允许你把Dotclear 1.2的安装文件导入你当前的博客。这个过程将取代当前博客的分类、博客列表、条目和评论。

输入要导入的Dotclear 1.2数据库所在的服务器URL，然后输入数据库名称，连接到数据库的用户名和密码以及在Dotclear 1.2安装中使用的数据库表前缀。你可以在Dotclear 1.2安装的 conf/config.php 文件中找到这些信息。

点击 现在导入我的博客 按钮，开始导入过程。

你可以使用 一次性导入的条目数量 字段，减少或增加一次导入的条目数量，这取决于你Dotclear 1.2安装的访问速度。

Wordpress导入

这个功能允许你将一个Wordpress装置导入到你当前的博客中。这个过程将取代当前博客的类别、博客列表、条目和评论。

输入要导入的Wordpress数据库所在的服务器URL，然后输入数据库名称、连接到数据库的用户名和密码以及Wordpress安装中使用的数据库表前缀。

你可以设置一些关于类别导入和一次导入的条目数量的导入选项。您还可以定义条目和评论内容使用的语法（HTML、Wiki或其他，取决于当前安装和激活的语法）。

点击 现在导入我的博客 ，开始导入过程。

注意：导出功能现在位于维护 页面中。
