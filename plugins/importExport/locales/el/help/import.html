Εισαγωγή επίπεδου αρχείου

Εισάγει ένα ιστολόγιο ή μια πλήρη εγκατάσταση Dotclear από ένα επίπεδο αρχείο.

Ενιαίο ιστολόγιο

Επιλέξτε το αρχείο αντιγράφου ασφαλείας (που εξήχθη από άλλη εγκατάσταση του Dotclear) ή επιλέξτε ένα από τον ριζικό κατάλογο του διαχειριστή πολυμέσων και κάντε κλικ στο κουμπί Εισαγωγή για να ξεκινήσει η διαδικασία εισαγωγής.

Σημείωση: το περιεχόμενο που εισάγεται κατά τη διάρκεια αυτής της διαδικασίας θα προστεθεί στο υπάρχον περιεχόμενο του τρέχοντος ιστολογίου, χωρίς να χαθεί τίποτα.

Πολλαπλά blogs

Επιλέξτε το πλήρες αρχείο αντιγράφου ασφαλείας από τον υπολογιστή σας ή επιλέξτε ένα από τον ριζικό φάκελο του διαχειριστή πολυμέσων, εισαγάγετε τον κωδικό πρόσβασής σας και κάντε κλικ στο κουμπί Εισαγωγή για να ξεκινήσει η διαδικασία πλήρους εισαγωγής.

Προειδοποίηση: όλο το περιεχόμενο του τρέχοντος ιστολογίου θα διαγραφεί και θα αντικατασταθεί από το περιεχόμενο του αρχείου αντιγράφων ασφαλείας. Θα διατηρηθούν μόνο οι τρέχοντες χρήστες. Πρόσθετοι χρήστες από το αρχείο αντιγράφων ασφαλείας θα προστεθούν στη λίστα των τρεχόντων χρηστών.

Εισαγωγή τροφών RSS ή Atom

Προσθέστε ένα περιεχόμενο τροφοδοσίας στο τρέχον ιστολόγιο.

Εισάγετε τη διεύθυνση URL της ροής RSS ή Atom που θέλετε να εισαγάγετε και κάντε κλικ στο Import. Το περιεχόμενο που βρίσκεται στην τροφοδοσία θα προστεθεί στο τρέχον περιεχόμενο του ιστολογίου σας.

Εισαγωγή Dotclear 1.2

Αυτή η λειτουργία σας επιτρέπει να εισαγάγετε μια εγκατάσταση Dotclear 1.2 στο τρέχον ιστολόγιό σας. Αυτή η διαδικασία θα αντικαταστήσει τις κατηγορίες, τα blogrolls, τις καταχωρήσεις και τα σχόλια του τρέχοντος ιστολογίου.

Εισάγετε τη διεύθυνση URL του διακομιστή όπου βρίσκεται η βάση δεδομένων Dotclear 1.2 που θέλετε να εισαγάγετε, στη συνέχεια εισάγετε το όνομα της βάσης δεδομένων, το όνομα χρήστη και τον κωδικό πρόσβασης για τη σύνδεση στη βάση δεδομένων και το πρόθεμα των πινάκων της βάσης δεδομένων που χρησιμοποιείται στην εγκατάσταση του Dotclear 1.2. Μπορείτε να βρείτε αυτές τις πληροφορίες στο αρχείο conf/config.php από την εγκατάσταση του Dotclear 1.2.

Κάντε κλικ στο κουμπί Import my blog now για να ξεκινήσει η διαδικασία εισαγωγής.

Μπορείτε να μειώσετε ή να αυξήσετε τον αριθμό των καταχωρίσεων που εισάγονται ταυτόχρονα χρησιμοποιώντας το πεδίο Αριθμός καταχωρίσεων για ταυτόχρονη εισαγωγή , ανάλογα με την ταχύτητα πρόσβασης της εγκατάστασής σας στο Dotclear 1.2.

Εισαγωγή Wordpress

Αυτή η λειτουργία σας επιτρέπει να εισαγάγετε μια εγκατάσταση Wordpress στο τρέχον ιστολόγιό σας. Αυτή η διαδικασία θα αντικαταστήσει τις κατηγορίες, τις λίστες ιστολογίων, τις καταχωρήσεις και τα σχόλια του τρέχοντος ιστολογίου.

Εισάγετε τη διεύθυνση URL του διακομιστή όπου βρίσκεται η βάση δεδομένων του Wordpress που θέλετε να εισαγάγετε, στη συνέχεια εισάγετε το όνομα της βάσης δεδομένων, το όνομα χρήστη και τον κωδικό πρόσβασης για τη σύνδεση στη βάση δεδομένων και το πρόθεμα των πινάκων της βάσης δεδομένων που χρησιμοποιείται στην εγκατάσταση του Wordpress.

Μπορείτε να ρυθμίσετε μερικές επιλογές εισαγωγής σχετικά με τις κατηγορίες εισαγωγής και τον αριθμό των καταχωρήσεων που θα εισαχθούν ταυτόχρονα. Μπορείτε επίσης να ορίσετε τη σύνταξη που χρησιμοποιείται για τις καταχωρήσεις και το περιεχόμενο των σχολίων (HTML, Wiki ή άλλες ανάλογα με τις τρέχουσες εγκατεστημένες και ενεργοποιημένες συντάξεις).

Κάντε κλικ στο Import my blog now για να ξεκινήσει η διαδικασία εισαγωγής.

Σημείωση: οι λειτουργίες εξαγωγής βρίσκονται τώρα στη σελίδαΣυντήρηση .
