Impor file datar

Mengimpor blog atau instalasi Dotclear lengkap dari file datar.

Blog tunggal

Pilih file cadangan (diekspor dari instalasi Dotclear lain) atau pilih salah satu dari direktori root manajer media Anda dan klik tombol Import untuk memulai proses impor.

Catatan: konten yang diimpor selama proses ini akan ditambahkan ke konten yang sudah ada di blog saat ini, tidak ada yang hilang.

Beberapa blog

Pilih file cadangan lengkap dari komputer Anda atau pilih salah satu dari folder root manajer media Anda, masukkan kata sandi dan klik tombol Impor untuk memulai proses impor lengkap.

Peringatan: seluruh konten blog saat ini akan dihapus dan digantikan oleh konten file cadangan. Hanya pengguna saat ini yang akan disimpan. Pengguna tambahan dari file cadangan akan ditambahkan ke daftar pengguna saat ini.

Impor umpan RSS atau Atom

Menambahkan konten feed ke blog yang sedang berjalan.

Masukkan URL feed RSS atau Atom yang ingin Anda impor dan klik Impor. Konten yang ditemukan di feed akan ditambahkan ke konten blog Anda saat ini.

Impor Dotclear 1.2

Fitur ini memungkinkan Anda untuk mengimpor instalasi Dotclear 1.2 ke dalam blog Anda saat ini. Proses ini akan menggantikan kategori, blogroll, entri, dan komentar pada blog saat ini.

Masukkan URL server di mana database Dotclear 1.2 yang akan diimpor berada, kemudian masukkan nama database, nama pengguna dan kata sandi untuk terhubung ke database dan awalan tabel database yang digunakan dalam instalasi Dotclear 1.2. Anda dapat menemukan informasi ini di file conf/config.php dari instalasi Dotclear 1.2 Anda.

Klik tombol Impor blog saya sekarang untuk memulai proses impor.

Anda dapat mengurangi atau menambah jumlah entri yang diimpor sekaligus menggunakan bidang Jumlah entri yang akan diimpor sekaligus , tergantung pada kecepatan akses instalasi Dotclear 1.2 Anda.

Impor Wordpress

Fitur ini memungkinkan Anda untuk mengimpor instalasi Wordpress ke dalam blog Anda saat ini. Proses ini akan menggantikan kategori, blogroll, entri, dan komentar pada blog saat ini.

Masukkan URL server tempat database Wordpress yang akan diimpor berada, lalu masukkan nama database, nama pengguna dan kata sandi untuk menyambung ke database, serta awalan tabel database yang digunakan dalam instalasi Wordpress.

Anda dapat mengatur beberapa opsi impor terkait impor kategori dan jumlah entri yang akan diimpor sekaligus. Anda juga dapat menentukan sintaks yang digunakan untuk entri dan konten komentar (HTML, Wiki, atau lainnya, tergantung pada sintaks yang sedang diinstal dan diaktifkan).

Klik Impor blog saya sekarang untuk memulai proses impor.

Catatan: fungsi ekspor sekarang berada di halamanMaintenance .
