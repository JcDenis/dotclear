Importar um ficheiro texto

Importa um blog ou uma instalação de Dotclear completa a partir de um arquivo plano.

Blog único

Selecione o arquivo de backup (exportado de outra instalação de Dotcle) ou selecione um no diretório raiz do seu gerenciador de mídia e clique no botão Importar para iniciar o processo de importação.

Nota: o conteúdo importado durante este processo será adicionado ao conteúdo existente do blog atual, nada será perdido.

Vários blogs

Selecione o arquivo completo de backup no seu computador ou selecione um da pasta raiz do seu gerenciador de mídia, digite sua senha e clique no botão Importar para iniciar o processo completo de importação.

Aviso: todo o conteúdo do blog atual será excluído e substituído pelo conteúdo do arquivo de backup. Apenas os usuários atuais serão mantidos. Usuários adicionais do arquivo de backup serão adicionados à lista de usuários atual.

Importação de RSS ou Atom

Adicionar um conteúdo do feed ao blog atual.

Insira a URL do feed RSS ou Atom que deseja importar e clique em Importar. O conteúdo do feed será adicionado ao conteúdo do seu blog atual.

Importação Dotclear 1.2

Esta função permite importar a instalação do Dotclear 1.2 para o seu blog atual. Este processo irá substituir as categorias, blogrolls, postagens e comentários do blog atual.

Informe a URL do servidor onde o Dotclear 1. banco de dados a ser importado está localizado, depois insira o nome do banco de dados o nome do usuário e senha para conectar ao banco de dados e ao prefixo das tabelas de banco de dados usado no Dotclear 1. instalação. Você pode encontrar essa informação no arquivo conf/config.php da sua instalação Dotclear 1.2.

Clique no botão Importar meu blog agora para iniciar o processo de importação.

Você pode diminuir ou aumentar a quantidade de entradas importadas de uma vez usando o número de entradas para importar de uma vez , dependendo da velocidade de acesso do seu Dotclear 1. instalação.

Importação de Wordpress

Esse recurso permite importar uma instalação do Wordpress para seu blog atual. Este processo irá substituir as categorias, blogrolls, postagens e comentários do blog atual.

Digite a URL do servidor onde o banco de dados Wordpress a importar está localizado, em seguida, insira o nome do banco de dados. o nome de usuário e senha para conectar ao banco de dados e o prefixo das tabelas do banco de dados usado na instalação do Wordpress.

Você pode configurar algumas opções de importação sobre as categorias de importação e o número de postagens a serem importadas de uma só vez. Você também pode definir a sintaxe usada para as postagens e o conteúdo dos comentários (HTML, Wiki ou outros dependendo da sintaxe atualmente instalada e ativada).

Clique em Importar meu blog agora para iniciar o processo de importação.

Nota: as funções de exportação agora estão localizadas na página deManutenção.
