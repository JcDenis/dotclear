Korterfaili import

Impordib blogi või täieliku Dotcleari paigalduse lamefailist.

Ühtne blogi

Valige varukoopiafail (mis on eksporditud teisest Dotcleari paigaldusest) või valige see oma meediahalduri juurkataloogist ja klõpsake importimise alustamiseks nupul Import.

Märkus: selle protsessi käigus imporditud sisu lisatakse praeguse blogi olemasolevale sisule, midagi ei lähe kaduma.

Mitu blogi

Valige täielik varukoopiafail arvutist või valige see meediahalduri juurkaustast, sisestage oma parool ja klõpsake nupule Import, et alustada täielikku impordiprotsessi.

Hoiatus: kogu praeguse blogi sisu kustutatakse ja asendatakse varukoopia faili sisuga. Säilitatakse ainult praegused kasutajad. Täiendavad kasutajad varukoopiadokumendist lisatakse praeguste kasutajate loendisse.

RSS- või Atom-voo importimine

Lisage praegusele blogile sööda sisu.

Sisestage importida soovitud RSS- või Atom-voo URL ja klõpsake nuppu Import. Sisu, mis leitakse feedist, lisatakse teie praegusele blogi sisule.

Dotclear 1.2 import

See funktsioon võimaldab teil importida Dotcleari 1.2 paigalduse oma praegusesse blogisse. See protsess asendab praeguse blogi kategooriad, blogirullid, kirjed ja kommentaarid.

Sisestage serveri URL, kus asub imporditav Dotclear 1.2 andmebaas, seejärel sisestage andmebaasi nimi, kasutajanimi ja parool andmebaasiga ühendumiseks ning Dotclear 1.2 paigaldamisel kasutatud andmebaasi tabelite eesliide. Selle teabe leiate oma Dotcleari 1.2 installimise failist conf/config.php.

Vajutage nupule Import my blog now, et alustada impordiprotsessi.

Saate vähendada või suurendada korraga imporditavate kirjete arvu, kasutades väljal Number of entries to import at once (korraga imporditavate kirjete arv ), sõltuvalt teie Dotcleari 1.2 paigalduse juurdepääsukiirusest.

Wordpressi import

See funktsioon võimaldab teil importida Wordpressi installi oma praegusesse blogisse. See protsess asendab praeguse blogi kategooriad, blogirullid, kirjed ja kommentaarid.

Sisestage serveri URL, kus asub imporditav Wordpressi andmebaas, seejärel sisestage andmebaasi nimi, kasutajanimi ja parool andmebaasiga ühendumiseks ning andmebaasi tabelite eesliide, mida kasutatakse Wordpressi paigaldamisel.

Saate määrata mõned impordivalikud seoses kategooriate importimisega ja korraga imporditavate kirjete arvuga. Samuti saate määrata kirjete ja kommentaaride sisu jaoks kasutatava süntaksi (HTML, Wiki või muud sõltuvalt hetkel paigaldatud ja aktiveeritud süntaksist).

Klõpsake importimise alustamiseks nuppu Import my blog now.

Märkus: ekspordifunktsioonid asuvad nüüd leheküljelHooldus.
