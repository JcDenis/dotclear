Importação de ficheiros simples

Importa um blog ou uma instalação completa do Dotclear a partir de um ficheiro simples.

Blogue único

Seleccione o ficheiro de cópia de segurança (exportado de outra instalação Dotclear) ou seleccione um do diretório raiz do seu gestor de média e clique no botão Import para iniciar o processo de importação.

Nota: o conteúdo importado durante este processo será adicionado ao conteúdo existente do blogue atual, nada se perderá.

Vários blogues

Seleccione o ficheiro de cópia de segurança completa a partir do seu computador ou seleccione um ficheiro a partir da pasta de raiz do seu gestor multimédia, introduza a sua palavra-passe e clique no botão Importar para iniciar o processo de importação completa.

Atenção: todo o conteúdo do blogue atual será eliminado e substituído pelo conteúdo do ficheiro de cópia de segurança. Apenas os utilizadores actuais serão mantidos. Os utilizadores adicionais do ficheiro de cópia de segurança serão adicionados à lista de utilizadores actuais.

Importação de feeds RSS ou Atom

Adicionar um conteúdo de feed ao blogue atual.

Introduza o URL do feed RSS ou Atom que pretende importar e clique em Import. O conteúdo encontrado no feed será adicionado ao conteúdo atual do seu blogue.

Importação do Dotclear 1.2

Esta funcionalidade permite-lhe importar uma instalação Dotclear 1.2 para o seu blogue atual. Este processo irá substituir as categorias, blogrolls, entradas e comentários do blog atual.

Introduza o URL do servidor onde se encontra a base de dados Dotclear 1.2 a importar, depois introduza o nome da base de dados, o nome de utilizador e a palavra-passe para se ligar à base de dados e o prefixo das tabelas da base de dados utilizadas na instalação Dotclear 1.2. Pode encontrar esta informação no ficheiro conf/config.php da sua instalação Dotclear 1.2.

Clique no botão Import my blog now para iniciar o processo de importação.

Pode diminuir ou aumentar a quantidade de entradas importadas de uma só vez usando o campo Number of entries to import at once , dependendo da velocidade de acesso da sua instalação Dotclear 1.2.

Importação do Wordpress

Esta funcionalidade permite-lhe importar uma instalação do Wordpress para o seu blogue atual. Este processo substituirá as categorias, os blogrolls, as entradas e os comentários do blog atual.

Introduza o URL do servidor onde se encontra a base de dados do Wordpress a importar e, em seguida, introduza o nome da base de dados, o nome de utilizador e a palavra-passe para ligar à base de dados e o prefixo das tabelas da base de dados utilizadas na instalação do Wordpress.

Pode configurar algumas opções de importação relativamente à importação de categorias e ao número de entradas a importar de uma só vez. Pode também definir a sintaxe utilizada para o conteúdo das entradas e dos comentários (HTML, Wiki ou outras, consoante as sintaxes atualmente instaladas e activadas).

Clique em Importar o meu blogue agora para iniciar o processo de importação.

Nota: as funções de exportação estão agora localizadas na páginaMaintenance .
