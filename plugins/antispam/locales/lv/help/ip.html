IP filtrs

Šis filtrs ļauj manuāli iekļaut melnajā vai baltajā sarakstā IP adreses. Komentārs, kas nāk no melnajā sarakstā norādītās IP adreses, nekavējoties tiks uzskatīts par nevēlamu. Komentārs, kas nāk no baltā saraksta sadaļā norādītas IP adreses, tiks uzskatīts par uzticamu, un citi pretsurogātpasta filtri to neapstrādās.

Lai pievienotu IP, vispirms atlasiet vajadzīgo cilni (Melnais saraksts vai Baltais saraksts) un ievadiet IP adresi. Izvēles rūtiņa Globālais IP ir pieejama, ja esat superadministrators, un tā ļaus jums pievienot IP adresi visiem jūsu Dotclear instalācijas emuāriem.

Varat pievienot IP adreses ar tīkla masku, lai bloķētu vai atļautu IP adrešu diapazonu. Piemēram:

  • 127.0.0.0/255.0.0.0 vai saīsinātā forma 127/8 atpazīs visas IP adreses, piemēram, 127.xxx. xxx.xxx
  • 127.0.0.0/255.255.0.0 vai saīsinātā forma 127.0/16 atpazīs visas IP adreses, piemēram, 127. xxx
  • 127.0.0.0/255.255.255.0 vai saīsinātā forma 127.0.0/24 atpazīs visas IP adreses, piemēram, 127.0.0.xxx
