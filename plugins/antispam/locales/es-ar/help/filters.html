Filtros

Rastreos justos
Este filtro comprueba si los trackbacks que se introducen contienen un enlace a tu entrada. Si no es el caso, serán automáticamente baneados, lo que significa que no aparecerán en la lista de comentarios basura.
Búsqueda de enlaces
Este filtro funciona un poco como IP Lookup excepto que en lugar de comprobar la dirección IP del autor, comprobará los enlaces contenidos en el comentario (incluido el sitio web del autor) aparecen en el sitio web de reputación URI SURBL. Este filtro no puede configurarse.
