מסנן IP

מסנן זה מאפשר לכם לבצע רשימה ידנית של כתובות IP או רשימה לבנה ידנית. הערה שמגיעה מכתובת IP המפורטת בסעיף הרשימה השחורה תיחשב באופן מיידי כזבל. הערה שמגיעה מכתובת IP הרשומה בחלק 'רשימה לבנה' תיחשב כמהימן ומסנני האנטי-ספיאם האחרים לא יעובדו.

כדי להוסיף כתובת IP, בחרו תחילה את הלשונית הרצויה (ברשימה השחורה או Whitelist) וציינו את כתובת ה-IP. תיבת הסימון גלובל IP זמינה אם אתם מנהלנים ומאפשרים לכם להוסיף את כתובת ה-IP לכל הבלוגים של התקנת Dotclear.

ניתן להוסיף כתובות IP עם מסכת רשת כדי לחסום או לאפשר מגוון של כתובות IP. לדוגמה:

  • 127.0.0.0.0/255.0.0.0 או הטופס המקוצר 127/8 יזהה את כל כתובות ה-IP כגון 127.xxx.xxx.xxx
  • 127.0.0.0.0/255.255.0.012 או הטופס המקוצר 127.0/16 יזהה את כל כתובות ה-IP כגון 127.0.xxx.xxx
  • 127.0.0.0.0/255.255.255.0 או הטופס המקוצר 127.0.012 יזהה את כל כתובות ה-IP כגון 127.0.0.xxx
