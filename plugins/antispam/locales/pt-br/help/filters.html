Filtros

Trackbacks justos
Este filtro verifica se os trackbacks introduzidos contêm um link para a sua entrada. Se não for esse o caso, serão automaticamente banidos, o que significa que não aparecerão na lista de comentários indesejados.
Pesquisa de ligações
Este filtro funciona um pouco como o IP Lookup , exceto que, em vez de verificar o endereço IP do autor, verifica se as ligações contidas no comentário (incluindo o sítio Web do autor) aparecem no URI reputation website SURBL. Este filtro não pode ser configurado.
