Anti-spam

O spam nos comentários e nos trackbacks passou de uma questão anedótica a um problema enorme. Hoje em dia, é obrigatório ter um conjunto de filtros para verificar se um comentário é potencialmente um lixo ou não. Este é o objetivo do plugin Antispam.

Informações
A ligação Junk comments leva-o para uma lista de todos os comentários não solicitados (nesta página, poderá executar acções em lote nos comentários não solicitados).
A ligação Comentários publicados leva-o para uma lista de todos os comentários publicados (poderá executar acções em lote nesta página).
O botão Eliminar todos os spams irá remover todas as mensagens de spam de uma só vez.
Filtros de spam disponíveis
Por defeito, o Dotclear vem com 6 filtros, incluindo 2 filtros desactivados. Pode adicionar mais filtros mais tarde.
Ordem: Os filtros são aplicados de cima para baixo. É possível modificar a ordem dos filtros. Tenha em atenção que alguns filtros predefinidos (IP Lookup, Links Lookup e Akismet) requerem acesso à Netowrk e, por isso, podem demorar algum tempo. É preferível deixá-los para o fim da lista. Além disso, o filtro IP Filter decide quais as mensagens que não são spam, pelo que é melhor deixá-lo no topo da lista.
Ativo: Marque a caixa para ativar um filtro e desmarque-a para o desativar.
Auto Del.: Se esta caixa estiver selecionada, as mensagens consideradas como lixo por este filtro serão automaticamente eliminadas sem qualquer hipótese de verificar se não se trata de um falso positivo.
Configuração de filtros: Alguns filtros podem (e, em alguns casos, devem) ser configurados para serem eficientes. Clique neste ícone para aceder à página de configuração do filtro.
