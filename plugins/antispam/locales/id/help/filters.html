Filter

Umpan Balik yang Adil
Filter ini memeriksa apakah trackback yang masuk berisi tautan ke entri Anda. Jika tidak demikian, komentar tersebut akan secara otomatis diblokir, yang berarti komentar tersebut tidak akan muncul dalam daftar komentar sampah.
Pencarian Tautan
Filter ini bekerja seperti IP Lookup kecuali bahwa alih-alih memeriksa alamat IP penulis, filter ini akan memeriksa tautan yang terdapat dalam komentar (termasuk situs web penulis) yang muncul di situs web reputasi URI SURBL. Filter ini tidak dapat dikonfigurasi.
