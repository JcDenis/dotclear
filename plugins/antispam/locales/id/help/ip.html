Filter IP

Filter ini memungkinkan Anda untuk membuat daftar hitam atau daftar putih alamat IP secara manual. Komentar yang berasal dari alamat IP yang terdaftar di bagian daftar hitam akan segera dianggap sebagai sampah. Komentar yang berasal dari alamat IP yang terdaftar di bagian daftar putih akan dianggap sebagai tepercaya dan filter antispam lainnya tidak akan memprosesnya.

Untuk menambahkan IP, pertama-tama pilih tab yang Anda inginkan (Daftar Hitam atau Daftar Putih) dan masukkan alamat IP. Kotak centang Global IP tersedia jika Anda adalah administrator super dan akan memungkinkan Anda untuk menambahkan alamat IP ke semua blog instalasi Dotclear Anda.

Anda dapat menambahkan alamat IP dengan mask jaringan untuk memblokir atau mengizinkan rentang alamat IP. Sebagai contoh:

  • 127.0.0.0/255.0.0.0 atau bentuk singkatnya 127/8 akan mengenali semua alamat IP seperti 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 atau bentuk singkatnya 127.0/16 akan mengenali semua alamat IP seperti 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0 atau bentuk singkatnya 127.0.0/24 akan mengenali semua alamat IP seperti 127.0.0.xxx
