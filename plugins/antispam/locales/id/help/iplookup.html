Filter Pencarian IP

Pencarian IP adalah filter yang memeriksa alamat IP penulis komentar di daftar bersama yang disebut Daftar Blackhole berbasis DNS (DNSBL) (juga dikenal sebagai Daftar Blackhole Real-time (RBL)). Secara default, filter ini menggunakan server berikut ini: sbl-xbl.spamhaus.org dan bsb.spamlookup.net. Anda bisa menemukan daftar server yang ada di halaman ini.

Anda bisa menambahkan sebanyak mungkin server yang Anda inginkan, dipisahkan dengan tanda koma. Jangan lupa untuk mengklik Simpan untuk menyimpan modifikasi Anda.
