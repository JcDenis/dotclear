Антиспам

Spam у коментарима и линкови за праћење је прешао из анегдотског проблема са масивним проблемом. У данашње време, обавезно је имати скуп филтера за проверу да ли је коментар потенцијално смеће или не. Ово је сврха прикључка Антиспама.

Информације
Junk коментари веза ће вас одвести на листу свих нежељених коментара (ћете моћи да покренете серије акција на нежељене коментаре на овој страници).
The Published comments link will take you to a list of all published comments (you will be able to run batch actions in this page).
Обриши све spam дугме уклања све спам поруке одједном.
Доступни spam филтери
Подразумевано, Дот долази са 6 филтера, укључујући 2 деактивирана филтера. Можете додати још касније.
Order: Filters are applied from top to bottom. You can modify the filters order. Note that some default filters (IP Lookup, Links Lookup and Akismet) require access to the Netowrk and therefore can take some time. It is better to leave them towards the end of the list. In addition, the IP Filter filter decides which messages are not spam, so it's better to leave it at the top of the list.
Active: Проверите оквир за активирање филтера, непроверите га да би га деактивирали.
Ауто Дел.: Ако је ова кућица попуњена, поруке које се сматрају отпадом од овог филтера биће аутоматски избрисане без икакве шансе да се уверите да није лажно позитиван.
филтер конфигурација: Неки филтери могу (и, у неким случајевима, морају) бити конфигурисани да буду ефикасни. Кликните на ову икону да бисте приступили страници конфигурације филтера.
