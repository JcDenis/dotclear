Filter för IP-sökning

IP Lookup är ett filter som kontrollerar IP-adressen för författaren till en kommentar på delade listor som kallas DNS-based Blackhole List (DNSBL) (även känd som Real-time Blackhole List (RBL)). Som standard använder filtret följande servrar: sbl-xbl.spamhaus.org och bsb.spamlookup.net. Du hittar en lista över befintliga servrar på på den här sidan.

Du kan lägga till så många servrar som du vill, åtskilda med kommatecken. Glöm inte att klicka på Spara för att ta hänsyn till dina ändringar.
