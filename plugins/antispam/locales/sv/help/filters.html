Filter

Fair Trackbacks
Detta filter kontrollerar om inkommande trackbacks innehåller en länk till din post. Om så inte är fallet kommer de automatiskt att förbjudas, vilket innebär att de inte kommer att visas i listan över skräpkommentarer.
Sök efter länkar
Detta filter fungerar lite som IP Lookup förutom att istället för att kontrollera författarens IP-adress, kommer det att kontrollera länkarna som finns i kommentaren (inklusive författarens webbplats) visas i URI rykte webbplats SURBL. Detta filter kan inte konfigureras.
