Filtr IP

Tento filtr umožňuje ručně vytvořit černou nebo bílou listinu IP adres. Komentář pocházející z IP adresy uvedené na černé listině bude okamžitě považován za nevyžádanou poštu. Komentář přicházející z IP adresy uvedené v sekci white list bude považován za důvěryhodný a ostatní antispamové filtry jej nebudou zpracovávat.

Chcete-li přidat IP adresu, nejprve vyberte požadovanou kartu (Blacklist nebo Whitelist) a zadejte IP adresu. Zaškrtávací políčko Global IP je k dispozici, pokud jste superadministrátor, a umožní vám přidat IP adresu do všech blogů vaší instalace Dotclear.

Můžete přidat IP adresy se síťovou maskou a blokovat nebo povolit rozsah IP adres. Například:

  • 127.0.0.0/255.0.0.0 nebo zkrácený tvar 127/8 rozpozná všechny IP adresy jako 127.xxx.xxx.xxx.
  • 127.0.0.0/255.255.0.0 nebo zkrácený tvar 127.0/16 rozpozná všechny IP adresy jako 127.0.xxx.xxx.
  • 127.0.0.0/255.255.255.0 nebo zkrácený tvar 127.0.0/24 rozpozná všechny IP adresy jako 127.0.0.xxx.
