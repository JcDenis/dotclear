Фильтр поиска по IP

IP Lookup - это фильтр, который проверяет IP-адрес автора комментария в общих списках, называемых DNS-на основе Blackhole List (DNSBL) (также известный как Real-time Blackhole List (RBL)). По умолчанию фильтр использует следующие серверы: sbl-xbl.spamhaus.org и bsb. Список существующих серверов pamlookup.net. Вы можете найти на этой странице

Вы можете добавить столько серверов, сколько хотите, разделенных запятыми. Не забудьте нажать на Сохранить чтобы учесть Ваши изменения.
