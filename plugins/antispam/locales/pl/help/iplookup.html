Filtr wyszukiwania adresów IP

IP Lookup to filtr, który sprawdza adres IP autora komentarza na współdzielonych listach o nazwie DNS-based Blackhole List (DNSBL) (znanej również jako Real-time Blackhole List (RBL)). Domyślnie filtr korzysta z następujących serwerów: sbl-xbl.spamhaus.org i bsb.spamlookup.net. Listę istniejących serwerów można znaleźć na tej stronie.

Możesz dodać dowolną liczbę serwerów, oddzielając je przecinkami. Nie zapomnij kliknąć na Save , aby uwzględnić wprowadzone zmiany.
