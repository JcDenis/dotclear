Filtr IP

Filtr ten umożliwia ręczne umieszczanie adresów IP na czarnej lub białej liście. Komentarz pochodzący z adresu IP znajdującego się na czarnej liście zostanie natychmiast uznany za spam. Komentarz pochodzący z adresu IP wymienionego w sekcji białej listy zostanie uznany za zaufany i inne filtry antyspamowe nie będą go przetwarzać.

Aby dodać adres IP, najpierw wybierz odpowiednią zakładkę (Czarna lista lub Biała lista) i wprowadź adres IP. Pole wyboru Global IP jest dostępne, jeśli jesteś super administratorem i pozwoli Ci dodać adres IP do wszystkich blogów Twojej instalacji Dotclear.

Można dodać adresy IP z maską sieci, aby zablokować lub zezwolić na zakres adresów IP. Na przykład:

  • 127.0.0.0/255.0.0.0 lub skrócona forma 127/8 rozpozna wszystkie adresy IP, takie jak 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 lub skrócona forma 127.0/16 rozpozna wszystkie adresy IP, takie jak 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0 lub skrócona forma 127.0.0/24 rozpozna wszystkie adresy IP, takie jak 127.0.0.xxx
