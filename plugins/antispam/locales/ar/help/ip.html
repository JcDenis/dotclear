فلتر IP

هذا الفلتر يسمح لك بالقائمة السوداء أو عناوين IP القائمة البيضاء يدويًا. التعليق القادم من عنوان IP الوارد في قسم القائمة السوداء سيعتبر فوراً غير مرغوب فيه. التعليق القادم من عنوان IP الوارد في قسم القائمة البيضاء سوف يعتبر موثوقا به و فلاتر مكافحة البريد المزعج الأخرى لن تعالجه.

لإضافة IP ، أولاً حدد علامة التبويب التي تريدها (القائمة السوداء أو القائمة البيضاءوأدخل عنوان IP مربع الاختيار العالمي للIP متاح إذا كنت المسؤول الممتاز وستسمح لك بإضافة عنوان IP إلى جميع مدونات تثبيت Dotclear.

يمكنك إضافة عناوين IP مع قناع الشبكة لمنع أو السماح بمجموعة من عناوين IP على سبيل المثال:

  • 127.0.0.0/255.0.0 أو النموذج المختصر 127/8 سوف يتعرف على جميع عناوين IP مثل 127.xxx.xxx.xxxx
  • 127.0.0.0/255.0.0 أو النموذج المختصر 127.0/16 سيتعرف على جميع عناوين IP مثل 127.0.xxxx.xxx
  • 127.0.0.0/255.255.255.0 أو النموذج المختصر 127.0.0/24 سوف يتعرف على جميع عناوين IP مثل 127.0.xxx
