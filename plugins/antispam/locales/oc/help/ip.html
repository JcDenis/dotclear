Filtre IP

Aqueste filtre vos permet de metre manualament las adreças IP en lista negra o blanca. Un comentari venent d'una adreça IP listada dins la seccion de la lista negra serà immediatament considerat coma de brossa. Un comentari venent d'una adreça IP listada dins la seccion de la lista blanca serà considerat coma de fisança e los autres filtres antispam lo tractaràn pas.

Per apondre una IP, seleccionatz d'en primièr l'onglet que volètz (Lista negra o Lista blanca) e picatz l'adreça IP. La bóstia de seleccion IP global es disponibla se sètz un super administrator e vos permetrà d'apondre l'adreça IP a totes los blògs de vòstra installacion Dotclear.

Podètz apondre d'adreças IP amb una masca de ret per blocar o autorizar una gama d'adreças IP. Per exemple:

  • 127.0.0.0/255.0.0.0 o la forma abreujada 127/8 reconeisserà totas las adreças IP coma 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 o la forma abreujada 127.0/16 reconeisserà totas las adreças IP coma 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0 o la forma abreujada 127.0.0/24 reconeisserà totas las adreças IP coma 127.0.0.xxx
