Filtres

Trackbacks justes
Aqueste filtre verifica se los trackbacks entrants contenon un ligam cap a vòstra entrada. S'es pas lo cas, seràn automaticament enebits, çò que significa qu'apareisseràn pas dins la lista dels comentaris bruts.
Recèrca de ligams
Aqueste filtre fonciona un pauc coma Recèrca IP levat qu'al luòc de verificar l'adreça IP de l'autor, verificarà los ligams contenguts dins lo comentari (inclusent lo sit web de l'autor) aparéisser dins lo URI reputation site SURBL. Aqueste filtre pòt pas èsser configurat.
