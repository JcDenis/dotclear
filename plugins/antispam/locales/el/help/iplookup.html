Φίλτρο αναζήτησης IP

Το IP Lookup είναι ένα φίλτρο που ελέγχει τη διεύθυνση IP του συντάκτη ενός σχολίου σε κοινόχρηστες λίστες που ονομάζονται DNS-based Blackhole List (DNSBL) (επίσης γνωστή ως Real-time Blackhole List (RBL)). Από προεπιλογή, το φίλτρο χρησιμοποιεί τους ακόλουθους διακομιστές: sbl-xbl.spamhaus.org και bsb.spamlookup.net. Μπορείτε να βρείτε μια λίστα με τους υπάρχοντες διακομιστές σε αυτή τη σελίδα.

Μπορείτε να προσθέσετε όσους διακομιστές θέλετε, διαχωρισμένους με κόμματα. Μην ξεχάσετε να κάνετε κλικ στο Αποθήκευση για να λάβετε υπόψη τις τροποποιήσεις σας.
