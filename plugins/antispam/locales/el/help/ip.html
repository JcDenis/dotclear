Φίλτρο IP

Αυτό το φίλτρο σας επιτρέπει να βάλετε χειροκίνητα διευθύνσεις IP σε μαύρη ή λευκή λίστα. Ένα σχόλιο που προέρχεται από μια διεύθυνση IP που περιλαμβάνεται στην ενότητα της μαύρης λίστας θα θεωρείται αμέσως ανεπιθύμητο. Ένα σχόλιο που προέρχεται από μια διεύθυνση IP που περιλαμβάνεται στην ενότητα της λευκής λίστας θα θεωρείται αξιόπιστο και τα άλλα φίλτρα antispam δεν θα το επεξεργάζονται.

Για να προσθέσετε μια IP, επιλέξτε πρώτα την καρτέλα που θέλετε (Μαύρη λίστα ή Λευκή λίστα) και εισαγάγετε τη διεύθυνση IP. Το πλαίσιο ελέγχου Global IP είναι διαθέσιμο εάν είστε υπερδιαχειριστής και θα σας επιτρέψει να προσθέσετε τη διεύθυνση IP σε όλα τα blogs της εγκατάστασης του Dotclear.

Μπορείτε να προσθέσετε διευθύνσεις IP με μάσκα δικτύου για να αποκλείσετε ή να επιτρέψετε ένα εύρος διευθύνσεων IP. Για παράδειγμα:

  • 127.0.0.0/255.0.0.0.0 ή η σύντομη μορφή 127/8 θα αναγνωρίσει όλες τις διευθύνσεις IP όπως 127.xxx.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0.0 ή η σύντομη μορφή 127.0/16 θα αναγνωρίσει όλες τις διευθύνσεις IP όπως 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.255.0 ή η σύντομη μορφή 127.0.0.0/24 θα αναγνωρίσει όλες τις διευθύνσεις IP όπως 127.0.0.0.xxx
