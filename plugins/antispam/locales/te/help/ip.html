IP ఫిల్టర్

ఈ ఫిల్టర్ మీరు మానసిక బ్లాక్లిస్ట్ లేదా whitelist IP చిరునామాను అనుమతిస్తుంది. బ్లాక్ జాబితా విభాగంలో జాబితా చేయబడిన ఒక IP చిరునామాల నుండి వచ్చే ఒక వ్యాఖ్యానం వెంటనే జున్నుగా పరిగణించబడుతుంది. తెలుపు జాబితా విభాగంలో జాబితా చేయబడిన ఒక IP చిరునామాల నుండి వచ్చే ఒక వ్యాఖ్యానం విశ్వసనీయంగా పరిగణించబడుతుంది మరియు ఇతర యాంటీస్పామ్ ఫిల్టర్లు దానిని ప్రక్రియ చేయదు.

ఒక IP ను జతచేయుటకు, మీరు కావలసిన టాబ్ ను మొదట ఎంచుకోండి (బ్లాక్జాబితా లేదా Whitelist) మరియు IP చిరునామాను ప్రవేశపెట్టండి. గ్లోబల్ IP చెక్ బాక్స్ ఉన్నట్లయితే మీరు ఒక సూపర్ పరిపాలకుడు ఉంటే మరియు మీ Dotclear స్థాపన యొక్క అన్ని బ్లాగులకు IP చిరునామాను జతచేయడానికి అనుమతిస్తుంది.

మీరు ఒక నెట్వర్క్ మాస్ట్తో IP చిరునామాను కలుపుకోవచ్చు లేదా IP చిరునామాల ఒక శ్రేణను అనుమతిస్తుంది. ఉదాహరణకు:

  • 127.0.0.0/255.0.0.0 లేదా సంప్రదించబడిన రూపాన్ని 127/8 వంటి అన్ని IP చిరునామాల వలె గుర్తించగలుగుతుంది 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 లేదా సంప్రదించబడిన రూపాన్ని 127.0/16 వంటి అన్ని IP చిరునామాల వలె గుర్తించగలుగుతుంది 127.0.xxx.xxx
  • 127.0.0.0.0/255.255.255.0 లేదా సంప్రదించబడిన రూపాన్ని 127.0.0/24 వంటి అన్ని IP చిరునామాల వలె గుర్తించగలుగుతుంది 127.0.0.xxx
