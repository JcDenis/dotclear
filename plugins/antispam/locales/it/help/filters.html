Filtri

Fair Trackback
Questo filtro controlla se inserire trackback contiene un link alla tua voce. Se non è così, saranno automaticamente banditi, il che significa che non appariranno nella lista dei commenti spazzatura.
Ricerca Link
Questo filtro funziona un po' come IP Lookup tranne che invece di controllare l'indirizzo IP dell'autore, controllerà i link contenuti nel commento (incluso il sito web dell'autore) appariranno nel sito web di reputazione URI SURBL. Questo filtro non può essere configurato.
