Filtro IP

Questo filtro consente di inserire manualmente gli indirizzi IP nella lista nera o nella whitelist. Un commento proveniente da un indirizzo IP elencato nella sezione della lista nera sarà immediatamente considerato spazzatura. Un commento proveniente da un indirizzo IP elencato nella sezione della lista bianca sarà considerato attendibile e gli altri filtri antispam non lo elaboreranno.

Per aggiungere un IP, prima selezionare la scheda che si desidera (Blacklist o Whitelist) e inserire l'indirizzo IP. La casella Global IP è disponibile se sei un super amministratore e ti permetterà di aggiungere l'indirizzo IP a tutti i blog della tua installazione Dotcleare.

È possibile aggiungere indirizzi IP con una maschera di rete per bloccare o consentire una serie di indirizzi IP. Per esempio:

  • 127.0.0.0/255.0.0.0 o il modulo abbreviato 127/8 riconoscerà tutti gli indirizzi IP come 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 o il modulo abbreviato 127.0/16 riconoscerà tutti gli indirizzi IP come 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0 o il modulo abbreviato 127.0.0/24 riconoscerà tutti gli indirizzi IP come 127.0.0.0.xxx
