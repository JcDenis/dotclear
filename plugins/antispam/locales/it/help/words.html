Parole Errate

Questo filtrerà i commenti in base alle parole usate nel nome dell'utente, indirizzo e-mail, URL del sito web o contenuto del commento.

Aggiungi una parola
Analogamente agli indirizzi IP che filtrano, basta aggiungere la parola che si desidera filtrare. Puoi anche, se hai i giusti permessi, definire parole globali che saranno usate per tutti i blog nella tua installazione Dotcleare.
Suggerimento: Puoi aggiungere un'espressione regolare alla tua lista di parole aggiungendo un / all'inizio e alla fine della parola. Ad esempio, /foo/ filtrerà tutte le parole contenenti foo.
Elenco delle parole errate
Elenco delle parole che elencano il commento come indesiderato.
Per rimuovere le parole da questo elenco, selezionare le parole e premere Elimina le parole selezionate.
