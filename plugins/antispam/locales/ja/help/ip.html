IP フィルタ

このフィルターでは、手動でブラックリストまたはホワイトリストの IP アドレスを指定できます。 ブラックリストセクションに記載されているIPアドレスからのコメントは、直ちにジャンクとみなされます。 ホワイトリストセクションに記載されているIPアドレスからのコメントは、信頼されているとみなされ、他のアンチスパムフィルタはそれを処理しません。

IPを追加するには、まずご希望のタブ（ブラックリスト または ホワイトリスト）を選択し、IPアドレスを入力してください。 Global IP のチェックボックスは、スーパー管理者であれば利用可能で、ドットクリアインストールの全てのブログにIPアドレスを追加することができます。

IP アドレスをネットワークマスクで追加すると、IP アドレスの範囲をブロックしたり許可したりすることができます。例えば:

  • 127.0.0.0/255.0.0.0 または省略形 127/8 は 127.xxx.xx.xxx のようなすべてのIPアドレスを認識します
  • 127.0.0.0/255.255.0.0 または省略形 127.0/16 は 127.0.0.xxx.xxのようなすべてのIPアドレスを認識します
  • 127.0.0.0/255.255.255.0 または省略形 127.0.0/24 は 127.0.0.xxx のようなすべてのIPアドレスを認識します
