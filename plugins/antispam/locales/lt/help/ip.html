IP filtras

Šis filtras leidžia rankiniu būdu sudaryti juodąjį arba baltąjį IP adresų sąrašą. Komentaras, gautas iš IP adreso, įtraukto į juodąjį sąrašą, iš karto bus laikomas nepageidaujamu. Komentaras, gautas iš IP adreso, įtraukto į baltojo sąrašo skiltį, bus laikomas patikimu ir kiti antispaudiniai filtrai jo neapdoros.

Norėdami pridėti IP adresą, pirmiausia pasirinkite norimą skirtuką (Juodasis sąrašas arba Baltasis sąrašas) ir įveskite IP adresą. Žymimasis langelis Global IP yra prieinamas, jei esate superadministratorius ir leis jums pridėti IP adresą į visus Dotclear diegimo tinklaraščius.

Galite pridėti IP adresus su tinklo kauke, kad blokuotumėte arba leistumėte IP adresų diapazoną. Pavyzdžiui:

  • 127.0.0.0.0/255.0.0.0.0 arba sutrumpinta forma 127/8 atpažins visus IP adresus, pvz., 127.xxx.xxx.xxx
  • 127.0.0.0.0/255.255.0.0.0 arba sutrumpinta forma 127.0/16 atpažins visus IP adresus, pvz., 127.0.xxx.xxx
  • 127.0.0.0.0/255.255.255.255.0 arba sutrumpinta forma 127.0.0/24 atpažins visus IP adresus, pvz., 127.0.0.0.xxx
