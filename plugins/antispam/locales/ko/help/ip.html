IP 필터

이 필터를 사용하면 IP 주소를 수동으로 블랙리스트 또는 화이트리스트에 추가할 수 있습니다. 블랙리스트 섹션에 나열된 IP 주소에서 보낸 댓글은 즉시 정크로 간주됩니다. 화이트리스트 섹션에 나열된 IP 주소에서 보낸 댓글은 신뢰할 수 있는 것으로 간주되며 다른 스팸 방지 필터에서 처리하지 않습니다.

IP를 추가하려면 먼저 원하는 탭(블랙리스트 또는 화이트리스트)을 선택한 후 IP 주소를 입력합니다. 글로벌 IP 확인란은 최고 관리자인 경우 사용할 수 있으며, 이 확인란을 선택하면 닷클리어 설치의 모든 블로그에 IP 주소를 추가할 수 있습니다.

네트워크 마스크로 IP 주소를 추가하여 다양한 IP 주소를 차단하거나 허용할 수 있습니다. 예를 들어

  • 127.0.0.0/255.0.0.0 또는 약식 형식 127/8 은 127.xxx.xxx.xxx와 같은 모든 IP 주소를 인식합니다.
  • 127.0.0.0/255.255.0.0 또는 약식 형식 127.0/16 은 127.0.xxx.xxx과 같은 모든 IP 주소를 인식합니다.
  • 127.0.0.0/255.255.255.0 또는 약식 형식 127.0.0/24 는 127.0.0.xxx와 같은 모든 IP 주소를 인식합니다.
