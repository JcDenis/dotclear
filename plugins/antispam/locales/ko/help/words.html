나쁜 단어

이렇게 하면 사용자 이름, 이메일 주소, 웹사이트 URL 또는 댓글 콘텐츠에 사용된 단어를 기준으로 댓글을 필터링합니다.

단어 추가
IP 주소 필터링과 마찬가지로 필터링하려는 단어를 추가하기만 하면 됩니다. 적절한 권한이 있는 경우 닷클리어 설치의 모든 블로그에 사용할 글로벌 단어를 정의할 수도 있습니다.
힌트: 단어의 시작과 끝에 / 을 추가하여 단어 목록에 정규식을 추가할 수 있습니다. 예를 들어 /foo/ 은 foo를 포함하는 모든 단어를 필터링합니다.
나쁜 단어 목록
댓글을 정크로 표시하는 단어 목록입니다.
이 목록에서 단어를 제거하려면 단어를 선택하고 선택한 단어 삭제를 누릅니다.
