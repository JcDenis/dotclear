IP filtar

Ovaj vam filtar omogućuje ručno stavljanje IP adresa na crnu ili bijelu listu. Komentar koji dolazi s IP adrese navedene u odjeljku crne liste odmah će se smatrati smećem. Komentar koji dolazi s IP adrese navedene u odjeljku bijele liste smatrat će se pouzdanim i drugi antispam filteri ga neće obraditi.

Da biste dodali IP, prvo odaberite karticu koju želite (Crna lista ili Bijela lista) i unesite IP adresu. Potvrdni okvir Global IP dostupan je ako ste super administrator i omogućit će vam dodavanje IP adrese svim blogovima vaše instalacije Dotclear.

Možete dodati IP adrese s mrežnom maskom kako biste blokirali ili dopustili niz IP adresa. Na primjer:

  • 127.0.0.0/255.0.0.0 ili skraćeni oblik 127/8 će prepoznati sve IP adrese poput 127.xxx. xxx.xxx
  • 127.0.0.0/255.255.0.0 ili skraćeni oblik 127.0/16 će prepoznati sve IP adrese poput 127.0.xxx. xxx
  • 127.0.0.0/255.255.255.0 ili skraćeni oblik 127.0.0/24 će prepoznati sve IP adrese poput 127.0. 0.xxx
