IP kereső szűrő

Az IP Lookup egy olyan szűrő, amely ellenőrzi a hozzászólás szerzőjének IP-címét a DNS-alapú Blackhole List (DNSBL) (más néven Real-time Blackhole List (RBL)) nevű megosztott listákon. Alapértelmezés szerint a szűrő a következő szervereket használja: sbl-xbl.spamhaus.org és bsb.spamlookup.net. A meglévő szerverek listáját a oldalon találja.

Annyi kiszolgálót adhat hozzá, amennyit csak akar, vesszővel elválasztva. Ne felejtsen el a Save gombra kattintani, hogy a módosításokat figyelembe vegyük.
