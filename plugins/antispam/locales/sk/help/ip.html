IP filter

Tento filter vám umožňuje manuálne pridať IP adresy na čiernu alebo bielu listinu. Komentár pochádzajúci z IP adresy uvedenej v sekcii čiernej listiny bude okamžite považovaný za nevyžiadanú poštu. Komentár pochádzajúci z IP adresy uvedenej v sekcii white list bude považovaný za dôveryhodný a ostatné antispamové filtre ho nespracujú.

Ak chcete pridať IP adresu, najprv vyberte požadovanú kartu (Blacklist alebo Whitelist) a zadajte IP adresu. Začiarkavacie políčko Global IP je dostupné, ak ste supersprávca a umožní vám pridať IP adresu do všetkých blogov vašej inštalácie Dotclear.

Môžete pridať adresy IP so sieťovou maskou a zablokovať alebo povoliť rozsah adries IP. Napríklad:

  • 127.0.0.0/255.0.0.0 alebo skrátená forma 127/8 rozpozná všetky adresy IP, napríklad 127.xxx. xxx.xxx
  • 127.0.0.0/255.255.0.0 alebo skrátená forma 127.0/16 rozpozná všetky adresy IP, ako napríklad 127.0. xxx
  • 127.0.0.0/255.255.255.0 alebo skrátená forma 127.0.0/24 rozpozná všetky adresy IP, napríklad 127.0.0.xxx
