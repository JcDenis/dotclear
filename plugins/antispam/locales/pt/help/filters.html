Filtros

Fair Trackbacks
Este filtro verifica se a entrada de trackbacks contém um link para a sua entrada. Se não for esse o caso, elas serão automaticamente proibidas, o que significa que não aparecerão na lista de comentários desnecessários.
Pesquisar Ligações
Este filtro funciona um pouco como uma aparência IP exceto que, em vez de verificar o endereço IP do autor, vai verificar os links contidos no comentário (incluindo o site do autor) que aparecem no site de reputação de URI SURBL. Este filtro não pode ser configurado.
