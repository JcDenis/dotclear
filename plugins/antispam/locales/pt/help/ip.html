Filtro de IP

Este filtro permite que você adicione manualmente endereços IP na lista negra ou na lista branca. Um comentário vindo de um endereço IP listado na seção de lista negra será considerado como lixo. Um comentário vindo de um endereço IP listado na seção Lista Branca será considerado confiável e os outros filtros antispam não o processarão.

Para adicionar um IP, primeiro selecione a aba que deseja (Lista negra ou Lista branca) e digite o endereço IP. A caixa de seleção do IP global está disponível se você é um super administrador e permitirá que você adicione o endereço IP a todos os blogs da sua instalação do DotPlay.

Você pode adicionar endereços IP com uma máscara de rede para bloquear ou permitir uma gama de endereços IP. Por exemplo:

  • 127.0.0.0/255.0.0.0 ou a forma abreviada 127/8 reconhecerão todos os endereços IP como 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 ou a forma abreviada 127.0/16 reconhecerá todos os endereços IP como 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.255.0 ou a forma abreviada 127.0.0/24 reconhecerá todos os endereços IP como 127.0.0.xxx
