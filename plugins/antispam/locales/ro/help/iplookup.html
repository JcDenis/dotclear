Filtru de căutare IP

IP Lookup este un filtru care verifică adresa IP a autorului unui comentariu pe listele partajate numite DNS-based Blackhole List (DNSBL) (cunoscută și sub numele de Real-time Blackhole List (RBL)). În mod implicit, filtrul utilizează următoarele servere: sbl-xbl.spamhaus.org și bsb.spamlookup.net. Puteți găsi o listă a serverelor existente pe această pagină.

Puteți adăuga oricâte servere doriți, separate prin virgule. Nu uitați să faceți clic pe Save pentru a lua în considerare modificările efectuate.
