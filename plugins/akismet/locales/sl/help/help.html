Filter Akismet

Akismet je posebej učinkovita storitev filtra neželene pošte, ki za delovanje potrebuje ključ API.

Če želite pridobiti ključ API, pojdite na stran za prijavo ključa API Akismet in izberite načrt, ki vam najbolj ustreza. Izpolniti boste morali obrazec. Ne pozabite zapisati svojega e-poštnega naslova, če želite prejeti API ključ. Če imate račun Wordpress, lahko svoje identifikatorje uporabite tudi za pridobitev ključa Akismet API. Če izberete paket »Osebno«, bo ključ brezplačen in ga boste lahko uporabljali na multiblogu.

Vnesite ključ API in shranite. Če je ključ prepoznan, bi morali dobiti naslednje sporočilo "Ključ API-ja preverjen". Vrnite se na seznam filtrov, da aktivirate Akismet, če še ni končan.
