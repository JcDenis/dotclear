Filtr Akismet

Akismet to szczególnie skuteczna usługa filtrowania spamu, która do działania wymaga klucza API.

Aby uzyskać klucz API, przejdź do strony rejestracji klucza API Akismet i wybierz plan, który najbardziej Ci odpowiada. Będziesz musiał wypełnić formularz. Nie zapomnij wpisać swojego adresu e-mail, aby otrzymać klucz API. Jeśli masz konto Wordpress, możesz również użyć swoich identyfikatorów, aby uzyskać klucz API Akismet. Jeśli wybierzesz plan "Personal", klucz będzie darmowy i będziesz mógł go używać na wielu blogach.

Wprowadź klucz API i zapisz. Jeśli klucz zostanie rozpoznany, powinien pojawić się następujący komunikat "Klucz API sprawdzony". Wróć do listy filtrów, aby aktywować Akismet, jeśli jeszcze tego nie zrobiłeś.
