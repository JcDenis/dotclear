Akismet filtrs

Akismet ir īpaši efektīvs surogātpasta filtra pakalpojums, kura darbībai nepieciešama API atslēga.

Lai iegūtu API atslēgu, dodieties uz Akismet API atslēgas pierakstīšanās lapu un izvēlieties sev vispiemērotāko plānu. Jums būs jāaizpilda veidlapa. Neaizmirstiet pierakstīt savu e-pasta adresi, lai saņemtu API atslēgu. Ja jums ir Wordpress konts, varat arī izmantot savus identifikatorus, lai iegūtu Akismet API atslēgu. Ja izvēlaties plānu "Personīgais", atslēga būs bez maksas un varēsiet to izmantot vairākos emuāros.

Ievadiet API atslēgu un saglabājiet. Ja atslēga tiek atpazīta, jums vajadzētu saņemt šādu ziņojumu "API atslēga pārbaudīta". Atgriezieties filtru sarakstā, lai aktivizētu Akismet, ja tas vēl nav izdarīts.
