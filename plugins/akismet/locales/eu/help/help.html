Akismet iragazkia

Akismet spam iragazki zerbitzu bereziki eraginkorra da, funtzionatzeko API gako bat behar duena.

API gakoa lortzeko, zoaz Akismet API gakoen erregistratzeko orrira eta aukeratu gehien egokitzen zaizun plana. Formulario bat bete beharko duzu. Ez ahaztu zure helbide elektronikoak idaztea API gakoa jasotzeko. Wordpress kontua baduzu, zure identifikatzaileak ere erabil ditzakezu Akismet API gakoa lortzeko. "Pertsonala" plana aukeratzen baduzu, giltza doan izango da eta multiblog batean erabili ahal izango duzu.

Sartu API gakoa eta gorde. Gakoa ezagutzen baduzu, "API gakoa egiaztatuta" mezua jaso beharko zenuke. Itzuli iragazkien zerrendara Akismet aktibatzeko, oraindik egin ez bada.
