అక్స్మెట్ ఫిల్టర్

అక్సెస్మెట్ (Akismet) ఒక తగిన స్పామ్ ఫిల్టర్ సేవను పనిచేయడానికి ఒక API కీ అవసరమవుతుంది.

API కీ పొందడానికి, అకిస్మ్యాట్ API కీ సంతకం పేజీ కు వెళ్ళి, మీరు అత్యధికంగా వసూలు చేయడానికి ప్రణాళిని ఎంచుకుంటారు. ఒక రూపాన్ని నింపుకోవాలి. API కీ అందుకునే క్రమంలో మీ మెయిల్ డౌన్ ను వ్రాయుటకు మర్చిపోవద్దు. మీరు ఒక వేర్డ్ప్రెస్ ఖాతా కలిగి ఉంటే, అక్సెస్మెట్ API కీ పొందడానికి మీ గుర్తింపు కూడా ఉపయోగించవచ్చు. "వ్యక్తిగత" ప్రణాళికాన్ని ఎంచుకుంటే, కీ ఉచితంగా ఉంటుంది మరియు మీరు ఒక మల్టీబ్లాగ్ మీద దానిని ఉపయోగించుకోవాలి.

API కీ ప్రవేశించి రక్షించుము. కీ గుర్తింపు పొందినట్లయితే, మీరు క్రింది సందేశం "API కీ పరిశీలించిన" పొందాలి. అక్సెస్మెట్ ను క్రియాశీలము చేయుటకు ఫిల్టర్ జాబితాలో తిరిగి వెళ్ళు
