Акизмет филтер

Акисмет је посебно ефикасна spam филтер услуга која захтева АПИ кључ за рад.

Да бисте добили API кључ, идите на АПИ кључ за АПИ кључ и изаберите план који вам највише одговара. Мораћете да попуните формулар. Не заборавите да запишете своје адресе е-поште како бисте примили API кључ. Ако имате налог за штампу, можете користити и идентификаторе да бисте добили API кључ за програмирање апликација AAIstember. Ако изаберете "Лични" план, кључ ће бити слободан и моћи ћете да га користите на мултиблогу.

Унесите тастер API и сачувајте. Ако је кључ препознат, требало би да добијете следећу поруку "АПИ кључ проверен". Вратите се на листу филтера да бисте активирали Акисмет ако то још увек није урађено.
