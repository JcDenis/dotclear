Akismet filter

Akismet je posebno učinkovit servis za filtriranje neželjene pošte koji zahtijeva API ključ za rad.

Da biste dobili API ključ, idite na stranicu za registraciju Akismet API ključa i odaberite plan koji vam najviše odgovara. Morat ćete ispuniti obrazac. Ne zaboravite zapisati svoju adresu e-pošte kako biste primili API ključ. Ako imate Wordpress račun, također možete koristiti svoje identifikatore za dobivanje Akismet API ključa. Ako odaberete "Osobni" plan, ključ će biti besplatan i moći ćete ga koristiti na multiblogu.

Unesite API ključ i spremite. Ako je ključ prepoznat, trebali biste dobiti sljedeću poruku "API ključ provjeren". Vratite se na popis filtara da aktivirate Akismet ako još nije gotovo.
