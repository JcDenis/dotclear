Filtre Akismet

Akismet és un servei de filtre de correu brossa especialment eficient que requereix una clau API per funcionar.

Per obtenir la clau API, aneu a la pàgina de registre de la clau API d'Akismet i trieu el pla que més us convingui. Haureu d'omplir un formulari. No oblideu anotar les vostres adreces de correu electrònic per rebre la clau API. Si teniu un compte de Wordpress, també podeu utilitzar els vostres identificadors per obtenir la clau API Akismet. Si trieu el pla "Personal", la clau serà gratuïta i la podreu utilitzar en un multiblog.

Introduïu la clau API i deseu. Si es reconeix la clau, hauríeu de rebre el missatge següent "Clau API comprovada". Torneu a la llista de filtres per activar Akismet si encara no s'ha fet.
