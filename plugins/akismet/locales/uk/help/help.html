Фільтр Akismet

Akismet - це особливо ефективний сервіс фільтрації спаму, для роботи якого потрібен ключ API.

Щоб отримати API-ключ, перейдіть за посиланням на сторінку реєстрації API-ключів Akismet та оберіть тарифний план, який вам найбільше підходить. Вам потрібно буде заповнити форму. Не забудьте вказати свої електронні адреси, щоб отримати ключ API. Якщо у вас є обліковий запис Wordpress, ви також можете використовувати свої ідентифікатори, щоб отримати ключ API Akismet. Якщо ви обираєте тарифний план "Персональний", ключ буде безкоштовним, і ви зможете використовувати його в мультиблозі.

Введіть ключ API та збережіть його. Якщо ключ буде розпізнано, ви повинні отримати наступне повідомлення "API-ключ перевірено". Поверніться до списку фільтрів, щоб активувати Akismet, якщо це ще не зроблено.
