Filtru Akismet

Akismet este un serviciu de filtrare a spam-ului deosebit de eficient, care necesită o cheie API pentru a funcționa.

Pentru a obține cheia API, accesați pagina de înregistrare a cheii API Akismet și alegeți planul care vi se potrivește cel mai bine. Va trebui să completați un formular. Nu uitați să vă scrieți adresa de e-mail pentru a primi cheia API. Dacă ai un cont Wordpress, poți folosi și identificatorii tăi pentru a obține cheia API Akismet. Dacă alegeți planul "Personal", cheia va fi gratuită și o veți putea folosi pe un multiblog.

Introduceți cheia API și salvați. Dacă cheia este recunoscută, ar trebui să primiți următorul mesaj: "API key checked" (Cheie API verificată). Reveniți la lista de filtre pentru a activa Akismet, dacă nu s-a făcut încă.
