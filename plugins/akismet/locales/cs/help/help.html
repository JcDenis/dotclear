Filtr Akismet

Akismet je obzvláště účinná služba filtrování spamu, která ke své činnosti vyžaduje klíč API.

Chcete-li získat klíč API, přejděte na na stránku pro registraci klíče API Akismet a vyberte si plán, který vám nejvíce vyhovuje. Budete muset vyplnit formulář. Nezapomeňte napsat svou e-mailovou adresu, abyste mohli obdržet klíč API. Pokud máte účet Wordpress, můžete k získání klíče Akismet API použít také své identifikátory. Pokud si vyberete plán "Personal", klíč bude zdarma a budete ho moci používat na více blozích.

Zadejte klíč API a uložte jej. Pokud je klíč rozpoznán, měla by se zobrazit následující zpráva "API key checked". Vraťte se do seznamu filtrů a aktivujte Akismet, pokud se tak ještě nestalo.
