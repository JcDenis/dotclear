Akismet filter

Akismet on eriti tõhus rämpspostifiltriteenus, mis vajab töötamiseks API-võtit.

API võtme saamiseks minge Akismeti API võtme registreerimise lehele ja valige teile kõige sobivam pakett. Peate täitma vormi. Ära unusta kirjutada oma e-posti aadressi, et saada API võti. Kui teil on Wordpressi konto, saate Akismet API võtme saamiseks kasutada ka oma identifikaatoreid. Kui valid "Personal" paketi, on võti tasuta ja saad seda kasutada mitme blogi puhul.

Sisestage API võti ja salvestage. Kui võti on tuvastatud, peaks teile ilmuma järgmine teade "API-võti kontrollitud". Mine tagasi filtrite loendisse, et aktiveerida Akismet, kui see pole veel tehtud.
