Φίλτρο Akismet

Το Akismet είναι μια ιδιαίτερα αποτελεσματική υπηρεσία φίλτρου spam που απαιτεί ένα κλειδί API για να λειτουργήσει.

Για να αποκτήσετε το κλειδί API, μεταβείτε στη σελίδα the Akismet API key signup page και επιλέξτε το πακέτο που σας ταιριάζει περισσότερο. Θα πρέπει να συμπληρώσετε μια φόρμα. Μην ξεχάσετε να γράψετε την ηλεκτρονική σας διεύθυνση για να λάβετε το κλειδί API. Αν έχετε λογαριασμό Wordpress, μπορείτε επίσης να χρησιμοποιήσετε τα αναγνωριστικά σας για να λάβετε το κλειδί Akismet API. Αν επιλέξετε το πακέτο "Personal", το κλειδί θα είναι δωρεάν και θα μπορείτε να το χρησιμοποιήσετε σε ένα multiblog.

Εισάγετε το κλειδί API και αποθηκεύστε. Εάν το κλειδί αναγνωριστεί, θα εμφανιστεί το ακόλουθο μήνυμα "API key checked". Επιστρέψτε στη λίστα φίλτρων για να ενεργοποιήσετε το Akismet αν δεν έχει γίνει ακόμα.
