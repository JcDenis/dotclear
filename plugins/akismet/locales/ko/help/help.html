아키스밋 필터

Akismet은 특히 효율적인 스팸 필터 서비스로, 작동하려면 API 키가 필요합니다.

API 키를 받으려면 Akismet API 키 가입 페이지 로 이동하여 가장 적합한 요금제를 선택하세요. 양식을 작성해야 합니다. API 키를 받으려면 이메일 주소를 적어두는 것을 잊지 마세요. 워드프레스 계정이 있는 경우 식별자를 사용하여 Akismet API 키를 받을 수도 있습니다. "개인" 요금제를 선택하면 키가 무료로 제공되며 멀티블로그에서 사용할 수 있습니다.

API 키를 입력하고 저장합니다. 키가 인식되면 "API 키 확인됨"이라는 메시지가 표시됩니다. 아직 완료되지 않은 경우 필터 목록으로 돌아가 Akismet을 활성화합니다.
