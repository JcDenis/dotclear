Филтър на Akismet

Akismet е особено ефективна услуга за филтриране на спам, която изисква API ключ, за да работи.

За да получите API ключа, отидете на страницата за регистрация на API ключа на Akismet и изберете най-подходящия за вас план. Ще трябва да попълните формуляр. Не забравяйте да запишете имейл адреса си, за да получите API ключа. Ако имате акаунт в Wordpress, можете да използвате и вашите идентификатори, за да получите API ключа на Akismet. Ако изберете плана "Персонален", ключът ще бъде безплатен и ще можете да го използвате в няколко блога.

Въведете API ключа и го запазете. Ако ключът е разпознат, трябва да получите следното съобщение "API key checked". Върнете се към списъка с филтри, за да активирате Akismet, ако това все още не е направено.
