Filter Akismet

Akismet adalah layanan filter spam yang sangat efisien yang membutuhkan kunci API untuk bekerja.

Untuk mendapatkan kunci API, kunjungi halaman pendaftaran kunci API Akismet dan pilih paket yang paling sesuai untuk Anda. Anda harus mengisi formulir. Jangan lupa untuk menuliskan alamat email Anda untuk menerima kunci API. Jika Anda memiliki akun Wordpress, Anda juga bisa menggunakan pengenal Anda untuk mendapatkan kunci API Akismet. Jika Anda memilih paket "Personal", kuncinya akan gratis dan Anda akan dapat menggunakannya pada multiblog.

Masukkan kunci API dan simpan. Jika kunci dikenali, Anda akan mendapatkan pesan berikut "Kunci API diperiksa". Kembali ke daftar filter untuk mengaktifkan Akismet jika belum selesai.
