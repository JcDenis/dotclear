Filtre Akismet

Akismet es un servici de filtratge de spam particularament eficient que demanda una clau API per foncionar.

Per obténer la clau API, anatz sus la pagina d'inscripcion de la clau API Akismet e causissètz lo plan que vos conven lo mai. Vos caldrà emplenar un formulari. Oblidetz pas d'escriure vòstra adreça de corrièl per fin de recebre la clau API. S'avètz un compte Wordpress, podètz tanben utilizar vòstres identificants per obténer la clau API Akismet. Se causissètz lo plan "Personal", la clau serà gratuita e la poiretz utilizar sus un multiblòg.

Picatz la clau API e enregistratz. Se la clau es reconeguda, deuriatz recebre lo messatge seguent "La clau API verificada". Tornatz a la lista de filtres per activar Akismet s'es pas encara fach.
