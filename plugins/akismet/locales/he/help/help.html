מסנן Akismet

Akismet הוא שירות מסנן דואר יעיל במיוחד שדורש מפתח API לעבודה.

כדי לקבל את מפתח ה-API, עברו ל- את עמוד ה-API של ה-API של Akisפגש את עמוד ובחרו את התוכנית המתאימה לכם ביותר. אתה תהיה חייב למלא טופס. אל תשכחו לרשום את כתובות הדואל שלכם כדי לקבל את מפתח ה-API. אם יש לכם חשבון פעיל, תוכלו גם להשתמש בזיהויים שלכם כדי לקבל את מפתח ה-API של Akismet. אם תבחרו בתוכנית "אישי", המפתח יהיה חופשי, ואתם תוכלו להשתמש בו בבלוג.

ציינו את מפתח ה-API ושמרו. אם המפתח מזוהה, עליכם לקבל את ההודעה שלהלן "בדיקת מפתח API". חזרו לרשימת הסינון כדי להפעיל את Akismet אם הוא עדיין לא נעשה.
