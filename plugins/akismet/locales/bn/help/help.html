Akismet ফিল্টার

Akismet একটি বিশেষ দক্ষ স্প্যাম ফিল্টার সার্ভিস যা কাজ করার জন্য একটি এপিআই কী প্রয়োজন.

API কী পেতে, Akismet API কী সাইনআপ পৃষ্ঠাতে যান এবং আপনার জন্য সবচেয়ে উপযুক্ত পরিকল্পনাটি বেছে নিন। আপনাকে একটি ফর্ম পূরণ করতে হবে। API কী পাওয়ার জন্য আপনার ইমেল ঠিকানা লিখতে ভুলবেন না। আপনার যদি একটি Wordpress অ্যাকাউন্ট থাকে, তাহলে আপনি Akismet API কী পেতে আপনার শনাক্তকারী ব্যবহার করতে পারেন। আপনি যদি "ব্যক্তিগত" পরিকল্পনা চয়ন করেন, তাহলে কীটি বিনামূল্যে হবে এবং আপনি এটি একটি মাল্টিব্লগে ব্যবহার করতে সক্ষম হবেন৷

API কী-টি লিখুন এবং সংরক্ষণ করুন. কী (key) স্বীকৃত হলে, আপনি নিম্নলিখিত বার্তাটি "এপিআই কী চেক করা" পেতে হবে. আকিসিত সক্রিয় করার জন্য ফিল্টার লিস্টে ফিরে যাও যদি তা না করা হয়.
