Фильтр Akismet

Akismet является особенно эффективным сервисом спам-фильтра, который требует API ключ для работы.

Чтобы получить API ключ, перейдите на страницу регистрации ключа Akismet API и выберите план, который подходит вам больше всего. Вы должны заполнить форму. Не забудьте записать свой адрес электронной почты для получения ключа API. Если у вас есть аккаунт Wordpress, вы также можете использовать идентификаторы для получения Akismet API ключ. Если вы выберете план "Личный", ключ будет свободным, и вы сможете использовать его в мультиблоге.

Введите ключ API и сохраните. Если ключ распознан, вы должны получить следующее сообщение "API key checked". Вернитесь к списку фильтров, чтобы активировать Akismet, если он еще не готов.
