Benutzer:Einstellungen

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Der Parameter nodragdrop kann beispielsweise auf der Registerkarte Meine Optionen der Seite Meine Einstellungen konfiguriert werden.

Die Seite hat zwei Registerkarten: Benutzereinstellungen und Globale Einstellungen zur Verwaltung der aktuellen Benutzereinstellungen bzw. der gemeinsamen Einstellungen für alle Benutzer.

Über das Dropdown-Menü Springen können Sie direkt auf die verschiedenen Seitenbereiche zugreifen.
