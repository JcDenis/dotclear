Пользователь: предпочтения

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Например, параметр nodragdrop может быть настроен на вкладке Мои опции Мои предпочтения страницы.

Страница имеет две вкладки: Настройки пользователя и Глобальные настройки для управления текущими предпочтениями пользователя и предпочтениями, общими для всех пользователей, соответственно.

Вы можете получить прямой доступ к различным разделам страницы, используя выпадающее меню Goto.
