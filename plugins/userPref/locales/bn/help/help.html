ব্যবহারকারী: পছন্দসমূহ

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

For instance, the nodragdrop parameter can be configured in the My options tab of the My preferences page.

The page has two tabs: User preferences and Global preferences to manage the current user preferences and the preferences common to every users, respectively.
