Pengguna:preferensi

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Misalnya, parameter nodragdrop dapat dikonfigurasi di tab Pilihan saya pada halaman Preferensi saya .

Halaman ini memiliki dua tab: Preferensi pengguna dan Preferensi global untuk mengelola preferensi pengguna saat ini dan preferensi yang umum bagi setiap pengguna.

Anda dapat langsung mengakses bagian halaman yang berbeda dengan menggunakan menu tarik-turun Goto .
