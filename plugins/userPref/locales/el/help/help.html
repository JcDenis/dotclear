Χρήστης:προτιμήσεις

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Για παράδειγμα, η παράμετρος nodragdrop μπορεί να ρυθμιστεί στην καρτέλα Οι επιλογές μου της σελίδας Οι προτιμήσεις μου .

Η σελίδα έχει δύο καρτέλες: Προτιμήσεις χρήστη και Παγκόσμιες προτιμήσεις για τη διαχείριση των τρεχουσών προτιμήσεων του χρήστη και των προτιμήσεων που είναι κοινές για όλους τους χρήστες, αντίστοιχα.

Μπορείτε να έχετε άμεση πρόσβαση στις διάφορες ενότητες της σελίδας χρησιμοποιώντας το αναπτυσσόμενο μενού Goto .
