Usuário:preferências

This page provides technical informations about parameters usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Por exemplo, o parâmetro nodragdrop pode ser configurado no separador As minhas opções da página As minhas preferências .

A página tem duas guias: Preferências do usuário e Preferências globais para gerenciar as preferências do usuário atual e as preferências comuns a todos os usuários, respectivamente.

Você pode acessar diretamente as diferentes seções de página usando o menu suspenso Ir.
