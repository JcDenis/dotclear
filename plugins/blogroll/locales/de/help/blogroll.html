Blogroll

Mit der Blogroll können Sie in Ihrem Blog eine Liste von Links zu anderen Blogs oder Websites anzeigen, die Sie Ihren Besuchern empfehlen möchten.

Blogroll
Your links and categories are displayed here. Categories are displayed in bold. You can reorder this list by dragging its items upwards or downwards. Once done, click on Save order. The order chosen here will be applied to links displayed on your blog with the Blogroll widget.
Click on a link or category name to edit it.
Ausgewählte Links löschen
Ermöglicht das Löschen von Links oder Kategorien, indem Sie die Kästchen neben den einzelnen Kategorien ankreuzen.

Einen Link hinzufügen

  • Titel: Titel des Links. Z.B.: Jules und Julie. Dieses Feld ist obligatorisch.
  • URL: Link-URL. Sie muss mit http:// oder https://beginnen. Dieses Feld ist obligatorisch.
  • Beschreibung: Kurze Beschreibung für den Link (wird in einem Tooltip angezeigt). Z.B.: Jules und Julie's Weltreise.
  • Sprache: Geben Sie hier einen zweistelligen Code für die Sprache der Website ein (z. B. en für Englisch).

Eine Kategorie hinzufügen

Links categories allow you to classify links displayed by the Blogroll widget.
They are displayed in bold characters. Enter the name of the category you would like to create and press Save.

Links importieren

Mit dieser Option können Sie Links aus einer OPML- (RSS-Feed-Format) oder XBEL-Datei (Format zum Teilen von Lesezeichen) importieren.
