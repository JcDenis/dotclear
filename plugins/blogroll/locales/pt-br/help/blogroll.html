Blogue

O Blogroll permite-lhe apresentar no seu blogue uma lista de ligações para outros blogues ou sítios Web que deseja sugerir aos seus visitantes.

Blogue
As suas ligações e categorias são apresentadas aqui. As categorias são apresentadas a negrito. Pode reordenar esta lista arrastando os itens para cima ou para baixo. Uma vez terminado, clique em Guardar ordem. A ordem escolhida aqui será aplicada aos links apresentados no seu blogue com o widget Blogroll.
Clique no nome de um link ou de uma categoria para o editar.
Eliminar ligações seleccionadas
Permite-lhe eliminar ligações ou categorias marcando as caixas junto a cada uma delas.

Anexar um link

  • Título: Título da ligação. Por exemplo: Jules e Julie. Este campo é obrigatório.
  • URL: URL da ligação. Deve começar por http:// ou https://. Este campo é obrigatório.
  • Descrição: Descrição curta para o link (será apresentada numa dica de ferramenta). Por exemplo: Viagem ao mundo de Jules e Julie.
  • Língua: Escreva aqui um código de dois caracteres para o idioma do sítio Web (por exemplo, en para inglês).

Adicionar uma categoria

As categorias de links permitem-lhe classificar os links apresentados pelo widget Blogroll.
São apresentadas em caracteres a negrito. Introduza o nome da categoria que pretende criar e prima Guardar.

Ligações de importação

Esta opção permite-lhe importar ligações de um ficheiro OPML (formato de feed RSS) ou XBEL (formato para partilhar marcadores).
