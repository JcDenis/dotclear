Bloco

Blogroll permite que você exiba em seu blog uma lista de links para outros blogs ou sites que você deseja sugerir aos seus visitantes.

Bloco
Your links and categories are displayed here. Categories are displayed in bold. You can reorder this list by dragging its items upwards or downwards. Once done, click on Save order. The order chosen here will be applied to links displayed on your blog with the Blogroll widget.
Click on a link or category name to edit it.
Suprimir os links seleccionados
Permite excluir links ou categorias, marcando as caixas ao lado de cada uma delas.

Adicionar um link

  • Título: Link de título. Por exemplo: Jules and Julie. Este campo é obrigatório.
  • URL: Link de URL. Deve começar com http:// ou https://. Este campo é obrigatório.
  • Descrição: Descrição curta para o link (será exibida na dica de ferramentas). Por exemplo: Jules e Julie World trip.
  • Idioma: Digite aqui um código de dois caracteres para o idioma do site (por exemplo, en para inglês).

Criar uma categoria

Links categories allow you to classify links displayed by the Blogroll widget.
They are displayed in bold characters. Enter the name of the category you would like to create and press Save.

Importar links

Esta opção permite importar links de um OPML (formato de feed RSS) ou XBEL (formato para compartilhar favoritos).
