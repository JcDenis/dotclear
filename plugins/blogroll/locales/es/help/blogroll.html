Enlaces

Blogroll le permite mostrar en su blog una lista de enlaces a otros blogs o sitios web que desee sugerir a sus visitantes.

Enlaces
Aquí se muestran sus enlaces y categorías. Las categorías aparecen en negrita. Puede reordenar esta lista arrastrando sus elementos hacia arriba o hacia abajo. Una vez hecho esto, haga clic en Guardar orden. El orden elegido aquí se aplicará a los enlaces mostrados en su blog con el widget Blogroll.
Haga clic en el nombre de un enlace o categoría para editarlo.
Borrar los enlaces seleccionados
Permite eliminar enlaces o categorías marcando las casillas situadas junto a cada uno de ellos.

Añadir un enlace

  • Título: Título del enlace. Ej: Jules y Julie. Este campo es obligatorio.
  • URL: URL del enlace. Debe comenzar por http:// o https://. Este campo es obligatorio.
  • Descripción: Breve descripción para el enlace (se mostrará en un tooltip). Por ejemplo El viaje de Jules y Julie por el mundo.
  • Idioma: Escriba aquí un código de dos caracteres para el idioma del sitio web (por ejemplo, en para el inglés).

Añadir una categoría

Las categorías de enlaces le permiten clasificar los enlaces mostrados por el widget Blogroll.
Se muestran en negrita. Introduzca el nombre de la categoría que desea crear y pulse Guardar.

Importar los enlaces

Esta opción le permite importar enlaces desde un archivo OPML (formato de fuente RSS) o XBEL (formato para compartir marcadores).
