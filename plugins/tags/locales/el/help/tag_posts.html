Διαθέσιμες ενέργειες σε μια ετικέτα

Το πρώτο μέρος της σελίδας σας επιτρέπει να τροποποιήσετε ή να διαγράψετε μια ετικέτα.

Μετονομασία
Εισάγετε μια νέα λέξη για την ετικέτα και κάντε κλικ στο OK. Αυτό θα τροποποιήσει όλες τις καταχωρήσεις που περιέχουν αυτή την ετικέτα.
Διαγράψτε αυτή την ετικέτα
Αυτό το κουμπί θα αφαιρέσει την ετικέτα από όλες τις καταχωρήσεις στις οποίες έχει εκχωρηθεί και θα τη διαγράψει από τη λίστα ετικετών.

Λίστα καταχωρήσεων με αυτή την ετικέτα

Από αυτή τη λίστα μπορείτε να εκτελέσετε όλες τις ενέργειες δέσμης που σας επιτρέπουν τα δικαιώματα του χρήστη σας:

  • Δημοσίευση: Δημοσίευση των επιλεγμένων καταχωρήσεων στο διαδίκτυο.
  • Δημοσίευση: Δημοσίευση των επιλεγμένων καταχωρήσεων στο διαδίκτυο.
  • Πρόγραμμα: Χρονοδιάγραμμα: Προγραμματίστε τις επιλεγμένες καταχωρήσεις να τεθούν σε απευθείας σύνδεση κατά την ημερομηνία δημοσίευσης.
  • Χαρακτηρίστε ως εκκρεμή: Σημειώστε επιλεγμένες καταχωρήσεις ως εκκρεμείς.
  • Σημειώστε ως επιλεγμένο/αποεπιλεγμένο: Επιλεγμένες/αποεπιλεγμένες./li>
  • Αλλαγή κατηγορίας: Τροποποίηση της κατηγορίας των επιλεγμένων καταχωρήσεων.
  • Αλλαγή γλώσσας: Τροποποίηση της γλώσσας των επιλεγμένων καταχωρήσεων.
  • Αλλαγή συγγραφέα: Τροποποιήστε τον συγγραφέα των επιλεγμένων καταχωρήσεων εισάγοντας το νέο αναγνωριστικό του συγγραφέα.
  • Διαγραφή: Διαγραφή: Διαγραφή των επιλεγμένων καταχωρήσεων (η λειτουργία αυτή δεν μπορεί να ακυρωθεί).
  • Προσθήκη/αφαίρεση ετικετών: Προσθήκη ή αφαίρεση ετικετών στις επιλεγμένες καταχωρήσεις.

Η παραπάνω λίστα ενδέχεται να περιέχει και άλλα στοιχεία, εάν έχουν προστεθεί πρόσθετες ενέργειες από πρόσθετα που είναι ενεργά στο σύστημά σας.
