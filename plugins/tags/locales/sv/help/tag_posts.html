Tillgängliga åtgärder för en tagg

Den första delen av sidan ger dig möjlighet att ändra eller ta bort en tagg.

Byt namn
Ange ett nytt ord för taggen och klicka på OK. Detta kommer att ändra alla poster som innehåller denna tagg.
Ta bort denna tagg
Den här knappen tar bort taggen från alla poster som den har tilldelats och tar bort den från tagglistan.

Lista över poster med denna tagg

Från den här listan kan du utföra alla batchåtgärder som din användares behörigheter tillåter dig att göra:

  • Publicera: Publicera de valda posterna online.
  • Avpublicera: Dra de valda posterna offline.
  • Schemalägg: Planera att de valda posterna ska läggas online vid publiceringsdatumet.
  • Markera som väntande: Markera valda poster som väntande.
  • Markera som markerad/ej markerad: Ange de markerade posterna som markerade/icke markerade./li>
  • Ändra kategori: Ändra kategori för de valda posterna.
  • Ändra språk: Ändra språket för de valda posterna.
  • Ändra författare: Ändra den valda postens författare genom att ange den nya författarens ID.
  • Radera: Ta bort de valda posterna (denna åtgärd kan inte avbrytas).
  • Lägg till/ta bort taggar: Lägg till eller ta bort taggar på de markerade posterna.

Ovanstående lista kan innehålla andra punkter om ytterligare åtgärder har lagts till av plugins som är aktiva på ditt system.
