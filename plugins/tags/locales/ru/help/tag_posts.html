Доступные действия на теге

Первая часть страницы позволяет изменять или удалять тег.

Переименовать
Введите новое слово для тега и нажмите OK. Это изменит все записи, содержащие этот тег.
Удалить этот тег
Эта кнопка удалит тег из всех записей, которым он присвоен и удалит его из списка тегов.

Список записей с этим тегом

Из этого списка вы можете выполнить все пакетные действия, разрешенные вашим пользователем:

  • Опубликовать: Опубликовать выбранные записи онлайн.
  • Снять публикацию: Потяните выбранные записи оффлайн.
  • Расписание: Расписание выбранных записей, которые будут заданы онлайн на дату публикации.
  • Отметить в ожидании: отметить выбранные записи в ожидании.
  • Отметить как выбранные/невыбранные: Установить выбранные записи как выбранные/невыбранные./li>
  • Изменить категорию: Изменить категорию выбранных записей.
  • Изменить язык: Изменить язык выбранных записей.
  • Автор изменения: Изменить автора выбранных записей, введя новый авторский ID.
  • Удалить: Удалить выбранные записи (эту операцию нельзя отменить).
  • Добавить/Удалить теги: Добавить или удалить теги из выбранных записей.

Вышеприведенный список может содержать другие элементы, если дополнительные действия были добавлены плагинами, активными в системе.
