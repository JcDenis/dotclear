Etiquetas

Você pode atribuir uma etiqueta a uma entrada:

  • Preenchendo o campo Marcadores . Este campo tem uma função de preenchimento automático : insira algumas letras e o sistema irá propor tags existentes que contenham estas letras;
  • Clicando na Escolha da lista. Link para selecionar as tags existentes.

As tags têm de ser separadas por vírgulas.
