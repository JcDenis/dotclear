Ações disponíveis em uma etiqueta

A primeira parte da página permite modificar ou excluir uma tag.

Renomear
Digite uma nova palavra para a tag e clique em OK. Isto irá modificar todas as referências que contém esta tag.
Excluir essa tag
Este botão irá remover a tag de todos os itens a que é atribuído e irá excluí-la da lista de tags.

Lista de referências com esta tag

A partir dessa lista, você pode executar todas as ações em lote que as permissões do seu usuário permitem:

  • Publicar: Publique as entradas selecionadas online.
  • Despublicar: Puxe as entradas selecionadas offline.
  • Agendar: Agendar as entradas selecionadas para serem definidas na data de publicação.
  • Marcar como pendente: Marcar as entradas selecionadas como pendentes.
  • Marcar como selecionado/não selecionado: Definir as entradas selecionadas como selecionadas/não selecionadas./li>
  • Alterar a categoria: Modificar a categoria das entradas selecionadas.
  • Mudar idioma: Modifica o idioma das entradas selecionadas.
  • Mudar o autor: Modificar o autor das postagens selecionadas informando o ID do novo autor.
  • Excluir: Excluir as entradas selecionadas (esta operação não pode ser cancelada).
  • Adicionar/Remover tags: Adicionar ou remover tags nas postagens selecionadas.

A lista acima pode conter outros itens se ações adicionais foram adicionadas por plugins ativos no seu sistema.
