Etiquetas

Puede asignar una etiqueta a una entrada:

  • Rellenando el campo Etiquetas. Este campo dispone de una función de autocompletado : introduzca algunas letras y el sistema le propondrá las etiquetas existentes que contengan esas letras;
  • Haciendo clic en Elija de la lista. Enlace para seleccionar las etiquetas existentes.

Las etiquetas deben estar separadas por comas.
