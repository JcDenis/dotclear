Acções disponíveis numa etiqueta

A primeira parte da página permite-lhe modificar ou apagar uma etiqueta.

Mudar o nome
Introduza uma nova palavra para a etiqueta e clique em OK. Isto modificará todas as entradas que contêm esta etiqueta.
Eliminar esta etiqueta
Este botão remove a etiqueta de todas as entradas a que está atribuída e elimina-a da lista de etiquetas.

Lista de entradas com esta etiqueta

A partir desta lista, pode executar todas as acções em lote que as permissões do seu utilizador lhe permitem:

  • Publicar: Publicar as entradas seleccionadas online.
  • Cancelar publicação: Retira as entradas seleccionadas para offline.
  • Programar: Programar as entradas seleccionadas para serem colocadas online na data de publicação.
  • Marcar como pendente: Marcar as entradas seleccionadas como pendentes.
  • Marcar como selecionado/não selecionado: Definir as entradas seleccionadas como seleccionadas/não seleccionadas./li>
  • Alterar categoria: Modificar a categoria dos registos seleccionados.
  • Alterar a língua: Modificar o idioma das entradas seleccionadas.
  • Alterar autor: Modificar o autor das entradas seleccionadas, introduzindo o ID do novo autor.
  • Apagar: Eliminar os registos seleccionados (esta operação não pode ser cancelada).
  • Adicionar/Remover etiquetas: Adicionar ou remover etiquetas nas entradas seleccionadas.

A lista acima pode conter outros itens se tiverem sido acrescentadas acções adicionais por plugins activos no seu sistema.
