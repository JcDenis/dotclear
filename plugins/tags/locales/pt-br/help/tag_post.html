Etiquetas

Pode atribuir uma etiqueta a uma entrada:

  • Preenchendo o campo Tags . Este campo tem uma função de autocompletar : introduza algumas letras e o sistema proporá etiquetas existentes que contenham essas letras;
  • Clicando em Escolher da lista. Ligação para selecionar etiquetas existentes.

As etiquetas têm de ser separadas por vírgulas.
