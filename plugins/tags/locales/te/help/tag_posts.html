ఒక ట్యాగ్పై అందుబాటులో చర్యలు

పేజీ యొక్క మొదటి భాగాన్ని మీరు ఒక ట్యాగ్ను సవరించటానికి లేదా తొలగించటానికి బదులుతుంది.

పునఃనామము
ట్యాగ్కు ఒక కొత్త పదాన్ని ప్రవేశపెట్టండి మరియు సరైనసరైన క్లిక్ చేయుము. @ action: button
ఈ ట్యాగ్ను తొలగించు
This button will remove the tag from all the entries it is assigned to and will delete it from the tags list.

ఈ ట్యాగ్తో ఎంట్రీల జాబితా

ఈ జాబితా నుండి మీరు మీ వినియోగదారు అనుమతులను మీరు అన్ని బ్యాచ్ చర్యలు చేయవచ్చు:

  • ప్రచురణ: ఎంపికచేసిన ఎంట్రీలను ఆన్లైన్లో ప్రచురిస్తుంది.
  • అప్రచురణ: ఎంపికచేసిన ఎంట్రీలను ఆఫ్లైన్.
  • షెడ్యూల్డ్: షెడ్యూలు ఎంపికచేసిన ఎంట్రీలను ప్రచురణ తేదీ వద్ద ఆన్లైన్లో అమలు చేయాలి.
  • Mark as pending: Mark selected entries as pending.
  • Mark as selected/unselected: Set the selected entries as selected/unselected ./li>
  • మార్పిడి వర్గం: ఎంపిక చేసిన ఎంట్రీలను మార్చుకోండి.
  • మార్పిడి భాష: ఎంపికచేసిన ఎంట్రీల భాషను సవరించండి.
  • మారుతి రచయిత: క్రొత్త రచయిత యొక్క ID ప్రవేశపెట్టడం ద్వారా ఎంపికచేసిన ఎంట్రీలను మార్చుకోండి.
  • తొలగించండి: ఎంపికచేసిన ఎంట్రీలను తొలగించుము (ఈ ఆపరేషన్ రద్దు చేయలేము).
  • Add/Remove tags: Add or remove tags on the selected entries.

అదనపు చర్యలు ప్లగిన్లు మీ సిస్టం మీద క్రియాశీలంగా జోడించినట్లయితే పై జాబితా ఇతర అంశాలను కలిగివుంటుంది.
