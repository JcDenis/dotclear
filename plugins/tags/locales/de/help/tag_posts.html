Verfügbare Aktionen für ein Tag

Der erste Teil der Seite ermöglicht es Ihnen, ein Tag zu ändern oder zu löschen.

Umbenennen
Geben Sie ein neues Wort für die Markierung ein und klicken Sie auf OK. Dadurch werden alle Einträge, die diese Markierung enthalten, geändert.
Dieses Tag löschen
Mit dieser Schaltfläche wird die Markierung aus allen Einträgen entfernt, denen sie zugewiesen ist, und aus der Markierungsliste gelöscht.

Liste der Einträge mit diesem Tag

Von dieser Liste aus können Sie alle Batch-Aktionen durchführen, die Ihre Benutzerberechtigung zulässt:

  • Veröffentlichen: Veröffentlichen Sie die ausgewählten Einträge online.
  • Unpublish: Ziehen Sie die ausgewählten Einträge offline.
  • Zeitplan: Planen Sie die ausgewählten Einträge so, dass sie zum Veröffentlichungsdatum online gestellt werden.
  • Als ausstehend markieren: Ausgewählte Einträge als ausstehend markieren.
  • Als ausgewählt/abgewählt markieren: Ausgewählte Einträge als ausgewählt/unausgewählt markieren./li>
  • Kategorie ändern: Ändern Sie die Kategorie der ausgewählten Einträge.
  • Sprache ändern: Ändern Sie die Sprache der ausgewählten Einträge.
  • Autor ändern: Ändern Sie den Autor der ausgewählten Einträge, indem Sie die ID des neuen Autors eingeben.
  • Löschen: Löschen Sie die ausgewählten Einträge (dieser Vorgang kann nicht abgebrochen werden).
  • Tags hinzufügen/entfernen: Hinzufügen oder Entfernen von Tags zu den ausgewählten Einträgen.

Die obige Liste kann weitere Elemente enthalten, wenn zusätzliche Aktionen durch auf Ihrem System aktive Plugins hinzugefügt wurden.
