Tags

Sie können einem Eintrag eine Markierung zuweisen:

  • Durch Ausfüllen des Feldes Tags . Dieses Feld verfügt über eine Autovervollständigungsfunktion : Geben Sie einige Buchstaben ein, und das System schlägt vorhandene Tags vor, die diese Buchstaben enthalten;
  • Klicken Sie auf Wählen Sie aus der Liste. Link zur Auswahl vorhandener Tags.

Die Tags müssen durch Kommata getrennt sein.
