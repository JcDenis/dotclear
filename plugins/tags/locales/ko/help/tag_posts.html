태그에 새 단어를 입력하고 확인을 클릭합니다. 그러면 이 태그가 포함된 모든 항목이 수정됩니다.

이 버튼을 누르면 해당 태그가 할당된 모든 항목에서 해당 태그가 제거되고 태그 목록에서 삭제됩니다.

이 태그가 있는 항목 목록

이 목록에서 사용자의 권한이 허용하는 모든 일괄 작업을 수행할 수 있습니다:

  • 게시: 게시: 선택한 항목을 온라인에 게시합니다.
  • 게시 취소: 선택한 항목을 오프라인으로 가져옵니다.
  • 스케줄: 선택한 항목이 게시 날짜에 온라인으로 설정되도록 예약합니다.
  • 보류 중으로 표시합니다: 선택한 항목을 보류 중으로 표시합니다.
  • 선택됨/선택되지 않음으로 표시합니다: 선택한 항목을 선택됨/선택되지 않음으로 설정합니다./li>.
  • 카테고리를 변경합니다: 선택한 항목의 카테고리를 수정합니다.
  • 언어를 변경합니다: 선택한 항목의 언어를 수정합니다.
  • 작성자 변경: 새 작성자의 ID를 입력하여 선택한 항목의 작성자를 수정합니다.
  • 삭제: 삭제: 선택한 항목을 삭제합니다(이 작업은 취소할 수 없습니다).
  • 태그 추가/제거: 선택한 항목에 태그를 추가하거나 제거합니다.

시스템에서 활성화된 플러그인에 의해 추가 작업이 추가된 경우 위 목록에 다른 항목이 포함될 수 있습니다.
