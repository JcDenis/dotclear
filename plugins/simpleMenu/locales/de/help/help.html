Einfaches Menü

Mit dem einfachen Menü können Sie ein einstufiges Menü in Ihrem Blog anzeigen, sofern Ihr Thema dies unterstützt.

Liste der Menüpunkte
Hier werden Ihre Menüeinträge angezeigt.
Sie können diese Liste neu anordnen, indem Sie die Einträge nach oben oder unten ziehen.
Sie können auch die Bezeichnung, Beschreibung und URL jedes Eintrags ändern.
Klicken Sie anschließend auf Menü aktualisieren.
Ausgewählte Menüpunkte löschen
Ermöglicht das Löschen von Menüpunkten, indem Sie die Kästchen neben den einzelnen Punkten markieren.
Einen neuen Menüpunkt hinzufügen
Wählen Sie zunächst die Art des Artikels aus. Klicken Sie dann auf Weiter…. Gegebenenfalls müssen Sie dann einen Untertyp (Kategorie, Seite usw.) definieren. Wenn ja, tun Sie dies und klicken Sie auf Weiter…. Anschließend können Sie die Vorschläge für die Bezeichnung, die Beschreibung (nicht obligatorisch) und die URL des Menüpunkts ändern. Klicken Sie anschließend auf Diesen Eintrag hinzufügen.
  • Etikett: Der Ankertext. Zum Beispiel "Archiv". Dieses Feld ist obligatorisch.
  • Beschreibung: Eine kurze Beschreibung für den Menüpunkt (wird je nach ausgewähltem Thema unter der Bezeichnung angezeigt). Zum Beispiel: "Archive von 2004 bis 2011".
  • URL: Die URL des Artikels. Dieses Feld ist obligatorisch.
  • URL in einer neuen Registerkarte öffnen: Prüfen Sie, ob dieser Link in einer neuen Registerkarte oder einem neuen Fenster geöffnet werden soll.
