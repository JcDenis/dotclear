Menú sencillo

El menú simple le permite mostrar un menú de un solo nivel en su blog si su tema lo admite.

Lista de elementos del menú
Aquí se muestran los elementos de su menú.
Puede reordenar esta lista arrastrando sus elementos hacia arriba y hacia abajo.
También puedes modificar la etiqueta, descripción y URL de cada elemento.
Una vez hecho esto, haga clic en Menú Actualizar.
Borrar los elementos de menú seleccionados
Permite eliminar elementos del menú marcando las casillas situadas junto a cada uno de ellos.
Añadir un nuevo elemento de menú
Primero seleccione el tipo de artículo. Luego haga clic en Continuar…. Luego, si es necesario, es posible que tengas que definir un subtipo (categoría, página, etc.). Si es así, hazlo y haz clic en Continuar…. Luego, finalmente, puede modificar las propuestas realizadas para la etiqueta, la descripción (no obligatoria) y la URL del elemento del menú. Una vez hecho esto, haga clic en Agregar este elemento.
  • Etiqueta: El texto de anclaje. Por ejemplo "Archivos". Este campo es obligatorio.
  • Descripción: Una breve descripción del elemento del menú (se mostrará debajo de la etiqueta según el tema seleccionado). Por ejemplo: "Archivos de 2004 a 2011".
  • URL: La URL del elemento. Este campo es obligatorio.
  • Abrir URL en una nueva pestaña: Compruebe si este enlace debe abrirse en una pestaña nueva o en una ventana nueva.
