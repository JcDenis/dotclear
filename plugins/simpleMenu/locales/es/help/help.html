Menú sencillo

Menú simple le permite mostrar un menú de un solo nivel en su blog si su tema lo soporta.

Lista de elementos del menú
Aquí se muestran los elementos de su menú.
Puede reordenar esta lista arrastrando sus elementos arriba y abajo.
También puede modificar la etiqueta, la descripción y la URL de cada elemento.
Una vez hecho esto, haga clic en Actualizar menú.
Borrar los elementos de menú seleccionados
Permite eliminar elementos del menú marcando las casillas situadas junto a cada uno de ellos.
Añadir un nuevo elemento de menú
Seleccione primero el tipo de artículo. A continuación, haga clic en Continuar…. A continuación, si es necesario, puede que tenga que definir un subtipo (categoría, página, etc.). Si es así, hágalo y haga clic en Continuar…. A continuación, por último, puede modificar las propuestas realizadas para etiqueta, descripción (no obligatoria) y URL del elemento de menú. Una vez hecho esto, haga clic en Añadir este elemento.
  • Etiqueta: El texto de anclaje. Por ejemplo "Archivos". Este campo es obligatorio.
  • Descripción: Una breve descripción para el elemento del menú (se mostrará debajo de la etiqueta en función del tema seleccionado). Por ejemplo: "Archivos de 2004 a 2011".
  • URL: La URL del artículo. Este campo es obligatorio en .
  • Abrir URL en una nueva pestaña: Compruebe si este enlace debe abrirse en una pestaña nueva o en una ventana nueva.
