Menu sederhana

Menu sederhana memungkinkan Anda menampilkan menu satu tingkat di blog Anda jika tema Anda mendukungnya.

Daftar item menu
Di sini ditampilkan item menu Anda.
Anda dapat menyusun ulang daftar ini dengan menyeret itemnya ke atas dan ke bawah.
Anda juga dapat mengubah label, deskripsi, dan URL setiap item.
Setelah selesai, klik Perbarui menu.
Menghapus item menu yang dipilih
Memungkinkan Anda menghapus item menu dengan mencentang kotak di samping masing-masing item.
Menambahkan item menu baru
Pertama pilih jenis item. Kemudian klik Lanjutkan…. Kemudian, jika perlu, Anda mungkin harus menentukan subtipe (kategori, halaman, dll.). Jika ya, lakukan dan klik Lanjutkan…. Kemudian akhirnya Anda dapat memodifikasi proposal yang telah selesai untuk label, deskripsi (tidak wajib) dan URL item menu. Setelah selesai, klik Tambahkan item ini.
  • Label: Teks tautan. Misalnya "Arsip". Bidang ini wajib diisi.
  • Deskripsi: Deskripsi singkat untuk item menu (akan ditampilkan di bawah label tergantung pada tema yang dipilih). Misalnya: "Arsip dari tahun 2004 hingga 2011".
  • URL: URL item. Bidang ini wajib diisi.
  • Buka URL pada tab baru: Periksa apakah tautan ini harus dibuka di tab baru atau jendela baru.
