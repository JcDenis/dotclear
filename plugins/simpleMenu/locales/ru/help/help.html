Простое меню

Simple menu allows you to display a single level menu on your blog if your theme supports it.

Список пунктов меню
Here are displayed your menu items.
You can reorder this list by dragging its items up and down.
You can also modify the label, description and URL of each item.
Once done, click on Update menu.
Удалить выбранные пункты меню
Позволяет удалять пункты меню, установив флажки рядом с каждым из них.
Добавить новый пункт меню
First select the type of item. Then click on Continue…. Then, if necessary, you may have to define a sub-type (category, page, etc.). If so, do it and click on Continue…. Then finally you can modify proposals done for label, description (not mandatory) and URL of the menu item. Once done, click on Add this item.
  • Label: The anchor text. For example "Archives". This field is mandatory.
  • Description: A short description for the menu item (will be displayed under the label depending on the selected theme). For example: "Archives from 2004 to 2011".
  • URL: The item URL. This field is mandatory.
  • Открыть URL на новой вкладке: Проверьте, должна ли эта ссылка быть открыта в новой вкладке или в новом окне.
