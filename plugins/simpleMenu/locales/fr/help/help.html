Menu simple

Menu simple vous permet d’afficher un menu limité à un seul niveau sur votreblog, si votre thème le supporte.

Liste des items de menu
Voici les éléments de votre menu.
Vous pouvez réorganiser cette liste en faisant glisser les éléments vers le haut ou vers le bas.
Vous pouvez également modifier le libellé, la description et l'URL de chaque élément.
Une fois que vous avez terminé, cliquez sur Mettre à jour le menu.
Supprimer les items de menu sélectionnés
Vous permet de supprimer les items de menu que vous souhaitez en cochant les cases pour chacun d’eux.
Ajouter un item de menu
Sélectionnez d'abord le type d'article. Cliquez ensuite sur Continuer…. Ensuite, si nécessaire, vous devrez peut-être définir un sous-type (catégorie, page, etc.). Si c'est le cas, faites-le et cliquez sur Continuer…. Enfin, vous pouvez modifier les propositions faites pour le libellé, la description (non obligatoire) et l'URL de l'élément de menu. Une fois cela fait, cliquez sur Ajouter cet élément.
  • Étiquette : Le texte d'ancrage. Par exemple "Archives". Ce champ est obligatoire.
  • Description : Brève description de l'élément de menu (affichée sous l'étiquette en fonction du thème sélectionné). Par exemple : "Archives de 2004 à 2011" : "Archives de 2004 à 2011".
  • URL : L'URL de l'élément. Ce champ est obligatoire.
  • S'ouvre dans un nouvel onglet : Cochez si le lien doit s'ouvrir dans un nouvel onglet ou une nouvelle fenêtre.
