Menu simples

O menu simples permite que você exiba um menu de nível único em seu blog, se o seu tema for compatível.

Lista de itens de menu
Aqui são exibidos seus itens de menu.
Você pode reordenar esta lista arrastando seus itens para cima e para baixo.
Você também pode modificar o rótulo, a descrição e o URL de cada item.
Uma vez feito isso, clique em Menu Atualizar.
Eliminar itens de menu seleccionados
Permite-lhe eliminar itens de menu, assinalando as caixas junto a cada um deles.
Adicionar um novo item de menu
Primeiro selecione o tipo de item. Em seguida, clique em Continuar…. Então, se necessário, poderá ser necessário definir um subtipo (categoria, página, etc.). Se sim, faça isso e clique em Continue…. Finalmente você pode modificar as propostas feitas para rótulo, descrição (não obrigatório) e URL do item de menu. Uma vez feito isso, clique em Adicionar este item.
  • Rótulo: O texto âncora. Por exemplo "Arquivos". Este campo é obrigatório.
  • Descrição: Uma breve descrição para o item de menu (será exibida sob o rótulo dependendo do tema selecionado). Por exemplo: “Arquivos de 2004 a 2011”.
  • URL: O URL do item. Este campo é obrigatório.
  • Abrir URL num novo separador: Verificar se esta ligação deve ser aberta num novo separador ou numa nova janela.
