Einstellungen für Alarme

Freischaltung

Aktivieren Sie dieses Kontrollkästchen, um Warnmeldungen unter Aufgaben anzuzeigen, die verspätet sind oder nie ausgeführt wurden.

Hinweis: Unter Meine Einstellungenfinden Sie zusätzliche Optionen, mit denen Sie diese Aufgaben auf Ihrem Dashboard anzeigen können.

Frequenz

Plant die Anzeige von Warnmeldungen über zu erledigende Aufgaben.

Diese Häufigkeit kann für alle Aufgaben gleich sein (wählen Sie das Optionsfeld Eine Abrufzeit für alle Aufgaben verwenden ) oder für jede Aufgabe einzeln festgelegt werden.
