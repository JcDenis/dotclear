Настройки оповещений

Активация

Установите этот флажок для отображения предупреждений в задачах, которые поздно или которые никогда не были выполнены.

Примечание: в Мои настройки, дополнительные параметры позволяют отображать эти задачи на панели управления.

Частота

Запланирует отображение предупреждений задач, которые должны быть выполнены.

Эта частота может быть распространена для всех задач (выберите Использовать время напоминания для всех задач радион) или может быть установлена для каждой задачи.
