Ρυθμίσεις ειδοποίησης

Ενεργοποίηση

Μαρκάρετε αυτό το πλαίσιο για να εμφανίζονται μηνύματα ειδοποίησης κάτω από εργασίες που έχουν καθυστερήσει ή που δεν έχουν εκτελεστεί ποτέ.

Σημείωση: στη διεύθυνση Οι προτιμήσεις μου, πρόσθετες επιλογές σας επιτρέπουν να εμφανίζετε αυτές τις εργασίες στον πίνακα οργάνων σας.

Συχνότητα

Προγραμματίζει την εμφάνιση ειδοποιήσεων για τις εργασίες που πρέπει να γίνουν.

Αυτή η συχνότητα μπορεί να είναι κοινή για όλες τις εργασίες (επιλέξτε το κουμπί επιλογής Use one recall time for all tasks ) ή μπορεί να οριστεί για κάθε εργασία.
