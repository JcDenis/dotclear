위젯 사용

위젯을 사용하여 공개 페이지에 표시할 다양한 콘텐츠 블록을 추가할 수 있습니다. 위젯을 추가하려면 왼쪽의 사용 가능한 위젯 목록에서 이 페이지 오른쪽의 사이드바 중 하나로 위젯을 드래그합니다. 위젯을 위아래로 드래그하여 사이드바에서 위젯을 정렬할 수 있습니다. 변경 사항을 적용하려면 사이드바 업데이트를 누릅니다.

사이드바에 포함된 위젯에는 이름 옆의 화살표를 클릭하여 접근할 수 있는 구성 옵션이 있습니다.

위젯을 제거하려면 전용 영역 중 하나로 드래그하기만 하면 됩니다.

수정 후에는 사이드바 업데이트를 눌러 변경 사항을 저장하는 것을 잊지 마세요.

사이드바 재설정 버튼을 사용하여 사이드바를 원래 시스템에서 설정한 대로 재설정합니다(선택 사항을 확인하라는 팝업이 표시됩니다).

위젯 태그

테마 템플릿을 편집할 수 있는 경우 자체 구성으로 위젯을 템플릿 태그로 직접 추가할 수 있습니다.

템플릿에 위젯을 추가하려면 다음과 같은 코드를 작성해야 합니다:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">설정 값</setting>
  ...
</tpl:Widget>
