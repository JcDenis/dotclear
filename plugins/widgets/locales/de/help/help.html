Verwendung von Widgets

Widgets können verwendet werden, um verschiedene Inhaltsblöcke hinzuzufügen, die auf Ihren öffentlichen Seiten angezeigt werden sollen. Um ein Widget hinzuzufügen, ziehen Sie es aus der Liste Verfügbare Widgets auf der linken Seite in eine der Seitenleisten auf der rechten Seite dieser Seite. Sie können Ihre Widgets in einer Seitenleiste anordnen, indem Sie sie nach oben oder unten ziehen. Um Ihre Änderungen zu übernehmen, drücken Sie Seitenleisten aktualisieren.

Einmal in eine Seitenleiste eingefügt, verfügen Widgets über Konfigurationsoptionen, die Sie durch Klicken auf den Pfeil neben ihrem Namen erreichen können.

Um ein Widget zu entfernen, ziehen Sie es einfach in einen der dafür vorgesehenen Bereiche

Denken Sie daran, Ihre Änderungen nach jeder Änderung zu speichern, indem Sie auf Seitenleisten aktualisierendrücken.

Verwenden Sie die Schaltfläche Seitenleisten zurücksetzen , um die Seitenleisten so zurückzusetzen, wie sie ursprünglich in Ihrem System eingestellt waren (Sie werden in einem Popup-Fenster aufgefordert, Ihre Wahl zu bestätigen).

Widget-Tags

Wenn Sie die Möglichkeit haben, Ihre Themenvorlagen zu bearbeiten, können Sie Widgets direkt als Vorlagen-Tags mit ihrer eigenen Konfiguration hinzufügen.

Um ein Widget in Ihre Vorlage einzufügen, müssen Sie einen Code wie den folgenden schreiben:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Einstellwert</setting>
  ...
</tpl:Widget>
