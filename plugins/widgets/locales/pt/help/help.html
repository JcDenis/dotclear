Utilização de widgets

Widgets podem ser usados para adicionar vários blocos de conteúdo a serem exibidos em suas páginas públicas. Para adicionar um widget, arraste-o a partir das widgets disponíveis lista na esquerda para uma das barras laterais à direita desta página. Você pode ordenar os seus widgets em uma barra lateral arrastando-os para cima ou para baixo. Para aplicar as alterações, pressione Atualizar barras laterais.

Uma vez incluído na barra lateral, os widgets têm opções de configuração que você pode alcançar clicando na seta ao lado do nome.

Para remover um widget, simplesmente arraste-o para uma das zonas dedicadas

Lembre-se de salvar suas alterações após qualquer modificação pressionando Atualizar barras laterais

Use o botão Redefinir as barras laterais para redefinir as barras laterais, como elas foram configuradas originalmente no seu sistema (um pop-up pedirá que você confirme sua escolha).

Tags do Widget

Se você tiver permissão para editar seus modelos de temas, poderá adicionar diretamente widgets como tags de templates, com suas próprias configurações.

Para adicionar um widget em seu modelo, você precisa escrever código como este:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Definindo o valor</setting>
  ...
</tpl:Widget>
