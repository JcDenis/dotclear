Penggunaan widget

Widget dapat digunakan untuk menambahkan berbagai blok konten yang akan ditampilkan di halaman publik Anda. Untuk menambahkan widget, seret dari daftar Widget yang tersedia di sebelah kiri ke salah satu bilah sisi di sebelah kanan halaman ini. Anda dapat menyusun widget di bilah sisi dengan menyeretnya ke atas atau ke bawah. Untuk menerapkan perubahan Anda, tekan Perbarui bilah sisi.

Setelah disertakan di bilah sisi, widget memiliki opsi konfigurasi yang dapat Anda jangkau dengan mengeklik panah di samping namanya.

Untuk menghapus widget, cukup seret ke salah satu zona khusus

Ingatlah untuk menyimpan perubahan Anda setelah modifikasi apa pun dengan menekan Perbarui sidebar

Gunakan tombol Reset sidebar untuk mereset sidebar seperti yang telah ditetapkan pada sistem Anda (pop-up akan meminta Anda untuk mengonfirmasi pilihan Anda).

Tag Widget

Jika Anda diizinkan untuk mengedit templat tema Anda, Anda bisa langsung menambahkan widget sebagai tag templat, dengan konfigurasinya sendiri.

Untuk menambahkan widget di templat Anda, Anda perlu menulis kode seperti ini:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Nilai pengaturan</setting>
  ...
</tpl:Widget>
