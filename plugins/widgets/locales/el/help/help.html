Χρήση των widgets

Τα widgets μπορούν να χρησιμοποιηθούν για να προσθέσετε διάφορα μπλοκ περιεχομένου που θα εμφανίζονται στις δημόσιες σελίδες σας. Για να προσθέσετε ένα widget, σύρετέ το από τη λίστα Διαθέσιμα widgets στα αριστερά σε μία από τις πλευρικές γραμμές στα δεξιά αυτής της σελίδας. Μπορείτε να διατάξετε τα widgets σας σε μια πλευρική μπάρα σύροντάς τα προς τα πάνω ή προς τα κάτω. Για να εφαρμόσετε τις αλλαγές σας, πατήστε Ενημέρωση πλευρικών γραμμών.

Αφού συμπεριληφθούν σε μια πλαϊνή γραμμή, τα widgets διαθέτουν επιλογές διαμόρφωσης, τις οποίες μπορείτε να επιλέξετε κάνοντας κλικ στο βέλος δίπλα στο όνομά τους.

Για να αφαιρέσετε ένα widget, απλά σύρετέ το σε μία από τις ειδικές ζώνες

Θυμηθείτε να αποθηκεύσετε τις αλλαγές σας μετά από κάθε τροποποίηση πατώντας Ενημέρωση πλευρικών γραμμών

Χρησιμοποιήστε το κουμπί Επαναφορά πλευρικών γραμμών για να επαναφέρετε τις πλευρικές γραμμές όπως είχαν οριστεί αρχικά στο σύστημά σας (ένα αναδυόμενο παράθυρο θα σας ζητήσει να επιβεβαιώσετε την επιλογή σας).

Ετικέτες Widget

Αν σας επιτρέπεται να επεξεργάζεστε τα πρότυπα του θέματός σας, μπορείτε να προσθέσετε απευθείας widgets ως ετικέτες προτύπων, με τις δικές τους ρυθμίσεις.

Για να προσθέσετε ένα widget στο πρότυπό σας, θα πρέπει να γράψετε κώδικα ως εξής:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Τιμή ρύθμισης</setting>
  ...
</tpl:Widget>
