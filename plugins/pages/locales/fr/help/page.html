Rédaction de la page

Titre
Inscrivez le titre de la page. Ce champ est obligatoire.
Extrait
Ce champ est optionnel.
Contenu
Le contenu de la page. Ce champ est obligatoire.
Notes personnelles
Cette zone de texte sert à la prise de notes ou pense-bête divers. Elle ne sera jamais affichée sur le blog.
État de la page
Permet de choisir l'état de la page après enregistrement :
  • En attente : en attente de publication.
  • Planifié : La page sera mise en ligne à l'heure et à la date définies dans le champ Publié le.
  • Non publié : page hors ligne.
  • Publié : page en ligne.
Publié le
Vous pouvez ici modifier la date et l'heure de publication de la page. Si le statut de la page est Planifié, elle sera mise en ligne à la date et à l'heure indiquées.
Langue de la page
Sélectionnez le code linguistique de votre entrée. Par défaut, il s'agit de votre langue (en tant qu'utilisateur), mais vous pouvez le remplacer par le code linguistique de votre choix, par exemple "en" ou "es-ar". Ce code est utilisé pour l'affichage de l'entrée.
Formatage du texte
Choisissez la syntaxe de la page. Wiki est une syntaxe simplifiée et sera convertie en HTML valide. A moins que vous ne maîtrisiez parfaitement le HTML, nous vous conseillons de choisir la syntaxe Wiki. Voir la référence de la syntaxe Wiki pour plus d'informations.
Position de la page
Les pages sont classées en fonction de ce nombre. Ce champ est utilisé par le widget Pages(Ordre par position de page).
Fichiers attachés
Les pièces jointes sont des fichiers multimédias ajoutés à la page. Vous pouvez utiliser le bouton Ajouter des fichiers à cette entrée pour sélectionner les fichiers multimédias à ajouter à l'entrée.
Cette option apparaîtra après l'enregistrement de votre entrée.
Veuillez consulter l'aide du gestionnaire de médias pour plus d'informations.
Accepter les commentaires
Cochez cette case pour permettre à vos visiteurs de commenter cette page. Le paramètre global correspondant peut être trouvé dans les paramètres du blog.
Accepter les rétroliens
Un trackback est un moyen de laisser une partie de votre page sous forme de commentaire sur un autre blog. Cochez cette case pour permettre à d'autres personnes de faire un trackback sur votre page. Le paramètre global correspondant se trouve dans les paramètres du blog.
Masquer dans le widget des pages
Si cette option est cochée, la page sera active mais ne sera pas listée dans le widget Pages .
Mot de passe
Vous pouvez saisir un mot de passe pour votre page. Une page protégée par un mot de passe ne sera pas affichée sur votre blog, elle ne sera accessible qu'aux personnes à qui vous donnerez l'URL d'entrée (voir le lien Aller à cette page sur le site ) et le mot de passe.
URL spécifique
En cliquant sur son cadenas, vous pouvez déprotéger ce champ et choisir une autre URL pour votre page. Si l'URL que vous essayez d'utiliser est déjà utilisée par une autre page, un numéro lui sera ajouté.

Commentaires

Liste des commentaires
L'onglet Commentaires vous permet de lire et de modifier le statut des commentaires sur votre page. En fonction de vos autorisations, vous pouvez modifier, mettre en ligne ou hors ligne, supprimer ou classer comme indésirable un commentaire.
Ajouter un commentaire
Depuis l'onglet Ajouter un commentaire, vous pouvez ajouter ou répondre directement à un commentaire sans passer par votre blog. Cette interface vous permettra également de saisir votre commentaire en HTML, sans limitation. Vous devrez remplir les champs de la même manière que lors de l'édition d'un commentaire.
