Επεξεργασία σελίδας

Τίτλος σελίδας
Πληκτρολογήστε τον τίτλο της σελίδας. Αυτό το πεδίο είναι υποχρεωτικό.
Απόσπασμα
Αυτό το πεδίο είναι προαιρετικό.
Περιεχόμενο
Το περιεχόμενο της σελίδας σας. Αυτό το πεδίο είναι υποχρεωτικό.
Σημειώσεις
Αυτή η περιοχή κειμένου είναι για προσωπική χρήση, για σημειώσεις ή σημειώσεις. Αυτό που εισάγετε εδώ δεν θα εμφανιστεί ποτέ στο blog.
Κατάσταση σελίδας
Σας επιτρέπει να επιλέξετε την κατάσταση της σελίδας σας:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί (δεν έχει δημοσιευθεί ακόμα).
  • Προγραμματισμός: Η σελίδα θα οριστεί online την ώρα και την ημερομηνία που ορίστηκε στο πεδίο Δημοσιεύθηκε στο.
  • Αδημοσίευτο: Η σελίδα είναι εκτός σύνδεσης.
  • Δημοσιεύθηκε: Η σελίδα είναι online.
Ημερομηνία και ώρα δημοσίευσης
Εδώ μπορείτε να αλλάξετε την ημερομηνία και την ώρα δημοσίευσης της σελίδας. Εάν η κατάσταση της σελίδας είναι Προγραμματισμένη, θα είναι συνδεδεμένη στη δεδομένη ημερομηνία και ώρα.
Γλώσσα σελίδας
Επιλέξτε τον κωδικό γλώσσας της καταχώρησής σας. Είναι προεπιλεγμένο στη γλώσσα σας (ως χρήστη) αλλά μπορείτε να την αλλάξετε σε όποιον κωδικό θέλετε, π. χ. . "en" ή "es-ar". Αυτός ο κώδικας χρησιμοποιείται για την προβολή της καταχώρησης.
Μορφοποίηση κειμένου
Επιλέξτε τη σύνταξη σελίδας. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρη HTML. Αν δεν έχετε μια τέλεια κατανόηση της HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη Wiki. Δείτε την αναφορά σύνταξης του Wiki για περισσότερες πληροφορίες.
Θέση σελίδας
Παραγγελίες σελίδων σύμφωνα με αυτόν τον αριθμό. Το πεδίο αυτό χρησιμοποιείται από το γραφικό συστατικό Σελίδες (Θέση παραγγελίας κατά σελίδα).
Συνημμένα
Συνημμένα είναι αρχεία πολυμέσων που προστίθενται στη σελίδα. Μπορείτε να χρησιμοποιήσετε το κουμπί Προσθήκη αρχείων σε αυτήν την καταχώρηση για να επιλέξετε αρχεία πολυμέσων για να προσθέσετε στην καταχώρηση.
Αυτή η επιλογή θα εμφανιστεί μετά την αποθήκευση της καταχώρησής σας.
Παρακαλούμε ελέγξτε τη βοήθεια του διαχειριστή πολυμέσων για περισσότερες πληροφορίες.
Αποδοχή σχολίων
Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε στους επισκέπτες σας να σχολιάσουν σε αυτή τη σελίδα. Η αντίστοιχη καθολική ρύθμιση μπορεί να βρεθεί στις ρυθμίσεις του ιστολογίου.
Αποδοχή trackbacks
Ένα trackback είναι ένας τρόπος για να αφήσετε ένα τμήμα της σελίδας σας ως σχόλιο για άλλο blog. Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε σε άλλους να παρακολουθούν τη σελίδα σας. Η αντίστοιχη καθολική ρύθμιση θα βρεθεί στις ρυθμίσεις ιστολογίου.
Απόκρυψη σε σελίδες widget
Αν αυτή η επιλογή είναι ενεργοποιημένη, η σελίδα θα είναι ενεργή αλλά δεν θα εμφανίζεται στο widget σελίδα. (Automatic Translation)
Κωδικός
Μπορείτε να εισάγετε έναν κωδικό πρόσβασης για τη σελίδα σας. Μια σελίδα προστατευμένη με κωδικό πρόσβασης δεν θα εμφανίζεται στο ιστολόγιό σας, θα είναι προσβάσιμο μόνο σε εκείνους που θα δώσει την καταχώρηση URL για να (δείτε το Μεταβείτε σε αυτή τη σελίδα στον σύνδεσμο ιστοσελίδα) και τον κωδικό πρόσβασης.
Βασικό
Κάνοντας κλικ στο κλείδωμα του, μπορείτε να καταργήσετε την προστασία αυτού του πεδίου και να επιλέξετε ένα άλλο URL για τη σελίδα σας. Αν το URL που προσπαθείτε να χρησιμοποιήσετε χρησιμοποιείται ήδη από άλλη σελίδα, ένας αριθμός θα προστεθεί σε αυτό.

Σχόλια

Κατάλογος σχολίων
Από την καρτέλα Σχόλια μπορείτε να διαβάσετε και να αλλάξετε την κατάσταση σχόλια στη σελίδα σας. Ανάλογα με τα δικαιώματά σας, μπορείτε να επεξεργαστείτε, να βάλετε online ή χωρίς σύνδεση, να διαγράψετε ή να ταξινομήσετε ως σχόλιο ανεπιθύμητο.
Προσθέστε ένα σχόλιο
Από την καρτέλα Προσθέστε ένα σχόλιο μπορείτε να προσθέσετε ή να απαντήσετε απευθείας σε ένα σχόλιο χωρίς να περάσετε από το ιστολόγιό σας. Αυτό το περιβάλλον εργασίας σας επιτρέπει επίσης να εισάγετε το σχόλιό σας σε HTML, χωρίς περιορισμό. Θα πρέπει να συμπληρώσετε τα πεδία με τον ίδιο τρόπο όπως κατά την επεξεργασία ενός σχολίου.
