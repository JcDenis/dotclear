Επεξεργασία σελίδας

Τίτλος σελίδας
Πληκτρολογήστε τον τίτλο της σελίδας. Αυτό το πεδίο είναι υποχρεωτικό.
Απόσπασμα
Αυτό το πεδίο είναι προαιρετικό.
Περιεχόμενο
Το περιεχόμενο της σελίδας σας. Αυτό το πεδίο είναι υποχρεωτικό.
Σημειώσεις
Αυτή η περιοχή κειμένου είναι για προσωπική χρήση, για σημειώσεις ή σημειώσεις. Ό,τι εισάγετε εδώ δεν θα εμφανίζεται ποτέ στο ιστολόγιο.
Κατάσταση σελίδας
Σας επιτρέπει να επιλέξετε την κατάσταση της σελίδας σας:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί (δεν έχει δημοσιευθεί ακόμα).
  • Προγραμματισμένο: Η σελίδα θα ρυθμιστεί online την ώρα και την ημερομηνία που έχουν οριστεί στο πεδίο Δημοσιεύτηκε στο .
  • Αδημοσίευτο: Η σελίδα είναι εκτός σύνδεσης.
  • Δημοσιεύθηκε: Η σελίδα είναι online.
Ημερομηνία και ώρα δημοσίευσης
Εδώ μπορείτε να αλλάξετε την ημερομηνία και την ώρα δημοσίευσης της σελίδας. Εάν η κατάσταση της σελίδας είναι Προγραμματισμένη, θα είναι συνδεδεμένη τη δεδομένη ημερομηνία και ώρα.
Γλώσσα σελίδας
Επιλέξτε τον κωδικό γλώσσας της συμμετοχής σας. Είναι προεπιλεγμένη στη γλώσσα σας (ως χρήστης), αλλά μπορείτε να την αλλάξετε σε όποιον κωδικό γλώσσας θέλετε, π.χ. "en" ή "es-ar". Αυτός ο κωδικός χρησιμοποιείται για την εμφάνιση της καταχώρισης.
Μορφοποίηση κειμένου
Επιλέξτε τη σύνταξη της σελίδας. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρο HTML. Αν δεν έχετε τέλεια κατανόηση του HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη του Wiki. Δείτε την αναφορά σύνταξης Wiki για περισσότερες πληροφορίες.
Θέση σελίδας
Παραγγελίες σελίδων σύμφωνα με αυτόν τον αριθμό. Αυτό το πεδίο χρησιμοποιείται από το γραφικό στοιχείο Σελίδες (Σειρά ανά θέση σελίδας).
Συνημμένα
Τα συνημμένα είναι αρχεία πολυμέσων που προστίθενται στη σελίδα. Μπορείτε να χρησιμοποιήσετε το κουμπί Προσθήκη αρχείων σε αυτήν την καταχώρηση για να επιλέξετε αρχεία πολυμέσων για προσθήκη στην καταχώρηση.
Αυτή η επιλογή θα εμφανιστεί αφού αποθηκεύσετε την καταχώρισή σας.
Ελέγξτε τη βοήθεια του Media Manager για περισσότερες πληροφορίες.
Αποδοχή σχολίων
Ενεργοποιήστε αυτή την επιλογή για να επιτρέψετε στους επισκέπτες σας να σχολιάσουν σε αυτή τη σελίδα. Η αντίστοιχη καθολική ρύθμιση μπορεί να βρεθεί στις ρυθμίσεις του ιστολογίου.
Αποδοχή trackbacks
Το trackback είναι ένας τρόπος για να αφήσετε ένα τμήμα της σελίδας σας ως σχόλιο σε ένα άλλο ιστολόγιο. Επιλέξτε αυτό το πλαίσιο για να επιτρέψετε σε άλλους να παρακολουθούν τη σελίδα σας. Η αντίστοιχη καθολική ρύθμιση βρίσκεται στις ρυθμίσεις του ιστολογίου.
Απόκρυψη σε σελίδες widget
Αν αυτή η επιλογή είναι ενεργοποιημένη, η σελίδα θα είναι ενεργή αλλά δεν θα εμφανίζεται στο widget σελίδα. (Automatic Translation)
Κωδικός
Μπορείτε να εισάγετε έναν κωδικό πρόσβασης για τη σελίδα σας. Μια σελίδα που προστατεύεται με κωδικό πρόσβασης δεν θα εμφανίζεται στο ιστολόγιό σας, θα είναι προσβάσιμη μόνο σε αυτούς στους οποίους θα δώσετε τη διεύθυνση URL εισόδου (δείτε τον σύνδεσμο Μεταβείτε σε αυτήν τη σελίδα στον ιστότοπο ) και τον κωδικό πρόσβασης .
Βασικό
Κάνοντας κλικ στην κλειδαριά του, μπορείτε να καταργήσετε την προστασία αυτού του πεδίου και να επιλέξετε μια άλλη διεύθυνση URL για τη σελίδα σας. Εάν η διεύθυνση URL που προσπαθείτε να χρησιμοποιήσετε χρησιμοποιείται ήδη από άλλη σελίδα, θα προστεθεί ένας αριθμός σε αυτήν.

Σχόλια

Κατάλογος σχολίων
Από την καρτέλα Σχόλια μπορείτε να διαβάσετε και να αλλάξετε τα σχόλια κατάστασης στη σελίδα σας. Ανάλογα με τις άδειές σας, μπορείτε να επεξεργαστείτε, να τοποθετήσετε στο διαδίκτυο ή εκτός σύνδεσης, να διαγράψετε ή να ταξινομήσετε ως ανεπιθύμητο σχόλιο.
Προσθέστε ένα σχόλιο
Από την καρτέλα Προσθήκη σχολίου μπορείτε να προσθέσετε ή να απαντήσετε απευθείας σε ένα σχόλιο χωρίς να μεταβείτε στο ιστολόγιό σας. Αυτή η διεπαφή θα σας επιτρέπει επίσης να εισάγετε το σχόλιό σας σε HTML, χωρίς περιορισμό. Θα πρέπει να συμπληρώσετε τα πεδία με τον ίδιο τρόπο όπως όταν επεξεργάζεστε ένα σχόλιο.
