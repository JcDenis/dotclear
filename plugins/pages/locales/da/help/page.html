Redigering af en side

Sidens titel
Indtast sidens titel. Dette felt er obligatorisk.
Uddrag
Dette felt er valgfrit.
Indhold
Indholdet af din side. Dette felt er obligatorisk.
Noter
This text area is for personal use, for notes or memos. What you enter here will never be displayed on the blog.
Status på siden
Giver dig mulighed for at vælge din sidestatus:
  • Afventer: Publikationsstatus er endnu ikke besluttet (ikke publiceret endnu).
  • Scheduled: The page will be set online at the time and date set in the Published on field.
  • Ikke udgivet: Siden er offline.
  • Udgivet: Siden er online.
Udgivelsesdato og -tidspunkt
Here you can change the page publication date and time. If the page status is Scheduled, it will be online at the given date and time.
Sprog på siden
Select the language code of your entry. It defaults to your (as a user) language but you can change it to whatever language code you want, e.g. "en" or "es-ar". This code is used for the entry's display.
Formatering af tekst
Choose the page syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Position på siden
Orders pages according to this number. This field is used by the Pages widget (Order by page position).
Vedhæftede filer
Attachments are media files added to the page. You can use the Add files to this entry button to select media files to add to the entry.
This option will appear after you save your entry.
Please check the Media manager help for more information.
Accepter kommentarer
Markér dette felt for at give dine besøgende mulighed for at kommentere på denne side. Den tilsvarende globale indstilling kan findes i blogindstillingerne.
Accepter trackbacks
A trackback is a way to let a portion of your page as a comment on another blog. Check this box to allow others to trackback your page. The corresponding global setting is to be found in the blog settings.
Skjul i widget-sider
Hvis denne indstilling er markeret, vil siden være aktiv, men den vil ikke blive vist i widgetten Sider.
Adgangskode
You can enter a password for your page. A password protected page will not be displayed on your blog, it will only be accessible to those you will give the entry URL to (see the Go to this page on the site link) and password.
Basenavn
By clicking on its lock, you can unprotect this field and choose another URL for your page. If the URL you're trying to use is already used by another page, a number will be added to it.

Kommentarer

Liste over kommentarer
From the tab Comments you can read and change the status comments on your page. Depending on your permissions, you can edit, put online or offline, delete or classify as a comment undesirable.
Tilføj en kommentar
From the tab Add a comment you can add or answer directly to a comment without going through your blog. This interface will also allows you to enter your comment in HTML, without limitation. You will have to fill in the fields in the same way as when editing a comment.
