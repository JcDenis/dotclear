ページの編集

ページタイトル
ページタイトルに入力してください。このフィールドは必須です。
概要
このフィールドは任意です。
コンテンツ
ページの内容。このフィールドは必須です。
メモ
このテキストエリアは、個人使用、メモ用です。ここに入力した内容はブログには表示されません。
ページの状態
ページのステータスを選択できます：
  • 保留中: 出版状況はまだ決定されていない(未公開)。
  • スケジュール: ページは、 公開日 フィールドに設定された日時にオンラインに設定されます。
  • 未発表: ページはオフラインです。
  • 公開日: ページはオンラインです。
公開日時：
ここで、ページの公開日時を変更できます。ページのステータスが スケジュール済みの場合、指定した日時にオンラインになります。
ページ言語
エントリの言語コードを選択します。デフォルトでは (ユーザーとしての) 言語に設定されていますが、任意の言語コード (例: 「en」または「es-ar」) に変更できます。このコードはエントリの表示に使用されます。
テキストの書式設定
ページ構文を選択します。Wiki は簡略化された構文であり、有効な HTML に変換されます。HTML を完全に理解していない場合は、Wiki 構文を選択することをお勧めします。詳細については、Wiki 構文リファレンスを参照してください。
ページの位置
この番号に従ってページを並べ替えます。このフィールドは、 ページ ウィジェット (ページ位置で並べ替え) によって使用されます。
添付ファイル
添付ファイルは、ページに追加されたメディア ファイルです。エントリに追加するメディア ファイルを選択するには、[ このエントリにファイルを追加 ] ボタンを使用します。
このオプションは、エントリを保存した後に表示されます。
詳細については、メディア マネージャーのヘルプを確認してください。
コメントを承認する
訪問者がこのページにコメントできるようにするには、このボックスにチェックを入れてください。対応するグローバル設定はブログ設定で確認できます。
トラックバックを承認する
トラックバックは、ページの一部を別のブログにコメントとして投稿する方法です。他の人があなたのページをトラックバックできるようにするには、このボックスをオンにします。対応するグローバル設定は、ブログ設定にあります。
ウィジェットページで非表示
このオプションにチェックがついている場合、ページはアクティブになりますが、ページウィジェットには表示されません。
パスワード
ページにパスワードを入力できます。パスワードで保護されたページはブログに表示されず、エントリ URL ( サイトのこのページに移動 リンクを参照) とパスワードを伝えたユーザーのみがアクセスできます。
Basename
ロックをクリックすると、このフィールドの保護を解除し、ページに別の URL を選択できます。使用しようとしている URL がすでに別のページで使用されている場合は、番号が追加されます。

コメント

コメント一覧
タブ コメント から、ページのステータスコメントを読んだり変更したりできます。権限に応じて、編集、オンラインまたはオフラインにしたり、削除したり、不要なコメントとして分類したりできます。
コメントを追加
タブ コメントを追加 からは、ブログを経由せずにコメントに直接追加したり、コメントに返信したりできます。このインターフェイスでは、制限なく HTML でコメントを入力することもできます。コメントを編集するときと同じ方法でフィールドに入力する必要があります。
