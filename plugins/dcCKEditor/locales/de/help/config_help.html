Freischaltung

Aktivieren Sie dieses Feld, um verwandte Beiträge in diesem Blog zu verwenden.

Hinweis: Die Deaktivierung des Plugins entfernt nicht die zuvor im Administrationsbereich getroffene Auswahl.

Optionen

Sie können wählen, welche Schaltflächen angezeigt werden sollen. Seien Sie vorsichtig, diese Einstellungen gelten für alle Benutzer des Blogs.

Wie man es benutzt

Jeder Benutzer des Blogs muss das Plugin unter in seinen Einstellungen wählen.
