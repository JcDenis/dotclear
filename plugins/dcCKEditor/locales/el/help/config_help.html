Ενεργοποίηση

Τσεκάρετε αυτό το πλαίσιο για να αρχίσετε να χρησιμοποιείτε τις σχετικές αναρτήσεις σε αυτό το ιστολόγιο.

Σημείωση : η απενεργοποίηση του πρόσθετου δεν καταργεί την προηγούμενη επιλογή που έγινε στην περιοχή διαχείρισης.

Επιλογές

Μπορείτε να επιλέξετε ποια κουμπιά θα εμφανίζονται. Προσοχή, αυτές οι ρυθμίσεις αφορούν όλους τους χρήστες του ιστολογίου.

Πώς να το χρησιμοποιήσετε

Κάθε χρήστης του ιστολογίου πρέπει να επιλέξει το πρόσθετο στο τις προτιμήσεις του.
