Aktivasi

Centang kotak ini untuk mulai menggunakan postingan terkait di blog ini.

Catatan: menonaktifkan plugin tidak menghapus pilihan sebelumnya yang dibuat di area administrasi.

Pilihan

Anda dapat memilih tombol mana yang akan ditampilkan. Hati-hati, pengaturan ini untuk semua pengguna di blog.

Bagaimana cara menggunakannya

Setiap pengguna di blog harus memilih plugin di preferensinya.
