About:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Egyes paraméterek csak ezen az oldalon módosíthatók.

Az oldalnak két lapja van: Az aktuális blog beállításai (blog neve) és Globális beállítások az aktuális blog beállításainak, illetve a globális beállítások kezeléséhez.

A Goto legördülő menü segítségével közvetlenül elérheti a különböző oldalrészeket.
