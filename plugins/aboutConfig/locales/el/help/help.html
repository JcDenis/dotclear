This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Ορισμένες παράμετροι μπορούν να τροποποιηθούν μόνο σε αυτή τη σελίδα.

Η σελίδα έχει δύο καρτέλες: Ρυθμίσεις για το τρέχον ιστολόγιο (όνομα ιστολογίου) και Παγκόσμιες ρυθμίσεις για τη διαχείριση των ρυθμίσεων του τρέχοντος ιστολογίου και των παγκόσμιων ρυθμίσεων, αντίστοιχα.

Μπορείτε να έχετε άμεση πρόσβαση στις διάφορες ενότητες της σελίδας χρησιμοποιώντας το αναπτυσσόμενο μενού Goto .
