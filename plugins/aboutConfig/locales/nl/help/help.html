Over:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Sommige parameters kunnen alleen op deze pagina worden gewijzigd.

De pagina heeft twee tabbladen: Instellingen voor het huidige blog (blognaam) en Globale instellingen om respectievelijk de huidige bloginstellingen en de globale instellingen te beheren.

Je kunt direct naar de verschillende paginasecties gaan met het vervolgkeuzemenu Ga naar.
