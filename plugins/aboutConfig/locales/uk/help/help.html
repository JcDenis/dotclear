Про:Конфігурація

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Деякі параметри можна змінити лише на цій сторінці.

Сторінка має дві вкладки: Налаштування поточного блогу (назва блогу) та Глобальні налаштування для керування налаштуваннями поточного блогу та глобальними налаштуваннями відповідно.

Ви можете отримати прямий доступ до різних розділів сторінки за допомогою випадаючого меню Goto .
