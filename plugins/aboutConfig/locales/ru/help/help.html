О программе

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Некоторые параметры могут быть изменены только на этой странице.

Страница имеет две вкладки: Настройки для текущего блога (имя блога) и Глобальные настройки для управления текущими настройками блога и глобальными настройками, соответственно.

Вы можете получить прямой доступ к различным разделам страницы, используя выпадающее меню Goto.
