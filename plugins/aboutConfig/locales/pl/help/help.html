O:Konfiguracja

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Niektóre parametry można modyfikować tylko na tej stronie.

Strona ma dwie zakładki: Ustawienia bieżącego bloga (nazwa bloga) i Ustawienia globalne do zarządzania odpowiednio bieżącymi ustawieniami bloga i ustawieniami globalnymi.

Bezpośredni dostęp do różnych sekcji strony można uzyskać za pomocą rozwijanego menu Goto .
