O produktu:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Některé parametry lze upravit pouze na této stránce.

Stránka má dvě karty: Nastavení aktuálního blogu (název blogu) a Globální nastavení pro správu nastavení aktuálního blogu a globálního nastavení.

K jednotlivým sekcím stránky můžete přistupovat přímo pomocí rozevírací nabídky Goto .
