Om:Konfig

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Nogle parametre kan kun ændres på denne side.

Siden har to faner: Indstillinger for den aktuelle blog (blognavn) og Globale indstillinger til at administrere henholdsvis de aktuelle blogindstillinger og de globale indstillinger.

Du kan få direkte adgang til de forskellige sektioner på siden ved hjælp af rullemenuen Goto .
