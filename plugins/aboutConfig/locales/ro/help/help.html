Despre:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Anumiți parametri pot fi modificați numai pe această pagină.

Pagina are două file: Setări pentru blogul curent (numele blogului) și Setări globale pentru a gestiona setările blogului curent și, respectiv, setările globale.

Puteți accesa direct diferitele secțiuni ale paginii folosind meniul derulant Goto .
