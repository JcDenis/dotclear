Hakkında:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Bazı parametreler yalnızca bu sayfada değiştirilebilir.

Sayfada iki sekme bulunmaktadır: Geçerli blog için ayarlar (blog adı) ve Genel ayarlar sırasıyla geçerli blog ayarlarını ve genel ayarları yönetmek için.

Goto açılır menüsünü kullanarak farklı sayfa bölümlerine doğrudan erişebilirsiniz.
