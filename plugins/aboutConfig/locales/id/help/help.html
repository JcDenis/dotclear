Tentang: Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Beberapa parameter hanya dapat dimodifikasi pada halaman ini.

Halaman ini memiliki dua tab: Pengaturan untuk blog saat ini (nama blog) dan Pengaturan global untuk mengelola pengaturan blog saat ini dan pengaturan global.

Anda dapat langsung mengakses bagian halaman yang berbeda dengan menggunakan menu tarik-turun Goto .
