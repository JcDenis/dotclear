Apie:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Kai kuriuos parametrus galima keisti tik šiame puslapyje.

Puslapyje yra du skirtukai: Dabartinio tinklaraščio nustatymai (tinklaraščio pavadinimas) ir Visuotiniai nustatymai , skirti atitinkamai dabartinio tinklaraščio nustatymams ir visuotiniams nustatymams tvarkyti.

Galite tiesiogiai pasiekti skirtingus puslapio skyrius naudodami Goto išskleidžiamąjį meniu.
