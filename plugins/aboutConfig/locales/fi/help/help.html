Tietoja:Config

This page provides technical information about the configuration of the current blog (as well as configuration common to all the blogs). These parameters are usually managed by other pages in the administration interface.
Only modify something here if you know what you are doing!

Joitakin parametreja voi muuttaa vain tällä sivulla.

Sivulla on kaksi välilehteä: Nykyisen blogin asetukset (blogin nimi) ja Globaalit asetukset nykyisten blogiasetusten ja globaalien asetusten hallintaan.

Voit siirtyä suoraan sivun eri osioihin käyttämällä Goto -pudotusvalikkoa.
